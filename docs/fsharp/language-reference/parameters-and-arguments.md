## <a name="passing-by-reference"></a><span data-ttu-id="bdb67-167">Passer par référence</span><span class="sxs-lookup"><span data-stu-id="bdb67-167">Passing by Reference</span></span>

<span data-ttu-id="bdb67-168">Le passage F# d’une valeur par référence implique [types ByRef](byrefs.md), qui sont des types pointeur managés.</span><span class="sxs-lookup"><span data-stu-id="bdb67-168">Passing an F# value by reference involves [byrefs](byrefs.md), which are managed pointer types.</span></span> <span data-ttu-id="bdb67-169">Les instructions pour le type à utiliser sont les suivantes :</span><span class="sxs-lookup"><span data-stu-id="bdb67-169">Guidance for which type to use is as follows:</span></span>

- <span data-ttu-id="bdb67-170">Utilisez `inref<'T>` si vous devez uniquement lire le pointeur.</span><span class="sxs-lookup"><span data-stu-id="bdb67-170">Use `inref<'T>` if you only need to read the pointer.</span></span>
- <span data-ttu-id="bdb67-171">Utilisez `outref<'T>` si vous devez uniquement écrire dans le pointeur.</span><span class="sxs-lookup"><span data-stu-id="bdb67-171">Use `outref<'T>` if you only need to write to the pointer.</span></span>
- <span data-ttu-id="bdb67-172">Utilisez `byref<'T>` si vous avez besoin de lire et d’écrire dans le pointeur.</span><span class="sxs-lookup"><span data-stu-id="bdb67-172">Use `byref<'T>` if you need to both read from and write to the pointer.</span></span>

```fsharp
let example1 (x: inref<int>) = printfn "It's %d" x

let example2 (x: outref<int>) = x <- x + 1

let example3 (x: byref<int>) =
    printfn "It'd %d" x
    x <- x + 1

let test () =
    // No need to make it mutable, since it's read-only
    let x = 1
    example1 &x

    // Needs to be mutable, since we write to it
    let mutable y = 2
    example2 &y
```
