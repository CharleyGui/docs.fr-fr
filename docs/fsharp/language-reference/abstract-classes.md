# <a name="abstract-classes"></a><span data-ttu-id="5e084-103">Classes abstraites</span><span class="sxs-lookup"><span data-stu-id="5e084-103">Abstract Classes</span></span>

<span data-ttu-id="5e084-104">*Les classes abstraites* sont des classes qui laissent une partie ou la totalité des membres non implémentés, afin que les implémentations puissent être fournies par les classes dérivées.</span><span class="sxs-lookup"><span data-stu-id="5e084-104">*Abstract classes* are classes that leave some or all members unimplemented, so that implementations can be provided by derived classes.</span></span>

## <a name="syntax"></a><span data-ttu-id="5e084-105">Syntaxe</span><span class="sxs-lookup"><span data-stu-id="5e084-105">Syntax</span></span>

```fsharp
// Abstract class syntax.
[<AbstractClass>]
type [ accessibility-modifier ] abstract-class-name =
[ inherit base-class-or-interface-name ]
[ abstract-member-declarations-and-member-definitions ]

// Abstract member syntax.
abstract member member-name : type-signature
```
