// Create a slice from the beginning to index 5 (inclusive)

## <a name="slicing-multidimensional-arrays"></a><span data-ttu-id="96253-113">Découpage de tableaux multidimensionnels</span><span class="sxs-lookup"><span data-stu-id="96253-113">Slicing multidimensional arrays</span></span>

<span data-ttu-id="96253-117">Les exemples suivants montrent comment découper un tableau 2D :</span><span class="sxs-lookup"><span data-stu-id="96253-117">The following examples demonstrate how to slice a 2D array:</span></span>

```fsharp

let A = array2D [[1;2;3];[4;5;6];[7;8;9]]
printfn "Full matrix:\n %A" A

// Take the first row
let row0 = A.[0,*]
printfn "Row 0: %A" row0

printfn "%A" subA

// Take two rows and all columns
let subA' = A.[0..1,*]
printfn "%A" subA'

// Slice a 2x2 matrix out of the full 3x3 matrix
let twoByTwo = A.[0..1,0..1]
printfn "%A" twoByTwo
```

## <a name="defining-slices-for-other-data-structures"></a><span data-ttu-id="96253-120">Définition de tranches pour d’autres structures de données</span><span class="sxs-lookup"><span data-stu-id="96253-120">Defining slices for other data structures</span></span>

<span data-ttu-id="96253-121">La F# bibliothèque principale définit des tranches pour un ensemble limité de types.</span><span class="sxs-lookup"><span data-stu-id="96253-121">The F# core library defines slices for a limited set of types.</span></span> <span data-ttu-id="96253-122">Si vous souhaitez définir des tranches pour davantage de types de données, vous pouvez le faire soit dans la définition de type elle-même, soit dans une extension de type.</span><span class="sxs-lookup"><span data-stu-id="96253-122">If you wish to define slices for more data types, you can do so either in the type definition itself or in a type extension.</span></span>

<span data-ttu-id="96253-123">Par exemple, voici comment vous pouvez définir des secteurs pour la classe <xref:System.ArraySegment%601> pour permettre une manipulation pratique des données :</span><span class="sxs-lookup"><span data-stu-id="96253-123">For example, here's how you might define slices for the <xref:System.ArraySegment%601> class to allow for convenient data manipulation:</span></span>

type ArraySegment<'TItem> with
    member segment.GetSlice(start, finish) =
        let start = defaultArg start 0

### <a name="use-inlining-to-avoid-boxing-if-it-is-necessary"></a><span data-ttu-id="96253-124">Utilisez l’incorporation pour éviter le boxing si nécessaire.</span><span class="sxs-lookup"><span data-stu-id="96253-124">Use inlining to avoid boxing if it is necessary</span></span>

<span data-ttu-id="96253-125">Si vous définissez des tranches pour un type qui est en fait un struct, nous vous recommandons de `inline` le membre `GetSlice`.</span><span class="sxs-lookup"><span data-stu-id="96253-125">If you are defining slices for a type that is actually a struct, we recommend that you `inline` the `GetSlice` member.</span></span> <span data-ttu-id="96253-126">Le F# compilateur optimise les arguments facultatifs, évitant ainsi toute allocation de tas résultant du découpage.</span><span class="sxs-lookup"><span data-stu-id="96253-126">The F# compiler optimizes away the optional arguments, avoiding any heap allocations as a result of slicing.</span></span> <span data-ttu-id="96253-127">Cela est essentiel pour les constructions de découpage telles que les <xref:System.Span%601> qui ne peuvent pas être allouées sur le tas.</span><span class="sxs-lookup"><span data-stu-id="96253-127">This is critically important for slicing constructs such as <xref:System.Span%601> that cannot be allocated on the heap.</span></span>

```fsharp
open System

type ReadOnlySpan<'T> with
    // Note the 'inline' in the member definition
    member inline sp.GetSlice(startIdx, endIdx) =
        let s = defaultArg startIdx 0
        let e = defaultArg endIdx sp.Length
        sp.Slice(s, e - s)

type Span<'T> with
    // Note the 'inline' in the member definition
    member inline sp.GetSlice(startIdx, endIdx) =
        let s = defaultArg startIdx 0
        let e = defaultArg endIdx sp.Length
        sp.Slice(s, e - s)

let printSpan (sp: Span<int>) =
    let arr = sp.ToArray()
    printfn "%A" arr

let sp = [| 1; 2; 3; 4; 5 |].AsSpan()
printSpan sp.[0..] // [|1; 2; 3; 4; 5|]
printSpan sp.[..5] // [|1; 2; 3; 4; 5|]
printSpan sp.[0..3] // [|1; 2; 3|]
printSpan sp.[1..2] // |2; 3|]
```
