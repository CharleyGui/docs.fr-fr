- <span data-ttu-id="e9201-110">**Hébergement d’applications sur Azure-** Azure peut héberger la totalité de votre pile d’applications à partir d’applications Web et d’API vers des bases de données vers des services de stockage.</span><span class="sxs-lookup"><span data-stu-id="e9201-110">**Application hosting on Azure -** Azure can host your entire application stack from web applications and APIs to databases to storage services.</span></span> <span data-ttu-id="e9201-111">Azure prend en charge un large éventail de modèles d’hébergement de services entièrement managés pour les conteneurs et les ordinateurs virtuels.</span><span class="sxs-lookup"><span data-stu-id="e9201-111">Azure supports a variety of hosting models from fully managed services to containers to virtual machines.</span></span> <span data-ttu-id="e9201-112">Lorsque vous utilisez des services Azure entièrement gérés, vos applications peuvent tirer parti de l’évolutivité, de la haute disponibilité et de la sécurité intégrée à Azure.</span><span class="sxs-lookup"><span data-stu-id="e9201-112">When using fully managed Azure services, your applications can take advantage of the scalability, high-availability, and security built in to Azure.</span></span>
