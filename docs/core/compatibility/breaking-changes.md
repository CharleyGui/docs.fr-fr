# <a name="net-core-breaking-changes"></a>Modifications avec rupture .NET Core

Le sélecteur de version suivant fournit une liste filtrée de modifications avec rupture pour .NET Core, ASP.NET Core et EF Core. Les versions cibles incluent non seulement des versions majeures, telles que .NET Core 2,2 et .NET Core 3,0, mais également des versions préliminaires, telles que .NET Core 3,0 Preview 7.

Vous pouvez également afficher des problèmes individuels qui détaillent les modifications avec rupture dans les dépôts GitHub suivants :

- Pour .NET Core, la référentiel [dotnet/docs](https://github.com/dotnet/docs/issues?q=is%3Aopen+is%3Aissue+label%3Abreaking-change) .
- Pour ASP.NET Core, le référentiel [ASPNET/AspNetCore](https://github.com/aspnet/AspNetCore/issues?q=is%3Aopen+is%3Aissue+label%3Abreaking-change) .
- Pour Entity Framework Core, le référentiel [ASPNET/EntityFrameworkCore](https://github.com/aspnet/EntityFrameworkCore/issues?q=is%3Aopen+is%3Aissue+label%3Abreaking-change) .
