# <a name="breaking-changes-in-windows-forms-net-framework-to-net-core"></a>Dernières modifications de Windows Forms (.NET Framework à .NET Core)

La prise en charge de WPF et de Windows Forms a été ajoutée à .NET Core dans la version 3,0. Si vous migrez une application Windows Forms ou Windows Presentation Foundation à partir de .NET Framework vers .NET Core, les modifications avec rupture mentionnées dans cet article peuvent affecter votre application.

Les modifications avec rupture sont regroupées par la version de .NET Core dans laquelle elles ont été introduites.
