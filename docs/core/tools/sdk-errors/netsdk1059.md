# <a name="netsdk1059-project-contains-obsolete-net-cli-tool"></a>NETSDK1059 : le projet contient un outil CLI .NET obsolète

**Cet article s’applique à :** ✔️ Kit de développement logiciel (SDK) 2.1.100 .net Core et versions ultérieures

Lorsque le kit de développement logiciel (SDK) .NET émet un avertissement NETSDK1059, votre projet contient un outil CLI .NET obsolète. À compter de .NET Core 2,1, ces outils sont inclus dans le kit de développement logiciel (SDK) .NET et n’ont pas besoin d’être explicitement référencés par le projet. Vous trouverez plus d’informations sur la migration vers .NET Core 2,1 [ici](https://aka.ms/dotnetclitools-in-box).
