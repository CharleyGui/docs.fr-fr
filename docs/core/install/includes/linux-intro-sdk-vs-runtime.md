Installez le kit de développement logiciel (SDK) (qui comprend le Runtime) si vous souhaitez développer des applications .NET. Ou, si vous avez uniquement besoin d’exécuter des applications, installez le Runtime. Si vous installez le runtime, nous vous suggérons d’installer le **runtime ASP.net Core** , car il comprend des runtimes .net Core et ASP.net core.

Si vous avez déjà installé le kit de développement logiciel (SDK) ou le runtime, utilisez les `dotnet --list-sdks` `dotnet --list-runtimes` commandes et pour voir quelles versions sont installées. Pour plus d’informations, consultez [Comment vérifier que .net Core est déjà installé](../how-to-detect-installed-versions.md).
