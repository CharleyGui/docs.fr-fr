> [!IMPORTANT]
> Les installations du gestionnaire de package sont uniquement prises en charge sur l’architecture **x64** . D’autres architectures, telles que **ARM**, doivent installer .NET par d’autres moyens comme with Snap, un script d’installation ou une installation manuelle binaire.

Pour plus d’informations sur l’installation de .NET sans gestionnaire de package, consultez l’un des articles suivants :

- [Installez le kit de développement logiciel (SDK) .NET ou le Runtime .NET avec Snap.](../linux-snap.md)
- [Installez le kit de développement logiciel (SDK) .NET ou le Runtime .NET avec un script.](../linux-scripted-manual.md#scripted-install)
- [Installez le kit de développement logiciel (SDK) .NET ou le Runtime .NET manuellement.](../linux-scripted-manual.md#manual-install)
