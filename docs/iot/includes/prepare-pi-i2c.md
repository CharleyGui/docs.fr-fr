## <a name="prepare-the-raspberry-pi"></a>Préparer le Raspberry pi

Utilisez la `raspi-config` commande pour vous assurer que les deux services suivants sont activés :

- SSH
- I2C

Pour plus d’informations sur `raspi-config` , reportez-vous à la [documentation Raspberry pi](https://www.raspberrypi.org/documentation/configuration/raspi-config.md) <span class="docon docon-navigate-external x-hidden-focus"></span> .
