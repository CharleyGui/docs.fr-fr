# <a name="design-guidelines-for-exceptions"></a>Instructions de conception pour les exceptions
La gestion des exceptions présente de nombreux avantages par rapport aux rapports d’erreurs basés sur les valeurs de retour. Une bonne conception d’infrastructure aide le développeur d’applications à tirer parti des avantages des exceptions. Cette section présente les avantages des exceptions et présente les instructions permettant de les utiliser efficacement.
