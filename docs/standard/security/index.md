# <a name="security-in-net"></a>Sécurité dans .NET

Les common language runtime et .NET fournissent de nombreux services et classes utiles qui permettent aux développeurs d’écrire du code sécurisé, d’utiliser le chiffrement et de mettre en œuvre une sécurité basée sur les rôles.
