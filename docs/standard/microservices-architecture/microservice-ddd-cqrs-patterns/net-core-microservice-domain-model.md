## <a name="encapsulate-data-in-the-domain-entities"></a><span data-ttu-id="335ae-145">Encapsuler des données dans les entités de domaine</span><span class="sxs-lookup"><span data-stu-id="335ae-145">Encapsulate data in the Domain Entities</span></span>

<span data-ttu-id="335ae-149">Dans le code précédent, notez que beaucoup d’attributs sont en lecture seule ou privés, et peuvent uniquement être mis à jour par les méthodes de classe.
