# <a name="query-xml-trees-overview-linq-to-xml"></a>Vue d’ensemble de la requête d’arborescences XML (LINQ to XML)

Une fois que vous avez instancié une arborescence XML, l’écriture de requêtes est le moyen le plus efficace pour extraire des données de l’arborescence. De plus, l'interrogation combinée à la construction fonctionnelle vous permet de générer un nouveau document XML qui possède une forme différente du document d'origine.

Pour obtenir des informations générales sur les requêtes LINQ, consultez :

- [Introduction aux requêtes LINQ (C#)](../../csharp/programming-guide/concepts/linq/introduction-to-linq-queries.md)
- [Écriture de votre première requête LINQ (Visual Basic)](../../visual-basic/programming-guide/concepts/linq/writing-your-first-linq-query.md)
