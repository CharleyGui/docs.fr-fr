# <a name="removing-nodes-content-and-values-from-an-xml-document"></a>Suppression de nœuds, de contenu et de valeurs d'un document XML
Une fois qu'un DOM (Document Object Model) XML est en mémoire, vous pouvez supprimer des nœuds de l'arborescence ou supprimer du contenu et des valeurs de certains types de nœuds. Pour plus d'informations sur la suppression d'un nœud sans descendant ou d'une sous-arborescence entière d'un document, consultez [Suppression des nœuds du DOM](removing-nodes-from-the-dom.md). Pour plus d'informations sur la suppression d'attributs d'un élément, consultez [Suppression d'attributs d'un nœud d'élément dans le DOM](removing-attributes-from-an-element-node-in-the-dom.md). Pour plus d’informations sur la suppression de contenu d’un nœud sans supprimer le nœud, consultez [Suppression du contenu d’un nœud dans le DOM](removing-node-content-in-the-dom.md).
