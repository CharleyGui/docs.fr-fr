# <a name="xmlschemacollection-schema-compilation"></a><span data-ttu-id="46954-102">Compilation de schéma XmlSchemaCollection</span><span class="sxs-lookup"><span data-stu-id="46954-102">XmlSchemaCollection Schema Compilation</span></span>
<span data-ttu-id="46954-103">La classe **XmlSchemaCollection** est un cache ou une bibliothèque où des schémas XDR (XML-Data Reduced) et des schémas de langage XSD (XML Schema Definition) peuvent être stockés et validés.</span><span class="sxs-lookup"><span data-stu-id="46954-103">The **XmlSchemaCollection** is a cache or library where XML-Data Reduced (XDR) and XML Schema definition language (XSD) schemas can be stored and validated.</span></span> <span data-ttu-id="46954-104">**XmlSchemaCollection** améliore les performances en mettant les schémas en cache au lieu d'y accéder à partir d'un fichier ou d'une URL.</span><span class="sxs-lookup"><span data-stu-id="46954-104">**XmlSchemaCollection** improves performance by caching schemas in memory instead of accessing them from a file or URL.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="46954-105">Bien que la classe **XmlSchemaCollection** contienne à la fois des schémas XDR et des schémas XML, toute méthode ou propriété qui prend ou retourne un objet **XmlSchema** ne prend en charge que les schémas XML.</span><span class="sxs-lookup"><span data-stu-id="46954-105">Although the **XmlSchemaCollection** class stores both XDR schemas and XML Schemas, any method and property that takes or returns an **XmlSchema** object supports XML Schemas only.</span></span>

> <span data-ttu-id="46954-106">La classe <xref:System.Xml.Schema.XmlSchemaCollection> est désormais obsolète et a été remplacée par la classe <xref:System.Xml.Schema.XmlSchemaSet>.</span><span class="sxs-lookup"><span data-stu-id="46954-106">The <xref:System.Xml.Schema.XmlSchemaCollection> class is now obsolete and has been replaced with the <xref:System.Xml.Schema.XmlSchemaSet> class.</span></span> <span data-ttu-id="46954-107">Pour plus d’informations sur la classe <xref:System.Xml.Schema.XmlSchemaSet>, consultez [XmlSchemaSet pour la compilation de schémas](../../../../docs/standard/data/xml/xmlschemaset-for-schema-compilation.md).</span><span class="sxs-lookup"><span data-stu-id="46954-107">For more information about the <xref:System.Xml.Schema.XmlSchemaSet> class see, [XmlSchemaSet for Schema Compilation](../../../../docs/standard/data/xml/xmlschemaset-for-schema-compilation.md).</span></span>

## <a name="validate-xml-documents-using-xmlschemacollection"></a><span data-ttu-id="46954-120">Validation des documents XML à l'aide de XmlSchemaCollection</span><span class="sxs-lookup"><span data-stu-id="46954-120">Validate XML Documents Using XmlSchemaCollection</span></span>

### <a name="improved-performance"></a><span data-ttu-id="46954-122">Performances améliorées</span><span class="sxs-lookup"><span data-stu-id="46954-122">Improved Performance</span></span>  
 <span data-ttu-id="46954-123">Si vous procédez à la validation de plusieurs documents par rapport au même schéma, il est recommandé d'utiliser **XmlSchemaCollection** car elle offre de meilleures performances grâce à la mise en cache des schémas.</span><span class="sxs-lookup"><span data-stu-id="46954-123">It is recommended, if you are validating more than one document against the same schema, that you use the **XmlSchemaCollection** because it provides better performance by caching schemas in memory.</span></span>  
  
 <span data-ttu-id="46954-124">L’exemple de code suivant crée l’objet **XmlSchemaCollection**, ajoute des schémas à la collection et définit la propriété **Schemas**.</span><span class="sxs-lookup"><span data-stu-id="46954-124">The following code example creates the **XmlSchemaCollection** object, adds schemas to the collection, and sets the **Schemas** property.</span></span>  
  
```vb  
Dim tr as XmlTextReader = new XmlTextReader("Books.xml")  
Dim vr as XmlValidatingReader = new XmlValidatingReader(tr)  
Dim xsc as XmlSchemaCollection = new XmlSchemaCollection  
xsc.Add("urn:bookstore-schema", "Books.xsd")  
vr.Schemas.Add(xsc)  
```  
  
```csharp  
XmlTextReader tr = new XmlTextReader("Books.xml");  
XmlValidatingReader vr = new XmlValidatingReader(tr);  
XmlSchemaCollection xsc = new XmlSchemaCollection();  
xsc.Add("urn:bookstore-schema", "Books.xsd");    
vr.Schemas.Add(xsc);  
```
