# <a name="introduction-tothe-docker-application-life-cycle"></a>Introduction au cycle de vie d’application Docker

*Le cycle de vie des applications en conteneur est un parcours qui commence par le développeur. Le développeur choisit d’implémenter des conteneurs et Docker, car il élimine les frictions dans les déploiements et les opérations informatiques, qui au final vous aide à être plus souples, plus productif end-to-end et plus rapide de tout le monde.*
