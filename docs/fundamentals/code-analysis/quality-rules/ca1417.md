## <a name="how-to-fix-violations"></a><span data-ttu-id="f3107-116">Comment corriger les violations</span><span class="sxs-lookup"><span data-stu-id="f3107-116">How to fix violations</span></span>

<span data-ttu-id="f3107-117">Si le marshaling de données de chaîne modifiées vers l’appelant est requis, transmettez la chaîne par référence à la place.</span><span class="sxs-lookup"><span data-stu-id="f3107-117">If marshaling of modified string data back to the caller is required, pass the string by reference instead.</span></span> <span data-ttu-id="f3107-118">Dans le cas contraire, le <xref:System.Runtime.InteropServices.OutAttribute> peut être supprimé sans aucune autre modification.</span><span class="sxs-lookup"><span data-stu-id="f3107-118">Otherwise, the <xref:System.Runtime.InteropServices.OutAttribute> can be removed without any other changes.</span></span>

```csharp
 // Violation
[DllImport("MyLibrary")]
private static extern void Foo([Out] string s);

// Fixed: passed by reference
[DllImport("MyLibrary")]
private static extern void Foo(out string s);

// Fixed: marshaling data back to caller is not required
[DllImport("MyLibrary")]
private static extern void Foo(string s);
```
