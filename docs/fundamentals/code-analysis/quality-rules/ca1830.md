## <a name="how-to-fix-violations"></a><span data-ttu-id="78dba-116">Comment corriger les violations</span><span class="sxs-lookup"><span data-stu-id="78dba-116">How to fix violations</span></span>

<span data-ttu-id="78dba-117">Supprimez le inutile `ToString()` de l’appel.</span><span class="sxs-lookup"><span data-stu-id="78dba-117">Delete the unnecessary `ToString()` from the invocation.</span></span>

```csharp
using System.Text;

class C
{
    int _value;

    // Violation
    public void Log(StringBuilder destination)
    {
        destination.Append("Value: ").Append(_value.ToString()).AppendLine();
    }

    // Fixed
    public void Log(StringBuilder destination)
    {
        destination.Append("Value: ").Append(_value).AppendLine();
    }
}
```
