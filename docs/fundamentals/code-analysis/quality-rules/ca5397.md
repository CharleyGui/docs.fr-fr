### <a name="enumeration-name-violation"></a><span data-ttu-id="a6996-133">Violation de nom d’énumération</span><span class="sxs-lookup"><span data-stu-id="a6996-133">Enumeration name violation</span></span>

```csharp
using System;
using System.Security.Authentication;

public class ExampleClass
{
    public void ExampleMethod()
    {
        // CA5397 violation for using Tls11
        SslProtocols protocols = SslProtocols.Tls11 | SslProtocols.Tls12;
    }
}
```

### <a name="solution"></a><span data-ttu-id="a6996-135">Solution</span><span class="sxs-lookup"><span data-stu-id="a6996-135">Solution</span></span>

```csharp
using System;
using System.Security.Authentication;

public class TestClass
{
    public void Method()
    {
        // Let the operating system decide what TLS protocol version to use.
        // See https://docs.microsoft.com/dotnet/framework/network-programming/tls
        SslProtocols sslProtocols = SslProtocols.None;
    }
}
```
