## <a name="how-to-fix-violations"></a><span data-ttu-id="0525d-118">Comment corriger les violations</span><span class="sxs-lookup"><span data-stu-id="0525d-118">How to fix violations</span></span>

<span data-ttu-id="0525d-119">Pour corriger une violation, substituez la méthode de la classe de base <xref:System.IEquatable%601.Equals%2A> et implémentez-la en appelant l' <xref:System.IEquatable%601.Equals%2A?displayProperty=fullName> implémentation.</span><span class="sxs-lookup"><span data-stu-id="0525d-119">To fix a violation, override the base class <xref:System.IEquatable%601.Equals%2A> method and implement it by invoking the <xref:System.IEquatable%601.Equals%2A?displayProperty=fullName> implementation.</span></span>
<span data-ttu-id="0525d-120">Par exemple, les deux extraits de code suivants illustrent une violation de la règle et comment la corriger :</span><span class="sxs-lookup"><span data-stu-id="0525d-120">For example, the following two code snippets show a violation of the rule and how to fix it:</span></span>

```csharp
using System;

public class B : IEquatable<B>
{
    public B(int value)
    {
        Value = value;
    }

    public int Value { get; }

    public virtual bool Equals(B other) => Value == other.Value;

    public sealed override bool Equals(object obj) => Equals(obj as B);
}

public sealed class C : B, IEquatable<C>
{
    public C(string name, int value)
        : base(value)
    {
        Name = name;
    }

    public string Name { get; }

    public bool Equals(C other) => base.Equals(other) && Name == other.Name;
}
```

<span data-ttu-id="0525d-121">L’extrait de code suivant montre les deux mêmes classes avec la violation traitée.</span><span class="sxs-lookup"><span data-stu-id="0525d-121">The following code snippet shows the same two classes with the violation addressed.</span></span>

```csharp
using System;

public class B : IEquatable<B>
{
    public B(int value)
    {
        Value = value;
    }

    public int Value { get; }

    public virtual bool Equals(B other) => Value == other.Value;

    public sealed override bool Equals(object obj) => Equals(obj as B);
}

public sealed class C : B, IEquatable<C>
{
    public C(string name, int value)
        : base(value)
    {
        Name = name;
    }

    public string Name { get; }

    public bool Equals(C other) => base.Equals(other) && Name == other.Name;

    // This line fixes the violation.
    public sealed override bool Equals(B other) => Equals(other as C);
}
```
