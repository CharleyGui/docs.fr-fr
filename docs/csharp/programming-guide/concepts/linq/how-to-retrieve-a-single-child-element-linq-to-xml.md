 <span data-ttu-id="3231c-108">L'exemple suivant illustre l'utilisation de la méthode <xref:System.Xml.Linq.XContainer.Element%2A>.</span><span class="sxs-lookup"><span data-stu-id="3231c-108">The following example demonstrates the use of the <xref:System.Xml.Linq.XContainer.Element%2A> method.</span></span> <span data-ttu-id="3231c-109">Cet exemple prend l'arborescence XML nommée `po` et recherche le premier élément nommé `Comment`.</span><span class="sxs-lookup"><span data-stu-id="3231c-109">This example takes the XML tree named `po` and finds the first element named `Comment`.</span></span>

## <a name="example"></a><span data-ttu-id="3231c-113">Exemple</span><span class="sxs-lookup"><span data-stu-id="3231c-113">Example</span></span>
