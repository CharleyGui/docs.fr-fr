# <a name="compiler-error-cs1001"></a><span data-ttu-id="552f4-102">Erreur du compilateur CS1001</span><span class="sxs-lookup"><span data-stu-id="552f4-102">Compiler Error CS1001</span></span>

<span data-ttu-id="552f4-103">Identificateur attendu</span><span class="sxs-lookup"><span data-stu-id="552f4-103">Identifier expected</span></span>

<span data-ttu-id="552f4-104">Vous n’avez pas fourni un identificateur.</span><span class="sxs-lookup"><span data-stu-id="552f4-104">You did not supply an identifier.</span></span> <span data-ttu-id="552f4-105">Un identificateur correspond au nom d’une classe, d’un struct, d’un espace de noms, d’une méthode, d’une variable, etc. que vous fournissez.</span><span class="sxs-lookup"><span data-stu-id="552f4-105">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>

<span data-ttu-id="552f4-106">Dans l’exemple suivant, une classe simple est déclarée, mais aucun nom ne lui est attribué :</span><span class="sxs-lookup"><span data-stu-id="552f4-106">The following example declares a simple class but does not give the class a name:</span></span>

```csharp
public class //CS1001
{
    public int Num { get; set; }
    void MethodA() {}
}
```

<span data-ttu-id="552f4-107">L’exemple suivant génère l’erreur CS1001 car, quand vous déclarez un enum, vous devez spécifier des membres :</span><span class="sxs-lookup"><span data-stu-id="552f4-107">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>

```csharp
public class Program
{
    enum Colors
    {
        'a', 'b' // CS1001, 'a' is not a valid int identifier
        // The following line shows examples of valid identifiers:
        // Blue, Red, Orange
    };

    public static void Main()
    {
    }
}
```

<span data-ttu-id="552f4-108">Les noms de paramètres sont requis même si le compilateur ne les utilise pas (par exemple, dans une définition d’interface).</span><span class="sxs-lookup"><span data-stu-id="552f4-108">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="552f4-109">Ces paramètres sont nécessaires afin que les programmeurs qui consomment l’interface connaissent la signification des paramètres.</span><span class="sxs-lookup"><span data-stu-id="552f4-109">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>

```csharp
interface IMyTest
{
    void TestFunc1(int, int);  // CS1001
    // Use the following line instead:
    // void TestFunc1(int a, int b);
}

class CMyTest : IMyTest
{
    void IMyTest.TestFunc1(int a, int b)
    {
    }
}
```
