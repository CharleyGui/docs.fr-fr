# <a name="compiler-error-cs1943"></a><span data-ttu-id="a0b72-103">Erreur du compilateur CS1943</span><span class="sxs-lookup"><span data-stu-id="a0b72-103">Compiler Error CS1943</span></span>

<span data-ttu-id="a0b72-104">Une expression de type 'type' n’est pas autorisée dans une clause from ultérieure d’une expression de requête avec le type source 'type'.</span><span class="sxs-lookup"><span data-stu-id="a0b72-104">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="a0b72-105">L’inférence de type a échoué lors de l’appel à la méthode 'method'.</span><span class="sxs-lookup"><span data-stu-id="a0b72-105">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="a0b72-106">Toutes les variables de portée doivent représenter des types requêtables.</span><span class="sxs-lookup"><span data-stu-id="a0b72-106">All range variables must represent queryable types.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a0b72-107">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="a0b72-107">To correct this error</span></span>  
  
1. <span data-ttu-id="a0b72-108">Vérifiez qu’il s’agit d’un type requêtable qui implémente `IEnumerable`, `IEnumerable<T>` ou une interface dérivée, ou d’un type disposant d’un modèle de requête spécialement défini.</span><span class="sxs-lookup"><span data-stu-id="a0b72-108">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>

## <a name="example"></a><span data-ttu-id="a0b72-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="a0b72-110">Example</span></span>  

 <span data-ttu-id="a0b72-111">Le code suivant génère l’erreur CS1943 :</span><span class="sxs-lookup"><span data-stu-id="a0b72-111">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
