## <a name="example"></a><span data-ttu-id="5a396-106"> Exemple</span><span class="sxs-lookup"><span data-stu-id="5a396-106">Example</span></span>  
 <span data-ttu-id="5a396-107">L’exemple suivant génère l’avertissement CS1691.</span><span class="sxs-lookup"><span data-stu-id="5a396-107">The following example generates CS1691.</span></span>  
  
```csharp  
// CS1691.cs  
public class C  
{  
    int i = 1;  
    public static void Main()  
    {  
        C myC = new C();  
#pragma warning disable 151  // CS1691  
// Try the following line instead:  
// #pragma warning disable 1645
        myC.i++;  
#pragma warning restore 151  // CS1691  
// Try the following line instead:  
// #pragma warning restore 1645
    }  
}  
```
