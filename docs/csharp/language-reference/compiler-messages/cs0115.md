## <a name="example"></a><span data-ttu-id="1f2dd-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="1f2dd-107">Example</span></span>

 <span data-ttu-id="1f2dd-108">L’exemple suivant génère l’erreur CS0115.</span><span class="sxs-lookup"><span data-stu-id="1f2dd-108">The following sample generates CS0115.</span></span> <span data-ttu-id="1f2dd-109">Vous pouvez la résoudre de deux façons différentes :</span><span class="sxs-lookup"><span data-stu-id="1f2dd-109">You can resolve CS0115 in one of two ways:</span></span>

```csharp
// CS0115.cs
namespace MyNamespace
{
    abstract public class MyClass1
    {
        public abstract int f();
    }

    abstract public class MyClass2
    {
        public override int f()   // CS0115
        {
            return 0;
        }

        public static void Main()
        {
        }
    }
}
```
