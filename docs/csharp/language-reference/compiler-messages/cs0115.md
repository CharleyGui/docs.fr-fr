## <a name="example"></a><span data-ttu-id="41897-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="41897-106">Example</span></span>

 <span data-ttu-id="41897-107">L’exemple suivant génère l’erreur CS0115.</span><span class="sxs-lookup"><span data-stu-id="41897-107">The following sample generates CS0115.</span></span> <span data-ttu-id="41897-108">Vous pouvez la résoudre de deux façons différentes :</span><span class="sxs-lookup"><span data-stu-id="41897-108">You can resolve CS0115 in one of two ways:</span></span>

- <span data-ttu-id="41897-110">Utilisez `MyClass1` comme classe de base pour `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="41897-110">Use `MyClass1` as a base class for `MyClass2`.</span></span>

```csharp
// CS0115.cs
namespace MyNamespace
{
    abstract public class MyClass1
    {
        public abstract int f();
    }

    abstract public class MyClass2
    {
        public override int f()   // CS0115
        {
            return 0;
        }

        public static void Main()
        {
        }
    }
}
```
