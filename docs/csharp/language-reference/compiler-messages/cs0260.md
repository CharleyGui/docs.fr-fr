# <a name="compiler-error-cs0260"></a><span data-ttu-id="1019d-103">Erreur du compilateur CS0260</span><span class="sxs-lookup"><span data-stu-id="1019d-103">Compiler Error CS0260</span></span>

<span data-ttu-id="1019d-104">Modificateur partiel manquant dans la déclaration de type 'type' ; il existe une autre déclaration partielle de ce type</span><span class="sxs-lookup"><span data-stu-id="1019d-104">Missing partial modifier on declaration of type 'type'; another partial declaration of this type exists</span></span>  
  
 <span data-ttu-id="1019d-105">Cette erreur indique que vous avez déclaré plusieurs classes qui ont le même nom.</span><span class="sxs-lookup"><span data-stu-id="1019d-105">This error indicates that you have declared multiple classes that have the same name.</span></span> <span data-ttu-id="1019d-106">En outre, au moins une déclaration contient le modificateur `partial` (mais pas toutes).</span><span class="sxs-lookup"><span data-stu-id="1019d-106">In addition, at least one but not all of the declarations contains the `partial` modifier.</span></span> <span data-ttu-id="1019d-107">Si vous souhaitez définir une classe en plusieurs parties, vous devez déclarer chaque partie en utilisant le mot clé `partial`.</span><span class="sxs-lookup"><span data-stu-id="1019d-107">If you want to define a class in several parts, you must declare each part by using the keyword `partial`.</span></span>  
  
 <span data-ttu-id="1019d-108">Cette erreur se produit également si vous déclarez une classe qui porte par erreur le même nom qu’une classe partielle qui est déclarée à un autre emplacement du même espace de noms.</span><span class="sxs-lookup"><span data-stu-id="1019d-108">This error also occurs if you declare a class and accidentally give it the same name as a partial class that's declared elsewhere in the same namespace.</span></span>  
  
 <span data-ttu-id="1019d-109">L’exemple suivant génère l’erreur CS0260 :</span><span class="sxs-lookup"><span data-stu-id="1019d-109">The following sample generates CS0260:</span></span>  

```csharp
// CS0260.cs  
// You must mark both parts of the definition of class C
// by using the partial keyword.  
  
// The following line causes CS0260. To resolve the error, add  
// the 'partial' keyword to the declaration.  
class C
{  
}  
  
partial class C  
{  
}  
```
