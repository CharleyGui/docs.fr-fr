# <a name="compiler-error-cs8411"></a><span data-ttu-id="9ed2f-102">Erreur du compilateur CS8411</span><span class="sxs-lookup"><span data-stu-id="9ed2f-102">Compiler Error CS8411</span></span>

<span data-ttu-id="9ed2f-103">L’instruction foreach asynchrone ne peut pas fonctionner sur des variables de type’type', car’type’ne contient pas de définition d’instance publique appropriée pour’GetAsyncEnumerator'</span><span class="sxs-lookup"><span data-stu-id="9ed2f-103">Asynchronous foreach statement cannot operate on variables of type 'type' because 'type' does not contain a suitable public instance definition for 'GetAsyncEnumerator'</span></span>

<span data-ttu-id="9ed2f-104">`await foreach` l’instruction ne fonctionne que sur les types ayant une définition de `GetAsyncEnumerator` , tels que `IAsyncEnumerable<T>` .</span><span class="sxs-lookup"><span data-stu-id="9ed2f-104">`await foreach` statement operates only on types having a definition of `GetAsyncEnumerator`, such as `IAsyncEnumerable<T>`.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="9ed2f-105">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="9ed2f-105">To correct this error</span></span>

<span data-ttu-id="9ed2f-106">Remplacez `await foreach` par `foreach`.</span><span class="sxs-lookup"><span data-stu-id="9ed2f-106">Replace `await foreach` with `foreach`.</span></span>

## <a name="example"></a><span data-ttu-id="9ed2f-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="9ed2f-107">Example</span></span>

```csharp
using System.Collections.Generic;
using System.Threading.Tasks;

class Program
{
    async Task Example(IAsyncEnumerator<int> enumerator)
    {
        // error CS8411: Asynchronous foreach statement cannot operate on variables
        // of type 'IAsyncEnumerator<T>' because 'IAsyncEnumerator<T>' does not
        // contain a suitable public instance definition for 'GetAsyncEnumerator'
        await foreach (int i in enumerator)
        {
        }
    }
}
```
