# <a name="compiler-error-cs0571"></a><span data-ttu-id="ec648-102">Erreur du compilateur CS0571</span><span class="sxs-lookup"><span data-stu-id="ec648-102">Compiler Error CS0571</span></span>

 <span data-ttu-id="ec648-104">Certains opérateurs ont des noms internes.</span><span class="sxs-lookup"><span data-stu-id="ec648-104">Certain operators have internal names.</span></span> <span data-ttu-id="ec648-105">Par exemple, **op_Increment** est le nom interne de l’opérateur ++.</span><span class="sxs-lookup"><span data-stu-id="ec648-105">For example, **op_Increment** is the internal name of the ++ operator.</span></span> <span data-ttu-id="ec648-106">Vous ne devez pas utiliser ni appeler explicitement ces noms de méthodes.</span><span class="sxs-lookup"><span data-stu-id="ec648-106">You should not use or explicitly call such method names.</span></span>  
  
 <span data-ttu-id="ec648-107">L’exemple suivant génère l’erreur CS0571 :</span><span class="sxs-lookup"><span data-stu-id="ec648-107">The following sample generates CS0571:</span></span>  
  
```csharp  
// CS0571.cs  
public class MyClass  
{  
   public static MyClass operator ++ (MyClass c)  
   {  
      return null;  
   }  
  
   public static int prop  
   {  
      get  
      {  
         return 1;  
      }  
      set  
      {  
      }  
   }  
  
   public static void Main()  
   {  
      op_Increment(null);   // CS0571  
      // use the increment operator as follows  
      // MyClass x = new MyClass();  
      // x++;  
  
      set_prop(1);      // CS0571  
      // try the following line instead  
      // prop = 1;  
   }  
}  
```
