# <a name="compiler-error-cs1579"></a><span data-ttu-id="e1bd6-103">Erreur du compilateur CS1579</span><span class="sxs-lookup"><span data-stu-id="e1bd6-103">Compiler Error CS1579</span></span>

<span data-ttu-id="e1bd6-104">L’instruction foreach ne peut pas fonctionner sur des variables de type 'type1', car 'type2' ne contient pas de définition publique pour 'identifier'</span><span class="sxs-lookup"><span data-stu-id="e1bd6-104">foreach statement cannot operate on variables of type 'type1' because 'type2' does not contain a public definition for 'identifier'</span></span>

<span data-ttu-id="e1bd6-105">Pour pouvoir itérer au sein d’une collection à l’aide de l’instruction [foreach](../keywords/foreach-in.md), la collection doit réunir les conditions suivantes :</span><span class="sxs-lookup"><span data-stu-id="e1bd6-105">To iterate through a collection using the [foreach](../keywords/foreach-in.md) statement, the collection must meet the following requirements:</span></span>

- <span data-ttu-id="e1bd6-106">Son type doit inclure une méthode `GetEnumerator` sans paramètre publique dont le retour est de type classe, struct ou interface.</span><span class="sxs-lookup"><span data-stu-id="e1bd6-106">Its type must include a public parameterless `GetEnumerator` method whose return type is either class, struct, or interface type.</span></span>
- <span data-ttu-id="e1bd6-107">Le type de retour de la méthode `GetEnumerator` doit contenir une propriété publique nommée `Current` et une méthode sans paramètre publique nommée `MoveNext`, dont le type de retour est <xref:System.Boolean>.</span><span class="sxs-lookup"><span data-stu-id="e1bd6-107">The return type of the `GetEnumerator` method must contain a public property named `Current` and a public parameterless method named `MoveNext` whose return type is <xref:System.Boolean>.</span></span>

## <a name="example"></a><span data-ttu-id="e1bd6-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="e1bd6-108">Example</span></span>

<span data-ttu-id="e1bd6-109">L’exemple suivant génère une erreur CS1579 car la classe `MyCollection` ne contient pas la méthode `GetEnumerator` publique :</span><span class="sxs-lookup"><span data-stu-id="e1bd6-109">The following sample generates CS1579 because the `MyCollection` class doesn't contain the public `GetEnumerator` method:</span></span>

```csharp  
// CS1579.cs  
using System;  
public class MyCollection
{  
   int[] items;  
   public MyCollection()
   {  
      items = new int[5] {12, 44, 33, 2, 50};  
   }  
  
   // Delete the following line to resolve.  
   MyEnumerator GetEnumerator()  
  
   // Uncomment the following line to resolve:  
   // public MyEnumerator GetEnumerator()
   {  
      return new MyEnumerator(this);  
   }  
  
   // Declare the enumerator class:  
   public class MyEnumerator
   {  
      int nIndex;  
      MyCollection collection;  
      public MyEnumerator(MyCollection coll)
      {  
         collection = coll;  
         nIndex = -1;  
      }  
  
      public bool MoveNext()
      {  
         nIndex++;  
         return (nIndex < collection.items.Length);  
      }  
  
      public int Current => collection.items[nIndex];
   }  
  
   public static void Main()
   {  
      MyCollection col = new MyCollection();  
      Console.WriteLine("Values in the collection are:");  
      foreach (int i in col)   // CS1579  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
