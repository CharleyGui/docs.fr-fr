CS0019 also is generated when the subtraction operator `-` is applied to a string.</span></span> <span data-ttu-id="a82d8-110">L’opérateur `+` d’addition peut être employé avec des opérands à cordes parce que cet opérateur est surchargé par la classe pour exécuter la `String` concatenation de corde.</span><span class="sxs-lookup"><span data-stu-id="a82d8-110">The addition operator `+` can be used with string operands because that operator is overloaded by the `String` class to perform string concatenation.</span></span>

```csharp
static void Main()
{
    bool result = true;
    if (result > 0) //CS0019
    {
        // Do something.
    }

    int i = 1;
    // You cannot compare an integer and a boolean value.
    if (i == true) //CS0019
    {
        //Do something...
    }

    string s = "Just try to subtract me.";
    float f = 100 - s; // CS0019
}
```
