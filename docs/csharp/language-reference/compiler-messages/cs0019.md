## <a name="example"></a><span data-ttu-id="a0b18-109">Exemple</span><span class="sxs-lookup"><span data-stu-id="a0b18-109">Example</span></span>

<span data-ttu-id="a0b18-110">Dans l’exemple suivant, CS0019 est généré dans trois emplacements, car [bool](../builtin-types/bool.md) en C# n’est pas convertible en [int](../builtin-types/integral-numeric-types.md). CS0019 est également généré quand l’opérateur de soustraction `-` est appliqué à une chaîne.</span><span class="sxs-lookup"><span data-stu-id="a0b18-110">In the following example, CS0019 is generated in three places because [bool](../builtin-types/bool.md) in C# is not convertible to [int](../builtin-types/integral-numeric-types.md). CS0019 is also generated when the subtraction operator `-` is applied to a string.</span></span> <span data-ttu-id="a0b18-111">L’opérateur d’addition `+` peut être utilisé avec des opérandes de chaîne, car cet opérateur est surchargé par la `String` classe pour effectuer la concaténation de chaînes.</span><span class="sxs-lookup"><span data-stu-id="a0b18-111">The addition operator `+` can be used with string operands because that operator is overloaded by the `String` class to perform string concatenation.</span></span>

```csharp
static void Main()
{
    bool result = true;
    if (result > 0) //CS0019
    {
        // Do something.
    }

    int i = 1;
    // You cannot compare an integer and a boolean value.
    if (i == true) //CS0019
    {
        //Do something...
    }

    string s = "Just try to subtract me.";
    float f = 100 - s; // CS0019
}
```
