# <a name="compiler-error-cs1708"></a><span data-ttu-id="3b694-103">Erreur du compilateur CS1708</span><span class="sxs-lookup"><span data-stu-id="3b694-103">Compiler Error CS1708</span></span>

<span data-ttu-id="3b694-104">Les mémoires tampons de taille fixe ne sont accessibles que via des variables locales ou des champs</span><span class="sxs-lookup"><span data-stu-id="3b694-104">Fixed size buffers can only be accessed through locals or fields</span></span>  
  
 <span data-ttu-id="3b694-105">Une nouvelle fonctionnalité de C# 2.0 vous permet de définir un tableau inline dans un `struct`.</span><span class="sxs-lookup"><span data-stu-id="3b694-105">A new feature in C# 2.0 is the ability to define an in-line array inside of a `struct`.</span></span> <span data-ttu-id="3b694-106">Ces tableaux sont accessibles par l’intermédiaire de variables locales ou de champs, et ne doivent pas être référencés comme valeurs intermédiaires dans la partie gauche d’une expression.</span><span class="sxs-lookup"><span data-stu-id="3b694-106">Such arrays can only be accessed via local variables or fields, and may not be referenced as intermediate values on the left-hand side of an expression.</span></span> <span data-ttu-id="3b694-107">Par contre, ils ne sont pas accessibles par les champs qui sont `static` ou `readonly`.</span><span class="sxs-lookup"><span data-stu-id="3b694-107">Also, the arrays cannot be accessed by fields that are `static` or `readonly`.</span></span>  
  
 <span data-ttu-id="3b694-108">Pour résoudre cette erreur, définissez une variable de tableau et assignez le tableau inline à la variable.</span><span class="sxs-lookup"><span data-stu-id="3b694-108">To resolve this error, define an array variable, and assign the in-line array to the variable.</span></span> <span data-ttu-id="3b694-109">Une autre solution consiste à supprimer le modificateur `static` ou `readonly` du champ qui représente le tableau inline.</span><span class="sxs-lookup"><span data-stu-id="3b694-109">Or, remove the `static` or `readonly` modifier from the field representing the in-line array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3b694-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="3b694-110">Example</span></span>  

 <span data-ttu-id="3b694-111">L’exemple suivant génère l’erreur CS1708.</span><span class="sxs-lookup"><span data-stu-id="3b694-111">The following sample generates CS1708.</span></span>  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
