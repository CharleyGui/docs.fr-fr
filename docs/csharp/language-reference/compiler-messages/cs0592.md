## <a name="example"></a><span data-ttu-id="74736-109">Exemple</span><span class="sxs-lookup"><span data-stu-id="74736-109">Example</span></span>

 <span data-ttu-id="74736-110">L’exemple suivant génère l’erreur CS0592 :</span><span class="sxs-lookup"><span data-stu-id="74736-110">The following sample generates CS0592:</span></span>  

```csharp
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.

{  
    public static void Main()  
    {  
    }  
}  
```
