<span data-ttu-id="cad4f-103">L’attribut 'attribute' n’est pas valide sur ce type de déclaration.</span><span class="sxs-lookup"><span data-stu-id="cad4f-103">Attribute 'attribute' is not valid on this declaration type.</span></span> <span data-ttu-id="cad4f-104">Il n’est valide que sur les déclarations 'type'.</span><span class="sxs-lookup"><span data-stu-id="cad4f-104">It is valid on 'type' declarations only.</span></span>

## <a name="example"></a><span data-ttu-id="cad4f-108">Exemples</span><span class="sxs-lookup"><span data-stu-id="cad4f-108">Example</span></span>

 <span data-ttu-id="cad4f-109">L’exemple suivant génère l’erreur CS0592 :</span><span class="sxs-lookup"><span data-stu-id="cad4f-109">The following sample generates CS0592:</span></span>  

```csharp
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.
public class A
{  
    public static void Main()  
    {  
    }  
}  
```
