# <a name="compiler-error-cs0038"></a><span data-ttu-id="ee854-102">Erreur du compilateur CS0038</span><span class="sxs-lookup"><span data-stu-id="ee854-102">Compiler Error CS0038</span></span>
<span data-ttu-id="ee854-103">Impossible d’accéder à un membre non statique de type externe 'type1' par l’intermédiaire du type imbriqué 'type2'</span><span class="sxs-lookup"><span data-stu-id="ee854-103">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>  
  
 <span data-ttu-id="ee854-104">Un champ dans une classe n’est pas automatiquement disponible pour une classe imbriquée.</span><span class="sxs-lookup"><span data-stu-id="ee854-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="ee854-105">Pour être disponible pour une classe imbriquée, le champ doit être de type [static](../keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="ee854-105">To be available to a nested class, the field must be [static](../keywords/static.md).</span></span> <span data-ttu-id="ee854-106">Sinon, vous devez créer une instance de la classe externe.</span><span class="sxs-lookup"><span data-stu-id="ee854-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="ee854-107">Pour plus d’informations, consultez [Types imbriqués](../../programming-guide/classes-and-structs/nested-types.md).</span><span class="sxs-lookup"><span data-stu-id="ee854-107">For more information, see [Nested Types](../../programming-guide/classes-and-structs/nested-types.md).</span></span>  
  
 <span data-ttu-id="ee854-108">L’exemple suivant génère l’erreur CS0038 :</span><span class="sxs-lookup"><span data-stu-id="ee854-108">The following sample generates CS0038:</span></span>  
  
```csharp  
// CS0038.cs  
class OuterClass  
{  
   public int count;  
   // try the following line instead  
   // public static int count;  
  
   class InnerClass  
   {  
      void func()  
      {  
         // or, create an instance  
         // OuterClass class_inst = new OuterClass();  
         // int count2 = class_inst.count;  
         int count2 = count;   // CS0038  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
