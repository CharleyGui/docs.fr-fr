# <a name="compiler-error-cs0038"></a>Erreur du compilateur CS0038

Impossible d’accéder à un membre non statique de type externe 'type1' par l’intermédiaire du type imbriqué 'type2'

 Un champ dans une classe n’est pas automatiquement disponible pour une classe imbriquée. Pour être disponible pour une classe imbriquée, le champ doit être de type [static](../keywords/static.md). Sinon, vous devez créer une instance de la classe externe. Pour plus d’informations, voir [Nested Types](../../programming-guide/classes-and-structs/nested-types.md).

 L’exemple suivant génère l’erreur CS0038 :

```csharp
// CS0038.cs
class OuterClass
{
    public int count;
    // Try the following line instead.
    // public static int count;

    class InnerClass
    {
        void Func()
        {
            // or, create an instance
            // OuterClass class_inst = new OuterClass();
            // int count2 = class_inst.count;
            int count2 = count;   // CS0038
        }
    }

    public static void Main()
    {
    }
}
```
