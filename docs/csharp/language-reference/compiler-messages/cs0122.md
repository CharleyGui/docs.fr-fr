# <a name="compiler-error-cs0122"></a>Erreur du compilateur CS0122

'member' est inaccessible en raison de son niveau de protection

 Le [modificateur d’accès](../keywords/modifiers.md) d’un membre de classe empêche l’accès au membre. Pour plus d’informations, consultez la page [Modificateurs d’accès](../../programming-guide/classes-and-structs/access-modifiers.md).

 Cela peut être dû à l’omission de l’indicateur de compilateur **/out** sur la cible d’un assembly friend (cause non illustrée dans l’exemple ci-dessous). Pour plus d’informations, consultez [assemblys friend](../../../standard/assembly/friend.md) et [/outC# (options du compilateur)](../compiler-options/out-compiler-option.md).

## <a name="example"></a>Exemple

 L’exemple suivant génère l’erreur CS0122 :

```csharp
// CS0122.cs
public class MyClass
{
    // Make public to resolve CS0122.
    void MyMethod()
    {
    }
}

public class MyClass2
{
    public static int Main()
    {  
        var a = new MyClass();  
        // MyMethod is private.
        a.MyMethod();   // CS0122
        return 0;
   }
}
```
