# <a name="compiler-error-cs0266"></a><span data-ttu-id="f6502-103">Erreur du compilateur CS0266</span><span class="sxs-lookup"><span data-stu-id="f6502-103">Compiler Error CS0266</span></span>

<span data-ttu-id="f6502-104">Impossible de convertir implicitement le type 'type1' en 'type2'.</span><span class="sxs-lookup"><span data-stu-id="f6502-104">Cannot implicitly convert type 'type1' to 'type2'.</span></span> <span data-ttu-id="f6502-105">Une conversion explicite existe (un cast est-il manquant ?)</span><span class="sxs-lookup"><span data-stu-id="f6502-105">An explicit conversion exists (are you missing a cast?)</span></span>

 <span data-ttu-id="f6502-106">Cette erreur se produit quand votre code tente une conversion entre deux types qui ne peuvent pas être convertis implicitement, mais où une conversion explicite est disponible.</span><span class="sxs-lookup"><span data-stu-id="f6502-106">This error occurs when your code tries to convert between two types that cannot be implicitly converted, but where an explicit conversion is available.</span></span> <span data-ttu-id="f6502-107">Pour plus d’informations, consultez [Cast et conversions de types](../../programming-guide/types/casting-and-type-conversions.md).</span><span class="sxs-lookup"><span data-stu-id="f6502-107">For more information, see [Casting and Type Conversions](../../programming-guide/types/casting-and-type-conversions.md).</span></span>

 <span data-ttu-id="f6502-108">Le code suivant montre des exemples qui génèrent des CS0266 :</span><span class="sxs-lookup"><span data-stu-id="f6502-108">The following code shows examples that generate CS0266:</span></span>

```csharp
// CS0266.cs
class MyClass
{
    public static void Main()
    {
        // You cannot implicitly convert a double to an integer.
        double d = 3.2;

        // The following line causes compiler error CS0266.
        int i1 = d;

        // However, you can resolve the error by using an explicit conversion.
        int i2 = (int)d;  

        // You cannot implicitly convert an object to a class type.
        object obj = new MyClass();

        // The following assignment statement causes error CS0266.
        MyClass myClass = obj;

        // You can resolve the error by using an explicit conversion.
        MyClass c = (MyClass)obj;

        // You cannot implicitly convert a base class object to a derived class type.
        MyClass mc = new MyClass();
        DerivedClass dc = new DerivedClass();

        // The following line causes compiler error CS0266.
        dc = mc;

        // You can resolve the error by using an explicit conversion.
        dc = (DerivedClass)mc;
    }  
}  
  
class DerivedClass : MyClass  
{  
}  
```
