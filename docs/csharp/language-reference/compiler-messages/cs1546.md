 <span data-ttu-id="9a4b8-109">Cet exemple de code consiste en un fichier .cpp, qui se compile en un fichier .dll, et en un fichier .cs qui utilise ce fichier .dll.</span><span class="sxs-lookup"><span data-stu-id="9a4b8-109">This code sample consists of a .cpp file, which compiles to a .dll, and a .cs file, which uses that .dll.</span></span> <span data-ttu-id="9a4b8-110">Le code suivant concerne le fichier .dll et définit une propriété accessible par le code dans le fichier .cs.</span><span class="sxs-lookup"><span data-stu-id="9a4b8-110">The following code is for the .dll file and defines a property to be accessed by the code in the .cs file.</span></span>

```cpp  
// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
