# <a name="compiler-warning-level-2-cs0108"></a>Avertissement du compilateur (niveau 2) CS0108

'member1' masque le membre hérité 'member2'. Utilisez le mot clé new si le masquage est intentionnel.  
  
 Une variable portant le même nom qu’une variable d’une classe de base a été déclarée. Cependant, le mot clé [new](../../../csharp/language-reference/keywords/new-modifier.md) n’a pas été utilisé. Cet avertissement vous informe que vous devez utiliser **new** ; la variable est déclarée comme si **new** avait été utilisé dans la déclaration.  
  
 L’exemple suivant génère l’avertissement CS0108 :  

```csharp
// CS0108.cs  
// compile with: /W:2  
using System;  
  
namespace x  
{  
   public class clx  
   {  
      public int i = 1;  
   }  
  
   public class cly : clx  
   {  
      public static int i = 2;   // CS0108, use the new keyword  
      // Use the following line instead:  
      // public static new int i = 2;  
  
      public static void Main()  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
