## <a name="example"></a><span data-ttu-id="d1cfb-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="d1cfb-106">Example</span></span>  

 <span data-ttu-id="d1cfb-107">L’exemple suivant génère l’erreur CS1018 et suggère plusieurs façons de la résoudre :</span><span class="sxs-lookup"><span data-stu-id="d1cfb-107">The following example generates CS1018, and suggests several ways to resolve the error:</span></span>  
  
```csharp  
// CS1018.cs  
public class C  
{  
}  
  
public class a : C  
{  
    public a(int i)  
    {  
    }  
  
    public a () :   // CS1018  
    // possible resolutions:  
    // public a () resolves by removing the colon  
    // public a () : base() calls C's parameterless constructor  
    // public a () : this(1) calls the assignment constructor of class a  
    {  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
