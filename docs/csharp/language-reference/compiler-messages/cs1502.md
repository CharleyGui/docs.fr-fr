# <a name="compiler-error-cs1502"></a><span data-ttu-id="f822d-102">Erreur du compilateur CS1502</span><span class="sxs-lookup"><span data-stu-id="f822d-102">Compiler Error CS1502</span></span>
<span data-ttu-id="f822d-103">La méthode surchargée correspondant le mieux à 'name' possède des arguments non valides</span><span class="sxs-lookup"><span data-stu-id="f822d-103">The best overloaded method match for 'name' has some invalid arguments</span></span>  
  
 <span data-ttu-id="f822d-104">Cette erreur se produit quand les types d’arguments passés à la méthode ne correspondent pas aux types de paramètres de cette méthode.</span><span class="sxs-lookup"><span data-stu-id="f822d-104">This error occurs when the argument types being passed to the method do not match the parameter types of that method.</span></span> <span data-ttu-id="f822d-105">Si la méthode appelée est surchargée, aucune des versions surchargées n’a de signature correspondant aux types d’arguments passés.</span><span class="sxs-lookup"><span data-stu-id="f822d-105">If the called method is overloaded, then none of the overloaded versions has a signature that matches the argument types being passed.</span></span>  
  
 <span data-ttu-id="f822d-106">Pour résoudre ce problème, effectuez l’une des opérations suivantes :</span><span class="sxs-lookup"><span data-stu-id="f822d-106">To resolve this problem, do one of the following:</span></span>  
  
- <span data-ttu-id="f822d-107">Vérifiez soigneusement les types des arguments passés.</span><span class="sxs-lookup"><span data-stu-id="f822d-107">Double-check the types of the arguments being passed.</span></span> <span data-ttu-id="f822d-108">Vérifiez qu’ils correspondent aux arguments de la méthode appelée.</span><span class="sxs-lookup"><span data-stu-id="f822d-108">Make sure that they match the arguments of the method being called.</span></span>
