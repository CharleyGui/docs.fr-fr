# <a name="compiler-error-cs1502"></a><span data-ttu-id="7ef71-103">Erreur du compilateur CS1502</span><span class="sxs-lookup"><span data-stu-id="7ef71-103">Compiler Error CS1502</span></span>
<span data-ttu-id="7ef71-104">La méthode surchargée correspondant le mieux à 'name' possède des arguments non valides</span><span class="sxs-lookup"><span data-stu-id="7ef71-104">The best overloaded method match for 'name' has some invalid arguments</span></span>  
  
 <span data-ttu-id="7ef71-105">Cette erreur se produit quand les types d’arguments passés à la méthode ne correspondent pas aux types de paramètres de cette méthode.</span><span class="sxs-lookup"><span data-stu-id="7ef71-105">This error occurs when the argument types being passed to the method do not match the parameter types of that method.</span></span> <span data-ttu-id="7ef71-106">Si la méthode appelée est surchargée, aucune des versions surchargées n’a de signature correspondant aux types d’arguments passés.</span><span class="sxs-lookup"><span data-stu-id="7ef71-106">If the called method is overloaded, then none of the overloaded versions has a signature that matches the argument types being passed.</span></span>  
  
 <span data-ttu-id="7ef71-107">Pour résoudre ce problème, effectuez l’une des opérations suivantes :</span><span class="sxs-lookup"><span data-stu-id="7ef71-107">To resolve this problem, do one of the following:</span></span>  
  
- <span data-ttu-id="7ef71-108">Vérifiez soigneusement les types des arguments passés.</span><span class="sxs-lookup"><span data-stu-id="7ef71-108">Double-check the types of the arguments being passed.</span></span> <span data-ttu-id="7ef71-109">Vérifiez qu’ils correspondent aux arguments de la méthode appelée.</span><span class="sxs-lookup"><span data-stu-id="7ef71-109">Make sure that they match the arguments of the method being called.</span></span>  
  
- <span data-ttu-id="7ef71-110">Le cas échéant, convertissez tous les paramètres sans correspondance à l’aide de la classe <xref:System.Convert>.</span><span class="sxs-lookup"><span data-stu-id="7ef71-110">If appropriate, convert any mismatched parameters using the <xref:System.Convert> class.</span></span>  
  
- <span data-ttu-id="7ef71-111">Le cas échéant,effectuez un cast de tout paramètre non concordant pour qu’il corresponde au type que la méthode attend.</span><span class="sxs-lookup"><span data-stu-id="7ef71-111">If appropriate, cast any mismatched parameters to match the type that the method is expecting.</span></span>  
  
- <span data-ttu-id="7ef71-112">Le cas échéant, définissez une autre version surchargée de la méthode pour qu’elle corresponde aux types de paramètres envoyés.</span><span class="sxs-lookup"><span data-stu-id="7ef71-112">If appropriate, define another overloaded version of the method to match the parameter types that are being sent.</span></span>

```csharp  
   {  
      public a(char i)  
      // try the following constructor instead  
      // public a(int i)  
      {  
      }

      {  
         a aa = new a(2222);   // CS1502  
      }  
   }  
}  
```
