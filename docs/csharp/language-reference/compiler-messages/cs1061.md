# <a name="compiler-error-cs1061"></a><span data-ttu-id="ea0cd-103">Erreur du compilateur CS1061</span><span class="sxs-lookup"><span data-stu-id="ea0cd-103">Compiler Error CS1061</span></span>
<span data-ttu-id="ea0cd-104">'type' ne contient pas de définition pour 'membre' et aucune méthode d’extension 'nom' acceptant un premier argument de type 'type' n’a été trouvée (une directive using ou une référence d’assembly est-elle manquante ?).</span><span class="sxs-lookup"><span data-stu-id="ea0cd-104">'type' does not contain a definition for 'member' and no extension method 'name' accepting a first argument of type 'type' could be found (are you missing a using directive or an assembly reference?).</span></span>  
  
 <span data-ttu-id="ea0cd-105">Cette erreur se produit quand vous essayez d’appeler une méthode ou d’accéder à un membre de classe qui n’existe pas.</span><span class="sxs-lookup"><span data-stu-id="ea0cd-105">This error occurs when you try to call a method or access a class member that does not exist.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ea0cd-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="ea0cd-106">Example</span></span>  
 <span data-ttu-id="ea0cd-107">L’exemple suivant génère l’erreur CS1061 parce que `TestClass1` n’a pas de méthode `DisplaySomething`.</span><span class="sxs-lookup"><span data-stu-id="ea0cd-107">The following example generates CS1061 because `TestClass1` does not have a `DisplaySomething` method.</span></span> <span data-ttu-id="ea0cd-108">En revanche, elle a une méthode nommée `WriteSomething`.</span><span class="sxs-lookup"><span data-stu-id="ea0cd-108">It does have a method that is called `WriteSomething`.</span></span> <span data-ttu-id="ea0cd-109">C’est peut-être ce que l’auteur de ce code source voulait écrire.</span><span class="sxs-lookup"><span data-stu-id="ea0cd-109">Perhaps that is what the author of this source code meant to write.</span></span>  
  
```csharp  
// cs1061.cs  
public class TestClass1  
{  
    // TestClass1 has one method, called WriteSomething.  
    public void WriteSomething(string s)  
    {  
        System.Console.WriteLine(s);  
    }  
}  
  
public class TestClass2  
{  
    // TestClass2 has one method, called DisplaySomething.  
    public void DisplaySomething(string s)  
    {  
        System.Console.WriteLine(s);  
    }  
}  
  
public class TestTheClasses  
{  
    public static void Main()  
    {  
        TestClass1 tc1 = new TestClass1();  
        TestClass2 tc2 = new TestClass2();  
        // The following call fails because TestClass1 does not have
        // a method called DisplaySomething.  
        tc1.DisplaySomething("Hello");      // CS1061  
  
        // To correct the error, change the method call to either
        // tc1.WriteSomething or tc2.DisplaySomething.  
        tc1.WriteSomething("Hello from TestClass1");  
        tc2.DisplaySomething("Hello from TestClass2");  
    }  
}  
```
