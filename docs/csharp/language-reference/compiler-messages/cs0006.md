# <a name="compiler-error-cs0006"></a>Erreur du compilateur CS0006
Le fichier de métadonnées 'nom_dll' est introuvable  
  
 Le programme a été compilé et a passé explicitement le nom d’un fichier qui contenait des métadonnées ; cependant, ce fichier .dll est introuvable. Pour plus d’informations, consultez l’article [/reference (C# Compiler Options) (/reference [Options du compilateur C#])](../compiler-options/reference-compiler-option.md).
