## <a name="example"></a><span data-ttu-id="ef266-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="ef266-108">Example</span></span>

<span data-ttu-id="ef266-109">Les exemples suivants génèrent CS8892 :</span><span class="sxs-lookup"><span data-stu-id="ef266-109">The following examples generates CS8892:</span></span>

```csharp
using System;
using System.Threading.Tasks;

public class Program
{
    // CS8892: Method 'Program.Main()' will not be used as an entry point because a synchronous entry point 'Program.Main(string[])' was found.
    public static async Task Main()
    {
        await Task.Delay(1);
    }

    public static void Main(string[] args)
    {
        Console.WriteLine(2);
    }
}
```

## <a name="how-to-fix-this-warning"></a><span data-ttu-id="ef266-110">Comment résoudre cet avertissement</span><span class="sxs-lookup"><span data-stu-id="ef266-110">How to fix this warning</span></span>

```csharp
    }
}
```
