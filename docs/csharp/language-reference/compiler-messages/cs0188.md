# <a name="compiler-error-cs0188"></a><span data-ttu-id="41f16-102">Erreur du compilateur CS0188</span><span class="sxs-lookup"><span data-stu-id="41f16-102">Compiler Error CS0188</span></span>

<span data-ttu-id="41f16-103">L’objet 'this' ne peut pas être utilisé avant que tous ses champs ne soient assignés</span><span class="sxs-lookup"><span data-stu-id="41f16-103">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="41f16-104">Tous les champs d’un `struct` doivent être assignés par un constructeur avant que le constructeur ne puisse appeler une méthode dans le `struct`.</span><span class="sxs-lookup"><span data-stu-id="41f16-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="41f16-105">Si vous voyez cette erreur lorsque vous tentez d’initialiser une propriété dans un constructeur de struct, la solution consiste à initialiser le champ de stockage de cette propriété directement à la place de la propriété elle-même.</span><span class="sxs-lookup"><span data-stu-id="41f16-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to initialize the backing field of that property directly instead of the property itself.</span></span>

## <a name="example"></a><span data-ttu-id="41f16-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="41f16-106">Example</span></span>

 <span data-ttu-id="41f16-107">L’exemple suivant génère l’erreur CS0188 :</span><span class="sxs-lookup"><span data-stu-id="41f16-107">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
