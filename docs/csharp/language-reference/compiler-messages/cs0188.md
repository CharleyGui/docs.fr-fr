 <span data-ttu-id="9147b-105">Si vous voyez cette erreur en essayant d’initialiser une propriété dans un constructeur structurant, la solution est d’initialiser le champ de soutien de cette propriété directement au lieu de la propriété elle-même.</span><span class="sxs-lookup"><span data-stu-id="9147b-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to initialize the backing field of that property directly instead of the property itself.</span></span>

## <a name="example"></a><span data-ttu-id="9147b-106"> Exemple</span><span class="sxs-lookup"><span data-stu-id="9147b-106">Example</span></span>

 <span data-ttu-id="9147b-107">L’exemple suivant génère l’erreur CS0188 :</span><span class="sxs-lookup"><span data-stu-id="9147b-107">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
