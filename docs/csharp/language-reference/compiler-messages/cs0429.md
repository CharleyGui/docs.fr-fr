<span data-ttu-id="17a8c-103">Code expression inaccessible détecté</span><span class="sxs-lookup"><span data-stu-id="17a8c-103">Unreachable expression code detected</span></span>  
  
 <span data-ttu-id="17a8c-104">Cette erreur se produit chaque fois qu’une partie d’une expression de votre code est inaccessible.</span><span class="sxs-lookup"><span data-stu-id="17a8c-104">This error occurs whenever part of an expression in your code is unreachable.</span></span> <span data-ttu-id="17a8c-105">Dans l’exemple suivant, la condition `false && myTest()` répond à ces critères, car la méthode `myTest()` ne sera jamais évaluée puisque la partie gauche de l’opération `&&` a toujours la valeur false.</span><span class="sxs-lookup"><span data-stu-id="17a8c-105">In the following example, the condition `false && myTest()` meets this criteria because the `myTest()` method will never get evaluated due to the fact that the left side of the `&&` operation is always false.</span></span> <span data-ttu-id="17a8c-106">Dès que l’opérateur `&&` évalue l’instruction `false` comme false, il arrête l’évaluation et n’évaluera jamais la partie droite.</span><span class="sxs-lookup"><span data-stu-id="17a8c-106">As soon as the `&&` operator evaluates the `false` statement as false, it stops the evaluation, and will never evaluate the right side.</span></span>
