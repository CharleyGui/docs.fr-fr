<span data-ttu-id="e4b89-104">Une référence a été créée dans l’assembly d’interopérabilité incorporé' \<assembly1> 'en raison d’une référence indirecte à cet assembly à partir de l’assembly' \<assembly2> '.</span><span class="sxs-lookup"><span data-stu-id="e4b89-104">A reference was created to embedded interop assembly '\<assembly1>' because of an indirect reference to that assembly from assembly '\<assembly2>'.</span></span> <span data-ttu-id="e4b89-105">Modifiez la propriété 'Incorporer les types interop' pour l’un des assemblys.</span><span class="sxs-lookup"><span data-stu-id="e4b89-105">Consider changing the 'Embed Interop Types' property on either assembly.</span></span>
