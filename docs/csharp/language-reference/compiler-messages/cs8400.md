# <a name="compiler-error-cs8400"></a>Erreur du compilateur CS8400

La fonctionnalité « fonctionnalité » n’est pas disponible en C# 8,0. Utilisez la version \<version> de langue ou une version ultérieure.

## <a name="to-correct-this-error"></a>Pour corriger cette erreur

Mettez à niveau votre projet vers au moins la version mentionnée dans l’erreur.
