# <a name="compiler-error-cs0703"></a><span data-ttu-id="16ddb-103">Erreur du compilateur CS0703</span><span class="sxs-lookup"><span data-stu-id="16ddb-103">Compiler Error CS0703</span></span>

<span data-ttu-id="16ddb-104">Accessibilité incohérente : le type de contrainte 'identifier' est moins accessible que 'identifier'</span><span class="sxs-lookup"><span data-stu-id="16ddb-104">Inconsistent accessibility: constraint type 'identifier' is less accessible than 'identifier'</span></span>  
  
 <span data-ttu-id="16ddb-105">Une contrainte ne peut pas forcer le paramètre générique à être moins accessible que la classe générique elle-même.</span><span class="sxs-lookup"><span data-stu-id="16ddb-105">A constraint may not force the generic parameter to be less accessible than the generic class itself.</span></span> <span data-ttu-id="16ddb-106">Dans l’exemple suivant, tandis que la classe générique C \<T> est déclarée public, la contrainte tente de forcer T à implémenter une interface interne.</span><span class="sxs-lookup"><span data-stu-id="16ddb-106">In the following example, while the generic class C\<T> is declared public, the constraint attempts to force T to implement an internal interface.</span></span> <span data-ttu-id="16ddb-107">Même si cela était autorisé, seuls les clients disposant d’un accès interne pourraient créer le paramètre pour la classe ; en conséquence, cette classe ne pourrait être utilisée que par les clients disposant d’un accès interne.</span><span class="sxs-lookup"><span data-stu-id="16ddb-107">Even if this were allowed, only clients with internal access would be able to create the parameter for the class, so in effect, the class could be used only by clients with internal access.</span></span>  
  
 <span data-ttu-id="16ddb-108">Pour supprimer cette erreur, vérifiez que le niveau d’accès de la classe générique n’est pas moins restrictif que les classes ou interfaces qui figurent dans les limites.</span><span class="sxs-lookup"><span data-stu-id="16ddb-108">To get rid of this error, make sure the access level of the generic class is not less restrictive than any classes or interfaces that appear in the bounds.</span></span>  
  
 <span data-ttu-id="16ddb-109">L’exemple suivant génère l’erreur CS0703 :</span><span class="sxs-lookup"><span data-stu-id="16ddb-109">The following sample generates CS0703:</span></span>  
  
```csharp  
// CS0703.cs  
internal interface I {}  
public class C<T> where T : I  // CS0703 – I is internal; C<T> is public  
{  
}  
```
