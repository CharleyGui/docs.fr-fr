## <a name="example"></a><span data-ttu-id="5faee-106">Exemples</span><span class="sxs-lookup"><span data-stu-id="5faee-106">Example</span></span>  
 <span data-ttu-id="5faee-107">L’exemple suivant génère l’erreur CS0050, car aucun modificateur d’accessibilité n’est fourni pour `MyClass` et son accessibilité adopte donc par défaut la valeur `private`.</span><span class="sxs-lookup"><span data-stu-id="5faee-107">The following sample generates CS0050 because no accessibility modifier is supplied for `MyClass` and its accessibility therefore defaults to `private`.</span></span>

```csharp  
// try the following line instead

{  
}

{  
    public static MyClass MyMethod()   // CS0050  
    {  
        return new MyClass();  
    }  
  
    public static void Main() { }  
}  
```
