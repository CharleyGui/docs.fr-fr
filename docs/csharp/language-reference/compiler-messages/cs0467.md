# <a name="compiler-warning-level-2-cs0467"></a><span data-ttu-id="7abb0-103">Avertissement du compilateur (niveau 2) CS0467</span><span class="sxs-lookup"><span data-stu-id="7abb0-103">Compiler Warning (level 2) CS0467</span></span>
<span data-ttu-id="7abb0-104">Ambiguïté entre la méthode 'method' et l’élément 'non-method' qui n’est pas une méthode.</span><span class="sxs-lookup"><span data-stu-id="7abb0-104">Ambiguity between method 'method' and non-method 'non-method'.</span></span> <span data-ttu-id="7abb0-105">Utilisation du groupe de méthodes.</span><span class="sxs-lookup"><span data-stu-id="7abb0-105">Using method group.</span></span>  
  
 <span data-ttu-id="7abb0-106">Les membres hérités de différentes interfaces avec la même signature provoquent une erreur d’ambiguïté.</span><span class="sxs-lookup"><span data-stu-id="7abb0-106">Inherited members from different interfaces that have the same signature  cause an ambiguity error.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7abb0-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="7abb0-107">Example</span></span>  
 <span data-ttu-id="7abb0-108">L’exemple suivant génère l’avertissement du compilateur CS0467.</span><span class="sxs-lookup"><span data-stu-id="7abb0-108">The following example generates CS0467.</span></span>  
  
```csharp  
// CS0467.cs  
interface IList
{  
    int Count { get; set; }  
}  
  
interface ICounter  
{  
    void Count(int i);  
}  
  
interface IListCounter : IList, ICounter {}  
  
class Driver
{  
    void Test(IListCounter x)  
    {  
        // The following line causes the warning. The assignment also  
        // causes an error because you can't assign a value to a method.  
        x.Count = 1;  
        x.Count(3);
        // To resolve the warning, you can change the name of the method or
        // the property.  
  
        // You can also disambiguate by specifying IList or ICounter.  
        ((IList)x).Count = 1;  
        ((ICounter)x).Count(3);  
    }  
  
    static void Main()
    {  
    }  
}  
```
