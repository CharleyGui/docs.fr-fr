# <a name="compiler-error-cs8403"></a>Erreur du compilateur CS8403

La méthode’Method’avec un bloc itérateur doit être’Async’pour retourner' {IAsyncEnumerable \<T> | IAsyncEnumerator \<T> }»

## <a name="to-correct-this-error"></a>Pour corriger cette erreur

Marquez votre méthode avec le `async` modificateur.
