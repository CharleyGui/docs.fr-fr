# <a name="compiler-error-cs0445"></a><span data-ttu-id="c50fa-102">Erreur du compilateur CS0445</span><span class="sxs-lookup"><span data-stu-id="c50fa-102">Compiler Error CS0445</span></span>
<span data-ttu-id="c50fa-103">Impossible de modifier le résultat d’une conversion unboxing</span><span class="sxs-lookup"><span data-stu-id="c50fa-103">Cannot modify the result of an unboxing conversion</span></span>  
  
 <span data-ttu-id="c50fa-104">Le résultat d’une conversion unboxing est une variable temporaire.</span><span class="sxs-lookup"><span data-stu-id="c50fa-104">The result of an unboxing conversion is a temporary variable.</span></span> <span data-ttu-id="c50fa-105">Le compilateur vous empêche de modifier ces variables, car toute modification serait supprimée en même temps que la variable temporaire.</span><span class="sxs-lookup"><span data-stu-id="c50fa-105">The compiler prevents you from modifying such variables because any modification would go away when the temporary variable goes away.</span></span> <span data-ttu-id="c50fa-106">Pour résoudre ce problème, déclarez une nouvelle variable de type valeur pour stocker l’expression intermédiaire et assigner le résultat de la conversion unboxing à cette variable.</span><span class="sxs-lookup"><span data-stu-id="c50fa-106">To fix this, declare a new value-type variable to store the intermediate expression, and assign the result of the unboxing conversion to that variable.</span></span>  
  
 <span data-ttu-id="c50fa-107">Le code suivant génère l’erreur CS0455.</span><span class="sxs-lookup"><span data-stu-id="c50fa-107">The following code generates CS0455.</span></span>  
  
```csharp  
// CS0445.CS  
class UnboxingTest  
{  
    public static void Main()  
    {  
        Point p;  
        p.x = 1;  
        p.y = 2;  
        object obj = p;  
        // The following line generates CS0445, because the result  
        // of unboxing obj is a temporary variable.  
        ((Point)obj).x = 2;  
  
        // The following lines resolve the error.  
  
        // Store the result of the unboxing conversion in p2.  
        Point p2;
        p2 = (Point)obj;  
        // Then you can modify the unboxed value.  
        p2.x = 2;  
    }  
}  
  
struct Point  
{  
    public int x, y;  
}  
```
