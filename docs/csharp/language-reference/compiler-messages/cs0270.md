# <a name="compiler-error-cs0270"></a>Erreur du compilateur CS0270
La taille du tableau ne peut pas être spécifiée dans une déclaration de variable (essayez d’initialiser avec une expression 'new')  
  
 Cette erreur se produit quand une taille est spécifiée dans le cadre d’une déclaration de tableau. Pour résoudre le, utilisez l’expression [de l’opérateur New](../operators/new-operator.md) .  
  
 L’exemple suivant génère l’erreur CS0270 :  
  
```csharp  
// CS0270.cs  
// compile with: /t:module  
  
public class Test  
{  
   int[10] a;   // CS0270  
   // To resolve, use the following line instead:  
   // int[] a = new int[10];  
}  
```
