# <a name="compiler-error-cs1674"></a><span data-ttu-id="3e7b3-103">Erreur du compilateur CS1674</span><span class="sxs-lookup"><span data-stu-id="3e7b3-103">Compiler Error CS1674</span></span>
<span data-ttu-id="3e7b3-104">'T' : le type utilisé dans une instruction using doit être convertible implicitement en 'System.IDisposable'</span><span class="sxs-lookup"><span data-stu-id="3e7b3-104">'T': type used in a using statement must be implicitly convertible to 'System.IDisposable'</span></span>  
  
 <span data-ttu-id="3e7b3-105">L’[instruction using](../keywords/using-statement.md) doit être utilisée pour garantir la suppression d’un objet à la fin du bloc `using` ; par conséquent, seuls les types pouvant être supprimés peuvent être utilisés dans ce type d’instruction.</span><span class="sxs-lookup"><span data-stu-id="3e7b3-105">The [using Statement](../keywords/using-statement.md) is intended to be used to ensure the disposal of an object at the end of the `using` block, thus, only types which are disposable may be used in such a statement.</span></span> <span data-ttu-id="3e7b3-106">Par exemple, les types valeur ne peuvent pas être supprimés, et les paramètres de type qui ne sont pas contraints à être des classes ne peuvent pas être considérés comme pouvant être supprimés.</span><span class="sxs-lookup"><span data-stu-id="3e7b3-106">For example, value types are not disposable, and type parameters which are not constrained to be classes may not be assumed to be disposable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3e7b3-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="3e7b3-107">Example</span></span>

## <a name="example"></a><span data-ttu-id="3e7b3-109">Exemple</span><span class="sxs-lookup"><span data-stu-id="3e7b3-109">Example</span></span>  
 <span data-ttu-id="3e7b3-110">L’exemple suivant génère l’erreur CS1674.</span><span class="sxs-lookup"><span data-stu-id="3e7b3-110">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674_b.cs  
using System;  
class C {  
   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }  
}  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   public static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
