## <a name="example"></a><span data-ttu-id="e4e73-111">Exemple</span><span class="sxs-lookup"><span data-stu-id="e4e73-111">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B {}  
class C {}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```
