 <span data-ttu-id="a9ed2-105">Si une variable est de type [const](../keywords/const.md), elle est également [static](../keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="a9ed2-105">If a variable is [const](../keywords/const.md), it is also [static](../keywords/static.md).</span></span> <span data-ttu-id="a9ed2-106">Si vous souhaitez une variable **const** et **static**, déclarez cette variable comme **const** ; si vous souhaitez uniquement une variable **static**, marquez-la comme **static**.</span><span class="sxs-lookup"><span data-stu-id="a9ed2-106">If you want a **const** and **static** variable, just declare that variable as **const**; if all you want is a **static** variable, just mark it **static**.</span></span>  
  
 <span data-ttu-id="a9ed2-107">L’exemple suivant génère l’erreur CS0504 :</span><span class="sxs-lookup"><span data-stu-id="a9ed2-107">The following sample generates CS0504:</span></span>

```csharp  
      static const int i = 0;   // CS0504, cannot be both static and const

}  
```
