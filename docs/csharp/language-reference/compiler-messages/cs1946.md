# <a name="compiler-error-cs1946"></a><span data-ttu-id="9434c-103">Erreur du compilateur CS1946</span><span class="sxs-lookup"><span data-stu-id="9434c-103">Compiler Error CS1946</span></span>

<span data-ttu-id="9434c-104">Une expression de méthode anonyme ne peut pas être convertie en arborescence d’expressions.</span><span class="sxs-lookup"><span data-stu-id="9434c-104">An anonymous method expression cannot be converted to an expression tree.</span></span>

<span data-ttu-id="9434c-105">Une [méthode anonyme](../operators/delegate-operator.md) représente un ensemble d’instructions, mais une arborescence d’expressions ne doit pas contenir d’instruction.</span><span class="sxs-lookup"><span data-stu-id="9434c-105">An [anonymous method](../operators/delegate-operator.md) represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="9434c-106">Par conséquent, une méthode anonyme ne peut pas être représentée par une arborescence d’expressions.</span><span class="sxs-lookup"><span data-stu-id="9434c-106">Therefore an anonymous method cannot be represented by an expression tree.</span></span>

<span data-ttu-id="9434c-107">Pour corriger cette erreur, remplacez la méthode anonyme par une [expression lambda](../operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="9434c-107">To correct this error, change the anonymous method to a [lambda expression](../operators/lambda-expressions.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="9434c-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="9434c-108">Example</span></span>

<span data-ttu-id="9434c-109">L’exemple suivant génère l’erreur CS1946 :</span><span class="sxs-lookup"><span data-stu-id="9434c-109">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
