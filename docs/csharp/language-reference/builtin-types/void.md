# <a name="void-c-reference"></a>vide (référence C)

Vous `void` utilisez comme type de retour d’une [méthode](../../programming-guide/classes-and-structs/methods.md) (ou une [fonction locale](../../programming-guide/classes-and-structs/local-functions.md)) pour spécifier que la méthode ne retourne pas une valeur.

[!code-csharp[void method](snippets/VoidType.cs#VoidExample)]

Vous pouvez `void` également utiliser comme type référent pour déclarer un pointeur à un type inconnu. Pour plus d’informations, consultez [Types pointeur](../../programming-guide/unsafe-code-pointers/pointer-types.md).

Vous ne `void` pouvez pas utiliser comme type de variable.
