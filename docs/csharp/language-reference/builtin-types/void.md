# <a name="void-c-reference"></a>void (référence C#)

Vous utilisez `void` comme type de retour d’une [méthode](../../programming-guide/classes-and-structs/methods.md) (ou une [fonction locale](../../programming-guide/classes-and-structs/local-functions.md)) pour spécifier que la méthode ne retourne pas de valeur.

[!code-csharp[void method](snippets/VoidType.cs#VoidExample)]

Vous pouvez également utiliser `void` comme type référent pour déclarer un pointeur vers un type inconnu. Pour plus d’informations, consultez [Types pointeur](../../programming-guide/unsafe-code-pointers/pointer-types.md).

Vous ne pouvez pas utiliser `void` comme type de variable.
