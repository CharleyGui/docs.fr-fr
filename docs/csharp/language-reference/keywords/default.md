# <a name="default-c-reference"></a>default (référence C#)

Le mot clé `default` peut être utilisé de deux façons :

- Pour spécifier l’étiquette par défaut dans l' [ `switch` instruction](switch.md).
- Comme [opérateur ou littéral par défaut](../operators/default.md) pour produire la valeur par défaut d'un type.
