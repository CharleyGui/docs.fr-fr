## <a name="remarks"></a><span data-ttu-id="7a88f-122">Remarques</span><span class="sxs-lookup"><span data-stu-id="7a88f-122">Remarks</span></span>

<span data-ttu-id="7a88f-123">Hormis les types `object` et `string`, tous les types listés dans le tableau sont considérés comme des types simples.</span><span class="sxs-lookup"><span data-stu-id="7a88f-123">All of the types in the table, except `object` and `string`, are referred to as simple types.</span></span>  
  
<span data-ttu-id="7a88f-124">Les mots clés des types C# et leurs alias sont interchangeables.</span><span class="sxs-lookup"><span data-stu-id="7a88f-124">The C# type keywords and their aliases are interchangeable.</span></span> <span data-ttu-id="7a88f-125">Par exemple, vous pouvez déclarer une variable de type entier à l’aide de l’une des deux déclarations suivantes :</span><span class="sxs-lookup"><span data-stu-id="7a88f-125">For example, you can declare an integer variable by using either of the following declarations:</span></span>  

```csharp
int x = 123;
System.Int32 y = 123;
```

<span data-ttu-id="7a88f-126">Utilisez l’opérateur [typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator) pour obtenir l’instance de <xref:System.Type?displayProperty=nameWithType>, qui représente le type spécifié :</span><span class="sxs-lookup"><span data-stu-id="7a88f-126">Use the [typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>

```csharp
Type stringType = typeof(string);
Console.WriteLine(stringType.FullName);

Type doubleType = typeof(System.Double);
Console.WriteLine(doubleType.FullName);

// Output:
// System.String
// System.Double
```
