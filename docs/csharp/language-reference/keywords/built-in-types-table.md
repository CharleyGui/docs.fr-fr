## <a name="remarks"></a><span data-ttu-id="1a103-122">Notes</span><span class="sxs-lookup"><span data-stu-id="1a103-122">Remarks</span></span>

<span data-ttu-id="1a103-123">Hormis les types `object` et `string`, tous les types listés dans le tableau sont considérés comme des types simples.</span><span class="sxs-lookup"><span data-stu-id="1a103-123">All of the types in the table, except `object` and `string`, are referred to as simple types.</span></span>

<span data-ttu-id="1a103-124">Les types .NET et leurs alias de mots clés de type C# sont interchangeables.</span><span class="sxs-lookup"><span data-stu-id="1a103-124">The .NET types and their C# type keyword aliases are interchangeable.</span></span> <span data-ttu-id="1a103-125">Par exemple, vous pouvez déclarer une variable de type entier à l’aide de l’une des deux déclarations suivantes :</span><span class="sxs-lookup"><span data-stu-id="1a103-125">For example, you can declare an integer variable by using either of the following declarations:</span></span>

```csharp
int x = 123;
System.Int32 y = 123;
```

<span data-ttu-id="1a103-126">Utilisez l’opérateur [typeof](../operators/type-testing-and-cast.md#typeof-operator) pour obtenir l’instance de <xref:System.Type?displayProperty=nameWithType>, qui représente le type spécifié :</span><span class="sxs-lookup"><span data-stu-id="1a103-126">Use the [typeof](../operators/type-testing-and-cast.md#typeof-operator) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>

```csharp
Type stringType = typeof(string);
Console.WriteLine(stringType.FullName);

Type doubleType = typeof(System.Double);
Console.WriteLine(doubleType.FullName);

// Output:
// System.String
// System.Double
```
