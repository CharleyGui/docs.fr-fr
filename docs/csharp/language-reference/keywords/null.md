# <a name="null-c-reference"></a>null (référence C#)

Le mot clé `null` est un littéral qui représente une référence null, c’est-à-dire qui ne fait pas référence à un objet. `null` est la valeur par défaut des variables de type référence. Les types de valeur ordinaires ne peuvent pas être [nuls,](../builtin-types/nullable-value-types.md)à l’exception des types de valeur nuls .
