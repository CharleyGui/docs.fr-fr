## <a name="remarks"></a><span data-ttu-id="c62b7-130">Notes</span><span class="sxs-lookup"><span data-stu-id="c62b7-130">Remarks</span></span>  
  
- <span data-ttu-id="c62b7-131">La conversion numérique explicite peut entraîner une perte de précision ou la levée d’une exception, généralement <xref:System.OverflowException>.</span><span class="sxs-lookup"><span data-stu-id="c62b7-131">The explicit numeric conversion may cause loss of precision or result in throwing an exception, typically an <xref:System.OverflowException>.</span></span>  

- <span data-ttu-id="c62b7-132">Quand vous convertissez une valeur d’un type intégral en un autre type intégral, le résultat varie selon le [contexte de vérification](checked-and-unchecked.md) du dépassement.</span><span class="sxs-lookup"><span data-stu-id="c62b7-132">When you convert a value of an integral type to another integral type, the result depends on the overflow [checking context](checked-and-unchecked.md).</span></span> <span data-ttu-id="c62b7-133">Dans un contexte vérifié, la conversion réussit si la valeur source se situe dans la plage du type de destination.</span><span class="sxs-lookup"><span data-stu-id="c62b7-133">In a checked context, the conversion succeeds if the source value is within the range of the destination type.</span></span> <span data-ttu-id="c62b7-134">Sinon, une exception <xref:System.OverflowException> est levée.</span><span class="sxs-lookup"><span data-stu-id="c62b7-134">Otherwise, an <xref:System.OverflowException> is thrown.</span></span> <span data-ttu-id="c62b7-135">Dans un contexte non vérifié, la conversion réussit toujours et procède comme suit :</span><span class="sxs-lookup"><span data-stu-id="c62b7-135">In an unchecked context, the conversion always succeeds, and proceeds as follows:</span></span>

  - <span data-ttu-id="c62b7-136">Si le type source est plus grand que le type de destination, la valeur source est tronquée via l’abandon de ses bits « supplémentaires » les plus significatifs.</span><span class="sxs-lookup"><span data-stu-id="c62b7-136">If the source type is larger than the destination type, then the source value is truncated by discarding its "extra" most significant bits.</span></span> <span data-ttu-id="c62b7-137">Le résultat est ensuite traité en tant que valeur du type de destination.</span><span class="sxs-lookup"><span data-stu-id="c62b7-137">The result is then treated as a value of the destination type.</span></span>

  - <span data-ttu-id="c62b7-138">Si le type source est plus petit que le type de destination, la valeur source fait l’objet d’une extension de signe ou d’une extension de zéros, pour avoir la même taille que le type de destination.</span><span class="sxs-lookup"><span data-stu-id="c62b7-138">If the source type is smaller than the destination type, then the source value is either sign-extended or zero-extended so that it is the same size as the destination type.</span></span> <span data-ttu-id="c62b7-139">L’extension de signe est utilisée si le type source est signé. L’extension de zéro est utilisée si le type source est non signé.</span><span class="sxs-lookup"><span data-stu-id="c62b7-139">Sign-extension is used if the source type is signed; zero-extension is used if the source type is unsigned.</span></span> <span data-ttu-id="c62b7-140">Le résultat est ensuite traité en tant que valeur du type de destination.</span><span class="sxs-lookup"><span data-stu-id="c62b7-140">The result is then treated as a value of the destination type.</span></span>

  - <span data-ttu-id="c62b7-141">Si le type source a la même taille que le type de destination, la valeur source est traitée comme une valeur du type de destination.</span><span class="sxs-lookup"><span data-stu-id="c62b7-141">If the source type is the same size as the destination type, then the source value is treated as a value of the destination type.</span></span>
  
- <span data-ttu-id="c62b7-142">Quand vous convertissez une valeur de type `decimal` en type intégral, la valeur source est arrondie vers zéro à la valeur intégrale la plus proche.</span><span class="sxs-lookup"><span data-stu-id="c62b7-142">When you convert a `decimal` value to an integral type, this value is rounded towards zero to the nearest integral value.</span></span> <span data-ttu-id="c62b7-143">Si la valeur intégrale obtenue se trouve en dehors de la plage du type de destination, une exception <xref:System.OverflowException> est levée.</span><span class="sxs-lookup"><span data-stu-id="c62b7-143">If the resulting integral value is outside the range of the destination type, an <xref:System.OverflowException> is thrown.</span></span>  
  
- <span data-ttu-id="c62b7-144">Quand vous convertissez une valeur `double` ou `float` en type intégral, cette valeur est arrondie vers zéro, à la valeur intégrale la plus proche.</span><span class="sxs-lookup"><span data-stu-id="c62b7-144">When you convert a `double` or `float` value to an integral type, this value is rounded towards zero to the nearest integral value.</span></span> <span data-ttu-id="c62b7-145">Si la valeur intégrale résultante se situe en dehors de la plage du type de destination, le résultat varie selon le [contexte de vérification](checked-and-unchecked.md) du dépassement.</span><span class="sxs-lookup"><span data-stu-id="c62b7-145">If the resulting integral value is outside the range of the destination type, the result depends on the overflow [checking context](checked-and-unchecked.md).</span></span> <span data-ttu-id="c62b7-146">Dans un contexte vérifié, une exception <xref:System.OverflowException> est levée, alors que dans un contexte non vérifié, le résultat est une valeur non spécifiée du type de destination.</span><span class="sxs-lookup"><span data-stu-id="c62b7-146">In a checked context, an <xref:System.OverflowException> is thrown, while in an unchecked context, the result is an unspecified value of the destination type.</span></span>  
  
- <span data-ttu-id="c62b7-147">Quand vous convertissez `double` en `float`, la valeur `double` est arrondie à la valeur `float` la plus proche.</span><span class="sxs-lookup"><span data-stu-id="c62b7-147">When you convert `double` to `float`, the `double` value is rounded to the nearest `float` value.</span></span> <span data-ttu-id="c62b7-148">Si la valeur `double` est inférieure ou supérieure à la plage de valeurs autorisées pour le type de destination, la conversion a pour résultat la valeur zéro ou un nombre infini.</span><span class="sxs-lookup"><span data-stu-id="c62b7-148">If the `double` value is too small or too large to fit into the destination type, the result will be zero or infinity.</span></span>  
  
- <span data-ttu-id="c62b7-149">Dans le cas d’une conversion de `float` ou `double` en `decimal`, la valeur source est convertie en une représentation `decimal` et arrondie au nombre le plus proche après la 28ème décimale, si nécessaire.</span><span class="sxs-lookup"><span data-stu-id="c62b7-149">When you convert `float` or `double` to `decimal`, the source value is converted to `decimal` representation and rounded to the nearest number after the 28th decimal place if required.</span></span> <span data-ttu-id="c62b7-150">Selon la valeur de la valeur source, la conversion peut aboutir à l’un des résultats suivants :</span><span class="sxs-lookup"><span data-stu-id="c62b7-150">Depending on the value of the source value, one of the following results may occur:</span></span>  

  - <span data-ttu-id="c62b7-151">Si la valeur source est trop petite pour être représentée en tant que `decimal`, le résultat est zéro.</span><span class="sxs-lookup"><span data-stu-id="c62b7-151">If the source value is too small to be represented as a `decimal`, the result becomes zero.</span></span>  

  - <span data-ttu-id="c62b7-152">Si la valeur source est une valeur non numérique (NaN), un nombre infini ou une valeur trop grande pour être représentée au format `decimal`, une exception <xref:System.OverflowException> est levée.</span><span class="sxs-lookup"><span data-stu-id="c62b7-152">If the source value is NaN (not a number), infinity, or too large to be represented as a `decimal`, an <xref:System.OverflowException> is thrown.</span></span>  
  
- <span data-ttu-id="c62b7-153">Quand vous convertissez `decimal` en `float` ou `double`, la valeur `decimal` est arrondie à la valeur `double` ou `float` la plus proche.</span><span class="sxs-lookup"><span data-stu-id="c62b7-153">When you convert `decimal` to `float` or `double`, the `decimal` value is rounded to the nearest `double` or `float` value.</span></span>
