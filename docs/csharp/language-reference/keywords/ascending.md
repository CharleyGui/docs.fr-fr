# <a name="ascending-c-reference"></a>ascending (Référence C#)
Le mot clé contextuel `ascending` est utilisé dans la [clause orderby](./orderby-clause.md) dans les expressions de requête afin de spécifier que l’ordre de tri est du plus petit au plus grand. `ascending` étant l’ordre de tri par défaut, vous n’avez pas à le spécifier.
