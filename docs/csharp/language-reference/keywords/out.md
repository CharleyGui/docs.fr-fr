# <a name="out-c-reference"></a>out (référence C#)

Vous pouvez utiliser le mot clé `out` dans deux contextes :

- Comme [modificateur de paramètre](out-parameter-modifier.md), qui vous permet de passer un argument à une méthode par référence plutôt que par valeur.

- Dans les [déclarations d’un paramètre de type générique](out-generic-modifier.md) pour interfaces et délégués, qui indique qu’un paramètre de type est covariant.
