# <a name="-fullpaths-c-compiler-options"></a>-fullpaths (Options du compilateur C#)

Avec l’option **-fullpaths** , le compilateur spécifie le chemin d’accès complet au fichier lors de la liste des erreurs et des avertissements de compilation.  
  
## <a name="syntax"></a>Syntaxe  
  
```console  
-fullpaths  
```  
  
## <a name="remarks"></a>Notes  

 Par défaut, les erreurs et les avertissements générés par la compilation spécifient le nom du fichier dans lequel une erreur a été trouvée. L’option **-fullpaths** fait en sorte que le compilateur spécifie le chemin d’accès complet au fichier.  
  
 Cette option de compilateur n’est pas disponible dans Visual Studio et ne peut pas être changée par programmation.
