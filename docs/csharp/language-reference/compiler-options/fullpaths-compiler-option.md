# <a name="-fullpaths-c-compiler-options"></a>-fullpaths (Options du compilateur C#)
**L’option -fullpaths** amène le compilateur à spécifier le chemin complet vers le fichier lors de la liste des erreurs de compilation et des avertissements.  
  
## <a name="syntax"></a>Syntaxe  
  
```console  
-fullpaths  
```  
  
## <a name="remarks"></a>Notes   
 Par défaut, les erreurs et les avertissements générés par la compilation spécifient le nom du fichier dans lequel une erreur a été trouvée. **L’option -fullpaths** provoque le compilateur de spécifier le chemin complet vers le fichier.  
  
 Cette option de compilateur n’est pas disponible dans Visual Studio et ne peut pas être changée par programmation.
