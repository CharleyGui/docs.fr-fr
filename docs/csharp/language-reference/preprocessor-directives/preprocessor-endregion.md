# <a name="endregion-c-reference"></a>#endregion (référence C#)
`#endregion` marque la fin d’un bloc [#region](./preprocessor-region.md). Par exemple :  
  
```csharp
#region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
