# <a name="compiler-error-cs1922"></a><span data-ttu-id="2df24-102">Erreur du compilateur CS1922</span><span class="sxs-lookup"><span data-stu-id="2df24-102">Compiler Error CS1922</span></span>

<span data-ttu-id="2df24-103">Impossible d’initialiser un type 'type' avec un initialiseur de collection, car il n’implémente pas 'System.Collections.IEnumerable'</span><span class="sxs-lookup"><span data-stu-id="2df24-103">Collection initializer requires its type 'type' to implement System.Collections.IEnumerable.</span></span>

<span data-ttu-id="2df24-104">Pour utiliser un initialiseur de collection avec un type, le type doit implémenter `IEnumerable`.</span><span class="sxs-lookup"><span data-stu-id="2df24-104">In order to use a collection initializer with a type, the type must implement `IEnumerable`.</span></span> <span data-ttu-id="2df24-105">Cette erreur peut se produire si vous utilisez une syntaxe d’initialiseur de collection alors que vous voulez utiliser un initialiseur d’objet.</span><span class="sxs-lookup"><span data-stu-id="2df24-105">This error can occur if you accidentally use collection initializer syntax when you meant to use an object initializer.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="2df24-106">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="2df24-106">To correct this error</span></span>

- <span data-ttu-id="2df24-107">Si le type ne représente pas une collection, utilisez la syntaxe d’initialisation d’objet au lieu de la syntaxe d’initialiseur de collection.</span><span class="sxs-lookup"><span data-stu-id="2df24-107">If the type does not represent a collection, use object initializer syntax instead of collection initializer syntax.</span></span>

- <span data-ttu-id="2df24-108">Si le type ne représente pas une collection, modifiez-le pour qu’il implémente `IEnumerable` . Vous pourrez ensuite utiliser des initialiseurs de collection pour initialiser les objets de ce type.</span><span class="sxs-lookup"><span data-stu-id="2df24-108">If the type does represent a collection, modify it to implement `IEnumerable` before you can use collection initializers to initialize objects of that type.</span></span>

- <span data-ttu-id="2df24-109">Si le type représente une collection et si vous n’avez pas accès au code source, initialisez ses éléments à l’aide de leurs constructeurs de classe ou d’autres méthodes d’initialisation.</span><span class="sxs-lookup"><span data-stu-id="2df24-109">If the type represents a collection and you do not have access to the source code, just initialize its elements by using their class constructors or other initialization methods.</span></span>

## <a name="example"></a><span data-ttu-id="2df24-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="2df24-110">Example</span></span>

<span data-ttu-id="2df24-111">Le code suivant génère l’erreur CS1922 :</span><span class="sxs-lookup"><span data-stu-id="2df24-111">The following code produces CS1922:</span></span>

```csharp
// cs1922.cs
public class Test
{
    public static void Main()
    {
        // Collection initializer.
        var tc = new TestClass  {1,"hello"} ; // CS1922

        // Object initializer.
        var tc2 = new TestClass { memberA = 1, memberB = "hello" }; // OK
    }
}

public class TestClass
{
    public int memberA { get; set; }
    public string memberB { get; set; }
}
```
