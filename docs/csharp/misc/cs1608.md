# <a name="compiler-error-cs1608"></a>Erreur du compilateur CS1608
L'attribut Required n'est pas autorisé sur les types C#  
  
 <xref:System.Runtime.CompilerServices.RequiredAttributeAttribute> n’est pas autorisé sur les types définis dans le code C#.
