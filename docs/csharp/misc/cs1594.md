# <a name="compiler-error-cs1594"></a><span data-ttu-id="2f357-102">Erreur du compilateur CS1594</span><span class="sxs-lookup"><span data-stu-id="2f357-102">Compiler Error CS1594</span></span>
<span data-ttu-id="2f357-103">Le délégué 'délégué' utilise des arguments non valides</span><span class="sxs-lookup"><span data-stu-id="2f357-103">Delegate 'delegate' has some invalid arguments</span></span>  
  
 <span data-ttu-id="2f357-104">Le type d’un argument passé à un appel de [délégué](../language-reference/builtin-types/reference-types.md) ne correspond pas au type du paramètre dans la déclaration du délégué.</span><span class="sxs-lookup"><span data-stu-id="2f357-104">The type of an argument passed to a [delegate](../language-reference/builtin-types/reference-types.md) invocation does not agree with the type of the parameter in the delegate declaration.</span></span>  
  
 <span data-ttu-id="2f357-105">L’exemple suivant génère l’erreur CS1594 :</span><span class="sxs-lookup"><span data-stu-id="2f357-105">The following sample generates CS1594:</span></span>  
  
```csharp  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
