# <a name="compiler-error-cs0589"></a>Erreur du compilateur CS0589

Erreur interne du compilateur : étape 'PARSE'

 Essayez de déterminer si le compilateur échoue en raison de son incapacité à analyser une syntaxe inattendue. Si ce n’est pas le cas, essayez [Talk to Us](/visualstudio/ide/feedback-options).

> [!NOTE]
> Cette erreur de compilateur n’est plus utilisée dans Roslyn.
