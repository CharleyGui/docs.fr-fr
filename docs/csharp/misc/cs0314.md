## <a name="example"></a><span data-ttu-id="37470-109">Exemple</span><span class="sxs-lookup"><span data-stu-id="37470-109">Example</span></span>  
 <span data-ttu-id="37470-110">Le code suivant génère l’erreur CS0314 :</span><span class="sxs-lookup"><span data-stu-id="37470-110">The following code generates CS0314:</span></span>  
  
```csharp  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.  
public class C<T> : A<T> where T : ClassConstraint  
{ }  
```
