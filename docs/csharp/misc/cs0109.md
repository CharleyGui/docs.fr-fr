# <a name="compiler-warning-level-4-cs0109"></a>Avertissement du compilateur (niveau 4) CS0109
Le membre 'membre' ne masque pas un membre hérité. Le mot clé new n’est pas requis  
  
 Une déclaration de classe comprend le mot clé [new](../../csharp/language-reference/keywords/new-modifier.md) alors qu’elle ne remplace pas une déclaration existante dans une classe de base. Vous pouvez supprimer le mot clé **new** .  
  
 L’exemple suivant génère l’avertissement CS0109 :  
  
```csharp  
// CS0109.cs  
// compile with: /W:4  
namespace x  
{  
   public class a  
   {  
      public int i;  
   }  
  
   public class b : a  
   {  
      public new int i;  
      public new int j;   // CS0109  
      public static void Main()  
      {  
      }  
   }  
}  
```
