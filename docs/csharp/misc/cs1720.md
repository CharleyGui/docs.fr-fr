# <a name="compiler-warning-level-1-cs1720"></a><span data-ttu-id="e7187-102">Avertissement du compilateur (niveau 1) CS1720</span><span class="sxs-lookup"><span data-stu-id="e7187-102">Compiler Warning (level 1) CS1720</span></span>

<span data-ttu-id="e7187-103">L’expression provoquera toujours un System.NullReferenceException, car la valeur par défaut de 'generic type' est null</span><span class="sxs-lookup"><span data-stu-id="e7187-103">Expression will always cause a System.NullReferenceException because the default value of 'generic type' is null</span></span>  
  
 <span data-ttu-id="e7187-104">Si vous écrivez une expression qui implique la valeur par défaut d’une variable de type générique correspondant à un type référence (par exemple, une classe), cette erreur se produit.</span><span class="sxs-lookup"><span data-stu-id="e7187-104">If you write an expression involving the default of a generic type variable that is a reference type (for example, a class), this error will occur.</span></span> <span data-ttu-id="e7187-105">Prenons l'expression suivante :</span><span class="sxs-lookup"><span data-stu-id="e7187-105">Consider the following expression:</span></span>  
  
```csharp  
default(T).ToString()  
```  
  
 <span data-ttu-id="e7187-106">Étant donné que `T` est un type référence, sa valeur par défaut est Null ; par conséquent, si vous tentez de lui appliquer la méthode <xref:System.Object.ToString%2A> , cela lève une <xref:System.NullReferenceException>.</span><span class="sxs-lookup"><span data-stu-id="e7187-106">Since `T` is a reference type, its default value is null, and so attempting to apply the <xref:System.Object.ToString%2A> method to it will throw a <xref:System.NullReferenceException>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e7187-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="e7187-107">Example</span></span>  

 <span data-ttu-id="e7187-108">La contrainte de référence de classe sur le type `T` garantit que `T` est un type référence.</span><span class="sxs-lookup"><span data-stu-id="e7187-108">The class reference constraint on type `T` ensures that `T` is a reference type.</span></span>  
  
 <span data-ttu-id="e7187-109">L’exemple suivant génère l’avertissement CS1720.</span><span class="sxs-lookup"><span data-stu-id="e7187-109">The following sample generates CS1720.</span></span>  
  
```csharp  
// CS1720.cs  
using System;  
public class Tester
{  
    public static void GenericClass<T>(T t1) where T : class
    {  
        Console.WriteLine(default(T).ToString());  // CS1720  
    }  
    public static void Main() {}  
}  
```
