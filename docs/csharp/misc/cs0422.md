# <a name="compiler-warning-level-4-cs0422"></a>Avertissement du compilateur (niveau 4) CS0422

L'option /incremental n'est plus prise en charge

 La compilation incrémentale **(/incr** **ou/incrémentale**) n’est pas prise en charge dans C 2.0.

> [!NOTE]
> Cet avertissement de compilateur n’est plus utilisé à Roslyn.
