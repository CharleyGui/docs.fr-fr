# <a name="compiler-warning-level-4-cs0422"></a>Avertissement du compilateur (niveau 4) CS0422

L'option /incremental n'est plus prise en charge

 La compilation incrémentielle (**/incr** ou **/INCREMENTAL**) n’est pas prise en charge dans C# 2,0.

> [!NOTE]
> Cet avertissement du compilateur n’est plus utilisé dans Roslyn.
