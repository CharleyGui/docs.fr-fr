## <a name="example"></a><span data-ttu-id="c065a-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="c065a-106">Example</span></span>  

 <span data-ttu-id="c065a-107">L’exemple suivant génère l’erreur CS0730.</span><span class="sxs-lookup"><span data-stu-id="c065a-107">The following sample generates CS0730.</span></span> <span data-ttu-id="c065a-108">Il se compose de deux fichiers sources.</span><span class="sxs-lookup"><span data-stu-id="c065a-108">It consists of two source files.</span></span> <span data-ttu-id="c065a-109">Compilez tout d’abord le fichier bibliothèque `CS0730a.cs`, puis le fichier `CS0730.cs` faisant référence au fichier bibliothèque.</span><span class="sxs-lookup"><span data-stu-id="c065a-109">First, compile the library file `CS0730a.cs`, and the compile the file `CS0730.cs` referencing the library file.</span></span>  
  
```csharp  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```csharp  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```
