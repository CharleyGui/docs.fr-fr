# <a name="compiler-error-cs0155"></a>Erreur du compilateur CS0155
Le type intercepté ou levé doit être dérivé de System.Exception  
  
 Une tentative a été effectuée de passer un type de données qui ne dérive pas de **System.Exception** à un bloc [catch](../../csharp/language-reference/keywords/try-catch.md) . Seuls les types de données qui dérivent de **System.Exception** peuvent être passés à un bloc **catch** . Pour plus d’informations, consultez [Exceptions et gestion des exceptions](../../csharp/programming-guide/exceptions/index.md).  
  
 L’exemple suivant génère l’erreur CS0155 :  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
