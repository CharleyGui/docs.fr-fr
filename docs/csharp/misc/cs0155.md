# <a name="compiler-error-cs0155"></a><span data-ttu-id="c6d48-103">Erreur du compilateur CS0155</span><span class="sxs-lookup"><span data-stu-id="c6d48-103">Compiler Error CS0155</span></span>

<span data-ttu-id="c6d48-104">Le type intercepté ou levé doit être dérivé de System.Exception</span><span class="sxs-lookup"><span data-stu-id="c6d48-104">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="c6d48-105">Une tentative a été effectuée de passer un type de données qui ne dérive pas de **System.Exception** à un bloc [catch](../language-reference/keywords/try-catch.md) .</span><span class="sxs-lookup"><span data-stu-id="c6d48-105">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="c6d48-106">Seuls les types de données qui dérivent de **System.Exception** peuvent être passés à un bloc **catch** .</span><span class="sxs-lookup"><span data-stu-id="c6d48-106">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="c6d48-107">Pour plus d’informations, consultez [exceptions et gestion des exceptions](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="c6d48-107">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="c6d48-108">L’exemple suivant génère l’erreur CS0155 :</span><span class="sxs-lookup"><span data-stu-id="c6d48-108">The following sample generates CS0155:</span></span>  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
