# <a name="compiler-error-cs0557"></a><span data-ttu-id="6777d-103">Erreur du compilateur CS0557</span><span class="sxs-lookup"><span data-stu-id="6777d-103">Compiler Error CS0557</span></span>
<span data-ttu-id="6777d-104">La conversion définie par l’utilisateur dans le type 'class' est en double</span><span class="sxs-lookup"><span data-stu-id="6777d-104">Duplicate user-defined conversion in type 'class'</span></span>  
  
 <span data-ttu-id="6777d-105">Les routines de conversion en double ne sont pas autorisées dans une classe.</span><span class="sxs-lookup"><span data-stu-id="6777d-105">Duplicate conversion routines are not allowed in a class.</span></span>  
  
 <span data-ttu-id="6777d-106">L’exemple suivant génère l’erreur CS0557 :</span><span class="sxs-lookup"><span data-stu-id="6777d-106">The following example generates CS0557:</span></span>  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
