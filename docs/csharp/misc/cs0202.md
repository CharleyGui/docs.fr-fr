# <a name="compiler-error-cs0202"></a><span data-ttu-id="81b67-103">Erreur du compilateur CS0202</span><span class="sxs-lookup"><span data-stu-id="81b67-103">Compiler Error CS0202</span></span>

 <span data-ttu-id="81b67-105">Une fonction <xref:System.Collections.IEnumerable.GetEnumerator%2A> , utilisée pour activer l’utilisation d’instructions foreach, ne peut pas retourner un pointeur ou un tableau. Elle doit retourner une instance d’une classe qui est en mesure d’agir comme un énumérateur.</span><span class="sxs-lookup"><span data-stu-id="81b67-105">A <xref:System.Collections.IEnumerable.GetEnumerator%2A> function, used to enable the use of foreach statements, cannot return a pointer or array; it must return an instance of a class that is able to act as an enumerator.</span></span> <span data-ttu-id="81b67-106">Les conditions préalables pour servir d’énumérateur incluent une propriété Current publique et une méthode MoveNext publique.</span><span class="sxs-lookup"><span data-stu-id="81b67-106">The proper requirements to serve as an enumerator include a public Current property and a public MoveNext method.</span></span>

 <span data-ttu-id="81b67-109">L’exemple suivant génère l’erreur CS0202 :</span><span class="sxs-lookup"><span data-stu-id="81b67-109">The following sample generates CS0202:</span></span>  
  
```csharp  
// CS0202.cs  
  
public class C1  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public static implicit operator C1 (int c1)  
   {  
      return 0;  
   }  
}  
  
public class C2  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public C1[] GetEnumerator ()  
   // try the following line instead  
   // public C1 GetEnumerator ()  
   {  
      return null;  
   }  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
      C2 c2 = new C2();  
  
      foreach (C1 x in c2)   // CS0202  
      {  
         System.Console.WriteLine(x.Current);  
      }  
   }  
}  
```
