# <a name="compiler-error-cs0202"></a><span data-ttu-id="b08cf-102">Erreur du compilateur CS0202</span><span class="sxs-lookup"><span data-stu-id="b08cf-102">Compiler Error CS0202</span></span>

 <span data-ttu-id="b08cf-104">Une fonction <xref:System.Collections.IEnumerable.GetEnumerator%2A> , utilisée pour activer l’utilisation d’instructions foreach, ne peut pas retourner un pointeur ou un tableau.  <span data-ttu-id="b08cf-108">L’exemple suivant génère l’erreur CS0202 :</span><span class="sxs-lookup"><span data-stu-id="b08cf-108">The following sample generates CS0202:</span></span>  
  
```csharp  
// CS0202.cs  
  
public class C1  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public static implicit operator C1 (int c1)  
   {  
      return 0;  
   }  
}  
  
public class C2  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public C1[] GetEnumerator ()  
   // try the following line instead  
   // public C1 GetEnumerator ()  
   {  
      return null;  
   }  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
      C2 c2 = new C2();  
  
      foreach (C1 x in c2)   // CS0202  
      {  
         System.Console.WriteLine(x.Current);  
      }  
   }  
}  
```
