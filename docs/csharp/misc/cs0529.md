<span data-ttu-id="9ad18-104">L’interface héritée 'interface1' provoque un cycle dans la hiérarchie des interfaces de 'interface2'</span><span class="sxs-lookup"><span data-stu-id="9ad18-104">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="9ad18-105">La liste d’héritage pour une [interface](../language-reference/keywords/interface.md) comprend une référence directe ou indirecte à elle-même.</span><span class="sxs-lookup"><span data-stu-id="9ad18-105">The inheritance list for an [interface](../language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="9ad18-106">Une interface ne peut pas hériter d’elle-même.</span><span class="sxs-lookup"><span data-stu-id="9ad18-106">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="9ad18-107">L’exemple suivant génère l’erreur CS0529 :</span><span class="sxs-lookup"><span data-stu-id="9ad18-107">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
