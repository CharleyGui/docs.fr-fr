# <a name="compiler-error-cs1672"></a>Erreur du compilateur CS1672

Option non valide 'option' pour /platform ; doit être anycpu, x86, Itanium ou x64  
  
 Les options spécifient le type de processeur ; utilisez l’une des formes répertoriées.
