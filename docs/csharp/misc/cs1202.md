# <a name="compiler-warning-level-1-cs1202"></a>Avertissement du compilateur (niveau 1) CS1202

La fonctionnalité 'invalid feature' est déconseillée. Utilisez 'valid feature' à la place.  
  
 La fonctionnalité que vous essayez d’utiliser est désormais déconseillée. Mettez à jour votre code de manière à utiliser la fonctionnalité valide à la place.
