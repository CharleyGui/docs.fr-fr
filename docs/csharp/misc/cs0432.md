# <a name="compiler-error-cs0432"></a><span data-ttu-id="35c95-103">Erreur du compilateur CS0432</span><span class="sxs-lookup"><span data-stu-id="35c95-103">Compiler Error CS0432</span></span>
<span data-ttu-id="35c95-104">Alias 'identificateur' introuvable</span><span class="sxs-lookup"><span data-stu-id="35c95-104">Alias 'identifier' not found</span></span>  
  
 <span data-ttu-id="35c95-105">Cette erreur se produit quand vous placez « :: » à droite d’un identificateur qui n’est pas un alias.</span><span class="sxs-lookup"><span data-stu-id="35c95-105">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="35c95-106">Pour résoudre cette erreur, utilisez « . » à la place de « :: ».</span><span class="sxs-lookup"><span data-stu-id="35c95-106">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="35c95-107">L’exemple suivant génère l’erreur CS0432 :</span><span class="sxs-lookup"><span data-stu-id="35c95-107">The following example generates CS0432:</span></span>  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
