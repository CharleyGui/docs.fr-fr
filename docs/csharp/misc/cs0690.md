# <a name="compiler-error-cs0690"></a>Erreur du compilateur CS0690

Le fichier d’entrée 'fichier' contient des métadonnées non valides.  
  
 Vous pouvez ouvrir le fichier de métadonnées, mais en raison d’un problème de localisation, les métadonnées ont été endommagées. Cette erreur est comparable à l’erreur [CS0009](./cs0009.md), sauf que vous pouvez ouvrir le fichier de métadonnées.  
  
 Un exécutable de processus (PE) est un fichier exécutable.
