# <a name="compiler-error-cs1013"></a>Erreur du compilateur CS1013
Nombre non valide  
  
 Le compilateur a détecté un nombre dont le format est incorrect. Pour plus d’informations sur les types intégraux, consultez [Integral Types Table](../../csharp/language-reference/builtin-types/integral-numeric-types.md).  
  
## <a name="example"></a>Exemple  
 L’exemple suivant génère l’erreur CS1013 :  
  
```csharp  
// CS1013.cs  
class Sample  
{  
    static void Main()  
    {  
        int i = 0x;   // CS1013  
    }  
}  
```
