# <a name="compiler-warning-level-2-cs1571"></a>Avertissement du compilateur (niveau 2) CS1571
Le commentaire XML sur 'construction' a une balise param en double pour 'paramètre'  
  
 Lors de l’utilisation de l’option du compilateur [/doc](../language-reference/compiler-options/doc-compiler-option.md) , plusieurs commentaires ont été détectés pour le même paramètre de méthode. Supprimez l’une des lignes dupliquées.  
  
 L’exemple suivant génère l’avertissement CS1571 :  
  
```csharp  
// CS1571.cs  
// compile with: /W:2 /doc:x.xml  
  
/// <summary>help text</summary>  
public class MyClass  
{  
   /// <param name='Int1'>Used to indicate status.</param>  
   /// <param name='Char1'>An initial.</param>  
   /// <param name='Int1'>Used to indicate status.</param> // CS1571  
   public static void MyMethod(int Int1, char Char1)  
   {  
   }  
  
   /// <summary>help text</summary>  
   public static void Main ()  
   {  
   }  
}  
```
