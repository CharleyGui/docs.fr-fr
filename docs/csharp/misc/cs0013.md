# <a name="compiler-error-cs0013"></a>Erreur du compilateur CS0013
Erreur inattendue lors de l’écriture des métadonnées dans le fichier 'ModelStore.dll' -- 'Pas d’espace logique restant pour la création de chaînes utilisateur supplémentaires.  
  
 Le Runtime .NET n’a pas émis de métadonnées. Vérifiez que le chemin est correct et que le disque n’est pas saturé. Si le problème persiste, vous devrez peut-être réparer ou réinstaller Visual Studio et/ou .NET.
