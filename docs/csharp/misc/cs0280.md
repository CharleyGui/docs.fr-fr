 <span data-ttu-id="345b1-110">Le modèle « ressource » et `using` fournissent un autre exemple.</span><span class="sxs-lookup"><span data-stu-id="345b1-110">The "resource" pattern and `using` provide another example.</span></span> <span data-ttu-id="345b1-111">Le modèle « ressource » nécessite la méthode <xref:System.IDisposable.Dispose%2A> ; si vous définissez une propriété portant le même nom, vous obtenez cet avertissement.</span><span class="sxs-lookup"><span data-stu-id="345b1-111">The "resource" pattern requires the <xref:System.IDisposable.Dispose%2A> method; if you define a property with the same name, you will get this warning.</span></span>  
  
 <span data-ttu-id="345b1-112">Pour y remédier, assurez-vous que les signatures de méthode dans votre type correspondent aux signatures des méthodes correspondantes dans le modèle et assurez-vous de n’avoir aucune propriété portant le même nom qu’une méthode requise par le modèle.</span><span class="sxs-lookup"><span data-stu-id="345b1-112">To resolve this warning, ensure that the method signatures in your type match the signatures of the corresponding methods in the pattern, and ensure that you have no properties with the same name as a method required by the pattern.</span></span>  
  
## <a name="example"></a><span data-ttu-id="345b1-113">Exemple</span><span class="sxs-lookup"><span data-stu-id="345b1-113">Example</span></span>  

 <span data-ttu-id="345b1-114">L’exemple suivant génère l’avertissement CS0280.</span><span class="sxs-lookup"><span data-stu-id="345b1-114">The following sample generates CS0280.</span></span>  
  
```csharp  
// CS0280.cs  
using System;  
using System.Collections;  
  
public class ValidBase: IEnumerable  
{  
   IEnumerator IEnumerable.GetEnumerator()  
   {  
      yield return 0;  
   }  
  
   internal IEnumerator GetEnumerator()  
   {  
      yield return 0;  
   }  
}  
  
class Derived : ValidBase  
{  
   // field, not method  
   new public int GetEnumerator;  
}  
  
public class Test  
{  
   public static void Main()  
   {  
      foreach (int i in new Derived()) {}   // CS0280  
   }  
}  
```
