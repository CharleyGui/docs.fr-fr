## <a name="example"></a><span data-ttu-id="a4670-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="a4670-107">Example</span></span>  

 <span data-ttu-id="a4670-108">L’exemple suivant génère l’erreur CS1510 :</span><span class="sxs-lookup"><span data-stu-id="a4670-108">The following sample generates CS1510:</span></span>

```csharp  
   public static void M(ref int j)  
   {  
      j++;  
   }  
  
   public static void Main ()  
   {  
      M (ref 2);   // CS1510, can't pass a number as a ref parameter  
      // try the following to resolve the error  
      // M (ref j);  
   }  
}  
```
