# <a name="compiler-warning-level-1-cs0183"></a><span data-ttu-id="471ad-103">Avertissement du compilateur (niveau 1) CS0183</span><span class="sxs-lookup"><span data-stu-id="471ad-103">Compiler Warning (level 1) CS0183</span></span>
<span data-ttu-id="471ad-104">L’expression donnée est toujours du type fourni ('type')</span><span class="sxs-lookup"><span data-stu-id="471ad-104">The given expression is always of the provided ('type') type</span></span>  
  
 <span data-ttu-id="471ad-105">Si une instruction conditionnelle a toujours la valeur **true**, vous n’avez pas besoin de cette instruction conditionnelle.</span><span class="sxs-lookup"><span data-stu-id="471ad-105">If a conditional statement always evaluates to **true**, then you do not need a conditional statement.</span></span> <span data-ttu-id="471ad-106">Cet avertissement se produit quand vous essayez d’évaluer un type à l’aide de l’opérateur **is** .</span><span class="sxs-lookup"><span data-stu-id="471ad-106">This warning occurs when you try to evaluate a type using the **is** operator.</span></span> <span data-ttu-id="471ad-107">Si l’évaluation donne un type de valeur, le contrôle est inutile.</span><span class="sxs-lookup"><span data-stu-id="471ad-107">If the evaluation is a value type, then the check is unnecessary.</span></span>  
  
 <span data-ttu-id="471ad-108">L’exemple suivant génère l’avertissement CS0183 :</span><span class="sxs-lookup"><span data-stu-id="471ad-108">The following sample generates CS0183:</span></span>  
  
```csharp  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);
   }  
}  
```
