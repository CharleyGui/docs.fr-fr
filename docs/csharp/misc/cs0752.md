 <span data-ttu-id="9100b-104">Une méthode partielle ne peut pas avoir de paramètres [out](../language-reference/keywords/out-parameter-modifier.md) .</span><span class="sxs-lookup"><span data-stu-id="9100b-104">A partial method cannot have an [out](../language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="9100b-105">Les paramètres de sortie ne sont pas autorisés, car si la méthode partielle est supprimée par le compilateur, il n’y a aucune garantie que le paramètre de sortie soit assigné.</span><span class="sxs-lookup"><span data-stu-id="9100b-105">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>
