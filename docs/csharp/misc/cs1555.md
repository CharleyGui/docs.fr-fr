# <a name="compiler-error-cs1555"></a>Erreur du compilateur CS1555

Impossible de trouver 'classe' spécifié pour la méthode Main  
  
 Une classe a été spécifiée pour l’option [-main](../language-reference/compiler-options/main-compiler-option.md) du compilateur, mais le nom de la classe est introuvable dans le code source.
