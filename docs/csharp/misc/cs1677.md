 <span data-ttu-id="9e553-104">Cette erreur se produit lorsque le modificateur de type de paramètre dans une méthode anonyme ne correspond pas à celui utilisé dans la déclaration du délégué, vers lequel vous effectuez un cast de la méthode.</span><span class="sxs-lookup"><span data-stu-id="9e553-104">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9e553-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="9e553-105">Example</span></span>  

 <span data-ttu-id="9e553-106">L’exemple suivant génère l’erreur CS1677 :</span><span class="sxs-lookup"><span data-stu-id="9e553-106">The following sample generates CS1677:</span></span>  
  
```csharp  
// CS1677.cs  
delegate void D(int i);  
class Errors  
{  
    static void Main()
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```
