# <a name="compiler-error-cs1558"></a><span data-ttu-id="b6fe6-103">Erreur du compilateur CS1558</span><span class="sxs-lookup"><span data-stu-id="b6fe6-103">Compiler Error CS1558</span></span>

<span data-ttu-id="b6fe6-104">'class' ne dispose pas d’une méthode Main static appropriée</span><span class="sxs-lookup"><span data-stu-id="b6fe6-104">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="b6fe6-105">L’option de compilateur [-main](../language-reference/compiler-options/main-compiler-option.md) a spécifié une classe dans laquelle rechercher une méthode **main** .</span><span class="sxs-lookup"><span data-stu-id="b6fe6-105">The [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="b6fe6-106">Toutefois, la méthode [Main](../programming-guide/main-and-command-args/index.md) n’était pas définie correctement.</span><span class="sxs-lookup"><span data-stu-id="b6fe6-106">However, the [Main](../programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="b6fe6-107">L’exemple suivant génère l’erreur CS1558 en raison d’un type de retour non valide.</span><span class="sxs-lookup"><span data-stu-id="b6fe6-107">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
