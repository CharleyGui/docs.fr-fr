<span data-ttu-id="e07ff-103">Argument d’instance : impossible de convertir 'typeA' en 'typeB'.</span><span class="sxs-lookup"><span data-stu-id="e07ff-103">Instance argument: cannot convert from 'typeA' to 'typeB'.</span></span>  
  
 <span data-ttu-id="e07ff-104">Cette erreur est générée quand vous tentez d’appeler une méthode d’extension à partir d’une classe qu’elle n’étend pas.</span><span class="sxs-lookup"><span data-stu-id="e07ff-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="e07ff-105">Dans l’exemple suivant, la méthode d’extension est définie pour la classe dérivée `A`, mais pas pour la classe de base `B`.</span><span class="sxs-lookup"><span data-stu-id="e07ff-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e07ff-106">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="e07ff-106">To correct this error</span></span>  
  
1. <span data-ttu-id="e07ff-107">Créez une méthode d’extension pour le type dans lequel vous devez l’appeler ou placez l’appel dans un objet du type étendu par la méthode existante.</span><span class="sxs-lookup"><span data-stu-id="e07ff-107">Create a new extension method for the type where you have to invoke it, or else move the call into an object of the type that the existing method extends.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e07ff-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="e07ff-108">Example</span></span>  

 <span data-ttu-id="e07ff-109">Le code suivant génère les erreurs CS1928 et CS1929 :</span><span class="sxs-lookup"><span data-stu-id="e07ff-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
