# <a name="compiler-error-cs0065"></a><span data-ttu-id="17b2c-103">Erreur du compilateur CS0065</span><span class="sxs-lookup"><span data-stu-id="17b2c-103">Compiler Error CS0065</span></span>

<span data-ttu-id="17b2c-104">'événement' : la propriété event doit posséder à la fois des accesseurs add et remove</span><span class="sxs-lookup"><span data-stu-id="17b2c-104">'event' : event property must have both add and remove accessors</span></span>

<span data-ttu-id="17b2c-105">Un événement qui n’est pas un champ doit avoir les deux méthodes d’accès.</span><span class="sxs-lookup"><span data-stu-id="17b2c-105">An event that is not a field must have both access methods.</span></span>

<span data-ttu-id="17b2c-106">L’exemple suivant génère l’erreur CS0065 :</span><span class="sxs-lookup"><span data-stu-id="17b2c-106">The following sample generates CS0065:</span></span>

```csharp
// CS0065.cs
using System;
public delegate void EventHandler(object sender, int e);
public class MyClass
{
   public event EventHandler Click   // CS0065,
   {
      // to fix, uncomment the add and remove definitions
      /*
      add
      {
         Click += value;
      }
      remove
      {
         Click -= value;
      }
      */
   }

   public static void Main()
   {
   }
}
```
