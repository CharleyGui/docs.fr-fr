## <a name="example"></a><span data-ttu-id="1f5e3-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="1f5e3-107">Example</span></span>  

 <span data-ttu-id="1f5e3-108">L’exemple suivant génère l’erreur CS0540.</span><span class="sxs-lookup"><span data-stu-id="1f5e3-108">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
