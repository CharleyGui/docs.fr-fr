 <span data-ttu-id="2dd47-104">Un mot clé **add** ou **remove** dans une définition d’ [événement](../language-reference/keywords/event.md) doit avoir un corps.</span><span class="sxs-lookup"><span data-stu-id="2dd47-104">An **add** or **remove** keyword in an [event](../language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="2dd47-105">Pour plus d'informations, consultez [Événements](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="2dd47-105">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="2dd47-106">L’exemple suivant génère l’erreur CS0073 :</span><span class="sxs-lookup"><span data-stu-id="2dd47-106">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
