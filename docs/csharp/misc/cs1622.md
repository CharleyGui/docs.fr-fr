<span data-ttu-id="d74ef-104">Impossible de retourner une valeur à partir d’un itérateur.</span><span class="sxs-lookup"><span data-stu-id="d74ef-104">Cannot return a value from an iterator.</span></span> <span data-ttu-id="d74ef-105">Utilisez l’instruction yield return pour retourner une valeur ou yield break pour mettre fin à l’itération.</span><span class="sxs-lookup"><span data-stu-id="d74ef-105">Use the yield return statement to return a value, or yield break to end the iteration.</span></span>

 <span data-ttu-id="d74ef-108">L’exemple suivant génère l’erreur CS1622 :</span><span class="sxs-lookup"><span data-stu-id="d74ef-108">The following sample generates CS1622:</span></span>  
  
```csharp  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```
