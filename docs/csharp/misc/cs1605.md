# <a name="compiler-error-cs1605"></a>Erreur du compilateur CS1605
Impossible de passer 'variable' comme argument ref ou out, car il est en lecture seule  
  
 Une variable passée comme un paramètre [ref](../language-reference/keywords/ref.md) ou [out](../language-reference/keywords/out-parameter-modifier.md) doit être modifiée dans la méthode appelée. Il n’est donc pas possible de passer un paramètre en lecture seule en tant que `ref` ou `out`.
