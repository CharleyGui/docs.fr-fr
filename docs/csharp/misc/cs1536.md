# <a name="compiler-error-cs1536"></a>Erreur du compilateur CS1536
Type de paramètre non valide void  
  
 Il n’est pas nécessaire ou possible de spécifier un paramètre [void](../language-reference/keywords/void.md) autre qu’un pointeur void.  
  
 L’exemple suivant génère l’erreur CS1536 :  
  
```csharp  
// CS1536.cs  
class a  
{  
   public static int x( void )   // CS1536  
   // try the following line instead  
   // public static int x()  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
