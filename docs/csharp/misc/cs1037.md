# <a name="compiler-error-cs1037"></a>Erreur du compilateur CS1037

Opérateur surchargeable attendu  
  
 Lors de la spécification d’un commentaire avec [-doc](../language-reference/compiler-options/doc-compiler-option.md), le compilateur a rencontré un lien non valide.
