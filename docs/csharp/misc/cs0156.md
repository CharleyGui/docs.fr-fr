# <a name="compiler-error-cs0156"></a><span data-ttu-id="7b5b9-103">Erreur du compilateur CS0156</span><span class="sxs-lookup"><span data-stu-id="7b5b9-103">Compiler Error CS0156</span></span>

<span data-ttu-id="7b5b9-104">Une instruction throw sans argument n'est pas autorisée dans une clause finally qui est imbriquée dans la clause catch englobante la plus proche</span><span class="sxs-lookup"><span data-stu-id="7b5b9-104">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>  
  
 <span data-ttu-id="7b5b9-105">Une instruction [throw](../language-reference/keywords/throw.md) sans paramètre peut uniquement apparaître dans une clause **catch** qui n’accepte aucun paramètre.</span><span class="sxs-lookup"><span data-stu-id="7b5b9-105">A [throw](../language-reference/keywords/throw.md) statement with no parameters can only appear in a **catch** clause that takes no parameters.</span></span>  
  
 <span data-ttu-id="7b5b9-106">Pour plus d’informations, consultez [exceptions et gestion des exceptions](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="7b5b9-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="7b5b9-107">L’exemple suivant génère l’erreur CS0156 :</span><span class="sxs-lookup"><span data-stu-id="7b5b9-107">The following sample generates CS0156:</span></span>  
  
```csharp  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```
