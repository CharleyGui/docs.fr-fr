# <a name="compiler-error-cs0210"></a><span data-ttu-id="14dbe-103">Erreur du compilateur CS0210</span><span class="sxs-lookup"><span data-stu-id="14dbe-103">Compiler Error CS0210</span></span>

<span data-ttu-id="14dbe-104">Vous devez fournir un initialiseur dans une déclaration d'instruction fixed ou using</span><span class="sxs-lookup"><span data-stu-id="14dbe-104">You must provide an initializer in a fixed or using statement declaration</span></span>  
  
 <span data-ttu-id="14dbe-105">Vous devez déclarer et initialiser la variable dans une [instruction fixed](../language-reference/keywords/fixed-statement.md).</span><span class="sxs-lookup"><span data-stu-id="14dbe-105">You must declare and initialize the variable in a [fixed statement](../language-reference/keywords/fixed-statement.md).</span></span> <span data-ttu-id="14dbe-106">Pour plus d’informations, consultez [Pointeurs et code unsafe](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="14dbe-106">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="14dbe-107">L’exemple suivant génère l’erreur CS0210 :</span><span class="sxs-lookup"><span data-stu-id="14dbe-107">The following sample generates CS0210:</span></span>  
  
```csharp  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 <span data-ttu-id="14dbe-108">L’exemple suivant génère également l’erreur CS0210, car l’ [instruction using](../language-reference/keywords/using-statement.md) n’a pas d’initialiseur.</span><span class="sxs-lookup"><span data-stu-id="14dbe-108">The following sample also generates CS0210 because the [using statement](../language-reference/keywords/using-statement.md) has no initializer.</span></span>  
  
```csharp  
// CS0210b.cs  
  
using System.IO;  
class Test
{  
   static void Main()
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
