# <a name="compiler-error-cs0451"></a><span data-ttu-id="09bd1-103">Erreur du compilateur CS0451</span><span class="sxs-lookup"><span data-stu-id="09bd1-103">Compiler Error CS0451</span></span>

<span data-ttu-id="09bd1-104">La contrainte 'new()' ne peut pas être utilisée avec la contrainte 'struct'</span><span class="sxs-lookup"><span data-stu-id="09bd1-104">The 'new()' constraint cannot be used with the 'struct' constraint</span></span>  
  
 <span data-ttu-id="09bd1-105">Au moment de spécifier des contraintes sur le type d’un générique, la contrainte `new()` ne peut être utilisée qu’avec des contraintes de type classe, des contraintes de type interface, des contraintes de type référence et des contraintes de paramètre de type, mais pas avec des contraintes de type valeur.</span><span class="sxs-lookup"><span data-stu-id="09bd1-105">When specifying constraints on the type of a generic, the `new()` constraint may only be used with class type constraints, interface type constraints, reference type constraints, and type parameter constraints, but not with value type constraints.</span></span>  
  
## <a name="example"></a><span data-ttu-id="09bd1-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="09bd1-106">Example</span></span>  

 <span data-ttu-id="09bd1-107">L’exemple suivant génère l’erreur CS0451.</span><span class="sxs-lookup"><span data-stu-id="09bd1-107">The following sample generates CS0451.</span></span>  
  
```csharp  
// CS0451.cs  
using System;  
public class C4
{  
   public void F4<T>() where T : struct, new() {}   // CS0451  
}  
  
// OK  
public class C5  
{  
   public void F5<T>() where T : struct {}  
}  
  
public class C6  
{  
   public void F6<T>() where T : new() {}  
}  
```
