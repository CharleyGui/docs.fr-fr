# <a name="compiler-error-cs1588"></a>Erreur du compilateur CS1588
Impossible de déterminer le répertoire common language runtime -- 'raison'  
  
 Le compilateur ne peut pas récupérer le répertoire dans lequel le Common Language Runtime du .NET Framework est installé. Cette erreur indique que le Common Language Runtime n’est pas correctement installé.
