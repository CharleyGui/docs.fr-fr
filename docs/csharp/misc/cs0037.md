# <a name="compiler-error-cs0037"></a>Erreur du compilateur CS0037
Impossible de convertir une valeur null en 'type', car il s’agit d’un type valeur qui n’autorise pas les valeurs null  
  
 Le compilateur ne peut pas assigner `null` à un type valeur ; `null` peut uniquement être assigné à un [type référence](../language-reference/keywords/reference-types.md) ou à un [type valeur Nullable](../language-reference/builtin-types/nullable-value-types.md). [struct](../language-reference/builtin-types/struct.md) est un type valeur.
  
 L’exemple suivant génère l’erreur CS0037 :  
  
```csharp  
// CS0037.cs  
public struct s  
{  
}  
  
class a  
{  
   public static void Main()  
   {  
      int i = null;   // CS0037  
      s ss = null;    // CS0037  
   }  
}  
```
