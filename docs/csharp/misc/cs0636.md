# <a name="compiler-error-cs0636"></a><span data-ttu-id="78ecb-102">Erreur du compilateur CS0636</span><span class="sxs-lookup"><span data-stu-id="78ecb-102">Compiler Error CS0636</span></span>
<span data-ttu-id="78ecb-103">L'attribut FieldOffset ne peut être placé que sur des membres de types marqués avec StructLayout(LayoutKind.Explicit)</span><span class="sxs-lookup"><span data-stu-id="78ecb-103">The FieldOffset attribute can only be placed on members of types marked with the StructLayout(LayoutKind.Explicit)</span></span>  
  
 <span data-ttu-id="78ecb-104">Vous devez utiliser l’attribut **StructLayout (LayoutKind.Explicit)** sur la déclaration de struct si elle contient des membres marqués avec l’attribut **FieldOffset** .</span><span class="sxs-lookup"><span data-stu-id="78ecb-104">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="78ecb-105">Pour plus d’informations, consultez [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span><span class="sxs-lookup"><span data-stu-id="78ecb-105">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>  
  
 <span data-ttu-id="78ecb-106">L’exemple suivant génère l’erreur CS0636 :</span><span class="sxs-lookup"><span data-stu-id="78ecb-106">The following sample generates CS0636:</span></span>  
  
```csharp  
// CS0636.cs  
using System;  
using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]  
struct Worksheet  
{  
   [FieldOffset(4)]public int i;   // CS0636
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
