 <span data-ttu-id="ff6df-105">Vous avez utilisé un paramètre de type comme argument d’attribut, ce qui n’est pas autorisé.</span><span class="sxs-lookup"><span data-stu-id="ff6df-105">A type parameter was used as an attribute argument, which is not allowed.</span></span> <span data-ttu-id="ff6df-106">Utilisez un type non générique.</span><span class="sxs-lookup"><span data-stu-id="ff6df-106">Use a non-generic type.</span></span>  
  
 <span data-ttu-id="ff6df-107">L’exemple suivant génère l’erreur CS0416 :</span><span class="sxs-lookup"><span data-stu-id="ff6df-107">The following sample generates CS0416:</span></span>  
  
```csharp  
// CS0416.cs  
public class MyAttribute : System.Attribute  
{  
   public MyAttribute(System.Type t)  
   {  
   }  
}

{  
  
   [MyAttribute(typeof(G<T>))]  // CS0416  
   public void F()  
   {  
   }  
  
}  
```
