# <a name="compiler-error-cs0416"></a><span data-ttu-id="ea343-103">Erreur du compilateur CS0416</span><span class="sxs-lookup"><span data-stu-id="ea343-103">Compiler Error CS0416</span></span>

<span data-ttu-id="ea343-104">'paramètre_de_type' : un argument d’attribut ne peut pas utiliser de paramètres de type</span><span class="sxs-lookup"><span data-stu-id="ea343-104">'type parameter': an attribute argument cannot use type parameters</span></span>  
  
 <span data-ttu-id="ea343-105">Vous avez utilisé un paramètre de type comme argument d’attribut, ce qui n’est pas autorisé.</span><span class="sxs-lookup"><span data-stu-id="ea343-105">A type parameter was used as an attribute argument, which is not allowed.</span></span> <span data-ttu-id="ea343-106">Utilisez un type non générique.</span><span class="sxs-lookup"><span data-stu-id="ea343-106">Use a non-generic type.</span></span>  
  
 <span data-ttu-id="ea343-107">L’exemple suivant génère l’erreur CS0416 :</span><span class="sxs-lookup"><span data-stu-id="ea343-107">The following sample generates CS0416:</span></span>  
  
```csharp  
// CS0416.cs  
public class MyAttribute : System.Attribute  
{  
   public MyAttribute(System.Type t)  
   {  
   }  
}  
  
class G<T>  
{  
  
   [MyAttribute(typeof(G<T>))]  // CS0416  
   public void F()  
   {  
   }  
  
}  
```
