# <a name="compiler-error-cs0522"></a><span data-ttu-id="2d586-102">Erreur du compilateur CS0522</span><span class="sxs-lookup"><span data-stu-id="2d586-102">Compiler Error CS0522</span></span>
<span data-ttu-id="2d586-103">'constructeur' : les structures ne peuvent pas appeler les constructeurs de classe de base</span><span class="sxs-lookup"><span data-stu-id="2d586-103">'constructor' : structs cannot call base class constructors</span></span>  
  
 <span data-ttu-id="2d586-104">Un [struct](../language-reference/builtin-types/struct.md) ne peut pas appeler un constructeur de classe de base. Supprimez l’appel au constructeur de classe de base.</span><span class="sxs-lookup"><span data-stu-id="2d586-104">A [struct](../language-reference/builtin-types/struct.md) cannot call a base class constructor; remove the call to the base class constructor.</span></span>  
  
 <span data-ttu-id="2d586-105">L’exemple suivant génère l’erreur CS0522 :</span><span class="sxs-lookup"><span data-stu-id="2d586-105">The following sample generates CS0522:</span></span>  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```
