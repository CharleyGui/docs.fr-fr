# <a name="compiler-error-cs1034"></a>Erreur du compilateur CS1034
Limite du compilateur dépassée : la ligne ne peut pas compter plus de 'number' caractères  
  
 La limite du nombre de caractères autorisés sur une ligne s’élève à 16 777 214.
