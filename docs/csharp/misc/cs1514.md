# <a name="compiler-error-cs1514"></a>Erreur du compilateur CS1514

{ attendue  
  
 Le compilateur attendait une accolade ouvrante (`{`) qui n’a pas été trouvée.  
  
 L’exemple suivant génère l’erreur CS1514 :  
  
```csharp  
// CS1514.cs  
namespace x  
// CS1514, no open curly brace  
```
