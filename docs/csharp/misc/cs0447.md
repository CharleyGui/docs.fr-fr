<span data-ttu-id="8ea2e-104">Les attributs ne peuvent pas être utilisés sur des arguments de type, uniquement sur des paramètres de type</span><span class="sxs-lookup"><span data-stu-id="8ea2e-104">Attributes cannot be used on type arguments, only on type parameters</span></span>

<span data-ttu-id="8ea2e-107">La ligne de code suivante génère cette erreur.</span><span class="sxs-lookup"><span data-stu-id="8ea2e-107">The following line of code will generate this error.</span></span> <span data-ttu-id="8ea2e-108">Il est supposé que la classe `C`, qui est définie dans la ligne de code précédente, possède une méthode statique nommée `MyStaticMethod`.</span><span class="sxs-lookup"><span data-stu-id="8ea2e-108">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>
