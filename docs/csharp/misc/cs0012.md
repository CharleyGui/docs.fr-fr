 <span data-ttu-id="3a630-110">Ensuite :</span><span class="sxs-lookup"><span data-stu-id="3a630-110">Then:</span></span>

```csharp  
      return new A();  
   }

```  
  
 <span data-ttu-id="3a630-111">Ensuite :</span><span class="sxs-lookup"><span data-stu-id="3a630-111">Then:</span></span>
