 <span data-ttu-id="cca09-109">Puis :</span><span class="sxs-lookup"><span data-stu-id="cca09-109">Then:</span></span>

```csharp  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="cca09-110">Puis :</span><span class="sxs-lookup"><span data-stu-id="cca09-110">Then:</span></span>
