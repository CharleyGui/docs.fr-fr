# <a name="compiler-error-cs1563"></a>Erreur du compilateur CS1563
La sortie 'output file' ne possède aucun fichier source  
  
 L’option de compilateur [/out](../language-reference/compiler-options/out-compiler-option.md) a été spécifiée, mais elle n’est suivie d’aucun fichier de code source. Vous devez arrêter d’utiliser **/out**ou spécifier les fichiers de code source après **/out**.
