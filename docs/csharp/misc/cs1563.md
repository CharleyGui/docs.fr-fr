# <a name="compiler-error-cs1563"></a>Erreur du compilateur CS1563
La sortie 'output file' ne possède aucun fichier source  
  
 L’option du compilateur [-out](../language-reference/compiler-options/out-compiler-option.md) a été spécifiée, mais aucun fichier de code source ne suit. Vous devez arrêter d’utiliser **/out**ou spécifier les fichiers de code source après **/out**.
