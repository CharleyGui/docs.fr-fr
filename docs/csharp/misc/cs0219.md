<span data-ttu-id="aafda-103">La variable ’variable’ est assignée, mais sa valeur n’est jamais utilisée</span><span class="sxs-lookup"><span data-stu-id="aafda-103">The variable 'variable' is assigned but its value is never used</span></span>

 <span data-ttu-id="aafda-104">Le compilateur émet un avertissement de niveau trois lorsque vous déclarez et assignez une variable, mais ne l’utilisez pas.</span><span class="sxs-lookup"><span data-stu-id="aafda-104">The compiler issues a level-three warning, when you declare and assign a variable, but do not use it.</span></span>

 > [!NOTE]
 > <span data-ttu-id="aafda-105">Le compilateur ne génère cet avertissement que lorsque la valeur variable est une constante de compilateur.</span><span class="sxs-lookup"><span data-stu-id="aafda-105">The compiler generates this warning only when the variable value is a compile-time constant.</span></span> <span data-ttu-id="aafda-106">L’attribution d’une expression ou d’une méthode non constante à une variable locale facilite l’observation de ces expressions dans le débogénaire.</span><span class="sxs-lookup"><span data-stu-id="aafda-106">Assigning a non-constant expression or method result to a local variable makes it easier to observe those expressions in the debugger.</span></span> <span data-ttu-id="aafda-107">Il rend également le résultat accessible, empêchant la collecte des ordures tandis que cette variable est accessible.</span><span class="sxs-lookup"><span data-stu-id="aafda-107">It also makes the result reachable, preventing garbage collection while that variable is reachable.</span></span>

 <span data-ttu-id="aafda-108">L’échantillon suivant montre les cas quand et quand l’avertissement n’est pas généré :</span><span class="sxs-lookup"><span data-stu-id="aafda-108">The following sample shows the cases when and when not the warning is generated:</span></span>

```csharp
// CS0219.cs
// compile with: /W:3
public class MyClass
{
   public static void Main()
   {
      var interpolated = "Interpolated";
      var a = 0; // CS0219
      int b = GetZero(); // Doesn't generate a warning.
      var c = "Regular string"; // CS0219
      var d = $"Constant interpolated string"; // Doesn't generate a warning.
      var e = $"{interpolated} string"; // Doesn't generate a warning.
   }

   private static int GetZero()
   {
       return 0;
   }
}  
```
