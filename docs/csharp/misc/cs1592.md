# <a name="compiler-warning-level-1-cs1592"></a>Avertissement du compilateur (niveau 1) CS1592
Code XML incorrect dans le fichier de commentaires inclus -- 'reason'  
  
 Un problème, signalé comme ***raison***, a été trouvé dans le fichier spécifié par [ \<](../programming-guide/xmldoc/include.md) la balise include >.
