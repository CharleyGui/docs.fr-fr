## <a name="example"></a><span data-ttu-id="285f4-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="285f4-110">Example</span></span>  
 <span data-ttu-id="285f4-111">L’exemple suivant génère l’erreur CS0757 :</span><span class="sxs-lookup"><span data-stu-id="285f4-111">The following example generates CS0757:</span></span>

    public partial class C  
    {  
        // Defining declaration.  
        partial void Part();  
  
        // Implementing declaration.

        {  
            //...Do something.  
        }  
  
        // Second implementing declaration.  
        partial void Part() // CS0757  
        {  
            //...Do something.  
        }

        {  
            return 1;  
        }  
    }
