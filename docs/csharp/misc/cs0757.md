## <a name="example"></a><span data-ttu-id="48c05-109">Exemples</span><span class="sxs-lookup"><span data-stu-id="48c05-109">Example</span></span>  
 <span data-ttu-id="48c05-110">L’exemple suivant génère l’erreur CS0757 :</span><span class="sxs-lookup"><span data-stu-id="48c05-110">The following example generates CS0757:</span></span>  
  
```csharp  
// cs0757.cs  
using System;  
  
    public partial class C  
    {  
        // Defining declaration.  
        partial void Part();  
  
        // Implementing declaration.  
        partial void Part()  
        {  
            //...Do something.  
        }  
  
        // Second implementing declaration.  
        partial void Part() // CS0757  
        {  
            //...Do something.  
        }  
  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
