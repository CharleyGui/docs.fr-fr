# <a name="compiler-error-cs1906"></a>Erreur du compilateur CS1906
Option 'option' non valide ; la visibilité de la ressource doit être 'public' ou 'private'  
  
 Cette erreur indique une option de ligne de commande [/Resource (incorporer un fichier de ressources à la sortie)](../language-reference/compiler-options/resource-compiler-option.md) ou [-linkresource (lien vers la ressource .NET Framework)](../language-reference/compiler-options/linkresource-compiler-option.md) non valide. Vérifiez la syntaxe de l’option de ligne de commande **/resource** ou **/linkresource** et assurez-vous que le modificateur d’accessibilité utilisé est soit **public** ou `private`.
