# <a name="compiler-error-cs0220"></a><span data-ttu-id="1de5b-103">Erreur du compilateur CS0220</span><span class="sxs-lookup"><span data-stu-id="1de5b-103">Compiler Error CS0220</span></span>
<span data-ttu-id="1de5b-104">L'opération engendre un dépassement de capacité au moment de la compilation dans le mode checked</span><span class="sxs-lookup"><span data-stu-id="1de5b-104">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="1de5b-105">Une opération a été détectée par [checked](../language-reference/keywords/checked.md), qui est la valeur par défaut, et a entraîné une perte de données.</span><span class="sxs-lookup"><span data-stu-id="1de5b-105">An operation was detected by [checked](../language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="1de5b-106">Pour résoudre cette erreur, corrigez les entrées de l’assignation ou utilisez [unchecked](../language-reference/keywords/unchecked.md) .</span><span class="sxs-lookup"><span data-stu-id="1de5b-106">Either correct the inputs to the assignment or use [unchecked](../language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="1de5b-107">Pour plus d’informations, consultez [cochée et désactivée](../language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="1de5b-107">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="1de5b-108">L’exemple suivant génère l’erreur CS0220 :</span><span class="sxs-lookup"><span data-stu-id="1de5b-108">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
