## <a name="example"></a><span data-ttu-id="9fcd1-109">Exemple</span><span class="sxs-lookup"><span data-stu-id="9fcd1-109">Example</span></span>

<span data-ttu-id="9fcd1-110">L’exemple C++ suivant compile dans un fichier. dll.</span><span class="sxs-lookup"><span data-stu-id="9fcd1-110">The following C++ example compiles to a .dll file.</span></span> <span data-ttu-id="9fcd1-111">Notez que l’attribut [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) est mis en commentaire afin de générer l’erreur.</span><span class="sxs-lookup"><span data-stu-id="9fcd1-111">Note that the [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) attribute is commented out in order to generate the error.</span></span>

```cpp
// CPP0021.cpp
// compile with: /clr /LD
using namespace System::Reflection;
// Uncomment the following line to resolve
//[DefaultMember("myItem")]
public ref class MyClassMC
{
        public:
        property int myItem[int]
        {
            int get(int i){  return 5; }
            void set(int i, int value) {}
        }
};
```

<span data-ttu-id="9fcd1-112">L’exemple C# suivant appelle le fichier. dll.</span><span class="sxs-lookup"><span data-stu-id="9fcd1-112">The following C# example calls the .dll file.</span></span> <span data-ttu-id="9fcd1-113">Elle tente d’accéder à la classe via un indexeur, mais étant donné qu’aucun membre n’a été déclaré en tant qu’indexeur par défaut, l’erreur est générée.</span><span class="sxs-lookup"><span data-stu-id="9fcd1-113">It attempts to access the class via an indexer, but because no member was declared as the default indexer, the error is generated.</span></span> <span data-ttu-id="9fcd1-114">Vous pouvez résoudre l’erreur en supprimant les commentaires de la ligne `[DefaultMember("myItem")]` .</span><span class="sxs-lookup"><span data-stu-id="9fcd1-114">You can address the error by uncommenting the line `[DefaultMember("myItem")]`.</span></span>

// CS0021.cs
