# <a name="compiler-error-cs0021"></a><span data-ttu-id="75bfa-102">Erreur du compilateur CS0021</span><span class="sxs-lookup"><span data-stu-id="75bfa-102">Compiler Error CS0021</span></span>

<span data-ttu-id="75bfa-104">L’utilisateur a tenté d’accéder à une valeur via un indexeur dans un type de données qui ne prend pas en charge [Indexers](../programming-guide/indexers/index.md).</span><span class="sxs-lookup"><span data-stu-id="75bfa-104">An attempt was made to access a value through an indexer on a data type that does not support [Indexers](../programming-guide/indexers/index.md).</span></span>  
  
<span data-ttu-id="75bfa-105">L’erreur CS0021 peut se produire quand vous tentez d’utiliser un indexeur dans un assembly C++.</span><span class="sxs-lookup"><span data-stu-id="75bfa-105">You may get CS0021 when trying to use an indexer in a C++ assembly.</span></span> <span data-ttu-id="75bfa-106">Dans ce cas, décorez la classe C++ avec l’attribut `DefaultMember` pour que le compilateur C# puisse connaître l’indexeur par défaut.</span><span class="sxs-lookup"><span data-stu-id="75bfa-106">In this case, decorate the C++ class with the `DefaultMember` attribute so the C# compiler knows which indexer is the default.</span></span> <span data-ttu-id="75bfa-107">L’exemple suivant génère l’erreur CS0021 :</span><span class="sxs-lookup"><span data-stu-id="75bfa-107">The following sample generates CS0021.</span></span>  
  
## <a name="example"></a><span data-ttu-id="75bfa-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="75bfa-108">Example</span></span>  

<span data-ttu-id="75bfa-109">L’exemple C++ suivant compile dans un fichier. dll.</span><span class="sxs-lookup"><span data-stu-id="75bfa-109">The following C++ example compiles to a .dll file.</span></span> <span data-ttu-id="75bfa-110">Notez que l’attribut [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) est mis en commentaire afin de générer l’erreur.</span><span class="sxs-lookup"><span data-stu-id="75bfa-110">Note that the the [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) attribute is commented out in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  

<span data-ttu-id="75bfa-111">L’exemple C# suivant appelle le fichier. dll.</span><span class="sxs-lookup"><span data-stu-id="75bfa-111">The following C# example calls the .dll file.</span></span> <span data-ttu-id="75bfa-112">Elle tente d’accéder à la classe via un indexeur, mais étant donné qu’aucun membre n’a été déclaré en tant qu’indexeur par défaut, l’erreur est générée.</span><span class="sxs-lookup"><span data-stu-id="75bfa-112">It attempts to access the class via an indexer, but because no member was declared as the default indexer, the error is generated.</span></span> <span data-ttu-id="75bfa-113">Vous pouvez résoudre l’erreur en supprimant les marques de commentaire de la ligne `[DefaultMember("myItem")]`.</span><span class="sxs-lookup"><span data-stu-id="75bfa-113">You can address the error by uncommenting the line `[DefaultMember("myItem")]`.</span></span>
  
```csharp  
// CS0021.cs  
// compile with: /reference:CPP0021.dll  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```
