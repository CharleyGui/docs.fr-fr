# <a name="compiler-error-cs1565"></a>Erreur du compilateur CS1565
Options spécifiées en conflit : fichier de ressources Win32 ; icône Win32  
  
 Il n’est pas possible de spécifier à la fois l’option [de compilateur-win32res](../language-reference/compiler-options/win32res-compiler-option.md) et l’option de compilateur [-win32ico](../language-reference/compiler-options/win32icon-compiler-option.md) dans la même compilation.
