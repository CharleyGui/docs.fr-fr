# <a name="compiler-error-cs1559"></a>Erreur du compilateur CS1559
Impossible d’utiliser 'object' pour la méthode Main, car il est importé  
  
 Une classe non valide a été spécifiée pour l’option [de compilateur-main](../language-reference/compiler-options/main-compiler-option.md) ; la classe ne peut pas être utilisée comme emplacement pour la méthode [main](../programming-guide/main-and-command-args/index.md) .
