 <span data-ttu-id="5dc5d-104">Une variable [readonly](../language-reference/keywords/readonly.md) doit avoir la même utilisation de [static](../language-reference/keywords/static.md) que le constructeur dans lequel vous voulez l’initialiser.</span><span class="sxs-lookup"><span data-stu-id="5dc5d-104">A [readonly](../language-reference/keywords/readonly.md) variable must have the same [static](../language-reference/keywords/static.md) usage as the constructor in which you want to initialize it.</span></span> <span data-ttu-id="5dc5d-105">Pour plus d’informations, consultez [Constructeurs statiques](../programming-guide/classes-and-structs/static-constructors.md).</span><span class="sxs-lookup"><span data-stu-id="5dc5d-105">For more information, see [Static Constructors](../programming-guide/classes-and-structs/static-constructors.md).</span></span>  
  
 <span data-ttu-id="5dc5d-106">L’exemple suivant génère l’erreur CS0198 :</span><span class="sxs-lookup"><span data-stu-id="5dc5d-106">The following sample generates CS0198:</span></span>  
  
```csharp

{  
   public static readonly int TestInt = 6;  
  
   MyClass()  
   {  
      TestInt = 11;   // CS0198, constructor is not static and readonly field is  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
