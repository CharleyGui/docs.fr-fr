# <a name="compiler-error-cs0198"></a><span data-ttu-id="b9184-103">Erreur du compilateur CS0198</span><span class="sxs-lookup"><span data-stu-id="b9184-103">Compiler Error CS0198</span></span>

<span data-ttu-id="b9184-104">Les champs du champ readonly statique 'nom' ne peuvent pas être assignés (sauf s’ils appartiennent à un constructeur statique ou un initialiseur de variable)</span><span class="sxs-lookup"><span data-stu-id="b9184-104">Fields of static readonly field 'name' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="b9184-105">Une variable [readonly](../language-reference/keywords/readonly.md) doit avoir la même utilisation de [static](../language-reference/keywords/static.md) que le constructeur dans lequel vous voulez l’initialiser.</span><span class="sxs-lookup"><span data-stu-id="b9184-105">A [readonly](../language-reference/keywords/readonly.md) variable must have the same [static](../language-reference/keywords/static.md) usage as the constructor in which you want to initialize it.</span></span> <span data-ttu-id="b9184-106">Pour plus d’informations, consultez [Constructeurs statiques](../programming-guide/classes-and-structs/static-constructors.md).</span><span class="sxs-lookup"><span data-stu-id="b9184-106">For more information, see [Static Constructors](../programming-guide/classes-and-structs/static-constructors.md).</span></span>  
  
 <span data-ttu-id="b9184-107">L’exemple suivant génère l’erreur CS0198 :</span><span class="sxs-lookup"><span data-stu-id="b9184-107">The following sample generates CS0198:</span></span>  
  
```csharp  
// CS0198.cs  
class MyClass  
{  
   public static readonly int TestInt = 6;  
  
   MyClass()  
   {  
      TestInt = 11;   // CS0198, constructor is not static and readonly field is  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
