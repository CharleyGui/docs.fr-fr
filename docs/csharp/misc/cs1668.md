# <a name="compiler-warning-level-2-cs1668"></a>Avertissement du compilateur (niveau 2) CS1668
Chemin de recherche 'path' non valide spécifié dans 'path string'--'system error message'  
  
 Le chemin d’accès fourni à [-lib](../language-reference/compiler-options/lib-compiler-option.md) sur la ligne de commande n’était pas valide, ou un chemin d’accès dans la variable d’environnement lib n’est pas valide. Examinez le chemin utilisé pour vérifier qu’il existe et qu’il est accessible. Le message d’erreur entre guillemets simples correspond à l’erreur retournée par le système d’exploitation.
