# <a name="compiler-warning-level-2-cs1668"></a>Avertissement du compilateur (niveau 2) CS1668
Chemin de recherche 'path' non valide spécifié dans 'path string'--'system error message'  
  
 Le chemin à [/lib](../language-reference/compiler-options/lib-compiler-option.md) fourni dans la ligne de commande ou un chemin dans la variable d’environnement LIB n’est pas valide. Examinez le chemin utilisé pour vérifier qu’il existe et qu’il est accessible. Le message d’erreur entre guillemets simples correspond à l’erreur retournée par le système d’exploitation.
