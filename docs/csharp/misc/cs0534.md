# <a name="compiler-error-cs0534"></a><span data-ttu-id="8f021-103">Erreur du compilateur CS0534</span><span class="sxs-lookup"><span data-stu-id="8f021-103">Compiler Error CS0534</span></span>
<span data-ttu-id="8f021-104">'fonction1' n’implémente pas le membre abstrait hérité 'fonction2'</span><span class="sxs-lookup"><span data-stu-id="8f021-104">'function1' does not implement inherited abstract member 'function2'</span></span>  
  
 <span data-ttu-id="8f021-105">Une classe est nécessaire pour implémenter tous les membres [abstract](../language-reference/keywords/abstract.md) de la classe de base, sauf si la classe est également abstract.</span><span class="sxs-lookup"><span data-stu-id="8f021-105">A class is required to implement all the [abstract](../language-reference/keywords/abstract.md) members in the base class, unless the class is also abstract.</span></span>  
  
 <span data-ttu-id="8f021-106">L’exemple suivant génère l’erreur CS0534 :</span><span class="sxs-lookup"><span data-stu-id="8f021-106">The following sample generates CS0534:</span></span>  
  
```csharp  
// CS0534.cs  
namespace x  
{  
   abstract public class clx  
   {  
      public abstract void f();  
   }  
  
   public class cly : clx   // CS0534, no override for clx::f  
   {  
      // uncomment the following sample override to resolve CS0534  
      // override public void f()  
      // {  
      // }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
