<span data-ttu-id="b88fa-104">Il n’existe aucune conversion prédéfinie entre les types.</span><span class="sxs-lookup"><span data-stu-id="b88fa-104">There is no predefined conversion between types.</span></span> <span data-ttu-id="b88fa-105">Vous pouvez définir une conversion personnalisée entre ces types.</span><span class="sxs-lookup"><span data-stu-id="b88fa-105">You can define a custom conversion between those types.</span></span> <span data-ttu-id="b88fa-106">Pour plus d’informations, consultez [Opérateurs de conversion définie par l’utilisateur](../language-reference/operators/user-defined-conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="b88fa-106">For more information, see [User-defined conversion operators](../language-reference/operators/user-defined-conversion-operators.md).</span></span>  
  
 <span data-ttu-id="b88fa-107">L’exemple suivant génère l’erreur CS0030 :</span><span class="sxs-lookup"><span data-stu-id="b88fa-107">The following sample generates CS0030:</span></span>  
  
```csharp  
// CS0030.cs  
namespace x  
{  
   public class iii  
   {  
      /*  
      public static implicit operator iii(int aa)  
      {  
         return null;  
      }  
  
      public static implicit operator int(iii aa)  
      {  
         return 0;  
      }  
      */  
  
      public static iii operator ++(iii aa)  
      {  
         return (iii)0;   // CS0030  
         // uncomment the conversion routines to resolve CS0030  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
