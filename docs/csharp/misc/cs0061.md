 <span data-ttu-id="50ca7-105">Une construction [publique](../language-reference/keywords/public.md) doit retourner un objet accessible publiquement.</span><span class="sxs-lookup"><span data-stu-id="50ca7-105">A [public](../language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>

 <span data-ttu-id="50ca7-108">L’exemple suivant génère l’erreur CS0061.</span><span class="sxs-lookup"><span data-stu-id="50ca7-108">The following sample generates CS0061.</span></span>  
  
```csharp  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
