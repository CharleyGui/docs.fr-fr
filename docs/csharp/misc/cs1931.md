 <span data-ttu-id="6a22a-108">Le code suivant génère l’erreur CS1931, car l’identificateur `x` est utilisé à la fois en tant que variable locale dans `Main` et en tant que variable de portée dans l’expression de requête :</span><span class="sxs-lookup"><span data-stu-id="6a22a-108">The following code generates CS1931 because the identifier `x` is used both as a local variable in `Main` and as the range variable in the query expression:</span></span>
