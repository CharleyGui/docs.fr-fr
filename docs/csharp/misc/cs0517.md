# <a name="compiler-error-cs0517"></a>Erreur du compilateur CS0517

'class' ne possède pas de classe de base et ne peut pas appeler un constructeur de base  
  
 CS0517 peut se produire uniquement lorsque le Runtime .NET compile le code source pour la classe d’objet, qui est la seule classe qui n’a pas de classe de base.
