# <a name="compiler-error-cs1647"></a>Erreur du compilateur CS1647

Une expression est trop longue ou complexe pour compiler près de 'code'  
  
 Un dépassement de capacité de la pile s’est produit dans le compilateur qui traite votre code. Pour résoudre cette erreur, simplifiez votre code. Si votre code est valide, procurez-vous de l’aide sur le site de la [communauté des développeurs](https://developercommunity.visualstudio.com/spaces/61/index.html) ou [StackOverflow](https://stackoverflow.com/).
