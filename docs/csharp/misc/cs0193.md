# <a name="compiler-error-cs0193"></a><span data-ttu-id="53438-103">Erreur du compilateur CS0193</span><span class="sxs-lookup"><span data-stu-id="53438-103">Compiler Error CS0193</span></span>

<span data-ttu-id="53438-105">L' \* opérateur ou-> a été utilisé avec un type non-pointeur.</span><span class="sxs-lookup"><span data-stu-id="53438-105">The \* or -> operator was used with a nonpointer type.</span></span> <span data-ttu-id="53438-106">Pour plus d’informations, consultez [Types pointeur](../programming-guide/unsafe-code-pointers/pointer-types.md).</span><span class="sxs-lookup"><span data-stu-id="53438-106">For more information, see [Pointer types](../programming-guide/unsafe-code-pointers/pointer-types.md).</span></span>  
  
 <span data-ttu-id="53438-107">L’exemple suivant génère l’erreur CS0193 :</span><span class="sxs-lookup"><span data-stu-id="53438-107">The following sample generates CS0193:</span></span>  
  
```csharp  
// CS0193.cs  
using System;  
  
public struct Age  
{  
   public int AgeYears;  
   public int AgeMonths;  
   public int AgeDays;  
}  
  
public class MyClass  
{  
   public static void SetAge(ref Age anAge, int years, int months, int days)  
   {  
      anAge->Months = 3;   // CS0193, anAge is not a pointer  
      // try the following line instead  
      // anAge.AgeMonths = 3;  
   }  
  
   public static void Main()  
   {  
      Age MyAge = new Age();  
      Console.WriteLine(MyAge.AgeMonths);  
      SetAge(ref MyAge, 22, 4, 15);  
      Console.WriteLine(MyAge.AgeMonths);  
   }  
}  
```
