 <span data-ttu-id="fa5c9-105">Une déclaration d’ [interface](../language-reference/keywords/interface.md) explicite a été trouvée en dehors d’une [classe](../language-reference/keywords/class.md) ou d’une [structure](../language-reference/builtin-types/struct.md).</span><span class="sxs-lookup"><span data-stu-id="fa5c9-105">An explicit [interface](../language-reference/keywords/interface.md) declaration was found outside a [class](../language-reference/keywords/class.md) or [struct](../language-reference/builtin-types/struct.md).</span></span>  
  
 <span data-ttu-id="fa5c9-106">L’exemple suivant génère l’erreur CS0541 :</span><span class="sxs-lookup"><span data-stu-id="fa5c9-106">The following sample generates CS0541:</span></span>

```csharp  
      void F();

   }  
}  
```
