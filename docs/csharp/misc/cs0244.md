# <a name="compiler-error-cs0244"></a>Erreur du compilateur CS0244
'is' et 'as' ne sont pas valides sur les types pointeur  
  
 Les opérateurs [is](../../csharp/language-reference/operators/type-testing-and-cast.md#is-operator) et [As](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator) ne sont pas valides pour une utilisation sur les types de pointeurs. Pour plus d’informations, consultez [Pointeurs et code unsafe](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 L’exemple suivant génère l’erreur CS0244 :  
  
```csharp  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
