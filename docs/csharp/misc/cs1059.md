## <a name="example"></a><span data-ttu-id="bbb75-111">Exemple</span><span class="sxs-lookup"><span data-stu-id="bbb75-111">Example</span></span>  
 <span data-ttu-id="bbb75-112">L’exemple suivant génère l’erreur CS1059, car `i` est une constante, et non une variable, et `E` est un type `Enum` , dont les éléments sont également des valeurs constantes.</span><span class="sxs-lookup"><span data-stu-id="bbb75-112">The following example generates CS1059 because `i` is a constant, not a variable, and `E` is an `Enum` type, whose elements are also constant values.</span></span>  
  
```csharp  
// CS1059.cs  
    class Program  
    {  
        public enum E : sbyte  
        {  
            a = 1,  
            b = 2  
        }  
  
        static void Main(string[] args)  
        {  
            const int i = 0;  
            i++;            // CS1059  
            E test = E.a++; // CS1059  
        }  
    }  
```
