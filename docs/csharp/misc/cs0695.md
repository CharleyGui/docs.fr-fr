# <a name="compiler-error-cs0695"></a><span data-ttu-id="a9ff8-103">Erreur du compilateur CS0695</span><span class="sxs-lookup"><span data-stu-id="a9ff8-103">Compiler Error CS0695</span></span>

<span data-ttu-id="a9ff8-104">'generic type' ne peut pas implémenter 'generic interface' et 'generic interface', car ils peuvent être réunis pour des substitutions de paramètre de type</span><span class="sxs-lookup"><span data-stu-id="a9ff8-104">'generic type' cannot implement both 'generic interface' and 'generic interface' because they may unify for some type parameter substitutions</span></span>  
  
 <span data-ttu-id="a9ff8-105">Cette erreur se produit quand une classe générique implémente plusieurs paramétrages de la même interface générique et qu’il existe une substitution de paramètre de type qui rendrait les deux interfaces identiques.</span><span class="sxs-lookup"><span data-stu-id="a9ff8-105">This error occurs when a generic class implements more than one parameterization of the same generic interface, and there exists a type parameter substitution which would make the two interfaces identical.</span></span> <span data-ttu-id="a9ff8-106">Pour éviter cette erreur, implémentez une seule des interfaces ou modifiez les paramètres de type pour éviter le conflit.</span><span class="sxs-lookup"><span data-stu-id="a9ff8-106">To avoid this error, implement only one of the interfaces, or change the type parameters to avoid the conflict.</span></span>  
  
 <span data-ttu-id="a9ff8-107">L’exemple suivant génère l’erreur CS0695 :</span><span class="sxs-lookup"><span data-stu-id="a9ff8-107">The following sample generates CS0695:</span></span>  
  
```csharp  
// CS0695.cs  
// compile with: /target:library  
  
interface I<T>  
{  
}  
  
class G<T1, T2> : I<T1>, I<T2>  // CS0695  
{  
}  
```
