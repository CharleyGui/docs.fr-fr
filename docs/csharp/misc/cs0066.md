# <a name="compiler-error-cs0066"></a><span data-ttu-id="b9f85-102">Erreur du compilateur CS0066</span><span class="sxs-lookup"><span data-stu-id="b9f85-102">Compiler Error CS0066</span></span>
<span data-ttu-id="b9f85-103">'event' : l’événement doit être de type délégué</span><span class="sxs-lookup"><span data-stu-id="b9f85-103">'event': event must be of a delegate type</span></span>  
  
 <span data-ttu-id="b9f85-104">Le mot clé de l’événement nécessite un type [délégué](../language-reference/builtin-types/reference-types.md) .</span><span class="sxs-lookup"><span data-stu-id="b9f85-104">The event keyword requires a [delegate](../language-reference/builtin-types/reference-types.md) type.</span></span> <span data-ttu-id="b9f85-105">Pour plus d’informations, consultez [événements](../programming-guide/events/index.md) et [délégués](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="b9f85-105">For more information, see [Events](../programming-guide/events/index.md) and [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="b9f85-106">L’exemple suivant génère l’erreur CS0066 :</span><span class="sxs-lookup"><span data-stu-id="b9f85-106">The following sample generates CS0066:</span></span>  
  
```csharp  
// CS0066.cs  
using System;  
  
public class EventHandler  
{  
}  
  
// to fix the error, remove the event declaration and the  
// EventHandler class declaration, and uncomment the following line  
// public delegate void EventHandler();  
  
public class a  
{  
   public event EventHandler Click;   // CS0066  
  
   private void TestMethod()  
   {  
      if (Click != null)  
         Click();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
