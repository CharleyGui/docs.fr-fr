 <span data-ttu-id="06b4a-104">Après une directive [#line](../language-reference/preprocessor-directives/preprocessor-line.md) , seul un nom de fichier (entre guillemets doubles) ou un commentaire sur une ligne est autorisé.</span><span class="sxs-lookup"><span data-stu-id="06b4a-104">After a [#line](../language-reference/preprocessor-directives/preprocessor-line.md) directive, only a file name (in double quotation marks) or a single-line comment is allowed.</span></span>

```csharp

   static void Main()  
   {  
      #line 101 abc.cs   // CS1578

      intt i;          // error will be reported on line 101  
   }  
}  
```
