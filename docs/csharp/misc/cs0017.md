# <a name="compiler-error-cs0017"></a><span data-ttu-id="bdd48-103">Erreur du compilateur CS0017</span><span class="sxs-lookup"><span data-stu-id="bdd48-103">Compiler Error CS0017</span></span>

<span data-ttu-id="bdd48-104">Plusieurs points d’entrée sont définis dans le programme 'nom_fichier_sortie'.</span><span class="sxs-lookup"><span data-stu-id="bdd48-104">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="bdd48-105">Compilez avec l’option /main pour spécifier le type qui contient le point d’entrée.</span><span class="sxs-lookup"><span data-stu-id="bdd48-105">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="bdd48-106">Un programme ne peut avoir qu’une seule méthode [Main](../programming-guide/main-and-command-args/index.md) .</span><span class="sxs-lookup"><span data-stu-id="bdd48-106">A program can only have one [Main](../programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="bdd48-107">Pour résoudre cette erreur, vous pouvez supprimer toutes les méthodes main de votre code, à l’exception d’une, ou vous pouvez utiliser l’option de compilateur [-main](../language-reference/compiler-options/main-compiler-option.md) pour spécifier la méthode main à utiliser.</span><span class="sxs-lookup"><span data-stu-id="bdd48-107">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="bdd48-108">L’exemple suivant génère l’erreur CS0017 :</span><span class="sxs-lookup"><span data-stu-id="bdd48-108">The following sample generates CS0017:</span></span>  
  
```csharp  
// CS0017.cs  
// compile with: /target:exe  
public class clx  
{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
