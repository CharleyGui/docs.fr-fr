# <a name="compiler-error-cs0160"></a><span data-ttu-id="94ce8-102">Erreur du compilateur CS0160</span><span class="sxs-lookup"><span data-stu-id="94ce8-102">Compiler Error CS0160</span></span>
<span data-ttu-id="94ce8-103">Une clause catch précédente intercepte déjà toutes les exceptions de this ou d’un super type ('type')</span><span class="sxs-lookup"><span data-stu-id="94ce8-103">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
<span data-ttu-id="94ce8-104">Une série d’instructions [catch](../../csharp/language-reference/keywords/try-catch.md) doit être classée dans l’ordre décroissant de dérivation.</span><span class="sxs-lookup"><span data-stu-id="94ce8-104">A series of [catch](../../csharp/language-reference/keywords/try-catch.md) statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="94ce8-105">Par exemple, les objets les plus dérivés doivent figurer en premier.</span><span class="sxs-lookup"><span data-stu-id="94ce8-105">For example, the most derived objects must appear first.</span></span>
  
 <span data-ttu-id="94ce8-106">Pour plus d’informations, consultez [Exceptions et gestion des exceptions](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="94ce8-106">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="94ce8-107">L’exemple suivant génère l’avertissement CS0160 :</span><span class="sxs-lookup"><span data-stu-id="94ce8-107">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
