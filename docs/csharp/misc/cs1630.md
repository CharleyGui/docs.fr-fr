# <a name="compiler-error-cs1630"></a>Erreur du compilateur CS1630

Option 'option' non valide pour /errorreport ; doit avoir la valeur prompt, send, queue ou none  
  
 L’option de ligne [de commande-errorreport](../language-reference/compiler-options/errorreport-compiler-option.md) doit être suivie de **prompt**, **Send**, **queue**ou **None**, en spécifiant l’action que vous souhaitez effectuer lorsqu’une erreur interne du compilateur se produit.
