# <a name="compiler-error-cs0539"></a><span data-ttu-id="65245-102">Erreur du compilateur CS0539</span><span class="sxs-lookup"><span data-stu-id="65245-102">Compiler Error CS0539</span></span>
<span data-ttu-id="65245-103">'membre' dans une déclaration d’interface explicite n’est pas un membre d’interface</span><span class="sxs-lookup"><span data-stu-id="65245-103">'member' in explicit interface declaration is not a member of interface</span></span>  
  
 <span data-ttu-id="65245-104">Une tentative a été effectuée pour déclarer explicitement un membre d’ [interface](../language-reference/keywords/interface.md) qui n’existe pas.</span><span class="sxs-lookup"><span data-stu-id="65245-104">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md) member that does not exist.</span></span> <span data-ttu-id="65245-105">Vous devez supprimer la déclaration ou la modifier pour qu’elle fasse référence à un membre d’interface valide.</span><span class="sxs-lookup"><span data-stu-id="65245-105">You should either delete the declaration or change it so that it refers to a valid interface member.</span></span>  
  
 <span data-ttu-id="65245-106">L’exemple suivant génère l’avertissement CS0539 :</span><span class="sxs-lookup"><span data-stu-id="65245-106">The following sample generates CS0539:</span></span>  
  
```csharp  
// CS0539.cs

{  
   interface I  
   {  
      void m();  
   }  
  
   public class clx : I  
   {  
      void I.x()   // CS0539  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
