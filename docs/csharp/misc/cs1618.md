# <a name="compiler-error-cs1618"></a><span data-ttu-id="f7770-103">Erreur du compilateur CS1618</span><span class="sxs-lookup"><span data-stu-id="f7770-103">Compiler Error CS1618</span></span>
<span data-ttu-id="f7770-104">Impossible de créer un délégué avec 'méthode', car il a un attribut conditionnel</span><span class="sxs-lookup"><span data-stu-id="f7770-104">Cannot create delegate with 'method' because it has a Conditional attribute</span></span>  
  
 <span data-ttu-id="f7770-105">Impossible de créer un délégué avec une méthode conditionnelle, car la méthode n’existe peut-être pas dans certaines builds.</span><span class="sxs-lookup"><span data-stu-id="f7770-105">You cannot create a delegate with a conditional method because the method might not exist in some builds.</span></span>  
  
 <span data-ttu-id="f7770-106">L’exemple suivant génère l’erreur CS1618 :</span><span class="sxs-lookup"><span data-stu-id="f7770-106">The following sample generates CS1618:</span></span>  
  
```csharp  
// CS1618.cs  
using System;  
using System.Diagnostics;  
  
delegate void del();  
  
class MakeAnError {  
   public static void Main() {  
      del d = new del(ConditionalMethod);   // CS1618  
      // Invalid because on builds where DEBUG is not set,
      // there will be no "ConditionalMethod".  
   }  
   // To fix the error, remove the next line:  
   [Conditional("DEBUG")]  
   public static void ConditionalMethod()
   {  
      Console.WriteLine("Do something only in debug");  
   }  
}  
```
