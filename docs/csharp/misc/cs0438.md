 <span data-ttu-id="3dc8e-105">Cette erreur se produit quand un type dans un fichier source est en conflit avec un espace de noms dans un autre fichier source.</span><span class="sxs-lookup"><span data-stu-id="3dc8e-105">This error occurs when a type in a source file conflicts with a namespace in another source file.</span></span> <span data-ttu-id="3dc8e-106">Cela se produit généralement quand le type et/ou l’espace de noms proviennent d’un module ajouté.</span><span class="sxs-lookup"><span data-stu-id="3dc8e-106">This typically happens when one or both come from an added module.</span></span> <span data-ttu-id="3dc8e-107">Pour résoudre cette erreur, renommez le type ou l’espace de noms à l’origine du conflit.</span><span class="sxs-lookup"><span data-stu-id="3dc8e-107">To resolve, rename the type or the namespace that caused the conflict.</span></span>  
  
 <span data-ttu-id="3dc8e-108">L’exemple suivant génère l’erreur CS0438 :</span><span class="sxs-lookup"><span data-stu-id="3dc8e-108">The following example generates CS0438:</span></span>

```csharp

{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="3dc8e-110">Compilez ensuite ce fichier :</span><span class="sxs-lookup"><span data-stu-id="3dc8e-110">Then compile this file:</span></span>

{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="3dc8e-111">Enfin, compilez ce fichier :</span><span class="sxs-lookup"><span data-stu-id="3dc8e-111">And then compile this file:</span></span>

using System;  
public class Test  
{  
   public static void Main() {  
      Console.WriteLine(typeof(Util.A));   // CS0438  
   }  
}  
```
