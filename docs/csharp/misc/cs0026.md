## <a name="example"></a><span data-ttu-id="bb450-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="bb450-107">Example</span></span>  
 <span data-ttu-id="bb450-108">L’exemple suivant génère l’erreur CS0026 :</span><span class="sxs-lookup"><span data-stu-id="bb450-108">The following example generates CS0026:</span></span>

   public static void Main()

   }  
}
