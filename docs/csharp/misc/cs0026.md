## <a name="example"></a><span data-ttu-id="6082d-107"> Exemple</span><span class="sxs-lookup"><span data-stu-id="6082d-107">Example</span></span>  
 <span data-ttu-id="6082d-108">L’exemple suivant génère l’erreur CS0026 :</span><span class="sxs-lookup"><span data-stu-id="6082d-108">The following example generates CS0026:</span></span>

```csharp  
   public static void Main()

   }

```
