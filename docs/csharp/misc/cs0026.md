# <a name="compiler-error-cs0026"></a>Erreur du compilateur CS0026
Le mot clé 'this' n'est pas valide dans un initialiseur de propriété statique, de méthode statique ou de champ statique  
  
 Le mot clé [this](../language-reference/keywords/this.md) fait référence à un objet qui est une instance d’un type. Dans la mesure où les méthodes statiques sont indépendantes de toute instance de la classe conteneur, le mot clé « this » est sans signification et n’est donc pas autorisé. Pour plus d’informations, voir [Classes statiques et Membres](../programming-guide/classes-and-structs/static-classes-and-static-class-members.md) et [Objets](../programming-guide/classes-and-structs/objects.md)de classe statiques .  
  
## <a name="example"></a> Exemple  
 L’exemple suivant génère l’erreur CS0026 :  
  
```csharp  
// CS0026.cs  
public class A  
{  
   public static int i = 0;  
  
   public static void Main()  
   {  
// CS0026  
      this.i = this.i + 1;
      // Try the following line instead:  
      // i = i + 1;  
   }  
}  
```
