# <a name="compiler-error-cs1528"></a><span data-ttu-id="f77a0-103">Erreur du compilateur CS1528</span><span class="sxs-lookup"><span data-stu-id="f77a0-103">Compiler Error CS1528</span></span>

<span data-ttu-id="f77a0-104">; ou = attendu (impossible de spécifier des arguments de constructeur dans une déclaration)</span><span class="sxs-lookup"><span data-stu-id="f77a0-104">Expected ; or = (cannot specify constructor arguments in declaration)</span></span>  
  
 <span data-ttu-id="f77a0-105">Une référence à une classe a été spécifiée comme si un objet de la classe était créé.</span><span class="sxs-lookup"><span data-stu-id="f77a0-105">A reference to a class was formed as if an object to the class was being created.</span></span> <span data-ttu-id="f77a0-106">Par exemple, une tentative de passer une variable à un constructeur a eu lieu.</span><span class="sxs-lookup"><span data-stu-id="f77a0-106">For example, there was an attempt to pass a variable to a constructor.</span></span> <span data-ttu-id="f77a0-107">Utilisez l’opérateur [new](../language-reference/operators/new-operator.md) pour créer un objet d’une classe.</span><span class="sxs-lookup"><span data-stu-id="f77a0-107">Use the [new](../language-reference/operators/new-operator.md) operator to create an object of a class.</span></span>  
  
 <span data-ttu-id="f77a0-108">L’exemple suivant génère l’erreur CS1528 :</span><span class="sxs-lookup"><span data-stu-id="f77a0-108">The following sample generates CS1528:</span></span>  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```
