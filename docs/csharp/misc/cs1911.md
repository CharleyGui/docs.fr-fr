# <a name="compiler-warning-level-1-cs1911"></a><span data-ttu-id="c881c-102">Avertissement du compilateur (niveau 1) CS1911</span><span class="sxs-lookup"><span data-stu-id="c881c-102">Compiler Warning (level 1) CS1911</span></span>

<span data-ttu-id="c881c-103">L’accès au membre 'name' via un mot clé 'base' à partir d’une méthode anonyme, d’expression lambda, d’expression de requête ou d’un itérateur entraîne la génération d’un code non vérifiable.</span><span class="sxs-lookup"><span data-stu-id="c881c-103">Access to member 'name' through a 'base' keyword from an anonymous method, lambda expression, query expression, or iterator results in unverifiable code.</span></span> <span data-ttu-id="c881c-104">Si possible, déplacez l’accès vers une méthode d’assistance sur le type conteneur.</span><span class="sxs-lookup"><span data-stu-id="c881c-104">Consider moving the access into a helper method on the containing type.</span></span>  
  
 <span data-ttu-id="c881c-105">L’appel de fonctions virtuelles avec le mot clé `base` à l’intérieur du corps de méthode d’un itérateur ou de méthodes anonymes génère du code non vérifiable.</span><span class="sxs-lookup"><span data-stu-id="c881c-105">Calling virtual functions with the `base` keyword inside the method body of an iterator or anonymous methods will result in unverifiable code.</span></span> <span data-ttu-id="c881c-106">Le code non vérifiable ne s’exécute pas dans un environnement de confiance partielle.</span><span class="sxs-lookup"><span data-stu-id="c881c-106">Unverifiable code will fail to run in a partial trust environment.</span></span>  
  
 <span data-ttu-id="c881c-107">Pour résoudre l’erreur CS1911, une solution consiste à déplacer l’appel de fonction virtuelle vers une fonction d’assistance.</span><span class="sxs-lookup"><span data-stu-id="c881c-107">One resolution for CS1911 is to move the virtual function call to a helper function.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c881c-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="c881c-108">Example</span></span>  

 <span data-ttu-id="c881c-109">L’exemple suivant génère l’avertissement CS1911.</span><span class="sxs-lookup"><span data-stu-id="c881c-109">The following sample generates CS1911.</span></span>  
  
```csharp  
// CS1911.cs  
// compile with: /W:1  
using System;  
  
delegate void D();  
delegate D RetD();  
  
class B {  
   protected virtual void M() {  
      Console.WriteLine("B.M");  
   }  
}  
  
class Der : B {  
   protected override void M() {  
      Console.WriteLine("D.M");  
   }  
  
   void Test() { base.M(); }  
   D Test2() { return new D(base.M); }  
  
   public D CallBaseM() {  
      return delegate () { base.M(); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { Test(); };  
   }  
  
   public RetD DelToBaseM() {  
      return delegate () { return new D(base.M); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { return Test2(); };  
   }  
}  
  
class Program {  
   public static void Main() {  
      Der der = new Der();  
      D d = der.CallBaseM();  
      d();  
      RetD rd = der.DelToBaseM();  
      rd()();  
   }  
}  
```
