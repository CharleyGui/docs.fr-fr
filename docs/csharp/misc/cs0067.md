 <span data-ttu-id="f5e96-105">Un [événement](../language-reference/keywords/event.md) a été déclaré, mais n’a jamais été utilisé dans la classe au sein de laquelle il a été déclaré.</span><span class="sxs-lookup"><span data-stu-id="f5e96-105">An [event](../language-reference/keywords/event.md) was declared but never used in the class in which it was declared.</span></span>

```csharp

using System;  
delegate void MyDelegate();  
  
class MyClass  
{  
   public event MyDelegate evt;   // CS0067

   private void TestMethod()  
   {  
      if (evt != null)  
         evt();  
   }  
*/  
   public static void Main()  
   {  
   }  
}  
```
