# <a name="compiler-error-cs1575"></a><span data-ttu-id="ed73f-102">Erreur du compilateur CS1575</span><span class="sxs-lookup"><span data-stu-id="ed73f-102">Compiler Error CS1575</span></span>
<span data-ttu-id="ed73f-103">Une expression stackalloc exige la présence de [] après le type</span><span class="sxs-lookup"><span data-stu-id="ed73f-103">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="ed73f-104">La taille de l’allocation demandée, avec [stackalloc](../language-reference/operators/stackalloc.md), doit être spécifiée entre crochets.</span><span class="sxs-lookup"><span data-stu-id="ed73f-104">The size of the requested allocation, with [stackalloc](../language-reference/operators/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="ed73f-105">L’exemple suivant génère l’avertissement CS1575 :</span><span class="sxs-lookup"><span data-stu-id="ed73f-105">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
