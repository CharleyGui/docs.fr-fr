# <a name="compiler-error-cs0159"></a><span data-ttu-id="b58b9-103">Erreur du compilateur CS0159</span><span class="sxs-lookup"><span data-stu-id="b58b9-103">Compiler Error CS0159</span></span>
<span data-ttu-id="b58b9-104">Il n’existe pas d’étiquette 'label' dans la portée de l’instruction goto</span><span class="sxs-lookup"><span data-stu-id="b58b9-104">No such label 'label' within the scope of the goto statement</span></span>  
  
 <span data-ttu-id="b58b9-105">L’étiquette référencée par l’instruction [goto](../language-reference/keywords/goto.md) est introuvable dans la portée de l’instruction `goto` .</span><span class="sxs-lookup"><span data-stu-id="b58b9-105">The label referenced by the [goto](../language-reference/keywords/goto.md) statement could not be found within the scope of the `goto` statement.</span></span>  
  
 <span data-ttu-id="b58b9-106">L’exemple suivant génère l’erreur CS0159 :</span><span class="sxs-lookup"><span data-stu-id="b58b9-106">The following sample generates CS0159:</span></span>  
  
```csharp  
// CS0159.cs  
public class Class1  
{  
   public static void Main()  
   {  
      int i = 0;  
  
      switch (i)  
      {  
         case 1:  
            goto case 3;   // CS0159, case 3 label does not exist  
         case 2:  
            break;  
      }  
      goto NOWHERE;   // CS0159, NOWHERE label does not exist  
   }  
}  
```
