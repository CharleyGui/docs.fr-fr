## <a name="example"></a><span data-ttu-id="564b1-109">Exemples</span><span class="sxs-lookup"><span data-stu-id="564b1-109">Example</span></span>

<span data-ttu-id="564b1-110">L’exemple suivant génère l’erreur CS0837 :</span><span class="sxs-lookup"><span data-stu-id="564b1-110">The following example generates CS0837:</span></span>  
  
```csharp  
// cs0837.cs  
namespace TestNamespace  
{  
    public delegate void Del();  
  
    class Test  
    {  
        static int Main()  
        {  
            bool b1 = (() => { }) is Del;   // CS0837  
            bool b2 = delegate() { } is Del;// CS0837  
            Del d1 = () => { } as Del;      // CS0837  
            Del d2 = delegate() { } as Del; // CS0837  
            return 1;  
        }  
    }  
}  
```
