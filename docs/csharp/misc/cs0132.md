 <span data-ttu-id="4a222-107">L’exemple suivant génère l’erreur CS0132 :</span><span class="sxs-lookup"><span data-stu-id="4a222-107">The following sample generates CS0132:</span></span>  
  
```csharp  
// CS0132.cs

{  
   public class MyClass  
   {  
      public MyClass(int i)  
      {

      static MyClass2(int i)   // CS0132  
      {  
      }  
   }  
}  
```
