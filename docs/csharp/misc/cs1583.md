# <a name="compiler-error-cs1583"></a>Erreur du compilateur CS1583
'fichier' n’est pas un fichier de ressources Win32 valide  
  
 Cette erreur se produit quand vous spécifiez un nom de fichier avec l’option [-win32res](../language-reference/compiler-options/win32res-compiler-option.md) du compilateur qui n’est pas un fichier de ressources Win32 valide ou correctement mis en forme. Dans Visual Studio, le nom de fichier est spécifié dans le volet Application du Concepteur de projets.
