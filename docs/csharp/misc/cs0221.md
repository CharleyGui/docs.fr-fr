# <a name="compiler-error-cs0221"></a>Erreur du compilateur CS0221
La valeur de constante 'value' ne peut pas être convertie en 'type' (utilisez la syntaxe 'unchecked' pour la remplacer)  
  
 Une opération d’assignation qui entraînerait une perte de données a été détectée par [checked](../language-reference/keywords/checked.md), qui est activé par défaut. Corrigez l’assignation ou utilisez [unchecked](../language-reference/keywords/unchecked.md) pour résoudre cette erreur. Pour plus d’informations, consultez [cochée et désactivée](../language-reference/keywords/checked-and-unchecked.md).  
  
 L’exemple suivant génère l’erreur CS0221 :  
  
```csharp  
// CS0221.cs  
public class MyClass  
{  
   public static void Main()  
   {  
      // unchecked  
      // {  
         int a = (int)0xFFFFFFFF;   // CS0221  
         a++;  
      // }  
   }  
}  
```
