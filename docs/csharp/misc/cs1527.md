# <a name="compiler-error-cs1527"></a><span data-ttu-id="cfcfc-103">Erreur du compilateur CS1527</span><span class="sxs-lookup"><span data-stu-id="cfcfc-103">Compiler Error CS1527</span></span>

<span data-ttu-id="cfcfc-104">Les éléments définis dans un espace de noms ne peuvent pas être explicitement déclarés comme Private, protected, protected internal ou Private protected.</span><span class="sxs-lookup"><span data-stu-id="cfcfc-104">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span>
  
 <span data-ttu-id="cfcfc-105">Les déclarations de type d’un espace de noms peuvent avoir un accès [public](../language-reference/keywords/public.md) ou [internal](../language-reference/keywords/internal.md) .</span><span class="sxs-lookup"><span data-stu-id="cfcfc-105">Type declarations in a namespace can have either [public](../language-reference/keywords/public.md) or [internal](../language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="cfcfc-106">Si aucune accessibilité n’est spécifiée, la valeur par défaut est **internal** .</span><span class="sxs-lookup"><span data-stu-id="cfcfc-106">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="cfcfc-107">L’exemple suivant génère l’erreur CS1527 :</span><span class="sxs-lookup"><span data-stu-id="cfcfc-107">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="cfcfc-108">L’exemple suivant génère l’erreur CS1527, car quand aucun espace de noms n’est déclaré explicitement dans le code de votre programme, toutes les déclarations de type sont situées implicitement dans l’espace de noms global.</span><span class="sxs-lookup"><span data-stu-id="cfcfc-108">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4 {}  
private struct S1 {}  
```
