# <a name="compiler-error-cs1527"></a><span data-ttu-id="4d440-102">Erreur du compilateur CS1527</span><span class="sxs-lookup"><span data-stu-id="4d440-102">Compiler Error CS1527</span></span>

<span data-ttu-id="4d440-103">Les éléments définis dans un espace de noms ne peuvent pas être explicitement déclarés comme Private, protected, protected internal ou Private protected.</span><span class="sxs-lookup"><span data-stu-id="4d440-103">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span>
  
 <span data-ttu-id="4d440-104">Les déclarations de type d’un espace de noms peuvent avoir un accès [public](../language-reference/keywords/public.md) ou [internal](../language-reference/keywords/internal.md) .</span><span class="sxs-lookup"><span data-stu-id="4d440-104">Type declarations in a namespace can have either [public](../language-reference/keywords/public.md) or [internal](../language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="4d440-105">Si aucune accessibilité n’est spécifiée, la valeur par défaut est **internal** .</span><span class="sxs-lookup"><span data-stu-id="4d440-105">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="4d440-106">L’exemple suivant génère l’erreur CS1527 :</span><span class="sxs-lookup"><span data-stu-id="4d440-106">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="4d440-107">L’exemple suivant génère l’erreur CS1527, car quand aucun espace de noms n’est déclaré explicitement dans le code de votre programme, toutes les déclarations de type sont situées implicitement dans l’espace de noms global.</span><span class="sxs-lookup"><span data-stu-id="4d440-107">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4 {}  
private struct S1 {}  
```
