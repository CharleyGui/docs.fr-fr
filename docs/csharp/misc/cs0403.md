## <a name="example"></a><span data-ttu-id="b57f4-109">Exemple</span><span class="sxs-lookup"><span data-stu-id="b57f4-109">Example</span></span>  

 <span data-ttu-id="b57f4-110">L’exemple suivant génère l’erreur CS0403 :</span><span class="sxs-lookup"><span data-stu-id="b57f4-110">The following sample generates CS0403.</span></span>  
  
```csharp  
// CS0403.cs

class C<T>  
{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
