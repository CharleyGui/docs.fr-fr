# <a name="compiler-error-cs0577"></a><span data-ttu-id="0f892-103">Erreur du compilateur CS0577</span><span class="sxs-lookup"><span data-stu-id="0f892-103">Compiler Error CS0577</span></span>

<span data-ttu-id="0f892-104">L’attribut Conditional n’est pas valide sur 'function', car il s’agit d’un constructeur, d’un destructeur, d’un opérateur ou d’une implémentation d’interface explicite</span><span class="sxs-lookup"><span data-stu-id="0f892-104">The Conditional attribute is not valid on 'function' because it is a constructor, destructor, operator, or explicit interface implementation</span></span>  
  
 <span data-ttu-id="0f892-105">`Conditional` ne peut pas être appliqué aux méthodes spécifiées.</span><span class="sxs-lookup"><span data-stu-id="0f892-105">`Conditional` cannot be applied to the specified methods.</span></span>  
  
 <span data-ttu-id="0f892-106">Par exemple, vous ne pouvez pas utiliser certains attributs sur une définition d’interface explicite.</span><span class="sxs-lookup"><span data-stu-id="0f892-106">For example, you cannot use some attributes on an explicit interface definition.</span></span> <span data-ttu-id="0f892-107">L’exemple suivant génère l’erreur CS0577 :</span><span class="sxs-lookup"><span data-stu-id="0f892-107">The following sample generates CS0577:</span></span>  
  
```csharp  
// CS0577.cs  
// compile with: /target:library  
interface I  
{  
   void m();  
}  
  
public class MyClass : I  
{  
   [System.Diagnostics.Conditional("a")]   // CS0577  
   void I.m() {}  
}  
```
