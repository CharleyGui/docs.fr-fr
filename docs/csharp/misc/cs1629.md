 <span data-ttu-id="652d2-105">La spécification du langage C# n’autorise pas la présence de code unsafe dans les itérateurs.</span><span class="sxs-lookup"><span data-stu-id="652d2-105">The C# language specification does not allow unsafe code in iterators.</span></span>  
  
 <span data-ttu-id="652d2-106">L’exemple suivant génère l’erreur CS1629 :</span><span class="sxs-lookup"><span data-stu-id="652d2-106">The following sample generates CS1629:</span></span>  
  
```csharp  
// CS1629.cs  
// compile with: /unsafe
using System.Collections.Generic;  
class C
{  
   IEnumerator<int> IteratorMeth() {  
      int i;  
      unsafe  // CS1629  
      {  
         int *p = &i;  
         yield return *p;  
      }  
   }  
}  
```
