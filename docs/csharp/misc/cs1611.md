# <a name="compiler-error-cs1611"></a><span data-ttu-id="6482c-103">Erreur du compilateur CS1611</span><span class="sxs-lookup"><span data-stu-id="6482c-103">Compiler Error CS1611</span></span>
<span data-ttu-id="6482c-104">Le paramètre params ne peut pas être déclaré comme ref ou out</span><span class="sxs-lookup"><span data-stu-id="6482c-104">The params parameter cannot be declared as in ref or out</span></span>  
  
 <span data-ttu-id="6482c-105">Les mots clés [dans](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) ou [out](../language-reference/keywords/out-parameter-modifier.md) ne peuvent pas être utilisés avec le mot clé [params](../language-reference/keywords/params.md) .</span><span class="sxs-lookup"><span data-stu-id="6482c-105">The keywords [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) cannot be used with the [params](../language-reference/keywords/params.md) keyword.</span></span>  
  
 <span data-ttu-id="6482c-106">L’exemple suivant génère l’erreur CS1611 :</span><span class="sxs-lookup"><span data-stu-id="6482c-106">The following sample generates CS1611:</span></span>  
  
```csharp  
// CS1611.cs  
public class MyClass  
{  
   public static void Test(params ref int[] a)   // CS1611, remove ref  
   {  
   }  
  
   public static void Main()  
   {  
      Test(1);  
   }  
}  
```
