# <a name="compiler-error-cs0572"></a><span data-ttu-id="3e461-103">Erreur du compilateur CS0572</span><span class="sxs-lookup"><span data-stu-id="3e461-103">Compiler Error CS0572</span></span>
<span data-ttu-id="3e461-104">'type' : impossible de référencer un type par l’intermédiaire d’une expression ; essayez plutôt 'path_to_type'</span><span class="sxs-lookup"><span data-stu-id="3e461-104">'type' : cannot reference a type through an expression; try 'path_to_type' instead</span></span>  
  
 <span data-ttu-id="3e461-105">Une tentative a été effectuée pour accéder à un membre d’une classe via un identificateur, ce qui n’est pas autorisé dans cette situation.</span><span class="sxs-lookup"><span data-stu-id="3e461-105">An attempt was made to access a member of a class through an identifier, which is not permitted in this situation.</span></span>  
  
 <span data-ttu-id="3e461-106">L’exemple suivant génère l’erreur CS0572 :</span><span class="sxs-lookup"><span data-stu-id="3e461-106">The following sample generates CS0572:</span></span>  
  
```csharp  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
