# <a name="compiler-error-cs0174"></a>Erreur du compilateur CS0174
Une classe de base est requise pour une référence 'base'  
  
 Cette erreur se produit uniquement lorsque le Runtime .NET compile le code source de la `System.Object` classe, qui est la seule classe qui n’a pas de classe de base.
