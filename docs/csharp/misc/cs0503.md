# <a name="compiler-error-cs0503"></a><span data-ttu-id="e91e1-103">Erreur du compilateur CS0503</span><span class="sxs-lookup"><span data-stu-id="e91e1-103">Compiler Error CS0503</span></span>

 <span data-ttu-id="e91e1-105">Il est superflu de marquer une méthode membre à la fois comme [abstract](../language-reference/keywords/abstract.md) et [virtual](../language-reference/keywords/virtual.md) , car **abstract** implique **virtual**.</span><span class="sxs-lookup"><span data-stu-id="e91e1-105">It is redundant to mark a member method as both [abstract](../language-reference/keywords/abstract.md) and [virtual](../language-reference/keywords/virtual.md) because **abstract** implies **virtual**.</span></span>

   {  
      abstract virtual public void f();   // CS0503
