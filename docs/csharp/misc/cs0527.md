# <a name="compiler-error-cs0527"></a>Erreur du compilateur CS0527
Le type 'type' qui figure dans la liste des interfaces n’est pas une interface  
  
 Il est possible pour un [struct](../language-reference/keywords/struct.md) ou une [interface](../language-reference/keywords/interface.md) d’hériter d’une autre interface, mais pas à partir de n’importe quel autre type.  
  
 L’exemple suivant génère l’erreur CS0527 :  
  
```csharp  
// CS0527.cs  
// compile with: /target:library  
public struct clx : int {}   // CS0527 int not an interface  
```
