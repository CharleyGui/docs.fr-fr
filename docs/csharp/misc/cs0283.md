# <a name="compiler-error-cs0283"></a>Erreur du compilateur CS0283
Le type 'type' ne peut pas être déclaré const  
  
Le type spécifié dans une déclaration de [constante](../language-reference/keywords/const.md) doit être `byte`, `sbyte`, `ushort`, `short`, `uint`, `int`, `ulong`, `long`, `char`, `float`, `double`, `decimal`, `bool`, `string`, un type [enum](../language-reference/builtin-types/enum.md) ou un type référence auquel est assignée la valeur `null`. Chaque expression de constante doit produire une valeur du type cible ou d’un type qui est implicitement convertible en type cible.  
  
## <a name="example"></a>Exemple  
 L’exemple suivant génère l’erreur CS0283.  
  
```csharp  
// CS0283.cs  
struct MyTest  
{  
}  
class MyClass   
{  
    // To resolve the error but retain the "const-ness",  
    // change const to readonly.  
    const MyTest test = new MyTest();   // CS0283  
  
    public static int Main() {  
        return 1;  
    }  
}  
```
