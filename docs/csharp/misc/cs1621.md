# <a name="compiler-error-cs1621"></a><span data-ttu-id="d8329-103">Erreur du compilateur CS1621</span><span class="sxs-lookup"><span data-stu-id="d8329-103">Compiler Error CS1621</span></span>

<span data-ttu-id="d8329-104">L'instruction yield ne peut pas être utilisée dans une méthode anonyme ou une expression lambda</span><span class="sxs-lookup"><span data-stu-id="d8329-104">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
<span data-ttu-id="d8329-105">Vous ne pouvez pas utiliser l’instruction [yield](../language-reference/keywords/yield.md) dans une [méthode anonyme](../language-reference/operators/delegate-operator.md) ou une [expression lambda](../language-reference/operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="d8329-105">You cannot use the [yield](../language-reference/keywords/yield.md) statement in an [anonymous method](../language-reference/operators/delegate-operator.md) or a [lambda expression](../language-reference/operators/lambda-expressions.md).</span></span>
  
<span data-ttu-id="d8329-106">L’exemple suivant génère l’erreur CS1621 :</span><span class="sxs-lookup"><span data-stu-id="d8329-106">The following sample generates CS1621:</span></span>
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
