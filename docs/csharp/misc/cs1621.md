# <a name="compiler-error-cs1621"></a><span data-ttu-id="940fa-102">Erreur du compilateur CS1621</span><span class="sxs-lookup"><span data-stu-id="940fa-102">Compiler Error CS1621</span></span>
<span data-ttu-id="940fa-103">L'instruction yield ne peut pas être utilisée dans une méthode anonyme ou une expression lambda</span><span class="sxs-lookup"><span data-stu-id="940fa-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
<span data-ttu-id="940fa-104">Vous ne pouvez pas utiliser l’instruction [yield](../language-reference/keywords/yield.md) dans une [méthode anonyme](../language-reference/operators/delegate-operator.md) ou une [expression lambda](../programming-guide/statements-expressions-operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="940fa-104">You cannot use the [yield](../language-reference/keywords/yield.md) statement in an [anonymous method](../language-reference/operators/delegate-operator.md) or a [lambda expression](../programming-guide/statements-expressions-operators/lambda-expressions.md).</span></span>
  
<span data-ttu-id="940fa-105">L’exemple suivant génère l’erreur CS1621 :</span><span class="sxs-lookup"><span data-stu-id="940fa-105">The following sample generates CS1621:</span></span>
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
