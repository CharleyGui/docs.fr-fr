## <a name="example"></a><span data-ttu-id="492c2-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="492c2-108">Example</span></span>  
 <span data-ttu-id="492c2-109">L’exemple suivant génère l’erreur CS0241 :</span><span class="sxs-lookup"><span data-stu-id="492c2-109">The following sample generates CS0241.</span></span> <span data-ttu-id="492c2-110">En outre, l’exemple montre comment simuler, avec la surcharge, une méthode avec des arguments par défaut.</span><span class="sxs-lookup"><span data-stu-id="492c2-110">In addition, the sample shows how to simulate, with overloading, a method with default arguments.</span></span>  
  
```csharp  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {
      B x = new B();  
      x.Test();  
   }  
}  
```
