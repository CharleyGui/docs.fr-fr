<span data-ttu-id="f0e14-106">L’exemple suivant génère un avertissement CS0168 :</span><span class="sxs-lookup"><span data-stu-id="f0e14-106">The following sample generates a CS0168 warning:</span></span>  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
