# <a name="compiler-error-cs1553"></a>Erreur du compilateur CS1553
Déclaration n’est pas valide. Utilisez ' opérateur modificateur \<dest-type > (...' à la place  
  
 Le type de retour pour un [opérateur de conversion](../../csharp/language-reference/keywords/implicit.md) doit précéder immédiatement la liste de paramètres et *modificateur* est soit `implicit` ou **explicite**.  
  
 L’exemple suivant génère l’erreur CS1553 :  
  
```csharp  
// CS1553.cs  
class MyClass  
{  
   public static int implicit operator (MyClass f)   // CS1553  
   // try the following line instead  
   // public static implicit operator int (MyClass f)  
   {  
      return 6;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
