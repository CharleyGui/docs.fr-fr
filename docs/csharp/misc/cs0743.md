<span data-ttu-id="05a54-104">Mot clé contextuel 'on' attendu</span><span class="sxs-lookup"><span data-stu-id="05a54-104">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="05a54-105">Le modèle pour une clause `join` est `join`...`in`...`on`...`equals`, comme illustré dans cet exemple :</span><span class="sxs-lookup"><span data-stu-id="05a54-105">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

## <a name="example"></a><span data-ttu-id="05a54-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="05a54-108">Example</span></span>  

 <span data-ttu-id="05a54-109">Le code suivant génère l’erreur CS0743 :</span><span class="sxs-lookup"><span data-stu-id="05a54-109">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
