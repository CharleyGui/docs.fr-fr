## <a name="example"></a><span data-ttu-id="3bde7-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="3bde7-108">Example</span></span>  
 <span data-ttu-id="3bde7-109">Le code suivant génère l’erreur CS0743 :</span><span class="sxs-lookup"><span data-stu-id="3bde7-109">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
