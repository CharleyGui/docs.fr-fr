# <a name="compiler-error-cs1623"></a><span data-ttu-id="8b80d-103">Erreur du compilateur CS1623</span><span class="sxs-lookup"><span data-stu-id="8b80d-103">Compiler Error CS1623</span></span>

<span data-ttu-id="8b80d-104">Les itérateurs ne peuvent pas avoir de paramètres ref ou out</span><span class="sxs-lookup"><span data-stu-id="8b80d-104">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="8b80d-105">Cette erreur se produit si une méthode Iterator prend `in` un `ref` paramètre, ou `out` .</span><span class="sxs-lookup"><span data-stu-id="8b80d-105">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="8b80d-106">Pour éviter cette erreur, supprimez `in` le `ref` `out` mot clé, ou de la signature de la méthode.</span><span class="sxs-lookup"><span data-stu-id="8b80d-106">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8b80d-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="8b80d-107">Example</span></span>  

 <span data-ttu-id="8b80d-108">L’exemple suivant génère l’erreur CS1623 :</span><span class="sxs-lookup"><span data-stu-id="8b80d-108">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
