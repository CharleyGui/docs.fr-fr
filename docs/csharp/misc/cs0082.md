## <a name="example"></a><span data-ttu-id="0c842-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="0c842-106">Example</span></span>  
 <span data-ttu-id="0c842-107">L’exemple suivant génère l’erreur CS0082 :</span><span class="sxs-lookup"><span data-stu-id="0c842-107">The following example generates CS0082:</span></span>  
  
```csharp  
//cs0082.cs  
class MyClass  
{  
  
    //property  
    public int MyProp  
    {  
        get //CS0082  
        {  
            return 1;  
        }  
    }  
  
    //conflicting Get  
    public int get_MyProp()  
    {  
        return 2;  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
