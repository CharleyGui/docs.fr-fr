# <a name="compiler-error-cs0674"></a><span data-ttu-id="5bac4-103">Erreur du compilateur CS0674</span><span class="sxs-lookup"><span data-stu-id="5bac4-103">Compiler Error CS0674</span></span>
<span data-ttu-id="5bac4-104">N’utilisez pas 'System.ParamArrayAttribute'.</span><span class="sxs-lookup"><span data-stu-id="5bac4-104">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="5bac4-105">Utilisez plutôt le mot clé 'params'.</span><span class="sxs-lookup"><span data-stu-id="5bac4-105">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="5bac4-106">Le compilateur C# n’autorise pas l’utilisation de <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; utilisez [params](../language-reference/keywords/params.md) à la place.</span><span class="sxs-lookup"><span data-stu-id="5bac4-106">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="5bac4-107">L’exemple suivant génère l’erreur CS0674 :</span><span class="sxs-lookup"><span data-stu-id="5bac4-107">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
