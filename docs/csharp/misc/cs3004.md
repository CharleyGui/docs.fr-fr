# <a name="compiler-warning-level-1-cs3004"></a>Avertissement du compilateur (niveau 1) CS3004
Les caractères Unicode mélangés et décomposés ne sont pas conformes CLS  
  
 Seuls les caractères UNICODE composés sont autorisés dans les identificateurs [public](../language-reference/keywords/public.md), [protected](../language-reference/keywords/protected.md)ou `protected internal` afin d’être conformes à la spécification CLS (Common Language Specification). Pour plus d’informations sur la conformité CLS, consultez [indépendance du langage et composants indépendants du langage](../../standard/language-independence.md).
