# <a name="compiler-error-cs0110"></a><span data-ttu-id="11c0e-103">Erreur du compilateur CS0110</span><span class="sxs-lookup"><span data-stu-id="11c0e-103">Compiler Error CS0110</span></span>
<span data-ttu-id="11c0e-104">L’évaluation de la valeur de constante de 'const declaration' implique une définition circulaire</span><span class="sxs-lookup"><span data-stu-id="11c0e-104">The evaluation of the constant value for 'const declaration' involves a circular definition</span></span>  
  
 <span data-ttu-id="11c0e-105">La déclaration d’une variable [const](../language-reference/keywords/const.md) (`a`) ne peut pas faire référence à une autre variable const (`b`) qui fait également référence à (`a`).</span><span class="sxs-lookup"><span data-stu-id="11c0e-105">The declaration of a [const](../language-reference/keywords/const.md) variable (`a`) cannot reference another const variable (`b`) that also references (`a`).</span></span>  
  
 <span data-ttu-id="11c0e-106">L’exemple suivant génère l’erreur CS0110 :</span><span class="sxs-lookup"><span data-stu-id="11c0e-106">The following sample generates CS0110:</span></span>  
  
```csharp  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```
