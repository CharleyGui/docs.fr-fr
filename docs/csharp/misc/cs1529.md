## <a name="example"></a><span data-ttu-id="844dd-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="844dd-105">Example</span></span>  
 <span data-ttu-id="844dd-106">L’exemple suivant génère l’erreur CS1529 :</span><span class="sxs-lookup"><span data-stu-id="844dd-106">The following sample generates CS1529:</span></span>

```csharp  
    {  
        using Microsoft;

```
