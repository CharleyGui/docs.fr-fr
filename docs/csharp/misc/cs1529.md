## <a name="example"></a><span data-ttu-id="9fe7a-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="9fe7a-106">Example</span></span>  

 <span data-ttu-id="9fe7a-107">L’exemple suivant génère l’erreur CS1529 :</span><span class="sxs-lookup"><span data-stu-id="9fe7a-107">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs

{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}  
  
using System;                 // CS1529, place at the beginning of the file  
```
