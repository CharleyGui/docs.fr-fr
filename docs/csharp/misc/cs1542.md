# <a name="compiler-error-cs1542"></a>Erreur du compilateur CS1542
'dll' ne peut pas être ajouté à cet assembly, car il s’agit déjà d’un assembly ; utilisez plutôt l’option '/R'  
  
 Le fichier référencé avec l’option de compilateur [-addmodule](../language-reference/compiler-options/addmodule-compiler-option.md) n’a pas été généré avec [-target : module](../language-reference/compiler-options/target-module-compiler-option.md); Utilisez [-Reference](../language-reference/compiler-options/reference-compiler-option.md) pour référencer le fichier dans cette compilation.
