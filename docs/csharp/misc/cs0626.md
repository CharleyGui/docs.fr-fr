<span data-ttu-id="4fcca-104">La méthode, l’opérateur ou l’accesseur 'method' est marqué comme external et n’a pas d’attribut.</span><span class="sxs-lookup"><span data-stu-id="4fcca-104">Method, operator, or accessor 'method' is marked external and has no attributes on it.</span></span> <span data-ttu-id="4fcca-105">Ajoutez un attribut DllImport pour spécifier l’implémentation externe.</span><span class="sxs-lookup"><span data-stu-id="4fcca-105">Consider adding a DllImport attribute to specify the external implementation.</span></span>
  
 <span data-ttu-id="4fcca-106">Une méthode marquée `extern` doit également être marquée avec un attribut, par exemple [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) .</span><span class="sxs-lookup"><span data-stu-id="4fcca-106">A method marked `extern` should also be marked with an attribute, for example, the [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) attribute.</span></span>
  
 <span data-ttu-id="4fcca-107">L’attribut spécifie où la méthode est implémentée.</span><span class="sxs-lookup"><span data-stu-id="4fcca-107">The attribute specifies where the method is implemented.</span></span> <span data-ttu-id="4fcca-108">Au moment de l’exécution, ces informations sont nécessaires au programme.</span><span class="sxs-lookup"><span data-stu-id="4fcca-108">At run time, the program will need this information.</span></span>

```csharp

public class MyClass

   static extern public void M(); // CS0626  
   // try the following line  
   // [DllImport("mydll.dll")] static extern public void M();  
  
   public static void Main()  
   {  
   }  
}  
```
