# <a name="compiler-error-cs0239"></a>Erreur du compilateur CS0239
'member' : impossible de substituer le membre hérité 'inherited member', car il est sealed  
  
 Un membre ne peut pas se [substituer](../language-reference/keywords/override.md) à un membre hérité [sealed](../language-reference/keywords/sealed.md) . Pour plus d’informations, consultez [cochée et](../language-reference/keywords/checked-and-unchecked.md)désactivée.  
  
 L’exemple suivant génère l’erreur CS0239 :  
  
```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}  
  
class MyClass3 : MyClass2  
{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```
