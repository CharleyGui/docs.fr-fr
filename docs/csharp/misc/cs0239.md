# <a name="compiler-error-cs0239"></a><span data-ttu-id="95dcf-102">Erreur du compilateur CS0239</span><span class="sxs-lookup"><span data-stu-id="95dcf-102">Compiler Error CS0239</span></span>
<span data-ttu-id="95dcf-103">'member' : impossible de substituer le membre hérité 'inherited member', car il est sealed</span><span class="sxs-lookup"><span data-stu-id="95dcf-103">'member' : cannot override inherited member 'inherited member' because it is sealed</span></span>  
  
 <span data-ttu-id="95dcf-104">Un membre ne peut pas se [substituer](../language-reference/keywords/override.md) à un membre hérité [sealed](../language-reference/keywords/sealed.md) .</span><span class="sxs-lookup"><span data-stu-id="95dcf-104">A member cannot [override](../language-reference/keywords/override.md) a [sealed](../language-reference/keywords/sealed.md) inherited member.</span></span> <span data-ttu-id="95dcf-105">Pour plus d’informations, consultez [cochée et](../language-reference/keywords/checked-and-unchecked.md)désactivée.</span><span class="sxs-lookup"><span data-stu-id="95dcf-105">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="95dcf-106">L’exemple suivant génère l’erreur CS0239 :</span><span class="sxs-lookup"><span data-stu-id="95dcf-106">The following sample generates CS0239:</span></span>  
  
```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}  
  
class MyClass3 : MyClass2  
{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```
