# <a name="compiler-error-cs0035"></a><span data-ttu-id="0e8bc-103">Erreur du compilateur CS0035</span><span class="sxs-lookup"><span data-stu-id="0e8bc-103">Compiler Error CS0035</span></span>
<span data-ttu-id="0e8bc-104">L’opérateur 'opérateur' est ambigu pour un opérande de type 'type'</span><span class="sxs-lookup"><span data-stu-id="0e8bc-104">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
<span data-ttu-id="0e8bc-105">Plusieurs conversions sont disponibles et le compilateur ne sait pas laquelle choisir avant d’appliquer l’opérateur.</span><span class="sxs-lookup"><span data-stu-id="0e8bc-105">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span>
  
<span data-ttu-id="0e8bc-106">L’exemple suivant génère l’erreur CS0035 :</span><span class="sxs-lookup"><span data-stu-id="0e8bc-106">The following sample generates CS0035:</span></span>  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
