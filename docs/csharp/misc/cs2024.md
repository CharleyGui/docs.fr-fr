# <a name="compiler-error-cs2024"></a>Erreur du compilateur CS2024

Numéro d’alignement de section de fichier non valide '#'

Une valeur non valide a été passée à l’option du compilateur [-filealign](../language-reference/compiler-options/filealign-compiler-option.md) .

## <a name="example"></a>Exemple

L’exemple suivant génère l’erreur CS2024 :

```csharp
// CS2024.cs
// compile with: /filealign:ex
// CS2024 expected
class MyClass
{
    public static void Main()
    {
    }
}
```
