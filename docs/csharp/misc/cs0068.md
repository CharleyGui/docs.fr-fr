 <span data-ttu-id="e3f37-104">Un événement dans une interface ne peut pas avoir d’initialiseur.</span><span class="sxs-lookup"><span data-stu-id="e3f37-104">An event in an interface cannot have an initializer.</span></span> <span data-ttu-id="e3f37-105">Pour plus d’informations, consultez [Interfaces](../programming-guide/interfaces/index.md).</span><span class="sxs-lookup"><span data-stu-id="e3f37-105">For more information, see [Interfaces](../programming-guide/interfaces/index.md).</span></span>  
  
 <span data-ttu-id="e3f37-106">L’exemple suivant génère l’erreur CS0068 :</span><span class="sxs-lookup"><span data-stu-id="e3f37-106">The following sample generates CS0068:</span></span>  
  
```csharp  
// CS0068.cs  
  
delegate void MyDelegate();  
  
interface I  
{  
   event MyDelegate d = new MyDelegate(M.f);   // CS0068  
   // try the following line instead  
   // event MyDelegate d2;  
}  
  
class M  
{  
   event MyDelegate d = new MyDelegate(M.f);  
  
   public static void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
