 <span data-ttu-id="52ebe-104">Quand un membre de classe est marqué [extern](../language-reference/keywords/extern.md), cela signifie que la définition du membre se trouve dans un autre fichier.</span><span class="sxs-lookup"><span data-stu-id="52ebe-104">When a class member is marked [extern](../language-reference/keywords/extern.md), it means that the member's definition is located in another file.</span></span> <span data-ttu-id="52ebe-105">Un membre de classe marqué comme **extern** ne peut donc pas être défini dans la classe.</span><span class="sxs-lookup"><span data-stu-id="52ebe-105">Therefore, a class member marked as **extern** cannot be defined in the class.</span></span> <span data-ttu-id="52ebe-106">Supprimez le mot clé `extern` ou la définition.</span><span class="sxs-lookup"><span data-stu-id="52ebe-106">Either remove the `extern` keyword or delete the definition.</span></span> <span data-ttu-id="52ebe-107">Pour plus d’informations, consultez [Méthodes](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="52ebe-107">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>

```csharp  
   }

```
