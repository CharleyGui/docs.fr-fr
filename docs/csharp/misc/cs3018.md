# <a name="compiler-warning-level-1-cs3018"></a><span data-ttu-id="3633f-102">Avertissement du compilateur (niveau 1) CS3018</span><span class="sxs-lookup"><span data-stu-id="3633f-102">Compiler Warning (level 1) CS3018</span></span>

<span data-ttu-id="3633f-103">Impossible de marquer 'type' comme conforme CLS car il est membre du type non conforme CLS 'type'</span><span class="sxs-lookup"><span data-stu-id="3633f-103">'type' cannot be marked as CLS-Compliant because it is a member of non CLS-compliant type 'type'</span></span>  
  
 <span data-ttu-id="3633f-104">Cet avertissement se produit si une classe imbriquée avec l’attribut CLSCompliant défini sur `true` est déclarée comme membre d’une classe déclarée avec l’attribut CLSCompliant défini sur `false`.</span><span class="sxs-lookup"><span data-stu-id="3633f-104">This warning occurs if a nested class with the CLSCompliant attribute set to `true` is declared as a member of a class declared with the CLSCompliant attribute set to `false`.</span></span> <span data-ttu-id="3633f-105">Cela n’est pas autorisé car une classe imbriquée ne peut pas être conforme CLS si elle est membre d’une classe externe qui n’est pas conforme CLS.</span><span class="sxs-lookup"><span data-stu-id="3633f-105">This is not allowed, since a nested class cannot be CLS-compliant if it is a member of an outer class that is not CLS-compliant.</span></span> <span data-ttu-id="3633f-106">Pour remédier à cet avertissement, supprimez l’attribut CLSCompliant de la classe imbriquée ou remplacez `true` par `false`.</span><span class="sxs-lookup"><span data-stu-id="3633f-106">To resolve this warning, remove the CLSCompliant attribute from the nested class, or change it from `true` to `false`.</span></span> <span data-ttu-id="3633f-107">Pour plus d’informations sur la conformité CLS, consultez [indépendance du langage et composants indépendants du langage](../../standard/language-independence.md).</span><span class="sxs-lookup"><span data-stu-id="3633f-107">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="3633f-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="3633f-108">Example</span></span>  

 <span data-ttu-id="3633f-109">L’exemple suivant génère l’avertissement CS3018.</span><span class="sxs-lookup"><span data-stu-id="3633f-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
