# <a name="compiler-error-cs1530"></a>Erreur du compilateur CS1530
Le mot clé 'new' n'est pas autorisé dans les éléments définis dans un espace de noms  
  
 Il n’est pas utile de spécifier le mot clé [new](../language-reference/keywords/new-modifier.md) sur une construction qui se trouve dans un [espace de noms](../language-reference/keywords/namespace.md).  
  
 L’exemple suivant génère l’erreur CS1530 :  
  
```csharp  
// CS1530.cs  
namespace a  
{  
   new class i   // CS1530  
   {  
   }  
  
   // try the following instead  
   class ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
