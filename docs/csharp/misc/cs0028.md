# <a name="compiler-warning-level-4-cs0028"></a>Avertissement du compilateur (niveau 4) CS0028
'déclaration_fonction' possède une signature erronée pour un point d’entrée  
  
 La déclaration de méthode pour `Main` n’est pas valide : elle a été déclarée avec une signature non valide. `Main` doit être déclarée comme static et doit retourner [int](../../csharp/language-reference/builtin-types/integral-numeric-types.md) ou [void](../../csharp/language-reference/keywords/void.md). Pour plus d’informations, consultez [Main() et Arguments de ligne de commande](../../csharp/programming-guide/main-and-command-args/index.md).  
  
 L’exemple suivant génère l’erreur CS0028 :  
  
```csharp  
// CS0028.cs  
// compile with: /W:4 /warnaserror  
public class a  
{  
    public static double Main(int i)   // CS0028  
    // Try the following line instead:  
    // public static void Main()  
    {  
    }  
}  
```
