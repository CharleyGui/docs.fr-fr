# <a name="compiler-warning-level-2-cs0252"></a><span data-ttu-id="e60c6-103">Avertissement du compilateur (niveau 2) CS0252</span><span class="sxs-lookup"><span data-stu-id="e60c6-103">Compiler Warning (level 2) CS0252</span></span>
<span data-ttu-id="e60c6-104">Possibilité d’une comparaison de références involontaire ; pour obtenir une comparaison de valeurs, effectuez un cast de la partie gauche en type 'type'</span><span class="sxs-lookup"><span data-stu-id="e60c6-104">Possible unintended reference comparison; to get a value comparison, cast the left hand side to type 'type'</span></span>  
  
 <span data-ttu-id="e60c6-105">Le compilateur effectue une comparaison de références.</span><span class="sxs-lookup"><span data-stu-id="e60c6-105">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="e60c6-106">Si vous souhaitez comparer la valeur de chaînes, effectuez un cast de la partie gauche de l’expression en `type`.</span><span class="sxs-lookup"><span data-stu-id="e60c6-106">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="e60c6-107">L’exemple suivant génère l’avertissement CS0252 :</span><span class="sxs-lookup"><span data-stu-id="e60c6-107">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
