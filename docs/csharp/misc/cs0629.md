# <a name="compiler-error-cs0629"></a><span data-ttu-id="4ad42-103">Erreur du compilateur CS0629</span><span class="sxs-lookup"><span data-stu-id="4ad42-103">Compiler error CS0629</span></span>

<span data-ttu-id="4ad42-104">Le membre conditionnel 'member' ne peut pas implémenter le membre d’interface 'base class member' dans le type 'Type Name'</span><span class="sxs-lookup"><span data-stu-id="4ad42-104">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
<span data-ttu-id="4ad42-105">L' [attribut Conditional](../language-reference/attributes/general.md#conditional-attribute) ne peut pas être utilisé sur l’implémentation d’une interface.</span><span class="sxs-lookup"><span data-stu-id="4ad42-105">The [Conditional attribute](../language-reference/attributes/general.md#conditional-attribute) cannot be used on the implementation of an interface.</span></span>  
  
<span data-ttu-id="4ad42-106">L’exemple suivant génère l’erreur CS0629 :</span><span class="sxs-lookup"><span data-stu-id="4ad42-106">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
