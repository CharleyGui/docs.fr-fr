# <a name="compiler-error-cs0629"></a><span data-ttu-id="1337a-102">Erreur du compilateur CS0629</span><span class="sxs-lookup"><span data-stu-id="1337a-102">Compiler Error CS0629</span></span>
<span data-ttu-id="1337a-103">Le membre conditionnel 'member' ne peut pas implémenter le membre d’interface 'base class member' dans le type 'Type Name'</span><span class="sxs-lookup"><span data-stu-id="1337a-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="1337a-104">L’attribut [Conditional](../programming-guide/concepts/attributes/common-attributes.md#Conditional) ne peut pas être utilisé sur l’implémentation d’une interface.</span><span class="sxs-lookup"><span data-stu-id="1337a-104">The [Conditional](../programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="1337a-105">L’exemple suivant génère l’erreur CS0629 :</span><span class="sxs-lookup"><span data-stu-id="1337a-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
