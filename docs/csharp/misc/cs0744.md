<span data-ttu-id="898a5-103">Mot clé contextuel 'equals' attendu</span><span class="sxs-lookup"><span data-stu-id="898a5-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="898a5-104">Le modèle pour une clause `join` est `join`...`in`...`on`...`equals`, comme illustré dans cet exemple :</span><span class="sxs-lookup"><span data-stu-id="898a5-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

## <a name="example"></a><span data-ttu-id="898a5-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="898a5-107">Example</span></span>  
 <span data-ttu-id="898a5-108">L’exemple suivant génère l’erreur CS0744 :</span><span class="sxs-lookup"><span data-stu-id="898a5-108">The following code generates CS0744:</span></span>

using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {

    }  
}
