# <a name="compiler-error-cs0157"></a><span data-ttu-id="8d464-102">Erreur du compilateur CS0157</span><span class="sxs-lookup"><span data-stu-id="8d464-102">Compiler Error CS0157</span></span>
<span data-ttu-id="8d464-103">Le contrôle ne peut pas laisser le corps d'une clause finally</span><span class="sxs-lookup"><span data-stu-id="8d464-103">Control cannot leave the body of a finally clause</span></span>  
  
 <span data-ttu-id="8d464-104">Toutes les instructions d’une clause [finally](../language-reference/keywords/try-catch-finally.md) doivent être exécutées.</span><span class="sxs-lookup"><span data-stu-id="8d464-104">All of the statements in a [finally](../language-reference/keywords/try-catch-finally.md) clause must execute.</span></span> <span data-ttu-id="8d464-105">Pour plus d’informations, consultez [exceptions et gestion des exceptions](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="8d464-105">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="8d464-106">L’exemple suivant génère l’erreur CS0157 :</span><span class="sxs-lookup"><span data-stu-id="8d464-106">The following sample generates CS0157:</span></span>  
  
```csharp  
// CS0157.cs  
using System;  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
         }  
  
         finally  
         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```
