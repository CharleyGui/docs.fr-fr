 <span data-ttu-id="6c28a-105">La tentative de substitution de l’une des méthodes d’accesseur d’une propriété a échoué, car l’accesseur ne peut pas être substitué.</span><span class="sxs-lookup"><span data-stu-id="6c28a-105">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="6c28a-106">Cette erreur peut se produire si :</span><span class="sxs-lookup"><span data-stu-id="6c28a-106">This error can occur if:</span></span>

## <a name="example"></a><span data-ttu-id="6c28a-111">Exemple</span><span class="sxs-lookup"><span data-stu-id="6c28a-111">Example</span></span>  
 <span data-ttu-id="6c28a-112">L’exemple suivant génère l’erreur CS0546, car la classe de base ne déclare pas un accesseur set pour la propriété.</span><span class="sxs-lookup"><span data-stu-id="6c28a-112">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
