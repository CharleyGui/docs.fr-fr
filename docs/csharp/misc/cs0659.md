# <a name="compiler-warning-level-3-cs0659"></a><span data-ttu-id="1deb6-103">Avertissement du compilateur (niveau 3) CS0659</span><span class="sxs-lookup"><span data-stu-id="1deb6-103">Compiler Warning (level 3) CS0659</span></span>

<span data-ttu-id="1deb6-104">’classe’ se substitue à Object.Equals(object o) mais pas à Object.GetHashCode()</span><span class="sxs-lookup"><span data-stu-id="1deb6-104">'class' overrides Object.Equals(object o) but does not override Object.GetHashCode()</span></span>  

<span data-ttu-id="1deb6-105">Le compilateur a détecté une substitution de la <xref:System.Object.Equals%2A?displayProperty=nameWithType> méthode, mais aucune substitution de la <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> méthode.</span><span class="sxs-lookup"><span data-stu-id="1deb6-105">The compiler detected an override of the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method but no override of the <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="1deb6-106">Une substitution de <xref:System.Object.Equals%2A> implique que vous souhaitez également substituer <xref:System.Object.GetHashCode%2A> .</span><span class="sxs-lookup"><span data-stu-id="1deb6-106">An override of <xref:System.Object.Equals%2A> implies that you also want to override <xref:System.Object.GetHashCode%2A>.</span></span>  

<span data-ttu-id="1deb6-107">Le code suivant génère CS0659 :</span><span class="sxs-lookup"><span data-stu-id="1deb6-107">The following code generates CS0659:</span></span>  

```csharp
// CS0659.cs  
// compile with: /W:3 /target:library  
class Test
{  
   public override bool Equals(object o) { return true; }   // CS0659  
}  
  
// OK  
class Test2  
{  
   public override bool Equals(object o) { return true; }  
   public override int GetHashCode() { return 0; }  
}  
```
