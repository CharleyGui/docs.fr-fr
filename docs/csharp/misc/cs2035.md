# <a name="compiler-error-cs2035"></a>Erreur du compilateur CS2035

Erreur de syntaxe de ligne de commande : ' : \<number> 'manquant pour l’option’compiler_option'  
  
 Certaines options du compilateur nécessitent une valeur.  
  
## <a name="example"></a>Exemple  

 L’exemple suivant génère l’erreur CS2035 :  
  
```csharp  
// CS2035.cs  
// compile with: /baseaddress  
// CS2035 expected  
```
