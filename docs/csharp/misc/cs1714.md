# <a name="compiler-error-cs1714"></a>Erreur du compilateur CS1714

La classe de base ou l’interface de NomType1 n’a pas pu être résolue ou n’est pas valide  
  
 Vous implémentez NomType1 à partir d’une classe ou interface qui n’a pas pu être résolue (par exemple, le compilateur n’a pas pu la trouver) ou qui n’est pas valide. La solution consiste à déterminer laquelle de ces deux causes est à l’origine du problème, et de spécifier plus correctement l’emplacement du type ou de corriger les erreurs du compilateur dans la classe de base.
