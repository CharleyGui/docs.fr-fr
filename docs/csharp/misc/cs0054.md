# <a name="compiler-error-cs0054"></a>Erreur du compilateur CS0054
Accessibilité incohérente : le type de retour d’indexeur 'type' est moins accessible que l’indexeur 'indexeur'  
  
 Une construction publique doit retourner un objet accessible publiquement. Pour plus d’informations, consultez [Modificateurs d’accès](../programming-guide/classes-and-structs/access-modifiers.md).  
  
 L’exemple suivant génère l’erreur CS0054 :  
  
```csharp  
// CS0054.cs  
class MyClass  
// try the following line instead  
// public class MyClass  
{  
}  
  
public class MyClass3  
{  
   public MyClass this[int i]   // CS0054  
   {  
      get  
      {  
         return new MyClass();  
      }  
   }  
}  
  
public class MyClass2  
{  
   public static void Main()  
   {  
   }  
}  
```
