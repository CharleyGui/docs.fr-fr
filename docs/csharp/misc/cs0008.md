# <a name="compiler-error-cs0008"></a>Erreur du compilateur CS0008
Erreur inattendue lors de la lecture des métadonnées dans le fichier 'fichier' — 'description'  
  
 Une DLL a été ouverte pour la récupération des métadonnées, mais elle est endommagée et les données n’ont pas pu être lues. Pour plus d’informations, consultez [-Reference (options du compilateur C#)](../language-reference/compiler-options/reference-compiler-option.md).
