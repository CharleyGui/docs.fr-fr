# <a name="compiler-error-cs0558"></a><span data-ttu-id="d72a7-102">Erreur du compilateur CS0558</span><span class="sxs-lookup"><span data-stu-id="d72a7-102">Compiler Error CS0558</span></span>

 <span data-ttu-id="d72a7-104">Les deux **modificateurs** d’accès **static** et [public](/dotnet/csharp/language-reference/keywords) doivent être spécifiés sur les opérateurs définis par l’utilisateur.</span><span class="sxs-lookup"><span data-stu-id="d72a7-104">Both the **static** and **public** access [modifiers](/dotnet/csharp/language-reference/keywords) must be specified on user-defined operators.</span></span>

```csharp  
// CS0558.cs

{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
