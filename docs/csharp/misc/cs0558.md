# <a name="compiler-error-cs0558"></a><span data-ttu-id="c5e2e-103">Erreur du compilateur CS0558</span><span class="sxs-lookup"><span data-stu-id="c5e2e-103">Compiler Error CS0558</span></span>
<span data-ttu-id="c5e2e-104">L’opérateur défini par l’utilisateur 'opérateur' doit être déclaré static et public</span><span class="sxs-lookup"><span data-stu-id="c5e2e-104">User-defined operator 'operator' must be declared static and public</span></span>  
  
 <span data-ttu-id="c5e2e-105">Les deux **modificateurs** d’accès **static** et [public](/dotnet/csharp/language-reference/keywords) doivent être spécifiés sur les opérateurs définis par l’utilisateur.</span><span class="sxs-lookup"><span data-stu-id="c5e2e-105">Both the **static** and **public** access [modifiers](/dotnet/csharp/language-reference/keywords) must be specified on user-defined operators.</span></span>  
  
 <span data-ttu-id="c5e2e-106">L’exemple suivant génère l’erreur CS0558 :</span><span class="sxs-lookup"><span data-stu-id="c5e2e-106">The following sample generates CS0558:</span></span>  
  
```csharp  
// CS0558.cs  
namespace x  
{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
