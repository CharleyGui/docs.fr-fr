# <a name="compiler-error-cs0558"></a><span data-ttu-id="5c7a0-103">Erreur du compilateur CS0558</span><span class="sxs-lookup"><span data-stu-id="5c7a0-103">Compiler Error CS0558</span></span>
<span data-ttu-id="5c7a0-104">L’opérateur défini par l’utilisateur 'opérateur' doit être déclaré static et public</span><span class="sxs-lookup"><span data-stu-id="5c7a0-104">User-defined operator 'operator' must be declared static and public</span></span>  
  
 <span data-ttu-id="5c7a0-105">Les deux **modificateurs** d’accès **static** et [public](../language-reference/keywords/index.md) doivent être spécifiés sur les opérateurs définis par l’utilisateur.</span><span class="sxs-lookup"><span data-stu-id="5c7a0-105">Both the **static** and **public** access [modifiers](../language-reference/keywords/index.md) must be specified on user-defined operators.</span></span>  
  
 <span data-ttu-id="5c7a0-106">L’exemple suivant génère l’erreur CS0558 :</span><span class="sxs-lookup"><span data-stu-id="5c7a0-106">The following sample generates CS0558:</span></span>  
  
```csharp  
// CS0558.cs  
namespace x  
{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
