# <a name="compiler-error-cs2011"></a>Erreur du compilateur CS2011

Impossible d’ouvrir le fichier réponse 'file'  
  
 Un fichier réponse a été spécifié dans une compilation, mais le compilateur n’a pas pu localiser ni ouvrir le fichier.  
  
 Pour plus d’informations sur les fichiers réponse, consultez [@ (spécifier un fichier réponse)](../language-reference/compiler-options/response-file-compiler-option.md).
