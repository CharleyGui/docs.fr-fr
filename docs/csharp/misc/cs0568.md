# <a name="compiler-error-cs0568"></a><span data-ttu-id="f8751-102">Erreur du compilateur CS0568</span><span class="sxs-lookup"><span data-stu-id="f8751-102">Compiler Error CS0568</span></span>
<span data-ttu-id="f8751-103">Les structures ne peuvent pas contenir de constructeurs exempts de paramètres explicites</span><span class="sxs-lookup"><span data-stu-id="f8751-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="f8751-104">Chaque [struct](../language-reference/builtin-types/struct.md) a déjà un constructeur sans paramètre qui initialise l’objet à zéro.</span><span class="sxs-lookup"><span data-stu-id="f8751-104">Each [struct](../language-reference/builtin-types/struct.md) already has a parameterless constructor that initializes the object to zero.</span></span> <span data-ttu-id="f8751-105">Les constructeurs que vous pouvez créer pour un struct doivent donc accepter un ou plusieurs paramètres.</span><span class="sxs-lookup"><span data-stu-id="f8751-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="f8751-106">L’exemple suivant génère l’erreur CS0568 :</span><span class="sxs-lookup"><span data-stu-id="f8751-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
