## <a name="example"></a><span data-ttu-id="efee8-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="efee8-108">Example</span></span>  

 <span data-ttu-id="efee8-109">L’exemple suivant génère l’erreur CS0272 :</span><span class="sxs-lookup"><span data-stu-id="efee8-109">The following example generates CS0272:</span></span>  
  
```csharp  
// CS0272.cs  
public class MyClass  
{  
    public int Property  
    {  
        get { return 0; }  
        private set { }  
    }  
}

{  
    static void Main()  
    {  
        MyClass c = new MyClass();  
        c.Property = 10;      // CS0272  
        // To resolve, remove the previous line
        // or use an appropriate modifier on the set accessor.  
    }  
}  
```
