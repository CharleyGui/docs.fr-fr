## <a name="example"></a><span data-ttu-id="be63b-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="be63b-107">Example</span></span>  
 <span data-ttu-id="be63b-108">L’exemple suivant génère l’erreur CS0508.</span><span class="sxs-lookup"><span data-stu-id="be63b-108">The following sample generates CS0508.</span></span>  
  
```csharp  
// CS0508.cs  
// compile with: /target:library  
abstract public class Clx  
{  
   public int i = 0;  
   // Return type is int.  
   abstract public int F();  
}  
  
public class Cly : Clx  
{  
   public override double F()  
   {  
      return 0.0;   // CS0508  
   }  
}  
```
