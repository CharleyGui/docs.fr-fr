## <a name="example"></a><span data-ttu-id="67ac6-107"> Exemple</span><span class="sxs-lookup"><span data-stu-id="67ac6-107">Example</span></span>  
 <span data-ttu-id="67ac6-108">L’exemple suivant génère l’erreur CS0411 :</span><span class="sxs-lookup"><span data-stu-id="67ac6-108">The following sample generates CS0411:</span></span>  
  
```csharp  
// CS0411.cs  
class C  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```
