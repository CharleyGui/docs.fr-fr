 <span data-ttu-id="470ea-106">Cette erreur se produit si vous appelez une méthode générique sans fournir explicitement les arguments de type et que le compilateur ne peut pas déduire les arguments de type prévus.</span><span class="sxs-lookup"><span data-stu-id="470ea-106">This error occurs if you call a generic method without explicitly providing the type arguments and the compiler cannot infer which type arguments are intended.</span></span> <span data-ttu-id="470ea-107">Pour éviter cette erreur, ajoutez les arguments de type prévus entre crochets pointus.</span><span class="sxs-lookup"><span data-stu-id="470ea-107">To avoid this error, add the intended type arguments in angle brackets.</span></span>  
  
## <a name="example"></a><span data-ttu-id="470ea-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="470ea-108">Example</span></span>  
 <span data-ttu-id="470ea-109">L’exemple suivant génère l’erreur CS0411 :</span><span class="sxs-lookup"><span data-stu-id="470ea-109">The following sample generates CS0411:</span></span>  
  
```csharp  
// CS0411.cs  
class C  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```
