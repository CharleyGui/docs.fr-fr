 <span data-ttu-id="b1c83-105">Une [classe](../language-reference/keywords/class.md) dérivée d’une [interface](../language-reference/keywords/interface.md)n’implémente pas un ou plusieurs membres de l’interface.</span><span class="sxs-lookup"><span data-stu-id="b1c83-105">A [class](../language-reference/keywords/class.md) derived from an [interface](../language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="b1c83-106">Une classe doit implémenter tous les membres des interfaces dont elle est dérivée ; sinon, elle doit être déclarée `abstract`.</span><span class="sxs-lookup"><span data-stu-id="b1c83-106">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b1c83-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="b1c83-107">Example</span></span>  

 <span data-ttu-id="b1c83-108">L’exemple suivant génère l’erreur CS0535.</span><span class="sxs-lookup"><span data-stu-id="b1c83-108">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
