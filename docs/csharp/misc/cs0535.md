## <a name="example"></a><span data-ttu-id="0b6bf-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="0b6bf-107">Example</span></span>  
 <span data-ttu-id="0b6bf-108">L’exemple suivant génère l’erreur CS0535.</span><span class="sxs-lookup"><span data-stu-id="0b6bf-108">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK

   public void F() {}  
   public static void Main() {}  
}  
```
