# <a name="compiler-error-cs0175"></a>Erreur du compilateur CS0175
L'utilisation du mot clé 'base' n'est pas valide dans ce contexte  
  
 Le mot clé [base](../language-reference/keywords/base.md) doit être utilisé pour spécifier un membre spécifique de la classe de base. Pour plus d’informations, consultez [Constructeurs](../programming-guide/classes-and-structs/constructors.md).  
  
 L’exemple suivant génère l’erreur CS0175 :  
  
```csharp  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```
