# <a name="compiler-error-cs0175"></a><span data-ttu-id="c6726-103">Erreur du compilateur CS0175</span><span class="sxs-lookup"><span data-stu-id="c6726-103">Compiler Error CS0175</span></span>
<span data-ttu-id="c6726-104">L'utilisation du mot clé 'base' n'est pas valide dans ce contexte</span><span class="sxs-lookup"><span data-stu-id="c6726-104">Use of keyword 'base' is not valid in this context</span></span>  
  
 <span data-ttu-id="c6726-105">Le mot clé [base](../language-reference/keywords/base.md) doit être utilisé pour spécifier un membre spécifique de la classe de base.</span><span class="sxs-lookup"><span data-stu-id="c6726-105">The [base](../language-reference/keywords/base.md) keyword must be used to specify a particular member of the base class.</span></span> <span data-ttu-id="c6726-106">Pour plus d’informations, consultez [Constructeurs](../programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="c6726-106">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="c6726-107">L’exemple suivant génère l’erreur CS0175 :</span><span class="sxs-lookup"><span data-stu-id="c6726-107">The following sample generates CS0175:</span></span>  
  
```csharp  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```
