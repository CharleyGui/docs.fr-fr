# <a name="compiler-error-cs1021"></a>Erreur du compilateur CS1021

Constante intégrale trop grande  
  
Une valeur représentée par un littéral entier est supérieure à <xref:System.UInt64.MaxValue?displayProperty=nameWithType> .  
  
L’exemple suivant génère l’erreur CS1021 :  

```csharp
// CS1021.cs  
class Program
{
    static void Main(string[] args)
    {
        int a = 18_446_744_073_709_552_000;
    }
}  
```

Le code suivant génère également CS1021 :

```csharp
using System.Numerics;

class Program
{
    static void Main(string[] args)
    {
        var a = new BigInteger(18_446_744_073_709_552_000);
    }
}
```

Pour plus d’informations sur la façon d’instancier une <xref:System.Numerics.BigInteger?displayProperty=nameWithType> instance dont la valeur dépasse la plage des types numériques intégrés, consultez la section [instanciation d’un objet BigInteger](https://docs.microsoft.com/dotnet/api/System.Numerics.BigInteger#instantiating-a-biginteger-object) de la  <xref:System.Numerics.BigInteger> page de référence.
