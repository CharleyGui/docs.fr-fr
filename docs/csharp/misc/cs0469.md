# <a name="compiler-warning-level-2-cs0469"></a><span data-ttu-id="fce99-103">Avertissement du compilateur (niveau 2) CS0469</span><span class="sxs-lookup"><span data-stu-id="fce99-103">Compiler Warning (level 2) CS0469</span></span>
<span data-ttu-id="fce99-104">La valeur 'goto case' n’est pas implicitement convertible en type 'type'</span><span class="sxs-lookup"><span data-stu-id="fce99-104">The 'goto case' value is not implicitly convertible to type 'type'</span></span>  
  
 <span data-ttu-id="fce99-105">Quand vous utilisez `goto case`, une conversion implicite de la valeur du goto case vers le type du commutateur doit avoir lieu.</span><span class="sxs-lookup"><span data-stu-id="fce99-105">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fce99-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="fce99-106">Example</span></span>  
 <span data-ttu-id="fce99-107">L’exemple suivant génère l’avertissement CS0469.</span><span class="sxs-lookup"><span data-stu-id="fce99-107">The following sample generates CS0469.</span></span>  
  
```csharp  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
