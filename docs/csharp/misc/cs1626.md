# <a name="compiler-error-cs1626"></a>Erreur du compilateur CS1626
Impossible de générer une valeur dans le corps d'un bloc try avec une clause catch  
  
 L’instruction yield n’est pas autorisée dans un bloc try si une clause catch est associée au bloc try. Pour éviter cette erreur, soit déplacez l’énoncé de rendement du bloc d’essai/capture/enfin, ou retirez le bloc de capture.
  
 L’exemple suivant génère l’erreur CS1626 :  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
      finally
      {

      }
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
