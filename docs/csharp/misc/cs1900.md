# <a name="compiler-error-cs1900"></a>Erreur du compilateur CS1900
Le niveau d'avertissement doit être compris entre 0 et 4  
  
 L’option du compilateur [/warn](../language-reference/compiler-options/warn-compiler-option.md) peut uniquement prendre l’une des cinq valeurs possibles (0, 1, 2, 3 ou 4). Toute autre valeur passée à **/warn** génère l’erreur CS1900.  
  
 L’exemple suivant génère l’erreur CS1900 :  
  
```csharp  
// CS1900.cs  
// compile with: /W:5  
// CS1900 expected  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
