# <a name="compiler-error-cs1107"></a><span data-ttu-id="fb803-103">Erreur du compilateur CS1107</span><span class="sxs-lookup"><span data-stu-id="fb803-103">Compiler Error CS1107</span></span>
<span data-ttu-id="fb803-104">Un paramètre ne peut posséder qu’un modificateur 'modifier name'.</span><span class="sxs-lookup"><span data-stu-id="fb803-104">A parameter can only have one 'modifier name' modifier.</span></span>

## <a name="example"></a><span data-ttu-id="fb803-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="fb803-106">Example</span></span>  
 <span data-ttu-id="fb803-107">L’exemple suivant génère l’erreur CS1107 :</span><span class="sxs-lookup"><span data-stu-id="fb803-107">The following example generates CS1107:</span></span>  
  
```csharp  
// cs1107.cs  
public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
