## <a name="example"></a><span data-ttu-id="43fe6-112">Exemple</span><span class="sxs-lookup"><span data-stu-id="43fe6-112">Example</span></span>  

 <span data-ttu-id="43fe6-113">Le code suivant génère l’erreur CS0844 :</span><span class="sxs-lookup"><span data-stu-id="43fe6-113">The following code generates CS0844:</span></span>  
  
```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // CS0844  
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

## <a name="correct-the-error-by-using-thisnum-to-refer-to-the-class-field"></a><span data-ttu-id="43fe6-114">Corrigez l’erreur en utilisant `this.num` pour faire référence au champ de classe.</span><span class="sxs-lookup"><span data-stu-id="43fe6-114">Correct the error by using `this.num` to refer to the class field</span></span>

```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

```csharp  
        public void TestMethod()  
        {  
            num = 5; // Error fixed.
            int num2 = 6;
        }

        {  
            return 1;  
        }  
    }  
```
