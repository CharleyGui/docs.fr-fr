# <a name="compiler-error-cs0462"></a><span data-ttu-id="d9f5c-103">Erreur du compilateur CS0462</span><span class="sxs-lookup"><span data-stu-id="d9f5c-103">Compiler Error CS0462</span></span>
<span data-ttu-id="d9f5c-104">Les membres hérités 'member1' et 'member2' ont la même signature dans le type 'type' et ne peuvent donc pas être substitués</span><span class="sxs-lookup"><span data-stu-id="d9f5c-104">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="d9f5c-105">Cette erreur se produit avec l’introduction de génériques.</span><span class="sxs-lookup"><span data-stu-id="d9f5c-105">This error arises with the introduction of generics.</span></span> <span data-ttu-id="d9f5c-106">Normalement, vous ne pouvez pas avoir deux versions d’une méthode dans une classe avec la même signature.</span><span class="sxs-lookup"><span data-stu-id="d9f5c-106">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="d9f5c-107">Mais avec les génériques, vous pouvez spécifier une méthode générique qui peut dupliquer une autre méthode si elle est instanciée avec un type particulier.</span><span class="sxs-lookup"><span data-stu-id="d9f5c-107">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d9f5c-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="d9f5c-108">Example</span></span>  
 <span data-ttu-id="d9f5c-109">Quand `C<int>` est instancié, deux versions de la méthode `F` sont créées avec la même signature, donc le remplacement dans la classe `D` ne peut pas déterminer celle à laquelle appliquer le remplacement.</span><span class="sxs-lookup"><span data-stu-id="d9f5c-109">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="d9f5c-110">L’exemple suivant génère l’erreur CS0462.</span><span class="sxs-lookup"><span data-stu-id="d9f5c-110">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>
{  
   public override void F(int t) {}   // CS0462  
}  
```
