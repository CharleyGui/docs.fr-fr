<span data-ttu-id="aef1e-104">Impossible de convertir une expression lambda en une arborescence d’expression dont l’argument de type 'type' n’est pas un type délégué.</span><span class="sxs-lookup"><span data-stu-id="aef1e-104">Cannot convert lambda to an expression tree whose type argument 'type' is not a delegate type.</span></span>  
  
 <span data-ttu-id="aef1e-105">Si une expression lambda est convertie en une arborescence d’expression, elle doit disposer d’un type délégué pour son argument.</span><span class="sxs-lookup"><span data-stu-id="aef1e-105">If a lambda expression is converted to an expression tree, the expression tree must have a delegate type for its argument.</span></span> <span data-ttu-id="aef1e-106">De plus, l’expression lambda doit pouvoir être convertie en type délégué.</span><span class="sxs-lookup"><span data-stu-id="aef1e-106">Furthermore, the lambda expression must be convertible to the delegate type.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="aef1e-107">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="aef1e-107">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="aef1e-109">Exemple</span><span class="sxs-lookup"><span data-stu-id="aef1e-109">Example</span></span>  
 <span data-ttu-id="aef1e-110">L’exemple suivant génère l’erreur CS0835 :</span><span class="sxs-lookup"><span data-stu-id="aef1e-110">The following example generates CS0835:</span></span>  
  
```csharp  
// cs0835.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;

    }  
}  
```
