# <a name="compiler-error-cs0460"></a><span data-ttu-id="823b8-102">Erreur du compilateur CS0460</span><span class="sxs-lookup"><span data-stu-id="823b8-102">Compiler Error CS0460</span></span>
<span data-ttu-id="823b8-103">Les contraintes pour les méthodes d'implémentation d'interface override et explicite sont héritées de la méthode de base et ne peuvent donc pas être spécifiées directement</span><span class="sxs-lookup"><span data-stu-id="823b8-103">Constraints for override and explicit interface implementation methods are inherited from the base method, so they cannot be specified directly</span></span>  
  
 <span data-ttu-id="823b8-104">Quand une méthode générique qui fait partie d’une classe dérivée remplace une méthode dans la classe de base, vous ne pouvez pas spécifier de contraintes sur la méthode remplacée.</span><span class="sxs-lookup"><span data-stu-id="823b8-104">When a generic method that is part of a derived class overrides a method in the base class, you may not specify constraints on the overridden method.</span></span> <span data-ttu-id="823b8-105">La méthode de substitution dans la classe dérivée hérite ses contraintes de la méthode de la classe de base.</span><span class="sxs-lookup"><span data-stu-id="823b8-105">The override method in the derived class inherits its constraints from the method in the base class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="823b8-106"> Exemple</span><span class="sxs-lookup"><span data-stu-id="823b8-106">Example</span></span>  
 <span data-ttu-id="823b8-107">L’exemple suivant génère l’erreur CS0460.</span><span class="sxs-lookup"><span data-stu-id="823b8-107">The following sample generates CS0460.</span></span>  
  
```csharp  
// CS0460.cs  
// compile with: /target:library  
class BaseClass
{  
   BaseClass() { }  
}  
  
interface I  
{  
   void F1<T>() where T : BaseClass;  
   void F2<T>() where T : struct;  
   void F3<T>() where T : BaseClass;  
}  
  
class ExpImpl : I  
{  
   void I.F1<T>() where T : BaseClass {}   // CS0460  
   void I.F2<T>() where T : class {}  // CS0460  
}  
```
