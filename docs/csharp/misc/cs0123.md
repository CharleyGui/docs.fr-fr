# <a name="compiler-error-cs0123"></a><span data-ttu-id="21933-103">Erreur du compilateur CS0123</span><span class="sxs-lookup"><span data-stu-id="21933-103">Compiler Error CS0123</span></span>
<span data-ttu-id="21933-104">Aucune surcharge pour 'method' ne correspond au délégué 'delegate'</span><span class="sxs-lookup"><span data-stu-id="21933-104">No overload for 'method' matches delegate 'delegate'</span></span>  
  
 <span data-ttu-id="21933-105">Une tentative de création d’un délégué a échoué, car la signature correcte n’a pas été utilisée.</span><span class="sxs-lookup"><span data-stu-id="21933-105">An attempt to create a delegate failed because the correct signature was not used.</span></span> <span data-ttu-id="21933-106">Les instances d’un délégué doivent être déclarées avec la même signature que celle de la déclaration delegate.</span><span class="sxs-lookup"><span data-stu-id="21933-106">Instances of a delegate must be declared with the same signature as the delegate declaration.</span></span>  
  
 <span data-ttu-id="21933-107">Vous pouvez corriger cette erreur en ajustant la signature de la méthode ou du délégué.</span><span class="sxs-lookup"><span data-stu-id="21933-107">You can resolve this error by adjusting either the method or delegate signature.</span></span> <span data-ttu-id="21933-108">Pour plus d'informations, consultez [Délégués](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="21933-108">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="21933-109">L’exemple suivant génère l’erreur CS0123.</span><span class="sxs-lookup"><span data-stu-id="21933-109">The following sample generates CS0123.</span></span>  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
