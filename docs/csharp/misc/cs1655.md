# <a name="compiler-error-cs1655"></a><span data-ttu-id="66abd-103">Erreur du compilateur CS1655</span><span class="sxs-lookup"><span data-stu-id="66abd-103">Compiler Error CS1655</span></span>
<span data-ttu-id="66abd-104">Impossible de passer les champs de 'variable' en tant qu’argument ref ou out, car il s’agit d’un 'readonly variable type'</span><span class="sxs-lookup"><span data-stu-id="66abd-104">Cannot pass fields of 'variable' as a ref or out argument because it is a 'readonly variable type'</span></span>  
  
 <span data-ttu-id="66abd-105">Cette erreur se produit si vous tentez de passer un membre d’une variable [foreach](../language-reference/keywords/foreach-in.md) , d’une variable [using](../language-reference/keywords/using-statement.md) ou d’une variable [fixed](../language-reference/keywords/fixed-statement.md) à une fonction en tant qu’argument ref ou out.</span><span class="sxs-lookup"><span data-stu-id="66abd-105">This error occurs if you are attempting to pass a member of a [foreach](../language-reference/keywords/foreach-in.md) variable, a [using](../language-reference/keywords/using-statement.md) variable, or a [fixed](../language-reference/keywords/fixed-statement.md) variable to a function as a ref or out argument.</span></span> <span data-ttu-id="66abd-106">Étant donné que ces variables sont considérées en lecture seule dans ces contextes, cela n’est pas autorisé.</span><span class="sxs-lookup"><span data-stu-id="66abd-106">Because these variables are considered read-only in these contexts, this is not allowed.</span></span>  
  
 <span data-ttu-id="66abd-107">L’exemple suivant génère l’erreur CS1655 :</span><span class="sxs-lookup"><span data-stu-id="66abd-107">The following sample generates CS1655:</span></span>  
  
```csharp  
// CS1655.cs  
struct S
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }  
  
  public static void Main()  
  {  
     S[] sa = new S[10];  
     foreach(S s in sa)  
     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```
