# <a name="compiler-error-cs1534"></a>Erreur du compilateur CS1534
L’opérateur binaire surchargé 'opérateur' accepte deux paramètres  
  
 La définition d’un fichier binaire [opérateur](../../csharp/language-reference/operators/operator-overloading.md) doit prendre deux paramètres.  
  
 L’exemple suivant génère l’erreur CS1534 :  
  
```csharp  
// CS1534.cs  
class MyClass  
{  
   public static MyClass operator - (MyClass MC1, MyClass MC2, MyClass MC3)   // CS1534  
   // try the following line instead  
   // public static MyClass operator - (MyClass MC1, MyClass MC2)  
   {  
      return new MyClass();  
   }  
  
   public static int Main()  
   {  
      return 1;  
   }  
}  
```
