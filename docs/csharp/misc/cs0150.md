# <a name="compiler-error-cs0150"></a><span data-ttu-id="b54e1-102">Erreur du compilateur CS0150</span><span class="sxs-lookup"><span data-stu-id="b54e1-102">Compiler Error CS0150</span></span>
<span data-ttu-id="b54e1-103">Une valeur de constante est attendue</span><span class="sxs-lookup"><span data-stu-id="b54e1-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="b54e1-104">Une variable a été trouvée là où une constante était attendue.</span><span class="sxs-lookup"><span data-stu-id="b54e1-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="b54e1-105">Pour plus d'informations, consultez [switch](../language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="b54e1-105">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="b54e1-106">L’exemple suivant génère l’erreur CS0150 :</span><span class="sxs-lookup"><span data-stu-id="b54e1-106">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="b54e1-107">Cette erreur est également générée quand une taille de tableau est spécifiée avec une valeur de variable et initialisée avec un initialiseur de tableau.</span><span class="sxs-lookup"><span data-stu-id="b54e1-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="b54e1-108">Pour supprimer l’erreur, initialisez le tableau dans une ou plusieurs instructions distinctes.</span><span class="sxs-lookup"><span data-stu-id="b54e1-108">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;   
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
