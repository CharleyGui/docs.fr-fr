# <a name="compiler-error-cs0077"></a>Erreur du compilateur CS0077
L’opérateur as doit être utilisé avec un type référence ou un type Nullable ('int' est un type valeur qui n’autorise pas les valeurs null).  
  
 Un [type valeur](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator) a été passé à l’opérateur [as](../../csharp/language-reference/keywords/value-types.md). Comme `as` peut retourner [null](../../csharp/language-reference/keywords/null.md), seuls des [types référence](../../csharp/language-reference/keywords/reference-types.md) ou un type Nullable peuvent lui être passés. Pour plus d’informations sur les types Nullable, consultez [types Nullable](../../csharp/programming-guide/nullable-types/index.md).  
  
 L’exemple suivant génère l’erreur CS0077 :  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
