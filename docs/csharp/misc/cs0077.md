# <a name="compiler-error-cs0077"></a><span data-ttu-id="e6f31-102">Erreur du compilateur CS0077</span><span class="sxs-lookup"><span data-stu-id="e6f31-102">Compiler Error CS0077</span></span>
<span data-ttu-id="e6f31-103">L’opérateur as doit être utilisé avec un type référence ou un type Nullable ('int' est un type valeur qui n’autorise pas les valeurs null).</span><span class="sxs-lookup"><span data-stu-id="e6f31-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="e6f31-104">Un [type valeur](../language-reference/operators/type-testing-and-cast.md#as-operator) a été passé à l’opérateur [as](../language-reference/keywords/value-types.md).</span><span class="sxs-lookup"><span data-stu-id="e6f31-104">The [as](../language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="e6f31-105">Comme `as` peut retourner [null](../language-reference/keywords/null.md), seuls des [types référence](../language-reference/keywords/reference-types.md) ou un type Nullable peuvent lui être passés.</span><span class="sxs-lookup"><span data-stu-id="e6f31-105">Because `as` can return [null](../language-reference/keywords/null.md), it can only be passed [reference types](../language-reference/keywords/reference-types.md) or nullable type.</span></span> <span data-ttu-id="e6f31-106">Pour plus d’informations sur les types Nullable, consultez [types Nullable](../programming-guide/nullable-types/index.md).</span><span class="sxs-lookup"><span data-stu-id="e6f31-106">For more information about nullable types, see [Nullable Types](../programming-guide/nullable-types/index.md).</span></span>  
  
 <span data-ttu-id="e6f31-107">L’exemple suivant génère l’erreur CS0077 :</span><span class="sxs-lookup"><span data-stu-id="e6f31-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
