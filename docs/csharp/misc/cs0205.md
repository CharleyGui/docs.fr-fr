 <span data-ttu-id="56b36-104">Vous ne pouvez pas appeler une méthode [abstraite](../language-reference/keywords/abstract.md) , car elle n’a pas de corps de méthode.</span><span class="sxs-lookup"><span data-stu-id="56b36-104">You cannot call an [abstract](../language-reference/keywords/abstract.md) method because it does not have a method body.</span></span> <span data-ttu-id="56b36-105">Pour plus d’informations, consultez la page [Classes abstraites et scellées et membres de classe](../programming-guide/classes-and-structs/abstract-and-sealed-classes-and-class-members.md).</span><span class="sxs-lookup"><span data-stu-id="56b36-105">For more information, see [Abstract and Sealed Classes and Class Members](../programming-guide/classes-and-structs/abstract-and-sealed-classes-and-class-members.md).</span></span>  
  
 <span data-ttu-id="56b36-106">L’exemple suivant génère l’erreur CS0205 :</span><span class="sxs-lookup"><span data-stu-id="56b36-106">The following sample generates CS0205:</span></span>  
  
```csharp  
// CS0205.cs  
abstract public class MyClass  
{  
   abstract public void M();  
}  
  
public class MyClass2 : MyClass  
{  
   public override void M()  
   {  
      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
