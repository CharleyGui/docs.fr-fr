# <a name="compiler-error-cs1561"></a>Erreur du compilateur CS1561

Le nom de fichier de sortie est trop long ou n'est pas valide  
  
 Le nom de fichier de sortie ne peut pas dépasser 256 caractères et ne peut pas contenir de caractères incorrects, tels que le symbole Euro, un point d’interrogation ou une barre oblique inverse, entre autres.
