# <a name="compiler-error-cs0216"></a><span data-ttu-id="32f86-103">Erreur du compilateur CS0216</span><span class="sxs-lookup"><span data-stu-id="32f86-103">Compiler Error CS0216</span></span>

<span data-ttu-id="32f86-104">L’opérateur 'opérateur' exige qu’un opérateur correspondant 'opérateur_manquant' soit aussi défini</span><span class="sxs-lookup"><span data-stu-id="32f86-104">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="32f86-105">Un opérateur défini par l’utilisateur [==](../language-reference/operators/equality-operators#equality-operator-) requiert un opérateur défini [](../language-reference/operators/inequality-operators#inequality-operator-) par l’utilisateur, et vice versa.</span><span class="sxs-lookup"><span data-stu-id="32f86-105">A user-defined [==](../language-reference/operators/equality-operators#equality-operator-) operator requires a user-defined [!=](../language-reference/operators/inequality-operators#inequality-operator-) operator, and vice versa.</span></span>  
 <span data-ttu-id="32f86-106">Cela s’applique également à un opérateur [true](../language-reference/operators/true-false-operators.md) défini par l’utilisateur et à un opérateur [false](../language-reference/operators/true-false-operators.md) défini par l’utilisateur.</span><span class="sxs-lookup"><span data-stu-id="32f86-106">The same applies also to a user-defined [true](../language-reference/operators/true-false-operators.md) operator and a user-defined [false](../language-reference/operators/true-false-operators.md) operator.</span></span>  
  
 <span data-ttu-id="32f86-107">L’exemple suivant génère l’erreur CS0216 :</span><span class="sxs-lookup"><span data-stu-id="32f86-107">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator == (MyClass MyIntLeft, MyClass MyIntRight)   // CS0216  
   {  
      return MyIntLeft == MyIntRight;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator != (MyClass MyIntLeft, MyClass MyIntRight)  
   {  
      return MyIntLeft != MyIntRight;  
   }  
   */  
  
   public override bool Equals (object obj)  
   {  
      return base.Equals (obj);  
   }  
  
   public override int GetHashCode()  
   {  
      return base.GetHashCode();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
