# <a name="compiler-warning-level-1-cs0672"></a><span data-ttu-id="a4978-103">Avertissement du compilateur (niveau 1) CS0672</span><span class="sxs-lookup"><span data-stu-id="a4978-103">Compiler Warning (level 1) CS0672</span></span>
<span data-ttu-id="a4978-104">Le membre 'membre1' se substitue au membre obsolète 'membre2'.</span><span class="sxs-lookup"><span data-stu-id="a4978-104">Member 'member1' overrides obsolete member 'member2.</span></span> <span data-ttu-id="a4978-105">Ajoutez l’attribut Obsolete à 'membre1'</span><span class="sxs-lookup"><span data-stu-id="a4978-105">Add the Obsolete attribute to 'member1'</span></span>  
  
 <span data-ttu-id="a4978-106">Le compilateur a trouvé un `override` à une méthode marquée comme `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="a4978-106">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="a4978-107">Toutefois, la méthode de substitution n’était pas marquée comme obsolète.</span><span class="sxs-lookup"><span data-stu-id="a4978-107">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="a4978-108">La méthode de substitution va générer l’erreur [CS0612](./cs0612.md)si elle est appelée.</span><span class="sxs-lookup"><span data-stu-id="a4978-108">The overriding method will still generate [CS0612](./cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="a4978-109">Examinez vos déclarations de méthode et indiquez explicitement si une méthode (et toutes ses substitutions) doit être marquée `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="a4978-109">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="a4978-110">L’exemple suivant génère l’erreur CS0672 :</span><span class="sxs-lookup"><span data-stu-id="a4978-110">The following sample generates CS0672:</span></span>  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```
