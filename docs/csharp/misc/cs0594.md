# <a name="compiler-error-cs0594"></a>Erreur du compilateur CS0594
La constante à virgule flottante sort de la plage du type 'type'  
  
 Une valeur assignée à une variable à virgule flottante est trop grande pour les variables de ce type de données. Pour plus d’informations sur la plage de valeurs autorisées des types de données, consultez [Tableau des types intégraux](../../csharp/language-reference/builtin-types/integral-numeric-types.md) .  
  
 L’exemple suivant génère l’erreur CS0594 :  
  
```csharp  
// CS0594.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         float f = 6.77777777777E400;   // CS0594, value too large  
      }  
   }  
}  
```
