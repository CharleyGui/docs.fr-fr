# <a name="compiler-error-cs2012"></a>Erreur du compilateur CS2012

Impossible d’ouvrir 'file' en écriture  
  
 Lors de l’utilisation de l’option [de compilateur-bugreport ( :](../language-reference/compiler-options/bugreport-compiler-option.md) `file` , le fichier n’a pas pu être ouvert en écriture. Assurez-vous d’avoir spécifié un nom de fichier valide et que ce fichier n’est pas en lecture seule.
