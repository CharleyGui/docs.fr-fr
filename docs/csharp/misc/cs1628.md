 <span data-ttu-id="25910-105">Cette erreur se produit si vous utilisez `in` un `ref` paramètre, ou `out` à l’intérieur d’un bloc de méthode anonyme.</span><span class="sxs-lookup"><span data-stu-id="25910-105">This error occurs if you use an `in`, `ref`, or `out` parameter inside an anonymous method block.</span></span> <span data-ttu-id="25910-106">Pour éviter cette erreur, utilisez une variable locale ou une autre construction.</span><span class="sxs-lookup"><span data-stu-id="25910-106">To avoid this error, use a local variable or some other construct.</span></span>

  }  
}
