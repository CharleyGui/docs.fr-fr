# <a name="compiler-error-cs1625"></a><span data-ttu-id="58345-103">Erreur du compilateur CS1625</span><span class="sxs-lookup"><span data-stu-id="58345-103">Compiler Error CS1625</span></span>

<span data-ttu-id="58345-104">Impossible de générer dans le corps d’une clause finally</span><span class="sxs-lookup"><span data-stu-id="58345-104">Cannot yield in the body of a finally clause</span></span>  
  
 <span data-ttu-id="58345-105">Une instruction yield n’est pas autorisée dans le corps d’une clause finally.</span><span class="sxs-lookup"><span data-stu-id="58345-105">A yield statement is not allowed in the body of a finally clause.</span></span> <span data-ttu-id="58345-106">Pour éviter cette erreur, placez l’instruction yield en dehors de la clause finally.</span><span class="sxs-lookup"><span data-stu-id="58345-106">To avoid this error, move the yield statement out of the finally clause.</span></span>  
  
 <span data-ttu-id="58345-107">L’exemple suivant génère l’erreur CS1625 :</span><span class="sxs-lookup"><span data-stu-id="58345-107">The following sample generates CS1625:</span></span>  
  
```csharp  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
