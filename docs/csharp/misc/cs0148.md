# <a name="compiler-error-cs0148"></a>Erreur du compilateur CS0148
Le délégué 'délégué' ne possède pas de constructeur valide  
  
 Vous avez importé et utilisé un programme managé (qui utilise le common language runtime .NET Framework) créé avec un autre compilateur. Ce compilateur a autorisé un constructeur [délégué](../language-reference/keywords/delegate.md) incorrect. Pour plus d’informations, consultez [Délégués](../programming-guide/delegates/index.md).
