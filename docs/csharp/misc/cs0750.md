# <a name="compiler-error-cs0750"></a><span data-ttu-id="dc1f0-103">Erreur du compilateur CS0750</span><span class="sxs-lookup"><span data-stu-id="dc1f0-103">Compiler Error CS0750</span></span>

<span data-ttu-id="dc1f0-104">Une méthode partielle ne peut pas avoir de modificateurs d’accès ni de modificateurs virtual, abstract, override, new, sealed ou extern.</span><span class="sxs-lookup"><span data-stu-id="dc1f0-104">A partial method cannot have access modifiers or the virtual, abstract, override, new, sealed, or extern modifiers.</span></span>  
  
 <span data-ttu-id="dc1f0-105">En raison de leur comportement particulier, les méthodes partielles sont soumises à des restrictions concernant les modificateurs qu’elles peuvent accepter.</span><span class="sxs-lookup"><span data-stu-id="dc1f0-105">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="dc1f0-106">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="dc1f0-106">To correct this error</span></span>  
  
1. <span data-ttu-id="dc1f0-107">Supprimez le modificateur non autorisé de la déclaration de méthode.</span><span class="sxs-lookup"><span data-stu-id="dc1f0-107">Remove the unauthorized modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="dc1f0-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="dc1f0-108">Example</span></span>  

 <span data-ttu-id="dc1f0-109">L’exemple suivant génère l’erreur CS0750 :</span><span class="sxs-lookup"><span data-stu-id="dc1f0-109">The following example generates CS0750:</span></span>  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
