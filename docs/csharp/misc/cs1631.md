# <a name="compiler-error-cs1631"></a><span data-ttu-id="b26af-103">Erreur du compilateur CS1631</span><span class="sxs-lookup"><span data-stu-id="b26af-103">Compiler Error CS1631</span></span>
<span data-ttu-id="b26af-104">Impossible de générer une valeur dans le corps d'une clause catch</span><span class="sxs-lookup"><span data-stu-id="b26af-104">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="b26af-105">L’instruction yield n’est pas autorisée à partir du corps d’une clause catch.</span><span class="sxs-lookup"><span data-stu-id="b26af-105">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="b26af-106">Pour éviter cette erreur, placez l’instruction yield en dehors du corps de la clause catch.</span><span class="sxs-lookup"><span data-stu-id="b26af-106">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="b26af-107">L’exemple suivant génère l’erreur CS1631 :</span><span class="sxs-lookup"><span data-stu-id="b26af-107">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }
  
   public static void Main()
   {  
   }  
}  
```
