# <a name="compiler-error-cs1512"></a><span data-ttu-id="abf1b-102">Erreur du compilateur CS1512</span><span class="sxs-lookup"><span data-stu-id="abf1b-102">Compiler Error CS1512</span></span>
<span data-ttu-id="abf1b-103">Le mot clé 'base' n'est pas disponible dans le contexte actuel</span><span class="sxs-lookup"><span data-stu-id="abf1b-103">Keyword 'base' is not available in the current context</span></span>  
  
 <span data-ttu-id="abf1b-104">Le mot clé [base](../language-reference/keywords/base.md) a été utilisé en dehors d’une méthode, d’une propriété ou d’un constructeur.</span><span class="sxs-lookup"><span data-stu-id="abf1b-104">The [base](../language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>  
  
 <span data-ttu-id="abf1b-105">L’exemple suivant génère l’erreur CS1512 :</span><span class="sxs-lookup"><span data-stu-id="abf1b-105">The following example generates CS1512:</span></span>  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
