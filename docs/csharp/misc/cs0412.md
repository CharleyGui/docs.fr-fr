# <a name="compiler-error-cs0412"></a><span data-ttu-id="812a5-103">Erreur du compilateur CS0412</span><span class="sxs-lookup"><span data-stu-id="812a5-103">Compiler Error CS0412</span></span>

<span data-ttu-id="812a5-104">'generic' : un paramètre ou une variable locale ne peut pas avoir le même nom qu’un paramètre de type de méthode</span><span class="sxs-lookup"><span data-stu-id="812a5-104">'generic': a parameter or local variable cannot have the same name as a method type parameter</span></span>  
  
 <span data-ttu-id="812a5-105">Il existe un conflit de nom entre le paramètre de type d’une méthode générique et une variable locale, au niveau de la méthode ou de l’un des paramètres de la méthode.</span><span class="sxs-lookup"><span data-stu-id="812a5-105">There is a name conflict between the type parameter of a generic method and a local variable in the method or one of the method's parameters.</span></span> <span data-ttu-id="812a5-106">Pour éviter cette erreur, renommez les paramètres ou les variables locales incompatibles.</span><span class="sxs-lookup"><span data-stu-id="812a5-106">To avoid this error, rename any conflicting parameters or local variables.</span></span>  
  
## <a name="example"></a><span data-ttu-id="812a5-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="812a5-107">Example</span></span>  

 <span data-ttu-id="812a5-108">L’exemple suivant génère l’erreur CS0412 :</span><span class="sxs-lookup"><span data-stu-id="812a5-108">The following sample generates CS0412:</span></span>  
  
```csharp  
// CS0412.cs  
using System;  
  
class C  
{  
    // Parameter name is the same as method type parameter name  
    public void G<T>(int T)  // CS0412  
    {  
    }  
    public void F<T>()  
    {  
        // Method local variable name is the same as method type  
        // parameter name  
        double T = 0.0;  // CS0412  
        Console.WriteLine(T);  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
