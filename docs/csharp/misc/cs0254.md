# <a name="compiler-error-cs0254"></a><span data-ttu-id="f6ef9-103">Erreur du compilateur CS0254</span><span class="sxs-lookup"><span data-stu-id="f6ef9-103">Compiler Error CS0254</span></span>
<span data-ttu-id="f6ef9-104">La partie droite d'une assignation d'instruction fixed peut ne pas être une expression de cast</span><span class="sxs-lookup"><span data-stu-id="f6ef9-104">The right hand side of a fixed statement assignment may not be a cast expression</span></span>  
  
 <span data-ttu-id="f6ef9-105">La partie droite d’une expression [fixed](../language-reference/keywords/fixed-statement.md) ne peut pas utiliser un cast.</span><span class="sxs-lookup"><span data-stu-id="f6ef9-105">The right side of a [fixed](../language-reference/keywords/fixed-statement.md) expression may not use a cast.</span></span> <span data-ttu-id="f6ef9-106">Pour plus d’informations, consultez [Pointeurs et code unsafe](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="f6ef9-106">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="f6ef9-107">L’exemple suivant génère l’erreur CS0254 :</span><span class="sxs-lookup"><span data-stu-id="f6ef9-107">The following sample generates CS0254:</span></span>  
  
```csharp  
// CS0254.cs  
// compile with: /unsafe  
class Point  
{  
   public uint x, y;  
}  
  
class FixedTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      *p *= *p;  
   }  
  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
      pt.x = 5;  
      pt.y = 6;  
  
      fixed (int* p = (int*)&pt.x)   // CS0254  
      // try the following line instead  
      // fixed (uint* p = &pt.x)  
      {  
         SquarePtrParam ((int*)p);  
      }  
   }  
}  
```
