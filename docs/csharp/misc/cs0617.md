# <a name="compiler-error-cs0617"></a><span data-ttu-id="cc744-102">Erreur du compilateur CS0617</span><span class="sxs-lookup"><span data-stu-id="cc744-102">Compiler Error CS0617</span></span>
<span data-ttu-id="cc744-103">'référence' n’est pas un argument d’attribut nommé valide, car il n’est pas un type de paramètre d’attribut valide</span><span class="sxs-lookup"><span data-stu-id="cc744-103">'reference' is not a valid named attribute argument because it is not a valid attribute parameter type</span></span>  
  
 <span data-ttu-id="cc744-104">L’utilisateur a tenté d’accéder à un membre [privé](../language-reference/keywords/private.md) d’une classe d’attributs.</span><span class="sxs-lookup"><span data-stu-id="cc744-104">An attempt was made to access a [private](../language-reference/keywords/private.md) member of an attribute class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="cc744-105"> Exemple</span><span class="sxs-lookup"><span data-stu-id="cc744-105">Example</span></span>  
 <span data-ttu-id="cc744-106">L’exemple suivant génère l’erreur CS0617 :</span><span class="sxs-lookup"><span data-stu-id="cc744-106">The following sample generates CS0617.</span></span>  
  
```csharp  
// CS0617.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Struct |
                AttributeTargets.Class |  
                AttributeTargets.Interface)]  
public class MyClass : Attribute  
{  
   public int Name;  
  
   public MyClass (int sName)  
   {  
      Name = sName;  
      Bad = -1;  
      Bad2 = -1;  
   }  
  
   public readonly int Bad;  
   public int Bad2;  
}  
  
[MyClass(5, Bad=0)] class Class1 {}   // CS0617  
[MyClass(5, Bad2=0)] class Class2 {}  
```
