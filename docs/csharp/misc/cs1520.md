# <a name="compiler-error-cs1520"></a><span data-ttu-id="317b3-103">Erreur du compilateur CS1520</span><span class="sxs-lookup"><span data-stu-id="317b3-103">Compiler Error CS1520</span></span>
<span data-ttu-id="317b3-104">La méthode doit avoir un type de retour</span><span class="sxs-lookup"><span data-stu-id="317b3-104">Method must have a return type</span></span>  
  
 <span data-ttu-id="317b3-105">Une méthode qui est déclarée dans une classe, une structure ou une interface doit avoir un type de retour explicite.</span><span class="sxs-lookup"><span data-stu-id="317b3-105">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="317b3-106">Dans l’exemple suivant, la méthode Square a la valeur de retour [string](../language-reference/builtin-types/reference-types.md):</span><span class="sxs-lookup"><span data-stu-id="317b3-106">In the following example, the Square method has a return value of [string](../language-reference/builtin-types/reference-types.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="317b3-107">L’exemple suivant génère l’erreur CS1520 :</span><span class="sxs-lookup"><span data-stu-id="317b3-107">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="317b3-108">Cette erreur peut aussi se produire quand la casse du nom d’un constructeur diffère de celle la déclaration de classe ou de struct, comme dans l’exemple suivant.</span><span class="sxs-lookup"><span data-stu-id="317b3-108">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="317b3-109">Comme le nom n’est pas exactement le même que celui de la classe, le compilateur l’interprète comme une méthode normale, et non comme un constructeur, et génère l’erreur :</span><span class="sxs-lookup"><span data-stu-id="317b3-109">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
