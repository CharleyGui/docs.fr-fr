# <a name="compiler-error-cs0531"></a><span data-ttu-id="284a9-103">Erreur du compilateur CS0531</span><span class="sxs-lookup"><span data-stu-id="284a9-103">Compiler Error CS0531</span></span>

<span data-ttu-id="284a9-104">'membre' : les membres d’interface ne peuvent pas avoir de définition</span><span class="sxs-lookup"><span data-stu-id="284a9-104">'member' : interface members cannot have a definition</span></span>  
  
 <span data-ttu-id="284a9-105">Les méthodes déclarées dans une [interface](../language-reference/keywords/interface.md) doivent être implémentées dans une classe qui en hérite et non dans l’interface elle-même.</span><span class="sxs-lookup"><span data-stu-id="284a9-105">Methods that are declared in an [interface](../language-reference/keywords/interface.md) must be implemented in a class that inherits from it and not in the interface itself.</span></span>  
  
 <span data-ttu-id="284a9-106">L’exemple suivant génère l’erreur CS0531 :</span><span class="sxs-lookup"><span data-stu-id="284a9-106">The following sample generates CS0531:</span></span>  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
