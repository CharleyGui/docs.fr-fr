# <a name="compiler-error-cs1660"></a><span data-ttu-id="32ccd-103">Erreur du compilateur CS1660</span><span class="sxs-lookup"><span data-stu-id="32ccd-103">Compiler Error CS1660</span></span>

<span data-ttu-id="32ccd-104">Impossible de convertir un bloc de méthode anonyme en type 'type', car il ne s’agit pas d’un type délégué</span><span class="sxs-lookup"><span data-stu-id="32ccd-104">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="32ccd-105">Cette erreur se produit si vous essayez d’assigner ou de convertir un bloc de méthode anonyme en type qui n’est pas un type délégué.</span><span class="sxs-lookup"><span data-stu-id="32ccd-105">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="32ccd-106">L’exemple suivant génère l’erreur CS1660 :</span><span class="sxs-lookup"><span data-stu-id="32ccd-106">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
