<span data-ttu-id="e7743-104">Le type 'nom_type' doit être un type de référence afin d’être utilisé comme paramètre 'nom_paramètre' dans le type ou la méthode générique 'identificateur_générique'</span><span class="sxs-lookup"><span data-stu-id="e7743-104">The type 'type name' must be a reference type in order to use it as parameter 'parameter name' in the generic type or method 'identifier of generic'</span></span>
