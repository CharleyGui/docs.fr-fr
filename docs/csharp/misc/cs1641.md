 <span data-ttu-id="6cd6a-107">L’exemple suivant génère l’erreur CS1641 :</span><span class="sxs-lookup"><span data-stu-id="6cd6a-107">The following sample generates CS1641:</span></span>

   fixed int [] a;  // CS1641
