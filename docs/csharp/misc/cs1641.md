# <a name="compiler-error-cs1641"></a><span data-ttu-id="31ce3-103">Erreur du compilateur CS1641</span><span class="sxs-lookup"><span data-stu-id="31ce3-103">Compiler Error CS1641</span></span>

<span data-ttu-id="31ce3-104">Un champ de mémoire tampon de taille fixe doit utiliser le spécificateur de la taille du tableau après le nom du champ</span><span class="sxs-lookup"><span data-stu-id="31ce3-104">A fixed size buffer field must have the array size specifier after the field name</span></span>  
  
 <span data-ttu-id="31ce3-105">Contrairement aux tableaux normaux, les mémoires tampons de taille fixe exigent la spécification d’une taille constante au niveau du point de déclaration.</span><span class="sxs-lookup"><span data-stu-id="31ce3-105">Unlike regular arrays, fixed size buffers require a constant size to be specified at the declaration point.</span></span> <span data-ttu-id="31ce3-106">Pour résoudre cette erreur, ajoutez un littéral entier positif ou un entier positif constant et placez les crochets après l’identificateur.</span><span class="sxs-lookup"><span data-stu-id="31ce3-106">To resolve this error, add a positive integer literal or a constant positive integer and put the square brackets after the identifier.</span></span>  
  
 <span data-ttu-id="31ce3-107">L’exemple suivant génère l’erreur CS1641 :</span><span class="sxs-lookup"><span data-stu-id="31ce3-107">The following sample generates CS1641:</span></span>  
  
```csharp  
// CS1641.cs  
// compile with: /unsafe /target:library  
unsafe struct S {  
   fixed int [] a;  // CS1641  
  
   // OK  
   fixed int b [10];  
   const int c = 10;  
   fixed int d [c];  
}  
```
