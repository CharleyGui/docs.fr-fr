# <a name="compiler-warning-level-1-cs2023"></a>Avertissement du compilateur (niveau 1) CS2023
Option /noconfig ignorée, car elle était spécifiée dans un fichier réponse  
  
 L’option [de compilateur-noconfig](../language-reference/compiler-options/noconfig-compiler-option.md) a été spécifiée dans un fichier réponse, ce qui n’est pas autorisé.  
  
 Cet avertissement ne peut pas être supprimé par l’option [-nowarn](../language-reference/compiler-options/nowarn-compiler-option.md) .
