# <a name="compiler-warning-level-1-cs2023"></a>Avertissement du compilateur (niveau 1) CS2023
Option /noconfig ignorée, car elle était spécifiée dans un fichier réponse  
  
 L’option [/noconfig](../language-reference/compiler-options/noconfig-compiler-option.md) du compilateur a été spécifiée dans un fichier réponse, ce qui n’est pas autorisé.  
  
 Vous ne pouvez pas supprimer cet avertissement avec l’option [/nowarn](../language-reference/compiler-options/nowarn-compiler-option.md) .
