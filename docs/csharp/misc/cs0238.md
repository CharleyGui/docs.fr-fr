# <a name="compiler-error-cs0238"></a><span data-ttu-id="f1cf6-103">Erreur du compilateur CS0238</span><span class="sxs-lookup"><span data-stu-id="f1cf6-103">Compiler Error CS0238</span></span>
<span data-ttu-id="f1cf6-104">'member' ne peut pas être sealed car il ne s’agit pas d’une substitution</span><span class="sxs-lookup"><span data-stu-id="f1cf6-104">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="f1cf6-105">[sealed](../language-reference/keywords/sealed.md) a été utilisé sur un membre qui n’était pas marqué avec [override](../language-reference/keywords/override.md).</span><span class="sxs-lookup"><span data-stu-id="f1cf6-105">[sealed](../language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../language-reference/keywords/override.md).</span></span> <span data-ttu-id="f1cf6-106">Pour plus d’informations, consultez [Héritage](../programming-guide/classes-and-structs/inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="f1cf6-106">For more information, see [Inheritance](../programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="f1cf6-107">L’exemple suivant génère l’erreur CS0238 :</span><span class="sxs-lookup"><span data-stu-id="f1cf6-107">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
