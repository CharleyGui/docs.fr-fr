## <a name="example"></a><span data-ttu-id="3fa90-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="3fa90-105">Example</span></span>

<span data-ttu-id="3fa90-106">L’exemple suivant génère l’erreur CS0619 :</span><span class="sxs-lookup"><span data-stu-id="3fa90-106">The following sample generates CS0619:</span></span>  
  
```csharp
using System;

public class C
{
    [Obsolete("Use NewMethod instead", true)] // generates an error on use
    public static void OldMethod()
    {
    }

    // this is the method you should be using
    public static void NewMethod()
    {
    }  
}

class MyClass
{
   public static void Main()
   {
      C.OldMethod();   // CS0619
   }
}
```
