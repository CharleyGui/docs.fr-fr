# <a name="compiler-error-cs0538"></a><span data-ttu-id="b6e8d-103">Erreur du compilateur CS0538</span><span class="sxs-lookup"><span data-stu-id="b6e8d-103">Compiler Error CS0538</span></span>

<span data-ttu-id="b6e8d-104">'nom' dans une déclaration d’interface explicite n’est pas une interface</span><span class="sxs-lookup"><span data-stu-id="b6e8d-104">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="b6e8d-105">Une tentative a été effectuée pour déclarer explicitement une [interface](../language-reference/keywords/interface.md), mais aucune interface n’a été spécifiée.</span><span class="sxs-lookup"><span data-stu-id="b6e8d-105">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="b6e8d-106">L’exemple suivant génère l’erreur CS0538 :</span><span class="sxs-lookup"><span data-stu-id="b6e8d-106">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
