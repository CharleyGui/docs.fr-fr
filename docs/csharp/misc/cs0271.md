 <span data-ttu-id="1c073-104">Cette erreur se produit quand vous essayez d’accéder à un accesseur `get` inaccessible.</span><span class="sxs-lookup"><span data-stu-id="1c073-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="1c073-105">Pour résoudre cette erreur, augmentez l’accessibilité de l’accesseur ou modifiez l’emplacement d’appel.</span><span class="sxs-lookup"><span data-stu-id="1c073-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="1c073-106">Pour plus d’informations, consultez [accessibilité](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) et [Propriétés](../programming-guide/classes-and-structs/properties.md)de l’accesseur.</span><span class="sxs-lookup"><span data-stu-id="1c073-106">For more information, see [Accessor Accessibility](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="1c073-107">L’exemple suivant génère l’erreur CS0271 :</span><span class="sxs-lookup"><span data-stu-id="1c073-107">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)   
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
