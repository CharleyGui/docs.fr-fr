# <a name="compiler-error-cs0573"></a>Erreur du compilateur CS0573
'déclaration_champ' : impossible d’avoir des initialiseurs de champ d’instance dans des structures  
  
 Vous ne pouvez pas initialiser un champ d’instance d’un [struct](../language-reference/keywords/struct.md). Les champs de types valeur seront initialisés à leurs valeurs par défaut et les champs de type référence seront initialisés à `null`.  
  
## <a name="example"></a>Exemple  
 L’exemple suivant génère l’erreur CS0573 :  
  
```csharp  
// CS0573.cs  
namespace x  
{  
    public class clx  
    {  
        public static void Main()  
        {  
        }  
    }  
  
    public struct cly  
    {  
        clx a = new clx();   // CS0573  
        // clx a;            // OK  
        int i = 7;           // CS0573  
        // int i;            // OK  
    }  
}  
```
