# <a name="compiler-error-cs1015"></a>Erreur du compilateur CS1015
Type objet, chaîne ou classe attendu  
  
 L’utilisateur a tenté de passer un type de données prédéfini dans un bloc [catch](../../csharp/language-reference/keywords/try-catch.md) . Seuls les types de données qui dérivent de <xref:System.Exception?displayProperty=nameWithType> peuvent être passés dans un bloc `catch` . Pour plus d’informations sur les exceptions, consultez [Exceptions et gestion des exceptions](../../csharp/programming-guide/exceptions/index.md).  
  
## <a name="example"></a>Exemple  
 L’exemple suivant génère l’erreur CS1015 :  
  
```csharp  
// CS1015.cs  
class Sample  
{  
    static void Main()  
    {  
        try   
        {  
        }  
        catch(int)   // CS1015, int is not derived from System.Exception  
        {  
        }  
    }  
}  
```
