# <a name="compiler-error-cs0149"></a><span data-ttu-id="c0fdd-102">Erreur du compilateur CS0149</span><span class="sxs-lookup"><span data-stu-id="c0fdd-102">Compiler Error CS0149</span></span>
<span data-ttu-id="c0fdd-103">Nom de méthode attendu</span><span class="sxs-lookup"><span data-stu-id="c0fdd-103">Method name expected</span></span>  
  
 <span data-ttu-id="c0fdd-104">Quand vous créez un [délégué](../language-reference/keywords/delegate.md), spécifiez une méthode.</span><span class="sxs-lookup"><span data-stu-id="c0fdd-104">When creating a [delegate](../language-reference/keywords/delegate.md), specify a method.</span></span> <span data-ttu-id="c0fdd-105">Pour plus d’informations, consultez [Délégués](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="c0fdd-105">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="c0fdd-106">L’exemple suivant génère l’erreur CS0149 :</span><span class="sxs-lookup"><span data-stu-id="c0fdd-106">The following sample generates CS0149:</span></span>  
  
```csharp  
// CS0149.cs  
using System;  
  
delegate string MyDelegate(int i);  
  
class MyClass  
{  
   // class member-field of the declared delegate type  
   static MyDelegate dt;     
  
   public static void Main()  
   {  
      dt = new MyDelegate(17.45);   // CS0149  
      // try the following line instead  
      // dt = new MyDelegate(Func2);  
      F(dt);  
   }  
  
   public static string Func2(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void F(MyDelegate myFunc)  
   {  
      myFunc(8);  
   }  
}  
```
