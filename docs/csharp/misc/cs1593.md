# <a name="compiler-error-cs1593"></a><span data-ttu-id="add74-102">Erreur du compilateur CS1593</span><span class="sxs-lookup"><span data-stu-id="add74-102">Compiler Error CS1593</span></span>
<span data-ttu-id="add74-103">Le délégué 'del' ne prend pas les arguments 'nombre'</span><span class="sxs-lookup"><span data-stu-id="add74-103">Delegate 'del' does not take 'number' arguments</span></span>  
  
 <span data-ttu-id="add74-104">Le nombre d’arguments passés à un appel de [délégué](../language-reference/keywords/delegate.md) ne correspond pas au nombre de paramètres dans la déclaration du délégué.</span><span class="sxs-lookup"><span data-stu-id="add74-104">The number of arguments passed to a [delegate](../language-reference/keywords/delegate.md) invocation does not agree with the number of parameters in the delegate declaration.</span></span>  
  
 <span data-ttu-id="add74-105">L’exemple suivant génère l’avertissement CS1593 :</span><span class="sxs-lookup"><span data-stu-id="add74-105">The following sample generates CS1593:</span></span>  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
