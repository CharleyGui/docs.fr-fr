## <a name="example"></a><span data-ttu-id="21d22-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="21d22-108">Example</span></span>

<span data-ttu-id="21d22-109">Le code suivant génère l’erreur CS1955 car le code essaie d’appeler un champ et une propriété à l’aide de la [opérateur d’appel `()` ](../language-reference/operators/member-access-operators.md#invocation-operator-).</span><span class="sxs-lookup"><span data-stu-id="21d22-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation operator `()`](../language-reference/operators/member-access-operators.md#invocation-operator-).</span></span> <span data-ttu-id="21d22-110">Vous ne pouvez pas appeler un champ ou une propriété.</span><span class="sxs-lookup"><span data-stu-id="21d22-110">You cannot call a field or a property.</span></span> <span data-ttu-id="21d22-111">Utilisez le [opérateur d’accès au membre `.` ](../language-reference/operators/member-access-operators.md#member-access-operator-) pour accéder à la valeur qu’elle stocke.</span><span class="sxs-lookup"><span data-stu-id="21d22-111">Use the [member access operator `.`](../language-reference/operators/member-access-operators.md#member-access-operator-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
