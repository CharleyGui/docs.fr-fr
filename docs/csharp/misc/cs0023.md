# <a name="compiler-error-cs0023"></a>Erreur du compilateur CS0023
Impossible d’appliquer l’opérateur 'opérateur' à un opérande de type 'type'  
  
 L’utilisateur a tenté d’appliquer un opérateur à une variable dont le type n’est pas compatible avec l’opérateur. Pour plus d’informations, consultez [types](../programming-guide/types/index.md) et [opérateurs C#](../language-reference/operators/index.md).  
  
 L’exemple suivant génère l’erreur CS0023 :  
  
```csharp  
// CS0023.cs  
namespace x  
{  
   public class a  
   {  
      public static void Main()  
      {  
         string s = "hello";  
         s = -s;   // CS0023, minus operator not allowed on strings  
      }  
   }  
}  
```
