# <a name="compiler-error-cs0170"></a><span data-ttu-id="04b71-102">Erreur du compilateur CS0170</span><span class="sxs-lookup"><span data-stu-id="04b71-102">Compiler Error CS0170</span></span>
<span data-ttu-id="04b71-103">Utilisation d’un champ potentiellement non assigné 'field'</span><span class="sxs-lookup"><span data-stu-id="04b71-103">Use of possibly unassigned field 'field'</span></span>  
  
 <span data-ttu-id="04b71-104">Un champ dans une structure a été utilisé sans être au préalable initialisé.</span><span class="sxs-lookup"><span data-stu-id="04b71-104">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="04b71-105">Pour résoudre ce problème, déterminez d’abord le champ qui a été initialisé, puis initialisez-le avant d’essayer d’y accéder.</span><span class="sxs-lookup"><span data-stu-id="04b71-105">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="04b71-106">Pour plus d’informations sur l’initialisation des structs, consultez [structures](../programming-guide/classes-and-structs/structs.md) et [utilisation](../programming-guide/classes-and-structs/using-structs.md)de structs.</span><span class="sxs-lookup"><span data-stu-id="04b71-106">For more information about initializing structs, see [Structs](../programming-guide/classes-and-structs/structs.md) and [Using Structs](../programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
 <span data-ttu-id="04b71-107">L’exemple suivant génère l’erreur CS0170 :</span><span class="sxs-lookup"><span data-stu-id="04b71-107">The following sample generates CS0170:</span></span>  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
