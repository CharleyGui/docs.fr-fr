# <a name="compiler-error-cs0170"></a><span data-ttu-id="48e64-103">Erreur du compilateur CS0170</span><span class="sxs-lookup"><span data-stu-id="48e64-103">Compiler Error CS0170</span></span>
<span data-ttu-id="48e64-104">Utilisation d’un champ potentiellement non assigné 'field'</span><span class="sxs-lookup"><span data-stu-id="48e64-104">Use of possibly unassigned field 'field'</span></span>  
  
 <span data-ttu-id="48e64-105">Un champ dans une structure a été utilisé sans être au préalable initialisé.</span><span class="sxs-lookup"><span data-stu-id="48e64-105">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="48e64-106">Pour résoudre ce problème, déterminez d’abord le champ qui a été initialisé, puis initialisez-le avant d’essayer d’y accéder.</span><span class="sxs-lookup"><span data-stu-id="48e64-106">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="48e64-107">Pour plus d’informations sur l’initialisation des structs, consultez [types structure](../language-reference/builtin-types/struct.md).</span><span class="sxs-lookup"><span data-stu-id="48e64-107">For more information about initializing structs, see [Structure types](../language-reference/builtin-types/struct.md).</span></span>
  
 <span data-ttu-id="48e64-108">L’exemple suivant génère l’erreur CS0170 :</span><span class="sxs-lookup"><span data-stu-id="48e64-108">The following sample generates CS0170:</span></span>  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because
                                         //e.i was never assigned  
   }  
}  
```
