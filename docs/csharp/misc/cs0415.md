# <a name="compiler-error-cs0415"></a><span data-ttu-id="48624-103">Erreur du compilateur CS0415</span><span class="sxs-lookup"><span data-stu-id="48624-103">Compiler Error CS0415</span></span>

<span data-ttu-id="48624-104">L’attribut 'IndexerName' est valide uniquement sur un indexeur qui n’est pas une déclaration de membre d’interface explicite</span><span class="sxs-lookup"><span data-stu-id="48624-104">The 'IndexerName' attribute is valid only on an indexer that is not an explicit interface member declaration</span></span>  
  
 <span data-ttu-id="48624-105">Cette erreur se produit si vous utilisez un attribut IndexerName sur un indexeur correspondant à une implémentation explicite d’une interface.</span><span class="sxs-lookup"><span data-stu-id="48624-105">This error occurs if you use an IndexerName attribute on an indexer that was an explicit implementation of an interface.</span></span> <span data-ttu-id="48624-106">Cette erreur peut être évitée grâce à la suppression du nom d’interface de la déclaration de l’indexeur, si possible.</span><span class="sxs-lookup"><span data-stu-id="48624-106">This error may be avoided by removing the interface name from the declaration of the indexer, if possible.</span></span> <span data-ttu-id="48624-107">Pour plus d’informations, consultez [IndexerNameAttribute, classe](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="48624-107">For more information, see the [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="48624-108">L’exemple suivant génère l’erreur CS0415 :</span><span class="sxs-lookup"><span data-stu-id="48624-108">The following sample generates CS0415:</span></span>  
  
```csharp
// CS0415.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public interface IA  
{  
    int this[int index]  
    {  
        get;  
        set;  
    }  
}  
  
public class A : IA  
{  
    [IndexerName("Item")]  // CS0415  
    int IA.this[int index]  
    // Try this line instead:  
    // public int this[int index]  
    {  
        get { return 0; }  
        set { }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
