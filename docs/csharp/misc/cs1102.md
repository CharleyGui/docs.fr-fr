 <span data-ttu-id="e83a7-104">Quand le mot-clé `this` modifie le premier paramètre d’une méthode statique, il signale au compilateur que la méthode est une méthode d’extension.</span><span class="sxs-lookup"><span data-stu-id="e83a7-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="e83a7-105">Aucun autre modificateur n’est nécessaire ni autorisé sur le premier paramètre d’une méthode d’extension.</span><span class="sxs-lookup"><span data-stu-id="e83a7-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>

## <a name="example"></a><span data-ttu-id="e83a7-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="e83a7-108">Example</span></span>  
 <span data-ttu-id="e83a7-109">L’exemple suivant génère l’erreur CS1102 :</span><span class="sxs-lookup"><span data-stu-id="e83a7-109">The following example generates CS1102:</span></span>  
  
```csharp  
// cs1102.cs  
// Compile with: /target:library.  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(this out int i) {} // CS1102  
  
    //Single type parameter  
        public static void Test<T>(this out T t) {}// CS1102  
  
    //Multiple type parameters  
        public static void Test<T,U,V>(this out U u) {}// CS1102  
}  
```
