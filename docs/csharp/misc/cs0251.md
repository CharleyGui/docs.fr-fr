# <a name="compiler-warning-level-2-cs0251"></a><span data-ttu-id="995ce-103">Avertissement du compilateur (niveau 2) CS0251</span><span class="sxs-lookup"><span data-stu-id="995ce-103">Compiler Warning (level 2) CS0251</span></span>
<span data-ttu-id="995ce-104">Indexation d’un tableau avec un index négatif (les index de tableau commencent toujours à zéro)</span><span class="sxs-lookup"><span data-stu-id="995ce-104">Indexing an array with a negative index (array indices always start at zero)</span></span>  
  
 <span data-ttu-id="995ce-105">N’utilisez pas un nombre négatif pour indexer dans un tableau.</span><span class="sxs-lookup"><span data-stu-id="995ce-105">Do not use a negative number to index into an array.</span></span>  
  
 <span data-ttu-id="995ce-106">L’exemple suivant génère l’avertissement CS0251 :</span><span class="sxs-lookup"><span data-stu-id="995ce-106">The following sample generates CS0251:</span></span>  
  
```csharp  
// CS0251.cs  
// compile with: /W:2  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[] {1,2,3};
      try  
      {  
         myarray[-1]++;   // CS0251  
         // try the following line instead  
         // myarray[1]++;  
      }  
      catch (System.IndexOutOfRangeException e)  
      {  
         System.Console.WriteLine("{0}", e);  
      }  
   }  
}  
```
