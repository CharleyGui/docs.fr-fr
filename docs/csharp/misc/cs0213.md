## <a name="example"></a><span data-ttu-id="729be-106">Exemples</span><span class="sxs-lookup"><span data-stu-id="729be-106">Example</span></span>  
 <span data-ttu-id="729be-107">L’exemple suivant génère l’erreur CS0213 :</span><span class="sxs-lookup"><span data-stu-id="729be-107">The following sample generates CS0213.</span></span>  
  
```csharp  
// CS0213.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int i = 45;  
      fixed (int *j = &i) { }  // CS0213  
      // try the following line instead  
      // int* j = &i;  
  
      int[] a = new int[] {1,2,3};  
      fixed (int *b = a)  
      {  
         fixed (int *c = b) { }  // CS0213  
         // try the following line instead  
         // int *c = b;  
      }  
   }  
}  
```
