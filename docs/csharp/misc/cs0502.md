# <a name="compiler-error-cs0502"></a><span data-ttu-id="919b3-103">Erreur du compilateur CS0502</span><span class="sxs-lookup"><span data-stu-id="919b3-103">Compiler Error CS0502</span></span>

 <span data-ttu-id="919b3-105">Un membre de classe ne peut pas être à la fois [abstract](../language-reference/keywords/abstract.md) et [sealed](../language-reference/keywords/sealed.md).</span><span class="sxs-lookup"><span data-stu-id="919b3-105">A class member cannot be both [abstract](../language-reference/keywords/abstract.md) and [sealed](../language-reference/keywords/sealed.md).</span></span>  
  
 <span data-ttu-id="919b3-106">L’exemple suivant génère l’erreur CS0502 :</span><span class="sxs-lookup"><span data-stu-id="919b3-106">The following sample generates CS0502:</span></span>  
  
```csharp  
// CS0502.cs

{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
