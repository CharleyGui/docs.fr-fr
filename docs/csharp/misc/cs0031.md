# <a name="compiler-error-cs0031"></a><span data-ttu-id="5f983-103">Erreur du compilateur CS0031</span><span class="sxs-lookup"><span data-stu-id="5f983-103">Compiler Error CS0031</span></span>

<span data-ttu-id="5f983-104">Impossible de convertir la valeur de constante 'value' en 'type'.</span><span class="sxs-lookup"><span data-stu-id="5f983-104">Constant value 'value' cannot be converted to a 'type'.</span></span>

<span data-ttu-id="5f983-105">Une tentative d’assignation d’une valeur à une variable dont le type ne peut pas stocker la valeur a été effectuée.</span><span class="sxs-lookup"><span data-stu-id="5f983-105">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="5f983-106">Pour plus d’informations, consultez [Types](../programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="5f983-106">For more information, see [Types](../programming-guide/types/index.md).</span></span>

<span data-ttu-id="5f983-107">L’exemple suivant génère l’erreur CS0031 dans des contextes checked et unchecked :</span><span class="sxs-lookup"><span data-stu-id="5f983-107">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>

```csharp
// CS0031.cs
namespace CS0031
{
    public class Program
    {
        public static void Main()
        {
            int num = (int)2147483648M; //CS0031
            // Try using a larger numeric type instead.
            // long num = (long)2147483648M; //CS0031

            const decimal d = -10M; // Decimal literal
            unchecked
            {
                const byte b = (byte)d; // CS0031
                // For small values try using a signed byte instead.
                // const sbyte b = (sbyte)d;
            }
        }
    }
}
```
