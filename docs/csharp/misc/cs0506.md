# <a name="compiler-error-cs0506"></a><span data-ttu-id="a0352-103">Erreur du compilateur CS0506</span><span class="sxs-lookup"><span data-stu-id="a0352-103">Compiler Error CS0506</span></span>
<span data-ttu-id="a0352-104">'fonction1' : impossible de substituer le membre hérité 'fonction2', car il n’est pas marqué comme virtual, abstract ou override</span><span class="sxs-lookup"><span data-stu-id="a0352-104">'function1' : cannot override inherited member 'function2' because it is not marked "virtual", "abstract", or "override"</span></span>  
  
 <span data-ttu-id="a0352-105">Une méthode qui a été substituée n’était pas explicitement marquée comme **virtual**, **abstract**ou `override`.</span><span class="sxs-lookup"><span data-stu-id="a0352-105">A method was overridden that was not explicitly marked as **virtual**, **abstract**, or `override`.</span></span>  
  
 <span data-ttu-id="a0352-106">L’exemple suivant génère l’erreur CS0506 :</span><span class="sxs-lookup"><span data-stu-id="a0352-106">The following sample generates CS0506:</span></span>  
  
```csharp  
// CS0506.cs  
namespace MyNameSpace  
{  
   abstract public class ClassX  
   {  
      public int i = 0;  
  
      public int f()  
      {  
         return 0;  
      }  
      // Try the following definition for f() instead:  
      // abstract public int f();  
   }  
  
   public class ClassY : ClassX  
   {  
      public override int f()   // CS0506  
      {  
         return 0;  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
