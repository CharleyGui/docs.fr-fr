# <a name="compiler-error-cs0553"></a><span data-ttu-id="a03ac-103">Erreur du compilateur CS0553</span><span class="sxs-lookup"><span data-stu-id="a03ac-103">Compiler Error CS0553</span></span>
<span data-ttu-id="a03ac-104">'routine de conversion' : conversion définie par l’utilisateur vers/de la classe de base</span><span class="sxs-lookup"><span data-stu-id="a03ac-104">'conversion routine' : user defined conversion to/from base class</span></span>  
  
 <span data-ttu-id="a03ac-105">Les conversions définies par l’utilisateur en valeurs de classe de base ne sont pas autorisées. Ce type d’opérateur n’est pas nécessaire.</span><span class="sxs-lookup"><span data-stu-id="a03ac-105">User-defined conversions to values of a base class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="a03ac-106">L’exemple suivant génère l’erreur CS0553 :</span><span class="sxs-lookup"><span data-stu-id="a03ac-106">The following sample generates CS0553:</span></span>  
  
```csharp  
// CS0553.cs  
namespace x  
{  
   public class ii  
   {  
   }  
  
   public class a : ii  
   {  
      // delete the conversion routine to resolve CS0553  
      public static implicit operator ii(a aa) // CS0553  
      {  
         return new ii();  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
