# <a name="compiler-error-cs0466"></a><span data-ttu-id="4ac7c-103">Erreur du compilateur CS0466</span><span class="sxs-lookup"><span data-stu-id="4ac7c-103">Compiler Error CS0466</span></span>
<span data-ttu-id="4ac7c-104">'méthode1' ne doit pas avoir de paramètre params, car 'méthode2' n’en possède pas</span><span class="sxs-lookup"><span data-stu-id="4ac7c-104">'method1' should not have a params parameter since 'method2' does not</span></span>  
  
 <span data-ttu-id="4ac7c-105">Vous ne pouvez pas utiliser le paramètre `params` sur un membre de classe si l’interface implémentée ne l’utilise pas.</span><span class="sxs-lookup"><span data-stu-id="4ac7c-105">You cannot use `params` parameter on a class member if the implemented interface doesn't use it.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4ac7c-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="4ac7c-106">Example</span></span>  
 <span data-ttu-id="4ac7c-107">L’exemple suivant génère l’erreur CS0466 :</span><span class="sxs-lookup"><span data-stu-id="4ac7c-107">The following sample generates CS0466.</span></span>  
  
```csharp  
// CS0466.cs  
interface I  
{  
   void F1(params int[] a);  
   void F2(int[] a);  
}  
  
class C : I  
{  
   void I.F1(params int[] a) {}  
   void I.F2(params int[] a) {}   // CS0466  
   void I.F2(int[] a) {}   // OK  
  
   public static void Main()  
   {  
      I i = (I) new C();  
  
      i.F1(new int[] {1, 2} );  
      i.F2(new int[] {1, 2} );  
   }  
}  
```
