# <a name="compiler-error-cs1508"></a>Erreur du compilateur CS1508
L’identificateur de ressource 'identifier' a déjà été utilisé dans cet assembly  
  
 Dans une compilation, le même identificateur (***identificateur***) a été passé à deux options de compilateur [/Resource](../language-reference/compiler-options/resource-compiler-option.md) ou [-linkresource](../language-reference/compiler-options/linkresource-compiler-option.md) ou plus.  
  
 Par exemple, les options suivantes génèrent l’erreur CS1508 :  
  
```console  
/resource:anyfile.bmp,DuplicatIdent /linkresource:a.bmp,DuplicatIdent  
```
