# <a name="compiler-warning-level-3-cs0660"></a><span data-ttu-id="0b4aa-103">Avertissement du compilateur (niveau 3) CS0660</span><span class="sxs-lookup"><span data-stu-id="0b4aa-103">Compiler Warning (level 3) CS0660</span></span>

<span data-ttu-id="0b4aa-104">'class' définit l’opérateur == ou l’opérateur != mais ne se substitue pas à Object.Equals(object o)</span><span class="sxs-lookup"><span data-stu-id="0b4aa-104">'class' defines operator == or operator != but does not override Object.Equals(object o)</span></span>  
  
<span data-ttu-id="0b4aa-105">Le compilateur a détecté l’opérateur d’égalité ou d’inégalité défini par l’utilisateur, mais pas de substitution pour la <xref:System.Object.Equals%2A?displayProperty=nameWithType> méthode.</span><span class="sxs-lookup"><span data-stu-id="0b4aa-105">The compiler detected the user-defined equality or inequality operator, but no override for the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="0b4aa-106">Un opérateur d’égalité ou d’inégalité défini par l’utilisateur implique que vous souhaitez également substituer la <xref:System.Object.Equals%2A> méthode.</span><span class="sxs-lookup"><span data-stu-id="0b4aa-106">A user-defined equality or inequality operator implies that you also want to override the <xref:System.Object.Equals%2A> method.</span></span> <span data-ttu-id="0b4aa-107">Pour plus d’informations, consultez [comment définir l’égalité des valeurs pour un type](../programming-guide/statements-expressions-operators/how-to-define-value-equality-for-a-type.md).</span><span class="sxs-lookup"><span data-stu-id="0b4aa-107">For more information, see [How to define value equality for a type](../programming-guide/statements-expressions-operators/how-to-define-value-equality-for-a-type.md).</span></span>
  
<span data-ttu-id="0b4aa-108">L’exemple suivant génère l’avertissement CS0660 :</span><span class="sxs-lookup"><span data-stu-id="0b4aa-108">The following sample generates CS0660:</span></span>  

```csharp

{  
   public static bool operator == (object o, Test t)  
   {  
      return true;  
   }  
  
   // uncomment the Equals function to resolve  
   // public override bool Equals(object o)  
   // {  
   //    return true;  
   // }  
  
   public override int GetHashCode()  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
