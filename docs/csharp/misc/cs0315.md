## <a name="example"></a><span data-ttu-id="db85b-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="db85b-108">Example</span></span>  
 <span data-ttu-id="db85b-109">L’exemple suivant génère l’erreur CS0315 :</span><span class="sxs-lookup"><span data-stu-id="db85b-109">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{         
}

{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315

    }  
}
```  
