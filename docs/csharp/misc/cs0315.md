## <a name="example"></a><span data-ttu-id="8dfdd-109">Exemple</span><span class="sxs-lookup"><span data-stu-id="8dfdd-109">Example</span></span>  

 <span data-ttu-id="8dfdd-110">L’exemple suivant génère l’erreur CS0315 :</span><span class="sxs-lookup"><span data-stu-id="8dfdd-110">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }

public class Gen<T> where T : ClassConstraint  
{
}

    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315

    }
```  
