 <span data-ttu-id="a6a98-107">L’exemple suivant génère l’avertissement CS0176 :</span><span class="sxs-lookup"><span data-stu-id="a6a98-107">The following sample generates CS0176:</span></span>  
  
```csharp

    public static int num;  
}

{  
    public static void Main()

        MyClass2 mc2 = new MyClass2();  
        int i = mc2.num;   // CS0176

        // int i = MyClass2.num;  
    }  
}  
```
