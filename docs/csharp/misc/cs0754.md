# <a name="compiler-error-cs0754"></a><span data-ttu-id="54652-102">Erreur du compilateur CS0754</span><span class="sxs-lookup"><span data-stu-id="54652-102">Compiler Error CS0754</span></span>
<span data-ttu-id="54652-103">Une méthode partielle ne peut pas implémenter explicitement une méthode d’interface.</span><span class="sxs-lookup"><span data-stu-id="54652-103">A partial method may not explicitly implement an interface method.</span></span>  
  
 <span data-ttu-id="54652-104">Une méthode partielle ne peut pas être déclarée en tant qu’implémentation explicite d’une méthode définie dans une interface.</span><span class="sxs-lookup"><span data-stu-id="54652-104">A partial method cannot be declared as an explicit implementation of a method defined in an interface.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="54652-105">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="54652-105">To correct this error</span></span>  
  
1. <span data-ttu-id="54652-106">Supprimez la qualification d’interface explicite de la déclaration de méthode.</span><span class="sxs-lookup"><span data-stu-id="54652-106">Remove the explicit interface qualification from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="54652-107">Exemples</span><span class="sxs-lookup"><span data-stu-id="54652-107">Example</span></span>  
 <span data-ttu-id="54652-108">L’exemple suivant génère l’erreur CS0754 :</span><span class="sxs-lookup"><span data-stu-id="54652-108">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
