# <a name="compiler-error-cs0754"></a><span data-ttu-id="45dbb-103">Erreur du compilateur CS0754</span><span class="sxs-lookup"><span data-stu-id="45dbb-103">Compiler Error CS0754</span></span>

## <a name="example"></a><span data-ttu-id="45dbb-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="45dbb-108">Example</span></span>  

 <span data-ttu-id="45dbb-109">L’exemple suivant génère l’erreur CS0754 :</span><span class="sxs-lookup"><span data-stu-id="45dbb-109">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
