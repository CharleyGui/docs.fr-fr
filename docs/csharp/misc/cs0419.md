# <a name="compiler-warning-level-3-cs0419"></a><span data-ttu-id="921a2-103">Avertissement du compilateur (niveau 3) CS0419</span><span class="sxs-lookup"><span data-stu-id="921a2-103">Compiler Warning (level 3) CS0419</span></span>

<span data-ttu-id="921a2-104">Référence ambiguë dans l’attribut cref : 'nom_méthode1'.</span><span class="sxs-lookup"><span data-stu-id="921a2-104">Ambiguous reference in cref attribute: 'Method Name1'.</span></span>  <span data-ttu-id="921a2-105">'nom_méthode2', pris par défaut, mais peut aussi correspondre à d’autres surcharges, notamment 'nom_méthode'.</span><span class="sxs-lookup"><span data-stu-id="921a2-105">Assuming 'Method Name2', but could have also matched other overloads including 'Method Name3'.</span></span>  
  
 <span data-ttu-id="921a2-106">Dans un commentaire de documentation XML du code, une référence n’a pas pu être résolue.</span><span class="sxs-lookup"><span data-stu-id="921a2-106">In an XML documentation comment in the code, a reference could not be resolved.</span></span> <span data-ttu-id="921a2-107">Cela peut se produire si la méthode est surchargée ou si deux identificateurs portent le même nom.</span><span class="sxs-lookup"><span data-stu-id="921a2-107">This could occur if the method is overloaded, or if two different identifiers with the same name are found.</span></span> <span data-ttu-id="921a2-108">Pour résoudre l’avertissement, utilisez un nom qualifié pour désambiguïser la référence ou mettez la surcharge spécifique entre parenthèses.</span><span class="sxs-lookup"><span data-stu-id="921a2-108">To resolve the warning, use a qualified name to disambiguate the reference, or include the specific overload in parentheses.</span></span>  
  
 <span data-ttu-id="921a2-109">L’exemple suivant génère l’erreur CS0419 :</span><span class="sxs-lookup"><span data-stu-id="921a2-109">The following sample generates CS0419.</span></span>  
  
```csharp  
// cs0419.cs  
// compile with: /doc:x.xml /W:3  
interface I  
{  
   /// text for F(void)  
   void F();  
   /// text for F(int)  
   void F(int i);  
}  
/// text for class MyClass  
public class MyClass  
{  
   /// <see cref="I.F"/>  
   public static void MyMethod(int i)  
   {  
   }  
/* Try this instead:  
   /// <see cref="I.F(int)"/>  
   public static void MyMethod(int i)  
   {  
   }  
*/  
   /// text for Main  
   public static void Main ()  
   {  
   }  
}  
```
