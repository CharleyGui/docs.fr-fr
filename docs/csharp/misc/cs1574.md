<span data-ttu-id="6247b-104">Le commentaire XML sur 'construction' comporte une erreur de syntaxe au niveau de l’attribut cref 'nom'</span><span class="sxs-lookup"><span data-stu-id="6247b-104">XML comment on 'construct' has syntactically incorrect cref attribute 'name'</span></span>  
  
 <span data-ttu-id="6247b-105">Chaîne transmise à une balise CREF, par exemple dans une \<exception> balise, référencée par un membre qui n’est pas disponible dans l’environnement de génération actuel.</span><span class="sxs-lookup"><span data-stu-id="6247b-105">A string passed to a cref tag, for example, within an \<exception> tag, referred to a member that is not available within the current build environment.</span></span> <span data-ttu-id="6247b-106">La chaîne que vous passez à une balise cref doit correspondre au nom d’un membre ou d’un champ ayant une syntaxe correcte.</span><span class="sxs-lookup"><span data-stu-id="6247b-106">The string that you pass to a cref tag must be the syntactically correct name of a member or field.</span></span>  
  
 <span data-ttu-id="6247b-107">Pour plus d’informations, consultez [Balises recommandées pour commentaires de documentation](../programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).</span><span class="sxs-lookup"><span data-stu-id="6247b-107">For more information, see [Recommended Tags for Documentation Comments](../programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).</span></span>  
  
 <span data-ttu-id="6247b-108">L’exemple suivant génère l’erreur CS1574 :</span><span class="sxs-lookup"><span data-stu-id="6247b-108">The following sample generates CS1574:</span></span>  
  
```csharp

// compile with: /W:1 /doc:x.xml  
using System;  
  
/// <exception cref="System.Console.WriteLin">An exception class.</exception>   // CS1574

// /// <exception cref="System.Console.WriteLine">An exception class.</exception>

{  
}  
  
class TestClass  
{  
   public static void Main()  
   {  
      try  
      {  
      }  
      catch(EClass)  
      {  
      }  
   }  
}  
```
