# <a name="compiler-error-cs0668"></a><span data-ttu-id="336a8-103">Erreur du compilateur CS0668</span><span class="sxs-lookup"><span data-stu-id="336a8-103">Compiler Error CS0668</span></span>
<span data-ttu-id="336a8-104">Deux indexeurs ont des noms différents ; l'attribut IndexerName doit être utilisé avec le même nom sur chaque indexeur d'un type</span><span class="sxs-lookup"><span data-stu-id="336a8-104">Two indexers have different names; the IndexerName attribute must be used with the same name on every indexer within a type</span></span>  
  
 <span data-ttu-id="336a8-105">Les valeurs passées à l’attribut **IndexerName** doivent être les mêmes pour tous les indexeurs d’un type.</span><span class="sxs-lookup"><span data-stu-id="336a8-105">The values passed to the **IndexerName** attribute must be the same for all indexers in a type.</span></span> <span data-ttu-id="336a8-106">Pour plus d’informations sur l’attribut **IndexerName** , consultez [IndexerNameAttribute, classe](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="336a8-106">For more information on the **IndexerName** attribute, see [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="336a8-107">L’exemple suivant génère l’erreur CS0668 :</span><span class="sxs-lookup"><span data-stu-id="336a8-107">The following sample generates CS0668:</span></span>  
  
```csharp
// CS0668.cs  
using System;  
using System.Runtime.CompilerServices;  
  
class IndexerClass  
{  
   [IndexerName("IName1")]  
   public int this [int index]   // indexer declaration  
   {  
      get  
      {  
         return index;  
      }  
      set  
      {  
      }  
   }  
  
   [IndexerName("IName2")]  
   public int this [string s]    // CS0668, change IName2 to IName1  
   {  
      get  
      {  
         return int.Parse(s);  
      }  
      set  
      {  
      }  
   }  
  
   void Main()  
   {  
   }  
}  
```
