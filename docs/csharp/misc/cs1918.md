# <a name="compiler-error-cs1918"></a><span data-ttu-id="4656e-102">Erreur du compilateur CS1918</span><span class="sxs-lookup"><span data-stu-id="4656e-102">Compiler Error CS1918</span></span>

<span data-ttu-id="4656e-103">Les membres de la propriété 'name' de type 'type' ne peuvent pas être assignés avec un initialiseur d’objet car il s’agit d’un type valeur.</span><span class="sxs-lookup"><span data-stu-id="4656e-103">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="4656e-104">Cette erreur se produit quand vous essayez d’utiliser un initialiseur d’objet pour initialiser les propriétés d’un type struct qui est lui-même une propriété de la classe en cours d’initialisation.</span><span class="sxs-lookup"><span data-stu-id="4656e-104">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4656e-105">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="4656e-105">To correct this error</span></span>  
  
1. <span data-ttu-id="4656e-106">Si vous devez initialiser complètement les champs de la propriété dans l’initialiseur d’objet, remplacez le struct par un type de classe.</span><span class="sxs-lookup"><span data-stu-id="4656e-106">If you must fully initialize the fields of the property in the object initializer, change the struct to a class type.</span></span> <span data-ttu-id="4656e-107">Sinon, initialisez les membres du struct dans un appel de méthode distinct après avoir créé l’objet à l’aide de l’initialiseur d’objet.</span><span class="sxs-lookup"><span data-stu-id="4656e-107">Otherwise, initialize the struct members in a separate method call after you create the object by using the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4656e-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="4656e-108">Example</span></span>  

 <span data-ttu-id="4656e-109">L’exemple suivant génère l’erreur CS1918 :</span><span class="sxs-lookup"><span data-stu-id="4656e-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
