# <a name="compiler-warning-level-1-cs0197"></a>Avertissement du compilateur (niveau 1) CS0197
Le passage de 'argument' comme ref ou out ou la prise de son adresse peut provoquer une exception runtime, car il s’agit d’un champ d’une classe marshalée par référence  
  
 Toute classe qui dérive directement ou indirectement de <xref:System.MarshalByRefObject> est une classe marshalée par référence. Une telle classe peut être marshalée par référence au-delà des limites du processus et de l’ordinateur. Par conséquent, les instances de cette classe peut être des proxys vers des objets distants. Vous ne pouvez pas passer un champ d’un objet proxy en tant que [ref](../language-reference/keywords/ref.md) ou [out](../language-reference/keywords/out-parameter-modifier.md). Ainsi, vous ne pouvez pas passer les champs d’une telle classe en tant que `ref` ou `out` , sauf si l’instance est [This](../language-reference/keywords/this.md), qui ne peut pas être un objet proxy.  
  
## <a name="example"></a>Exemple  
 L’exemple suivant génère l’avertissement CS0197.  
  
```csharp  
// CS0197.cs  
// compile with: /W:1  
class X : System.MarshalByRefObject  
{  
   public int i;  
}  
  
class M  
{  
   public int i;  
   static void AddSeventeen(ref int i)  
   {  
      i += 17;  
   }  
  
   static void Main()  
   {  
      X x = new X();  
      x.i = 12;  
      AddSeventeen(ref x.i);   // CS0197  
  
      // OK  
      M m = new M();  
      m.i = 12;  
      AddSeventeen(ref m.i);  
   }  
}  
```
