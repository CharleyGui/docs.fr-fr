# <a name="compiler-warning-level-1-cs0197"></a><span data-ttu-id="9882e-103">Avertissement du compilateur (niveau 1) CS0197</span><span class="sxs-lookup"><span data-stu-id="9882e-103">Compiler Warning (level 1) CS0197</span></span>

<span data-ttu-id="9882e-104">Le passage de 'argument' comme ref ou out ou la prise de son adresse peut provoquer une exception runtime, car il s’agit d’un champ d’une classe marshalée par référence</span><span class="sxs-lookup"><span data-stu-id="9882e-104">Passing 'argument' as ref or out or taking its address may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="9882e-105">Toute classe qui dérive directement ou indirectement de <xref:System.MarshalByRefObject> est une classe marshalée par référence.</span><span class="sxs-lookup"><span data-stu-id="9882e-105">Any class that derives, directly or indirectly, from <xref:System.MarshalByRefObject> is a marshal-by-reference class.</span></span> <span data-ttu-id="9882e-106">Une telle classe peut être marshalée par référence au-delà des limites du processus et de l’ordinateur.</span><span class="sxs-lookup"><span data-stu-id="9882e-106">Such a class can be marshaled by reference across process and machine boundaries.</span></span> <span data-ttu-id="9882e-107">Par conséquent, les instances de cette classe peut être des proxys vers des objets distants.</span><span class="sxs-lookup"><span data-stu-id="9882e-107">Thus, instances of this class could be proxies to remote objects.</span></span> <span data-ttu-id="9882e-108">Vous ne pouvez pas passer un champ d’un objet proxy en tant que [ref](../language-reference/keywords/ref.md) ou [out](../language-reference/keywords/out-parameter-modifier.md). So, you cannot pass fields of such a class as `ref` or `out`, unless the instance is [this](../language-reference/keywords/this.md), which can not be a proxy object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9882e-109">Exemple</span><span class="sxs-lookup"><span data-stu-id="9882e-109">Example</span></span>  

 <span data-ttu-id="9882e-110">L’exemple suivant génère l’avertissement CS0197.</span><span class="sxs-lookup"><span data-stu-id="9882e-110">The following sample generates CS0197.</span></span>  
  
```csharp  
// CS0197.cs  
// compile with: /W:1  
class X : System.MarshalByRefObject  
{  
   public int i;  
}  
  
class M  
{  
   public int i;  
   static void AddSeventeen(ref int i)  
   {  
      i += 17;  
   }  
  
   static void Main()  
   {  
      X x = new X();  
      x.i = 12;  
      AddSeventeen(ref x.i);   // CS0197  
  
      // OK  
      M m = new M();  
      m.i = 12;  
      AddSeventeen(ref m.i);  
   }  
}  
```
