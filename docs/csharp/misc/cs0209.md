# <a name="compiler-error-cs0209"></a>Erreur du compilateur CS0209

Le type des variables locales déclaré dans une instruction fixed doit être un type pointeur  
  
 La variable que vous déclarez dans une [instruction fixed](../language-reference/keywords/fixed-statement.md) doit être un pointeur. Pour plus d’informations, consultez [Pointeurs et code unsafe](../programming-guide/unsafe-code-pointers/index.md).  
  
 L’exemple suivant génère l’erreur CS0209 :  
  
```csharp  
// CS0209.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0209  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```
