# <a name="compiler-error-cs2022"></a>Erreur du compilateur CS2022
Les options '/out' et '/target' doivent apparaître avant les noms de fichiers sources  
  
 Les options du compilateur [-out (définir le nom du fichier de sortie)](../language-reference/compiler-options/out-compiler-option.md) et [-cible (spécifier le format du fichier de sortie)](../language-reference/compiler-options/target-compiler-option.md) , quand elles sont spécifiées sur la ligne de commande, doivent précéder les fichiers de code source.
