# <a name="compiler-error-cs0196"></a>Erreur du compilateur CS0196
Un pointeur ne doit être indexé que par une seule valeur  
  
 Un pointeur ne peut pas avoir plusieurs index. Pour plus d’informations, consultez [types](../programming-guide/unsafe-code-pointers/pointer-types.md) de pointeurs  
  
 L’exemple suivant génère l’erreur CS0196 :  
  
```csharp  
// CS0196.cs  
public class MyClass  
{  
   public static void Main ()  
   {  
      int *i = null;  
      int j = 0;  
      j = i[1,2];   // CS0196  
      // try the following line instead  
      // j = i[1];  
   }  
}  
```
