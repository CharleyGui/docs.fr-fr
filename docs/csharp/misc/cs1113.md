## <a name="example"></a><span data-ttu-id="391ee-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="391ee-110">Example</span></span>  

 <span data-ttu-id="391ee-111">L’exemple suivant génère l’erreur CS1113 :</span><span class="sxs-lookup"><span data-stu-id="391ee-111">The following example generates CS1113:</span></span>

```csharp  
public static class Extensions  
{  
    public static S ExtMethod(this S s)  
    {  
        return s;  
    }

{  
}

{  
    static int Main()  
    {

    }

```
