# <a name="compiler-error-cs0838"></a><span data-ttu-id="732b4-103">Erreur du compilateur CS0838</span><span class="sxs-lookup"><span data-stu-id="732b4-103">Compiler Error CS0838</span></span>

<span data-ttu-id="732b4-104">Une arborescence de l’expression ne peut pas contenir un initialiseur de tableau à plusieurs dimensions.</span><span class="sxs-lookup"><span data-stu-id="732b4-104">An expression tree may not contain a multidimensional array initializer.</span></span>  
  
 <span data-ttu-id="732b4-105">Vous ne pouvez pas initialiser des tableaux multidimensionnels dans des arborescences d’expression à l’aide d’un initialiseur de tableau.</span><span class="sxs-lookup"><span data-stu-id="732b4-105">Multidimensional arrays in expression trees cannot be initialized by using an array initializer.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="732b4-106">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="732b4-106">To correct this error</span></span>  
  
1. <span data-ttu-id="732b4-107">Créez et initialisez le tableau avant de créer l’arborescence d’expression.</span><span class="sxs-lookup"><span data-stu-id="732b4-107">Create and initialize the array before creating the expression tree.</span></span>  
  
## <a name="example"></a><span data-ttu-id="732b4-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="732b4-108">Example</span></span>  

 <span data-ttu-id="732b4-109">L’exemple suivant génère l’erreur CS0838 :</span><span class="sxs-lookup"><span data-stu-id="732b4-109">The following example generates CS0838:</span></span>  
  
```csharp  
// cs0838.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace TestNamespace  
{  
    class Test  
    {  
        static int Main()  
        {  
  
            Expression<Func<int[,]>> expr =  
                () => new int[2, 2] { { 1, 2 }, { 3, 4 } }; // CS0838  
  
            // try the following 2 lines instead  
            int[,] nums = new int[2, 2] { { 1, 2 }, { 3, 4 } };  
            Expression<Func<int[,]>> expr2 = () => nums;
  
            return 1;  
        }  
    }  
}  
```
