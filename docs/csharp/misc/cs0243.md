# <a name="compiler-error-cs0243"></a><span data-ttu-id="719ba-102">Erreur du compilateur CS0243</span><span class="sxs-lookup"><span data-stu-id="719ba-102">Compiler Error CS0243</span></span>

<span data-ttu-id="719ba-103">L’attribut Conditional n’est pas valide sur 'méthode', car il s’agit d’une méthode override</span><span class="sxs-lookup"><span data-stu-id="719ba-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
 <span data-ttu-id="719ba-104">L’attribut [Conditional](../programming-guide/concepts/attributes/common-attributes.md#Conditional) n’est pas autorisé sur une méthode marquée avec le mot clé [override](../language-reference/keywords/override.md) .</span><span class="sxs-lookup"><span data-stu-id="719ba-104">The [Conditional](../programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="719ba-105">Pour plus d’informations, consultez [savoir quand utiliser les mots clés override et New](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="719ba-105">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="719ba-106">Le compilateur ne lie jamais à des méthodes override ; il lie uniquement à la méthode de base et le Common Language Runtime appelle la substitution comme il convient.</span><span class="sxs-lookup"><span data-stu-id="719ba-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="719ba-107">L’exemple suivant génère l’erreur CS0243 :</span><span class="sxs-lookup"><span data-stu-id="719ba-107">The following sample generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
