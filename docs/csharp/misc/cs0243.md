# <a name="compiler-error-cs0243"></a><span data-ttu-id="816fc-103">Erreur du compilateur CS0243</span><span class="sxs-lookup"><span data-stu-id="816fc-103">Compiler error CS0243</span></span>

<span data-ttu-id="816fc-104">L’attribut Conditional n’est pas valide sur 'méthode', car il s’agit d’une méthode override</span><span class="sxs-lookup"><span data-stu-id="816fc-104">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="816fc-105">L' <xref:System.Diagnostics.ConditionalAttribute> attribut n’est pas autorisé sur une méthode marquée avec le mot clé [override](../language-reference/keywords/override.md) .</span><span class="sxs-lookup"><span data-stu-id="816fc-105">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="816fc-106">Pour plus d’informations, consultez [savoir quand utiliser les mots clés override et New](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="816fc-106">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="816fc-107">Le compilateur ne se lie jamais aux méthodes override.</span><span class="sxs-lookup"><span data-stu-id="816fc-107">The compiler never binds to override methods.</span></span> <span data-ttu-id="816fc-108">Il se lie uniquement à la méthode de base, et le common language runtime appelle la substitution, le cas échéant.</span><span class="sxs-lookup"><span data-stu-id="816fc-108">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="816fc-109">Le code suivant génère CS0243 :</span><span class="sxs-lookup"><span data-stu-id="816fc-109">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
