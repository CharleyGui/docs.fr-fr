# <a name="compiler-error-cs0243"></a><span data-ttu-id="2c639-102">Erreur de compilateur CS0243</span><span class="sxs-lookup"><span data-stu-id="2c639-102">Compiler error CS0243</span></span>

<span data-ttu-id="2c639-103">L’attribut Conditional n’est pas valide sur 'méthode', car il s’agit d’une méthode override</span><span class="sxs-lookup"><span data-stu-id="2c639-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="2c639-104">L’attribut <xref:System.Diagnostics.ConditionalAttribute> n’est pas autorisé sur une méthode qui est marquée avec le mot clé [de remplacement.](../language-reference/keywords/override.md)</span><span class="sxs-lookup"><span data-stu-id="2c639-104">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="2c639-105">Pour plus d’informations, voir [Savoir quand utiliser Override et nouveaux mots clés](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="2c639-105">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="2c639-106">Le compilateur ne se lie jamais aux méthodes de remplacement.</span><span class="sxs-lookup"><span data-stu-id="2c639-106">The compiler never binds to override methods.</span></span> <span data-ttu-id="2c639-107">Il ne se lie qu’à la méthode de base, et le langage courant exécute l’heure d’exécution appelle le remplacement, le cas échéant.</span><span class="sxs-lookup"><span data-stu-id="2c639-107">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="2c639-108">Le code suivant génère CS0243 :</span><span class="sxs-lookup"><span data-stu-id="2c639-108">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
