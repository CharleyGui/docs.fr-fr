# <a name="compiler-error-cs0664"></a><span data-ttu-id="15709-102">Erreur du compilateur CS0664</span><span class="sxs-lookup"><span data-stu-id="15709-102">Compiler Error CS0664</span></span>
<span data-ttu-id="15709-103">Impossible de convertir implicitement un littéral de type double en type 'type' ; utilisez un suffixe 'suffixe' pour créer un littéral de ce type</span><span class="sxs-lookup"><span data-stu-id="15709-103">Literal of type double cannot be implicitly converted to type 'type'; use an 'suffix' suffix to create a literal of this type</span></span>  
  
 <span data-ttu-id="15709-104">Une affectation n’a pas pu être effectuée ; utilisez un suffixe pour corriger l’instruction.</span><span class="sxs-lookup"><span data-stu-id="15709-104">An assignment could not be completed; use a suffix to correct the instruction.</span></span> <span data-ttu-id="15709-105">La documentation de chaque type identifie le suffixe correspondant au type.</span><span class="sxs-lookup"><span data-stu-id="15709-105">The documentation for each type identifies the corresponding suffix for the type.</span></span> <span data-ttu-id="15709-106">Pour plus d’informations sur les conversions, consultez [Cast et](../programming-guide/types/casting-and-type-conversions.md)conversions de types.</span><span class="sxs-lookup"><span data-stu-id="15709-106">For more information on conversions, see [Casting and Type Conversions](../programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="15709-107">L’exemple suivant génère l’erreur CS0664 :</span><span class="sxs-lookup"><span data-stu-id="15709-107">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```
