<span data-ttu-id="b2273-104">On a tenté d’attribuer une valeur à une [propriété,](../programming-guide/classes-and-structs/using-properties.md)mais la propriété n’a pas d’accesseur fixe ou l’affectation se trouvait à l’extérieur du constructeur.</span><span class="sxs-lookup"><span data-stu-id="b2273-104">An attempt was made to assign a value to a [property](../programming-guide/classes-and-structs/using-properties.md), but the property does not have a set accessor or the assignment was outside of the constructor.</span></span> <span data-ttu-id="b2273-105">Pour résoudre l’erreur, ajoutez un accesseur set.</span><span class="sxs-lookup"><span data-stu-id="b2273-105">Resolve the error by adding a set accessor.</span></span> <span data-ttu-id="b2273-106">Pour plus d’informations, voir [Comment déclarer et utiliser lire écrire des propriétés](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).</span><span class="sxs-lookup"><span data-stu-id="b2273-106">For more information, see [How to declare and use read write properties](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).</span></span>

## <a name="examples"></a><span data-ttu-id="b2273-107">Exemples</span><span class="sxs-lookup"><span data-stu-id="b2273-107">Examples</span></span>
<span data-ttu-id="b2273-108">L’exemple suivant génère l’erreur CS0200 :</span><span class="sxs-lookup"><span data-stu-id="b2273-108">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="b2273-109">L’échantillon suivant utilise [des propriétés auto-mises en œuvre,](../programming-guide/classes-and-structs/auto-implemented-properties.md) [des initialisateurs d’objets,](../programming-guide/classes-and-structs/object-and-collection-initializers.md)et génère toujours CS0200 :</span><span class="sxs-lookup"><span data-stu-id="b2273-109">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md), [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md), and still generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

<span data-ttu-id="b2273-110">L’affectation à une propriété ou à un indexeur qui est lue seulement peut être réalisée en ajoutant un accesseur défini ou en assignant à la propriété dans le constructeur de l’objet.</span><span class="sxs-lookup"><span data-stu-id="b2273-110">Assignment to a property or indexer 'property' that is read only, can be achieved through adding a set accessor or by assigning to the property in the object's constructor.</span></span>

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
