## <a name="examples"></a><span data-ttu-id="88f82-107">Exemples</span><span class="sxs-lookup"><span data-stu-id="88f82-107">Examples</span></span>

<span data-ttu-id="88f82-108">L’exemple suivant génère l’erreur CS0200 :</span><span class="sxs-lookup"><span data-stu-id="88f82-108">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="88f82-109">L’exemple suivant utilise des [Propriétés implémentées automatiquement](../programming-guide/classes-and-structs/auto-implemented-properties.md) et des [initialiseurs d’objets](../programming-guide/classes-and-structs/object-and-collection-initializers.md) , tout en générant des CS0200 :</span><span class="sxs-lookup"><span data-stu-id="88f82-109">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md) and [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md) and still generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

<span data-ttu-id="88f82-110">Pour assigner à une propriété ou à un indexeur’propriété’en lecture seule, ajoutez un accesseur set ou assignez la valeur dans le constructeur de l’objet.</span><span class="sxs-lookup"><span data-stu-id="88f82-110">To assign to a property or indexer 'property' that's read-only, add a set accessor or assign the value in the object's constructor.</span></span>

```csharp
    public Example()
    {

    }
}
```
