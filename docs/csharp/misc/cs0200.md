# <a name="compiler-error-cs0200"></a>Erreur du compilateur CS0200
Propriété ou indexeur «propriété» ne peut pas être attribué à - il est lu seulement

On a tenté d’attribuer une valeur à une [propriété,](../programming-guide/classes-and-structs/using-properties.md)mais la propriété n’a pas d’accesseur fixe ou l’affectation se trouvait à l’extérieur du constructeur. Pour résoudre l’erreur, ajoutez un accesseur set. Pour plus d’informations, voir [Comment déclarer et utiliser lire écrire des propriétés](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).

## <a name="examples"></a>Exemples
L’exemple suivant génère l’erreur CS0200 :

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

L’échantillon suivant utilise [des propriétés auto-mises en œuvre,](../programming-guide/classes-and-structs/auto-implemented-properties.md) [des initialisateurs d’objets,](../programming-guide/classes-and-structs/object-and-collection-initializers.md)et génère toujours CS0200 :

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

L’affectation à une propriété ou à un indexeur qui est lue seulement peut être réalisée en ajoutant un accesseur défini ou en assignant à la propriété dans le constructeur de l’objet.

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
