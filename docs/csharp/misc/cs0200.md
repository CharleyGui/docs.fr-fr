## <a name="examples"></a><span data-ttu-id="af9b7-108">Exemples</span><span class="sxs-lookup"><span data-stu-id="af9b7-108">Examples</span></span>

<span data-ttu-id="af9b7-109">L’exemple suivant génère l’erreur CS0200 :</span><span class="sxs-lookup"><span data-stu-id="af9b7-109">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="af9b7-110">L’exemple suivant utilise des [Propriétés implémentées automatiquement](../programming-guide/classes-and-structs/auto-implemented-properties.md) et des [initialiseurs d’objets](../programming-guide/classes-and-structs/object-and-collection-initializers.md) , tout en générant des CS0200 :</span><span class="sxs-lookup"><span data-stu-id="af9b7-110">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md) and [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md) and still generates CS0200:</span></span>

```csharp

{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```
