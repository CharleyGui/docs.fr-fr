 <span data-ttu-id="11683-105">Certains types ne peuvent pas être appliqués à un tableau.</span><span class="sxs-lookup"><span data-stu-id="11683-105">There are some types that cannot be used as the type of an array.</span></span> <span data-ttu-id="11683-106">Ces types incluent **System.TypedReference** et **System.ArgIterator**.</span><span class="sxs-lookup"><span data-stu-id="11683-106">These types include **System.TypedReference** and **System.ArgIterator**.</span></span>

```csharp  
{  
   public static void Main()  
   {

   }

```
