# <a name="compiler-error-cs1545"></a><span data-ttu-id="7c093-103">Erreur du compilateur CS1545</span><span class="sxs-lookup"><span data-stu-id="7c093-103">Compiler Error CS1545</span></span>
<span data-ttu-id="7c093-104">La propriété, l’indexeur ou l’événement 'property' n’est pas pris en charge par le langage ; essayez d’appeler directement les méthodes d’accesseur 'set accessor' ou 'get accessor'</span><span class="sxs-lookup"><span data-stu-id="7c093-104">Property, indexer, or event 'property' is not supported by the language; try directly calling accessor methods 'set accessor' or 'get accessor'</span></span>  
  
 <span data-ttu-id="7c093-105">Le code utilise un objet ayant un [indexeur](../programming-guide/indexers/index.md) autre que celui par défaut et a tenté d’utiliser la syntaxe indexée.</span><span class="sxs-lookup"><span data-stu-id="7c093-105">The code is consuming an object that has a non-default [indexer](../programming-guide/indexers/index.md) and tried to use the indexed syntax.</span></span> <span data-ttu-id="7c093-106">Pour résoudre cette erreur, appelez la méthode d’accesseur `get` ou `set` de la propriété.</span><span class="sxs-lookup"><span data-stu-id="7c093-106">To resolve this error, call the property's `get` or `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7c093-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="7c093-107">Example</span></span>  
  
```cpp  
// CPP1545.cpp  
// compile with: /clr /LD  
// a Visual C++ program  
using namespace System;  
public ref struct Employee {  
   Employee( String^ s, int d ) {}  
  
   property String^ name {  
      String^ get() {  
         return nullptr;  
      }  
   }  
};  
  
public ref struct Manager {  
   property Employee^ Report [String^] {  
      Employee^ get(String^ s) {  
         return nullptr;  
      }  
  
      void set(String^ s, Employee^ e) {}  
   }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="7c093-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="7c093-108">Example</span></span>  
 <span data-ttu-id="7c093-109">L’exemple suivant génère l’erreur CS1545.</span><span class="sxs-lookup"><span data-stu-id="7c093-109">The following sample generates CS1545.</span></span>

```csharp  
   public static void Main() {  
      Manager Ed = new Manager();

   }  
}  
```
