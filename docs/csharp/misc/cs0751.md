# <a name="compiler-error-cs0751"></a><span data-ttu-id="bb106-102">Erreur du compilateur CS0751</span><span class="sxs-lookup"><span data-stu-id="bb106-102">Compiler Error CS0751</span></span>
<span data-ttu-id="bb106-103">Une méthode partielle doit être déclarée au sein d’une classe partielle ou d’une structure partielle</span><span class="sxs-lookup"><span data-stu-id="bb106-103">A partial method must be declared in a partial class or partial struct</span></span>  
  
 <span data-ttu-id="bb106-104">Il n’est pas possible de déclarer une méthode [partielle](../language-reference/keywords/partial-method.md) , sauf si elle est encapsulée dans une classe partielle ou un struct partiel.</span><span class="sxs-lookup"><span data-stu-id="bb106-104">It is not possible to declare a [partial](../language-reference/keywords/partial-method.md) method unless it is encapsulated inside a partial class or partial struct.</span></span>
