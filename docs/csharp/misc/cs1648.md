## <a name="example"></a><span data-ttu-id="7dd88-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="7dd88-106">Example</span></span>

 <span data-ttu-id="7dd88-107">L’exemple suivant génère l’erreur CS1648 :</span><span class="sxs-lookup"><span data-stu-id="7dd88-107">The following sample generates CS1648:</span></span>

```csharp
// CS1648.cs
public struct Inner
{
    public int i;
}

class Outer
{
    public readonly Inner inner = new Inner();
}

class D
{
    static void Main()
    {
        var outer = new Outer();
        outer.inner.i = 1;  // CS1648
    }
}
```
