# <a name="compiler-error-cs0127"></a><span data-ttu-id="041b2-103">Erreur du compilateur CS0127</span><span class="sxs-lookup"><span data-stu-id="041b2-103">Compiler Error CS0127</span></span>

<span data-ttu-id="041b2-104">Comme 'fonction' retourne void, un mot clé return ne peut pas être suivi d’une expression d’objet</span><span class="sxs-lookup"><span data-stu-id="041b2-104">Since 'function' returns void, a return keyword must not be followed by an object expression</span></span>  
  
 <span data-ttu-id="041b2-105">Une méthode avec un type de retour [void](../language-reference/builtin-types/void.md) ne peut pas retourner de valeur.</span><span class="sxs-lookup"><span data-stu-id="041b2-105">A method with a [void](../language-reference/builtin-types/void.md) return type cannot return a value.</span></span> <span data-ttu-id="041b2-106">Pour plus d’informations, consultez [Méthodes](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="041b2-106">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="041b2-107">L’exemple suivant génère l’erreur CS0127 :</span><span class="sxs-lookup"><span data-stu-id="041b2-107">The following sample generates CS0127:</span></span>  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
