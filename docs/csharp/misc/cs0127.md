# <a name="compiler-error-cs0127"></a>Erreur du compilateur CS0127
Comme 'fonction' retourne void, un mot clé return ne peut pas être suivi d’une expression d’objet  
  
 Une méthode avec un type de retour [void](../language-reference/keywords/void.md) ne peut pas retourner de valeur. Pour plus d’informations, consultez [Méthodes](../programming-guide/classes-and-structs/methods.md).  
  
 L’exemple suivant génère l’erreur CS0127 :  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
