# <a name="compiler-warning-level-2-cs3021"></a><span data-ttu-id="8777e-102">Avertissement du compilateur (niveau 2) CS3021</span><span class="sxs-lookup"><span data-stu-id="8777e-102">Compiler Warning (level 2) CS3021</span></span>

<span data-ttu-id="8777e-103">'type' n’a pas besoin d’un attribut CLSCompliant, car l’assembly n’a pas d’attribut CLSCompliant</span><span class="sxs-lookup"><span data-stu-id="8777e-103">'type' does not need a CLSCompliant attribute because the assembly does not have a CLSCompliant attribute</span></span>  
  
 <span data-ttu-id="8777e-104">Cet avertissement se produit si `[CLSCompliant(false)]` apparaît sur une classe dans un assembly qui n’a pas d’attribut CLSCompliant au niveau de l’assembly défini sur true (autrement dit, la ligne `[assembly: CLSCompliant(true)]` ).</span><span class="sxs-lookup"><span data-stu-id="8777e-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="8777e-105">Étant donné que l’assembly ne se déclare pas comme conforme CLS, il est inutile que son contenu se déclare comme non conforme, puisqu’il est déjà supposé être non conforme.</span><span class="sxs-lookup"><span data-stu-id="8777e-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="8777e-106">Pour plus d’informations sur la conformité CLS, consultez [indépendance du langage et composants indépendants du langage](../../standard/language-independence.md).</span><span class="sxs-lookup"><span data-stu-id="8777e-106">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>
  
 <span data-ttu-id="8777e-107">Pour supprimer cet avertissement, supprimez l’attribut ou ajoutez l’attribut de niveau assembly.</span><span class="sxs-lookup"><span data-stu-id="8777e-107">To get rid of this warning, remove the attribute or add the assembly level attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8777e-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="8777e-108">Example</span></span>  

 <span data-ttu-id="8777e-109">L’exemple suivant génère l’erreur CS3021 :</span><span class="sxs-lookup"><span data-stu-id="8777e-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
