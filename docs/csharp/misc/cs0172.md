# <a name="compiler-error-cs0172"></a><span data-ttu-id="60a68-102">Erreur du compilateur CS0172</span><span class="sxs-lookup"><span data-stu-id="60a68-102">Compiler Error CS0172</span></span>
<span data-ttu-id="60a68-103">Impossible de déterminer le type d’expression conditionnelle, car 'type1' et 'type2' sont convertis implicitement l’un en l’autre</span><span class="sxs-lookup"><span data-stu-id="60a68-103">Type of conditional expression cannot be determined because 'type1' and 'type2' implicitly convert to one another</span></span>  
  
<span data-ttu-id="60a68-104">Dans une instruction conditionnelle, vous devez être en mesure de convertir implicitement les types de chaque côté du `:` jeton.</span><span class="sxs-lookup"><span data-stu-id="60a68-104">In a conditional statement, you must be able to implicitly convert the types on either side of the `:` token.</span></span> <span data-ttu-id="60a68-105">En outre, il ne peut pas y avoir de conversions implicites réciproques; vous n’avez besoin que d’une seule conversion.</span><span class="sxs-lookup"><span data-stu-id="60a68-105">Also, there cannot be mutual implicit conversions; you only need one conversion.</span></span>
  
<span data-ttu-id="60a68-106">L’exemple suivant génère l’avertissement CS0172 :</span><span class="sxs-lookup"><span data-stu-id="60a68-106">The following sample generates CS0172:</span></span>
  
```csharp  
// CS0172.cs  
public class Square  
{  
   public class Circle  
   {  
      public static implicit operator Circle(Square aa)  
      {  
         return null;  
      }  
  
      public static implicit operator Square(Circle aa)  
      // using explicit resolves this error  
      // public static explicit operator Square(Circle aa)  
      {  
         return null;  
      }  
   }  
  
   public static void Main()  
   {  
      Circle aa = new Circle();  
      Square ii = new Square();  
      object o = (1 == 1) ? aa : ii;   // CS0172  
      // the following cast would resolve this error  
      // (1 == 1) ? aa : (Circle)ii;  
   }  
}  
```
