# <a name="compiler-error-cs1576"></a>Erreur du compilateur CS1576

Le numéro de ligne spécifié pour la directive #line est manquant ou non valide  
  
 Le compilateur a détecté une erreur concernant la valeur passée à la directive [#line](../language-reference/preprocessor-directives/preprocessor-line.md) .  
  
 L’exemple suivant génère l’erreur CS1576 :  
  
```csharp  
// CS1576.cs  
public class MyClass  
{  
   static void Main()  
   {  
      #line "abc.sc"         // CS1576  
      // try the following line instead  
      //#line  101 "abc.sc"  
      intt i;  // error will be reported on line 101  
   }  
}  
```
