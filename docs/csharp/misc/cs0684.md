# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="7913c-103">Avertissement du compilateur (niveau 1) CS0684</span><span class="sxs-lookup"><span data-stu-id="7913c-103">Compiler Warning (level 1) CS0684</span></span>

<span data-ttu-id="7913c-104">Interface 'interface' marquée avec 'CoClassAttribute' et non avec 'ComImportAttribute'</span><span class="sxs-lookup"><span data-stu-id="7913c-104">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="7913c-105">Si vous spécifiez **CoClassAttribute** sur une interface, vous devez également spécifier **ComImportAttribute**.</span><span class="sxs-lookup"><span data-stu-id="7913c-105">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="7913c-106">L’exemple suivant génère l’avertissement CS0684 :</span><span class="sxs-lookup"><span data-stu-id="7913c-106">The following sample generates CS0684:</span></span>  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
