# <a name="compiler-error-cs0637"></a><span data-ttu-id="66053-102">Erreur du compilateur CS0637</span><span class="sxs-lookup"><span data-stu-id="66053-102">Compiler Error CS0637</span></span>

<span data-ttu-id="66053-103">L’attribut FieldOffset n’est pas autorisé sur les champs static ou const.</span><span class="sxs-lookup"><span data-stu-id="66053-103">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="66053-104">L’attribut [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) ne peut pas être utilisé sur les champs marqués avec [static](../language-reference/keywords/static.md) ou [const](../language-reference/keywords/const.md).</span><span class="sxs-lookup"><span data-stu-id="66053-104">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../language-reference/keywords/static.md) or [const](../language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="66053-105">L’exemple suivant génère l’erreur CS0637 :</span><span class="sxs-lookup"><span data-stu-id="66053-105">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
