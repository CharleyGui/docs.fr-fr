# <a name="compiler-error-cs0449"></a><span data-ttu-id="98836-103">Erreur du compilateur CS0449</span><span class="sxs-lookup"><span data-stu-id="98836-103">Compiler Error CS0449</span></span>

<span data-ttu-id="98836-104">La contrainte 'class' ou 'struct' doit être placée avant toutes les autres contraintes</span><span class="sxs-lookup"><span data-stu-id="98836-104">The 'class' or 'struct' constraint must come before any other constraints</span></span>  
  
 <span data-ttu-id="98836-105">Les contraintes sur le paramètre de type d’un type ou d’une méthode générique doivent se produire dans un ordre spécifique : en premier `class` ou `struct` , le cas échéant, puis toutes les contraintes d’interface et enfin, toutes les contraintes de constructeur.</span><span class="sxs-lookup"><span data-stu-id="98836-105">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="98836-106">Cette erreur est provoquée par la contrainte `class` ou `struct` qui n’apparaît pas en premier.</span><span class="sxs-lookup"><span data-stu-id="98836-106">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="98836-107">Pour résoudre cette erreur, réorganisez les clauses de contrainte.</span><span class="sxs-lookup"><span data-stu-id="98836-107">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="98836-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="98836-108">Example</span></span>  

 <span data-ttu-id="98836-109">L’exemple suivant génère l’erreur CS0449.</span><span class="sxs-lookup"><span data-stu-id="98836-109">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```
