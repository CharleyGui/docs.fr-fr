## <a name="example"></a><span data-ttu-id="92a1f-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="92a1f-108">Example</span></span>

 <span data-ttu-id="92a1f-109">L’exemple suivant génère CS0459 lorsqu’une tentative est effectuée pour prendre l’adresse d’une variable locale en lecture seule dans une `foreach` boucle et dans un `fixed` bloc d’instructions :</span><span class="sxs-lookup"><span data-stu-id="92a1f-109">The following example generates CS0459 when an attempt is made to take the address of a read-only local variable in a `foreach` loop and in a `fixed` statement block:</span></span>

```csharp
// CS0459.cs
// compile with: /unsafe

class Program
{
    public unsafe void M1()
    {
        int[] ints = new int[] { 1, 2, 3 };
        foreach (int i in ints)
        {
            int *j = &i;  // CS0459
        }

        fixed (int *i = &_i)
        {
            int **j = &i;  // CS0459
        }
    }

    private int _i = 0;
}
```
