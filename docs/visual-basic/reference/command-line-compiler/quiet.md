# <a name="-quiet"></a><span data-ttu-id="f1076-102">-quiet</span><span class="sxs-lookup"><span data-stu-id="f1076-102">-quiet</span></span>

<span data-ttu-id="f1076-103">Empêche le compilateur d'afficher le code pour les erreurs et les avertissements liés à la syntaxe.</span><span class="sxs-lookup"><span data-stu-id="f1076-103">Prevents the compiler from displaying code for syntax-related errors and warnings.</span></span>

## <a name="syntax"></a><span data-ttu-id="f1076-104">Syntaxe</span><span class="sxs-lookup"><span data-stu-id="f1076-104">Syntax</span></span>

```console
-quiet
```

## <a name="remarks"></a><span data-ttu-id="f1076-105">Notes</span><span class="sxs-lookup"><span data-stu-id="f1076-105">Remarks</span></span>

<span data-ttu-id="f1076-106">Par défaut, l'option `-quiet` n'est pas activée.</span><span class="sxs-lookup"><span data-stu-id="f1076-106">By default, `-quiet` is not in effect.</span></span> <span data-ttu-id="f1076-107">Lorsque le compilateur signale une erreur ou un avertissement lié à la syntaxe, il renvoie également la ligne à partir du code source.</span><span class="sxs-lookup"><span data-stu-id="f1076-107">When the compiler reports a syntax-related error or warning, it also outputs the line from source code.</span></span> <span data-ttu-id="f1076-108">Pour les applications qui analysent la sortie du compilateur, il peut être plus commode pour le compilateur de générer uniquement le texte du diagnostic.</span><span class="sxs-lookup"><span data-stu-id="f1076-108">For applications that parse compiler output, it may be more convenient for the compiler to output only the text of the diagnostic.</span></span>

<span data-ttu-id="f1076-109">Dans l’exemple suivant, `Module1` génère une erreur qui comprend le code source lorsqu’elle est compilée sans `-quiet`.</span><span class="sxs-lookup"><span data-stu-id="f1076-109">In the following example, `Module1` outputs an error that includes source code when compiled without `-quiet`.</span></span>

```vb
Module Module1
    Sub Main()
        x()
    End Sub
End Module
```

<span data-ttu-id="f1076-110">Sortie :</span><span class="sxs-lookup"><span data-stu-id="f1076-110">Output:</span></span>

```console

        ~
```

<span data-ttu-id="f1076-111">Compilé avec `-quiet`, le compilateur génère uniquement les éléments suivants :</span><span class="sxs-lookup"><span data-stu-id="f1076-111">Compiled with `-quiet`, the compiler outputs only the following:</span></span>


> [!NOTE]
> <span data-ttu-id="f1076-112">L’option `-quiet` n’est pas disponible dans l’environnement de développement Visual Studio. elle est disponible uniquement lors de la compilation à partir de la ligne de commande.</span><span class="sxs-lookup"><span data-stu-id="f1076-112">The `-quiet` option is not available from within the Visual Studio development environment; it is available only when compiling from the command line.</span></span>
