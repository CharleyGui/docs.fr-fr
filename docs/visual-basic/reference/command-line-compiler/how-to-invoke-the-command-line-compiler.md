<span data-ttu-id="a2ea5-103">Vous pouvez appeler le compilateur de ligne de commande en tapant le nom de son fichier exécutable dans la ligne de commande, également appelée invite MS-DOS.</span><span class="sxs-lookup"><span data-stu-id="a2ea5-103">You can invoke the command-line compiler by typing the name of its executable file into the command line, also known as the MS-DOS prompt.</span></span> <span data-ttu-id="a2ea5-104">Si vous compilez à partir de l’invite de commandes Windows par défaut, vous devez taper le chemin d’accès complet au fichier exécutable.</span><span class="sxs-lookup"><span data-stu-id="a2ea5-104">If you compile from the default Windows Command Prompt, you must type the fully qualified path to the executable file.</span></span> <span data-ttu-id="a2ea5-105">Pour remplacer ce comportement par défaut, vous pouvez utiliser l’Invite de commandes développeur pour Visual Studio ou modifier la variable d’environnement PATH.</span><span class="sxs-lookup"><span data-stu-id="a2ea5-105">To override this default behavior, you can either use the Developer Command Prompt for Visual Studio, or modify the PATH environment variable.</span></span> <span data-ttu-id="a2ea5-106">Tous deux vous permettent de compiler à partir de n’importe quel répertoire en tapant simplement le nom du compilateur.</span><span class="sxs-lookup"><span data-stu-id="a2ea5-106">Both allow you to compile from any directory by simply typing the compiler name.</span></span>

## <a name="to-invoke-the-compiler-using-the-developer-command-prompt-for-visual-studio"></a><span data-ttu-id="a2ea5-107">Pour appeler le compilateur à l’aide de l’Invite de commandes développeur pour Visual Studio</span><span class="sxs-lookup"><span data-stu-id="a2ea5-107">To invoke the compiler using the Developer Command Prompt for Visual Studio</span></span>

2. <span data-ttu-id="a2ea5-109">Vous pouvez utiliser le Invite de commandes développeur pour Visual Studio pour accéder au compilateur à partir de n’importe quel répertoire sur votre ordinateur, si Visual Studio est installé.</span><span class="sxs-lookup"><span data-stu-id="a2ea5-109">You can use the Developer Command Prompt for Visual Studio to access the compiler from any directory on your machine, if Visual Studio is installed.</span></span>
