# <a name="reference-required-to-module-modulename-containing-the-type-membername"></a>Une référence au module' \<modulename> 'contenant le type' \<membername> 'est requise

Une référence au module' \<modulename> 'contenant le type' \<membername> 'est requise. Ajoutez-en une à votre projet.  
  
 Le projet contient un type qui dépend d’un autre type d’un assembly qui n’est pas inclus dans les références de projet.  
  
 **ID d’erreur :** BC30653  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ajoutez une référence de projet pour les types dépendants contenus dans le projet.
