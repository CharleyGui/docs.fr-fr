# <a name="set-parameter-cannot-be-declared-specifier"></a>Le paramètre’Set’ne peut pas être déclaré' \<specifier> '
Un modificateur non valide a été spécifié avec un paramètre dans une procédure de propriété `Set` .  
  
 **ID d’erreur :** BC31065  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez les modificateurs non valides, comme `ByRef`, `ParamArray`et `Optional`.
