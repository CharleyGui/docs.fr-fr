# <a name="character-character-0xhexadecimal-is-not-allowed-in-xml-literal"></a>Le caractère 'caractère' (0x\<hexadecimal>) n’est pas autorisé dans un littéral XML
Un caractère non pris en charge a été utilisé dans un littéral XML.  
  
 **ID d’erreur :** BC31176  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez le caractère non pris en charge.
