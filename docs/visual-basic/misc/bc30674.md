# <a name="end-synclock-must-be-preceded-by-a-matching-synclock"></a>'End SyncLock' doit être précédé d’un 'SyncLock' correspondant.
`SyncLock` commencent par le mot clé `SyncLock` et se terminent par la construction `End SyncLock` .  
  
 **ID d’erreur :** BC30674  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez que le bloc `SyncLock` commence par une instruction `SyncLock` .
