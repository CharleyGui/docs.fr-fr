# <a name="name-is-ambiguous-between-declarations-in-modules-modulename1-and-modulename2"></a>«\<nom >' est ambigu entre les déclarations dans les Modules'\<nom_module1 >' et '\<nom_module2 > »
Vous avez fourni un nom qui est ambigu et donc en conflit avec un autre nom. Le compilateur Visual Basic n’a pas les règles de résolution de conflit ; Vous devez lever l’ambiguïté des noms vous-même.  
  
 **ID d’erreur :** BC30562  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Utilisez un nom qualifié complet.
