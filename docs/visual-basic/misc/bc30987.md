# <a name="-expected"></a><span data-ttu-id="eafd5-102">'{' attendu</span><span class="sxs-lookup"><span data-stu-id="eafd5-102">'{' expected</span></span>

<span data-ttu-id="eafd5-103">Pour déclarer une instance d’un type nommé ou anonyme à l’aide d’un initialiseur d’objet, vous devez placer la liste des champs ou des propriétés et leurs valeurs initiales entre accolades ({et}).</span><span class="sxs-lookup"><span data-stu-id="eafd5-103">To declare an instance of a named or anonymous type by using an object initializer, you must enclose the list of fields or properties and their initial values in braces ({ and }).</span></span>  
  
```vb  
Dim client As New Customer() With {.Name = "Microsoft", .City = "Seattle"}  
Dim emp = New Employee() With {.Name = "Rob Young", .ID = 55555}
```  
