# <a name="string-length-exceeds-maximum-length-of-32767-characters-for-filesystem-apis"></a>La chaîne dépasse la longueur maximale (32767 caractères) autorisée pour les API 'FileSystem'
Le commentaire dépasse la longueur maximale de 32767 caractères.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Raccourcissez la chaîne.
