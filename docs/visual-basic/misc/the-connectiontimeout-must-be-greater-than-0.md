# <a name="the-connectiontimeout-must-be-greater-than-0"></a>Le ConnectionTimeout doit être supérieur à 0

Lors du chargement et du téléchargement de fichiers avec le [My. Computer. Network](xref:Microsoft.VisualBasic.Devices.Network), vous devez spécifier un `connectionTimeout` supérieur à `0` .  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Fournissez un `connectionTimeout` supérieur à `0`.
