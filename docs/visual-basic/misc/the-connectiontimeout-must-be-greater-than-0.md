# <a name="the-connectiontimeout-must-be-greater-than-0"></a>Le ConnectionTimeout doit être supérieur à 0
Quand le téléchargement des fichiers avec la [My.Computer.Network](xref:Microsoft.VisualBasic.Devices.Network), vous devez spécifier un `connectionTimeout` supérieur `0`.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Fournissez un `connectionTimeout` supérieur à `0`.
