# <a name="type-parameter-with-a-structure-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="77c18-102">Un paramètre de type avec une contrainte 'Structure' ne peut pas être utilisé en tant que contrainte</span><span class="sxs-lookup"><span data-stu-id="77c18-102">Type parameter with a 'Structure' constraint cannot be used as a constraint</span></span>
<span data-ttu-id="77c18-103">Un paramètre de type avec une contrainte `Structure` est utilisé en tant que contrainte pour un autre paramètre de type.</span><span class="sxs-lookup"><span data-stu-id="77c18-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="77c18-104">La contrainte `Structure` impose que l’argument de type passé à son paramètre de type soit un type valeur.</span><span class="sxs-lookup"><span data-stu-id="77c18-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="77c18-105">Toutefois, un type valeur ne peut pas être implémenté ou hérité. Son utilisation en tant que contrainte n’est donc pas significative, car il faudrait que l’autre paramètre de type l’implémente ou en hérite.</span><span class="sxs-lookup"><span data-stu-id="77c18-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="77c18-106">La seule interprétation significative de cette situation est que les deux arguments de type doivent être exactement du même type.</span><span class="sxs-lookup"><span data-stu-id="77c18-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="77c18-107">Si tel est le cas, votre type générique a besoin d’un seul paramètre de type.</span><span class="sxs-lookup"><span data-stu-id="77c18-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="77c18-108">L’instruction suivante peut générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="77c18-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="77c18-109">Le type passé à `t2` ne peut pas implémenter ou hériter du type passé à `t1`, car le type passé à `t1` doit être un type valeur.</span><span class="sxs-lookup"><span data-stu-id="77c18-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="77c18-110">**ID d’erreur :** BC32114</span><span class="sxs-lookup"><span data-stu-id="77c18-110">**Error ID:** BC32114</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="77c18-111">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="77c18-111">To correct this error</span></span>  
  
- <span data-ttu-id="77c18-112">Supprimez le paramètre de type contraint à `Structure` de la liste des contraintes sur l’autre paramètre de type.</span><span class="sxs-lookup"><span data-stu-id="77c18-112">Remove the type parameter constrained to `Structure` from the constraint list on the other type parameter.</span></span>  
  
- <span data-ttu-id="77c18-113">Si les deux paramètres de type nécessitent le même type de valeur, définissez le type générique avec un seul paramètre de type.</span><span class="sxs-lookup"><span data-stu-id="77c18-113">If both type parameters require the same value type, define the generic type with only one type parameter.</span></span>
