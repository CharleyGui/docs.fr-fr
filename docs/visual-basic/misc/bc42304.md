# <a name="xml-documentation-parse-error-error"></a>Erreur d’analyse de la documentation XML : \<error>

Les messages suivants peuvent apparaître avec cette erreur. Pour plus d’informations, consultez la rubrique spécifique :  
  
- [La balise de commentaire XML '\<tag>' apparaît plusieurs fois avec des attributs identiques dans le même bloc de commentaire XML](bc42305.md)  
  
- [La balise de commentaire XML '\<tag>' n’est pas autorisée sur un élément de langage '\<element>'](bc42306.md)  
  
- [Le paramètre de commentaire XML '\<parameter>' ne correspond pas à un paramètre pour l’instruction \<keyword> correspondante](bc42307.md)  
  
- [Le paramètre de commentaire XML doit avoir un attribut 'name'.](bc42308.md)  
  
- [Le commentaire XML a une balise avec un attribut 'cref' '\<attribute>' qui n’a pas pu être résolu](bc42309.md)  
  
- [L’étiquette de commentaire XML 'include' doit avoir un attribut '\<attribute>'](bc42310.md)  
  
 **ID d’erreur :** BC42304
