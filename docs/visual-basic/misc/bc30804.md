# <a name="variant-is-no-longer-a-supported-type-use-the-object-type-instead"></a>'Variant' n’est plus un type pris en charge ; utilisez le type 'Object' à la place

Le type de données `Object` remplace `Variant`.  
  
 **ID d’erreur :** BC30804  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Déclarez l’élément comme `Object`.
