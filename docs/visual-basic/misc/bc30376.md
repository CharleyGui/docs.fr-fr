# <a name="new-cannot-be-used-on-class-classname-because-it-contains-a-mustoverride-member-that-has-not-been-overridden"></a>'New' ne peut pas être utilisé sur la classe\<nom_classe >', car elle contient un membre 'MustOverride' qui n’a pas été substitué
Vous avez tenté d’utiliser `New` sur une classe qui contient un membre `MustOverride` qui n’a pas été substitué.  
  
 **ID d’erreur :** BC30376  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez l’instruction `New` .
