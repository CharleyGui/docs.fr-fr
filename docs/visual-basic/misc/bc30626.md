# <a name="namespace-statement-must-end-with-a-matching-end-namespace"></a>L’instruction 'Namespace' doit se terminer par un 'End Namespace' correspondant
Les blocs`Namespace` doivent se terminer par une instruction `End Namespace` .  
  
 **ID d’erreur :** BC30626  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ajoutez une instruction `End Namespace` à la fin du bloc `Namespace` .
