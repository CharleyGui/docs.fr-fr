# <a name="argument-value-pathname-contains-characters-that-are-not-valid-in-a-path-name"></a>La valeur d’argument '\<pathname>' contient des caractères qui ne sont pas valides dans un nom de chemin
Un nom de chemin contient un ou plusieurs caractères qui ne sont pas valides.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez le caractère du nom du chemin.
