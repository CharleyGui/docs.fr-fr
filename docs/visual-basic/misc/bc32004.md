# <a name="name-is-declared-in-project-projectname1-which-is-not-referenced-by-project-projectname2"></a>«\<nom >' est déclaré dans le projet '\<nom_projet1 > ', qui n’est pas référencé par le projet'\<nom_projet2 > »
Un élément de programmation est défini dans un projet auquel le projet actuel ne fait pas référence.  
  
 **ID d’erreur :** BC32004  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Dans le projet actif, ajoutez une référence au projet qui contient l’élément souhaité.
