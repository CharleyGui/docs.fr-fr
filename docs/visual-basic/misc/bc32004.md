# <a name="name-is-declared-in-project-projectname1-which-is-not-referenced-by-project-projectname2"></a>' \<name> 'est déclaré dans le projet' \<projectname1> ', qui n’est pas référencé par le projet' \<projectname2> '

Un élément de programmation est défini dans un projet auquel le projet actuel ne fait pas référence.  
  
 **ID d’erreur :** BC32004  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Dans le projet actif, ajoutez une référence au projet qui contient l’élément souhaité.
