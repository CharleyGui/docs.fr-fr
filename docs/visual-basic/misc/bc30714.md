# <a name="resume-statements-are-not-valid-in-the-immediate-window"></a>Les instructions 'Resume' ne sont pas valides dans la fenêtre Exécution
Les instructions Resume ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30714  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez les instructions `Resume` du code de débogage.
