# <a name="referenced-array-element-has-a-value-of-nothing"></a>L’élément de tableau référencé a la valeur 'Nothing'
L’élément de tableau référencé n’a pas été défini ou a précédemment été défini avec la valeur `Nothing`.  
  
 **ID d’erreur :** BC30744  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez que les valeurs du tableau sont correctement définies.
