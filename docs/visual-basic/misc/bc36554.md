# <a name="paramarray-cannot-be-applied-to-the-first-parameter-of-an-extension-method"></a><span data-ttu-id="1fb1a-102">'ParamArray' ne peut pas être appliqué au premier paramètre d’une méthode d’extension</span><span class="sxs-lookup"><span data-stu-id="1fb1a-102">'ParamArray' cannot be applied to the first parameter of an extension method</span></span>

<span data-ttu-id="1fb1a-103">'ParamArray' ne peut pas être appliqué au premier paramètre d’une méthode d’extension.</span><span class="sxs-lookup"><span data-stu-id="1fb1a-103">'ParamArray' cannot be applied to the first parameter of an extension method.</span></span> <span data-ttu-id="1fb1a-104">Le premier paramètre spécifie le type à étendre.</span><span class="sxs-lookup"><span data-stu-id="1fb1a-104">The first parameter specifies which type to extend.</span></span>

<span data-ttu-id="1fb1a-105">Le premier paramètre d’une méthode d’extension spécifie le type de données étendu par la méthode.</span><span class="sxs-lookup"><span data-stu-id="1fb1a-105">The first parameter of an extension method specifies the data type that the method extends.</span></span> <span data-ttu-id="1fb1a-106">Ainsi, le premier paramètre est obligatoire et ne peut pas être facultatif.</span><span class="sxs-lookup"><span data-stu-id="1fb1a-106">Therefore, the first parameter is required and cannot be optional.</span></span> <span data-ttu-id="1fb1a-107">Puisqu’un tableau de paramètres est automatiquement facultatif, il n’est pas valide en tant que premier argument de la méthode d’extension.</span><span class="sxs-lookup"><span data-stu-id="1fb1a-107">Because a parameter array is automatically optional, it is not valid as the first argument of an extension method.</span></span>

> [!NOTE]
> <span data-ttu-id="1fb1a-108">Quand la méthode est exécutée, l’instance du type de données étendu qui appelle la méthode devient l’argument du premier paramètre de la méthode.</span><span class="sxs-lookup"><span data-stu-id="1fb1a-108">When the method is executed, the instance of the extended data type that invokes the method becomes the argument for the first parameter of the method.</span></span> <span data-ttu-id="1fb1a-109">Par exemple, l’instance `greeting` dans `greeting.Print()` est l’argument du premier paramètre, `str`, dans la méthode d’extension `Public Sub Print (ByVal str As String)`.</span><span class="sxs-lookup"><span data-stu-id="1fb1a-109">For example, the instance `greeting` in `greeting.Print()` is the argument for the first parameter, `str`, in extension method `Public Sub Print (ByVal str As String)`.</span></span>

<span data-ttu-id="1fb1a-110">**ID d’erreur :** BC36554</span><span class="sxs-lookup"><span data-stu-id="1fb1a-110">**Error ID:** BC36554</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="1fb1a-111">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="1fb1a-111">To correct this error</span></span>

- <span data-ttu-id="1fb1a-112">Si le tableau de paramètres ne spécifie pas le type de données à étendre, ajoutez un nouveau premier paramètre qui spécifie ce type.</span><span class="sxs-lookup"><span data-stu-id="1fb1a-112">If the parameter array does not specify the data type you want to extend, add a new first parameter that specifies this type.</span></span>

  ```vb
  <Extension()>
  Public Sub AddTo(ByRef str As String, ByVal ParamArray addOns() As String)
  ' Concatenate the strings in addOns to str.
  End Sub
  ```

- <span data-ttu-id="1fb1a-113">Si le tableau de paramètres ne spécifie pas le type de données à étendre, envisagez de le modifier en tableau standard, ce qui exige un argument, au lieu d’un tableau de paramètres.</span><span class="sxs-lookup"><span data-stu-id="1fb1a-113">If the parameter array does specify the data type you want to extend, consider changing it to a regular array, requiring an argument, instead of a parameter array.</span></span> <span data-ttu-id="1fb1a-114">Les tableaux standard peuvent être étendus.</span><span class="sxs-lookup"><span data-stu-id="1fb1a-114">Regular arrays can be extended.</span></span>

  ```vb
  <Extension()>
  Public Function Sum(ByVal ints() As Integer) As Integer
      Dim total As Integer = 0
      For Each i As Integer In ints
          total = total + i
      Next i
      Return total
  End Function
  ```
