# <a name="expression-name-is-not-a-procedure-but-occurs-as-the-target-of-a-procedure-call"></a>L’expression '\<name>' n’est pas une procédure, mais elle intervient en tant que cible d’un appel de procédure
Vous avez effectué un appel de procédure vers une expression qui n’est pas une procédure.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Vérifiez que vous avez correctement orthographié le nom de la procédure.  
  
2. Fournissez la procédure qui <`name`> appels.
