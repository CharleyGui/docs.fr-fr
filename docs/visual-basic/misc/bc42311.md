# <a name="unable-to-create-xml-documentation-file-name-message"></a>Impossible de créer le fichier de documentation XML '\<nom >': \<message >
Le fichier de documentation XML ne peut pas être créé en raison d’une erreur.  
  
 **ID d’erreur :** BC42311  
  
 Les messages suivants peuvent apparaître avec cette erreur :  
  
- [Un bloc de commentaire XML ne peut être associé à aucun élément de langage prenant en charge l’application de commentaires de documentation XML](../../visual-basic/misc/bc42312.md)  
  
- [La balise de commentaire XML 'returns' n’est pas autorisée pour une propriété 'WriteOnly'](../../visual-basic/misc/bc42313.md)  
  
- [Le commentaire XML ne peut pas être appliqué plusieurs fois sur un partiel \<type >](../../visual-basic/misc/bc42314.md).  
  
- [La balise de commentaire XML 'returns' n’est pas autorisée pour un élément de langage 'declare sub'](../../visual-basic/misc/bc42315.md)  
  
- [Erreur d’analyse de documentation XML : Balise de début '\<balise >' n’a pas correspondante balise de fin](../../visual-basic/misc/bc42316.md).  
  
- [Le paramètre de type de commentaire XML \<paramètre > ne correspond pas à un paramètre de type sur le correspondant \<mot clé > instruction](../../visual-basic/misc/bc42317.md).  
  
- [Le paramètre de type de commentaire XML doit avoir un attribut 'name'](../../visual-basic/misc/bc42318.md)  
  
- [L’exception de commentaire XML doit avoir un attribut 'cref'](../../visual-basic/language-reference/error-messages/xml-comment-exception-must-have-a-cref-attribute.md)  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Pour obtenir des recommandations, consultez le message fourni.
