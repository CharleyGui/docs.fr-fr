# <a name="unable-to-create-xml-documentation-file-name-message"></a>Impossible de créer le fichier de documentation XML' \<name> ' : \<message>

Le fichier de documentation XML ne peut pas être créé en raison d’une erreur.  
  
 **ID d’erreur :** BC42311  
  
 Les messages suivants peuvent apparaître avec cette erreur :  
  
- [Le bloc de commentaires XML ne peut pas être associé à un élément de langage qui prend en charge l’application de commentaires de documentation XML](bc42312.md).  
  
- [La balise de commentaire XML’Returns’n’est pas autorisée pour une propriété’WriteOnly'](bc42313.md).  
  
- [Le commentaire XML ne peut pas être appliqué plusieurs fois sur \<type> un partiel ](bc42314.md).  
  
- [La balise de commentaire XML’Returns’n’est pas autorisée pour un élément de langage’Declare Sub'](bc42315.md).  
  
- [Erreur d’analyse de la documentation XML : la balise de début' \<tag> 'n’a pas de balise de fin correspondante](bc42316.md).  
  
- [Le paramètre de type \<parameter> de commentaire XML ne correspond pas à un paramètre de type sur l' \<keyword> instruction correspondante](bc42317.md).  
  
- [Le paramètre de type de commentaire XML doit avoir un attribut’name'](bc42318.md).  
  
- L' [exception de commentaire XML doit avoir un attribut’CREF'](../language-reference/error-messages/xml-comment-exception-must-have-a-cref-attribute.md).  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Pour obtenir des recommandations, consultez le message fourni.
