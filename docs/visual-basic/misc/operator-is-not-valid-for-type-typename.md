# <a name="operator-is-not-valid-for-type-typename"></a>Opérateur n’est pas valide pour le type '\<nom_type >'
Un opérateur n’est pas valide pour les types.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez que l’opérateur est compatible avec les types utilisés.
