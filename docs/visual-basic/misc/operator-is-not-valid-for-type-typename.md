# <a name="operator-is-not-valid-for-type-typename"></a>L’opérateur n’est pas valide pour le type' \<typename> '

Un opérateur n’est pas valide pour les types.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez que l’opérateur est compatible avec les types utilisés.
