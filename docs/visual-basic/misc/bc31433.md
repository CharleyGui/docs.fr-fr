# <a name="method-methodname-cannot-be-declared-partial-because-only-one-method-methodname-can-be-marked-partial"></a>La méthode '\<methodname>' ne peut pas être déclarée 'Partial', car une seule méthode '\<methodname>' peut être marquée comme 'Partial'

Vous ne pouvez avoir deux déclarations de méthode partielle ayant les mêmes nom et signature.  
  
 **ID d’erreur :** BC31433  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Modifiez le nom ou la signature de l’une des déclarations de méthode partielle.
