# <a name="project-projectname-makes-an-indirect-reference-to-assembly-assemblyname-which-contains-classname"></a>Le projet' \<projectname> 'établit une référence indirecte à l’assembly' \<assemblyname> 'qui contient' \<classname> '

Le projet' \<projectname> 'établit une référence indirecte à l’assembly' \<assemblyname> 'qui contient' \<classname> '. Ajoutez une référence à' \<assemblyname> 'à votre projet.  
  
 Ce projet contient des membres qui dépendent d’un assembly qui n’est pas contenu dans les références de projet.  
  
 **ID d’erreur :** BC31515  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ajoutez une référence à l’assembly en cliquant avec le bouton droit sur **Références** dans l’ **Explorateur de solutions** puis en cliquant sur **Ajouter une référence**.
