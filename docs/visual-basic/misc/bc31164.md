# <a name="expected-matching-closing-double-quote-for-xml-attribute-value"></a>Guillemet double fermant correspondant attendu pour une valeur d’attribut XML

Un caractère de guillemet fermant est manquant dans une valeur d’attribut qui est délimitée par des guillemets doubles.  
  
 **ID d’erreur :** BC31164  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ajoutez le caractère de guillemet double fermant pour délimiter la valeur d’attribut.
