# <a name="select-statements-are-not-valid-in-the-immediate-window"></a>Les instructions 'Select' ne sont pas valides dans la fenêtre Exécution
Les instructions`Select` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30718  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez les instructions `Select` du code de débogage.
