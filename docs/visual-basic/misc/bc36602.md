# <a name="readonly-variable-cannot-be-the-target-of-an-assignment-in-a-lambda-expression-inside-a-constructor"></a><span data-ttu-id="18c95-102">Une variable 'ReadOnly' ne peut pas être la cible d’une assignation dans une expression lambda au sein d’un constructeur</span><span class="sxs-lookup"><span data-stu-id="18c95-102">'ReadOnly' variable cannot be the target of an assignment in a lambda expression inside a constructor</span></span>

<span data-ttu-id="18c95-103">Vous avez fait référence à une variable `ReadOnly` dans une expression lambda, ce qui n’est pas autorisé.</span><span class="sxs-lookup"><span data-stu-id="18c95-103">You have referred to a `ReadOnly` variable from within a lambda expression, which is not permitted.</span></span> <span data-ttu-id="18c95-104">Le code suivant génère cette erreur en envoyant `ReadOnly` à la variable `m` comme argument d’un paramètre `ByRef` .</span><span class="sxs-lookup"><span data-stu-id="18c95-104">The following code causes this error by sending the `ReadOnly` variable `m` as the argument to a `ByRef` parameter.</span></span>  
  
```vb  
Class Class1  
  
    ReadOnly m As Integer  
  
    Sub New()  
        ' The use of m triggers the error.  
        Dim f = Function() Test(m)  
    End Sub  
  
    Function Test(ByRef n As Integer) As String  
    End Function  
  
End Class  
```  
  
 <span data-ttu-id="18c95-105">**ID d’erreur :** BC36602</span><span class="sxs-lookup"><span data-stu-id="18c95-105">**Error ID:** BC36602</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="18c95-106">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="18c95-106">To correct this error</span></span>  
  
- <span data-ttu-id="18c95-107">Si possible, remplacez le paramètre `n` dans la fonction `Test` par un paramètre `ByVal` , comme indiqué dans le code suivant.</span><span class="sxs-lookup"><span data-stu-id="18c95-107">If possible, change parameter `n` in function `Test` to a `ByVal` parameter, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix1  
  
        ReadOnly m As Integer  
  
        Sub New()  
            Dim f = Function() Test(m)  
        End Sub  
  
        Function Test(ByVal n As Integer) As String  
        End Function  
  
    End Class  
    ```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="18c95-108">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="18c95-108">To correct this error</span></span>  
  
- <span data-ttu-id="18c95-109">Assignez la variable `ReadOnly``m` à une variable locale dans le constructeur et utilisez la variable locale à la place de `m`, comme illustré dans le code suivant.</span><span class="sxs-lookup"><span data-stu-id="18c95-109">Assign the `ReadOnly` variable `m` to a local variable in the constructor, and use the local variable in place of `m`, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix2  
         ReadOnly m As Integer  
  
        Sub New()  
            Dim temp = m  
            Dim f = Function() Test(temp)  
        End Sub  
  
        Function Test(ByRef n As Integer) As String  
        End Function  
  
    End Class  
    ```
