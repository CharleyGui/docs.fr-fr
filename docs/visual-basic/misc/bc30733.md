# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>'\<MemberName > 'est déjà déclaré par'\<NomProcédure > ', qui a été généré pour ce \<ContainerName >

Un membre ayant le même nom qu’un membre déclaré par le common language runtime a été déclaré.

**ID d’erreur :** BC30733

## <a name="to-correct-this-error"></a>Pour corriger cette erreur

- Modifiez le nom du membre pour supprimer le conflit.
