# <a name="class-classname-must-declare-a-sub-new-because-its-base-class-classname2-does-not-have-an-accessible-sub-new-that-can-be-called-with-no-arguments"></a><span data-ttu-id="3ed13-102">La classe '\<classname>' doit déclarer un 'Sub New', car sa classe de base '\<classname2>' n’a pas de 'Sub New' accessible qu’il est possible d’appeler sans argument</span><span class="sxs-lookup"><span data-stu-id="3ed13-102">Class '\<classname>' must declare a 'Sub New' because its base class '\<classname2>' does not have an accessible 'Sub New' that can be called with no arguments</span></span>

<span data-ttu-id="3ed13-103">Une classe dérivée ne déclare pas de constructeur, et Visual Basic ne peut pas en générer une, car il n’y a aucun constructeur de classe de base qu’elle peut appeler.</span><span class="sxs-lookup"><span data-stu-id="3ed13-103">A derived class does not declare a constructor, and Visual Basic cannot generate one because there is no base class constructor it can call.</span></span>  
  
 <span data-ttu-id="3ed13-104">Quand une classe dérivée ne déclare pas de constructeur, Visual Basic tente de générer un constructeur sans paramètre implicite qui appelle `MyBase.New()` .</span><span class="sxs-lookup"><span data-stu-id="3ed13-104">When a derived class does not declare a constructor, Visual Basic attempts to generate an implicit parameterless constructor that calls `MyBase.New()`.</span></span> <span data-ttu-id="3ed13-105">S’il n’existe aucun constructeur accessible dans la classe de base qui peut être appelé sans arguments, ou s’il en existe plusieurs, Visual Basic ne peut pas générer de constructeur implicite.</span><span class="sxs-lookup"><span data-stu-id="3ed13-105">If there is no accessible constructor in the base class that can be called without arguments, or if there is more than one, Visual Basic cannot generate an implicit constructor.</span></span>  
  
 <span data-ttu-id="3ed13-106">**ID d’erreur :** BC30387</span><span class="sxs-lookup"><span data-stu-id="3ed13-106">**Error ID:** BC30387</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3ed13-107">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="3ed13-107">To correct this error</span></span>  
  
1. <span data-ttu-id="3ed13-108">Déclarez et implémentez au moins un constructeur `Sub New` quelque part dans la classe dérivée.</span><span class="sxs-lookup"><span data-stu-id="3ed13-108">Declare and implement at least one `Sub New` constructor somewhere in the derived class.</span></span>  
  
2. <span data-ttu-id="3ed13-109">Ajoutez un appel à un constructeur de classe de base, `MyBase.New()`, en tant que première ligne de chaque `Sub New`.</span><span class="sxs-lookup"><span data-stu-id="3ed13-109">Add a call to a base class constructor, `MyBase.New()`, as the first line of every `Sub New`.</span></span>
