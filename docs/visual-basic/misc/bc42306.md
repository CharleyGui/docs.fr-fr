# <a name="xml-comment-tag-tag-is-not-permitted-on-a-element-language-element"></a>Balise de commentaire XML '\<balise >' n’est pas autorisée sur un «\<élément >' élément de langage
Balise de commentaire XML \<balise > n’est pas autorisée sur un «\<élément >' élément. Le commentaire XML est ignoré.  
  
 Une balise de commentaire XML n’est pas prise en charge par l’élément de langage auquel elle est appliquée.  
  
 **ID d’erreur :** BC42306  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez la balise de commentaire XML non prise en charge.
