# <a name="argument-argumentname-cannot-be-a-multidimensional-array"></a>L’argument\<'ArgumentName > 'ne peut pas être un tableau multidimensionnel
Le rang d’un argument de tableau est supérieur à 1, mais il peut seulement être un tableau unidimensionnel.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Attribuez la valeur 1 au rang de l’argument.
