# <a name="option-infer-can-be-followed-only-by-on-or-off"></a>'Option Infer' ne peut être suivi que par 'On' ou 'Off'

Les instructions`Option Infer` doivent spécifier `On` ou `Off`. D’autres valeurs ne sont pas valides.  
  
 **ID d’erreur :** BC30979  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Spécifiez `Option Infer On` ou `Option Infer Off`.
