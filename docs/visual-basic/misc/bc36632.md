# <a name="nullable-parameters-must-specify-a-type"></a>Les paramètres Nullable doivent spécifier un type
Un paramètre comprend le modificateur de type nullable (?), mais ne spécifie pas de type.  
  
 **ID d’erreur :** BC36632  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Spécifiez un type lorsque vous déclarez le paramètre de type valeur Nullable.
