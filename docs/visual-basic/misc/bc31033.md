# <a name="interface-interfacename-can-be-implemented-only-once-by-this-type"></a>L’interface '\<interfacename>' ne peut être implémentée qu’une seule fois par ce type
L’interface spécifiée est déjà implémentée par ce type. Elle ne peut pas être implémentée à nouveau.  
  
 **ID d’erreur :** BC31033  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Implémentez cette interface une seule fois à l’aide de ce type.
