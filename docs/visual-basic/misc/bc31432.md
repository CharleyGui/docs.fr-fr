<span data-ttu-id="ad574-103">Le modificateur d’accès `Private` est nécessaire dans les déclarations de méthode partielle.</span><span class="sxs-lookup"><span data-stu-id="ad574-103">The access modifier `Private` is required in partial method declarations.</span></span> <span data-ttu-id="ad574-104">L’exemple suivant illustre l’utilisation de `Private` dans la signature de méthode et son implémentation.</span><span class="sxs-lookup"><span data-stu-id="ad574-104">The following example shows the use of `Private` in the method signature and its implementation.</span></span>  
  
```vb  
' Definition of the partial method signature.

    ' The body of the signature is empty.  
End Sub  
```  
  
```vb

Private Sub OnNameChanged()  
    MsgBox("Name was changed to " & Me.Name)  
End Sub  
```
