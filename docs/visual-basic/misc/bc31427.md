# <a name="syntax-error-in-conditional-compilation-expression"></a>Erreur de syntaxe dans l’expression de compilation conditionnelle
L’expression de compilation conditionnelle génère une erreur de syntaxe non spécifique.  
  
 **ID d’erreur:** BC31427  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Consultez la documentation de la compilation conditionnelle et des mots clés utilisés dans l’expression. Comparez l’exemple de code dans la documentation à la ligne source générant cette erreur.  
  
2. Si vous ne parvenez pas à identifier la cause de l’erreur, rassemblez des informations sur ses circonstances et avertissez les Services de support technique Microsoft.
