# <a name="classes-can-inherit-only-from-other-classes"></a>Les classes peuvent hériter uniquement d’autres classes
Une instruction `Inherits` spécifie un type autre qu’une classe en tant que type de base.  
  
 **ID d’erreur :** BC30258  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Remplacez le type de base par une classe qui est déjà définie.
