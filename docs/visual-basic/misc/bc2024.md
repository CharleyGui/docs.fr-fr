# <a name="the-default-assembly-reference-reference-has-already-been-automatically-added-ignored"></a>La référence d’assembly par défaut '\<référence >' a déjà été automatiquement ajoutée ; ignoré
L’utilisateur a tenté d’ajouter la référence d’assembly par défaut, mais l’assembly a déjà été ajouté. La tentative est donc ignorée.  
  
 **ID d’erreur :** BC2024  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez le code redondant.
