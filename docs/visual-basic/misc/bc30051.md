# <a name="paramarray-parameter-must-be-a-one-dimensional-array"></a>Le paramètre ParamArray doit être un tableau unidimensionnel.
Un argument `ParamArray` doit être déclaré comme type de tableau unidimensionnel. La déclaration spécifie plusieurs dimensions pour l’argument `ParamArray` .  
  
 **ID d’erreur :** BC30051  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez toutes les virgules comprises dans les parenthèses qui suivent le nom d’argument `ParamArray` .
