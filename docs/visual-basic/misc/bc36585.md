# <a name="elementname-is-not-a-parameter-of-extension-method-methodname-defined-in-typename"></a>' \<elementname> 'n’est pas un paramètre de la méthode d’extension' \<methodname> 'définie dans' \<typename> '

Un appel de procédure fournit un argument supplémentaire qui n’est pas défini par la méthode d’extension.  
  
 **ID d’erreur :** BC36585  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez l’argument superflu de la liste d’arguments.
