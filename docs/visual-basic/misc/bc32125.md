# <a name="cannot-implement-interfacename1membername-because-its-implementation-could-conflict-with-the-implementation-for-interfacename2membername-for-some-type-arguments"></a><span data-ttu-id="15f4b-102">Ne peut pas implémenter '\<nom_interface1 >.\< nom_membre >', car son implémentation pourrait être en conflit avec l’implémentation pour '\<nom_interface2 >.\< nom_membre >' pour certains arguments de type</span><span class="sxs-lookup"><span data-stu-id="15f4b-102">Cannot implement '\<interfacename1>.\<membername>' because its implementation could conflict with the implementation for '\<interfacename2>.\<membername>' for some type arguments</span></span>
<span data-ttu-id="15f4b-103">Une classe implémente plusieurs interfaces génériques, dont une hérite d’une autre, et deux implémentations d’un membre d’interface peuvent être en conflit pour certaines valeurs d’arguments de type.</span><span class="sxs-lookup"><span data-stu-id="15f4b-103">A class implements more than one generic interface, one of which inherits from another, and two implementations of an interface member could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="15f4b-104">Les instructions suivantes peuvent générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="15f4b-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface iFace1(Of t)  
    Sub testSub()  
End Interface  
Public Interface iFace2(Of u)  
    Inherits iFace1(Of u)  
End Interface  
Public Class testClass(Of y, z)  
    Implements iFace1(Of y), iFace2(Of z)  
    Public Sub testSuby() Implements iFace1(Of y).testSub  
    End Sub  
    Public Sub testSubz() Implements iFace1(Of z).testSub  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="15f4b-105">Étant donné que `iFace2` hérite de `iFace1` à l’aide de son propre paramètre de type (`u`), `testClass` implémente deux versions de `iFace1.testSub` avec des signatures identiques si le même argument de type est transmis à `y` et `z`.</span><span class="sxs-lookup"><span data-stu-id="15f4b-105">Because `iFace2` inherits from `iFace1` using its own type parameter (`u`), `testClass` would implement two versions of `iFace1.testSub` with identical signatures if the same type argument were passed to `y` and `z`.</span></span> <span data-ttu-id="15f4b-106">Cela produit une ambiguïté à propos de la version à laquelle il faut accéder.</span><span class="sxs-lookup"><span data-stu-id="15f4b-106">This would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="15f4b-107">**ID d’erreur :** BC32125</span><span class="sxs-lookup"><span data-stu-id="15f4b-107">**Error ID:** BC32125</span></span>
