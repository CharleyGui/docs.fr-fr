# <a name="modifier-is-not-valid-on-a-method-declaration"></a>'\<modificateur >' n’est pas valide dans une déclaration de méthode
Une instruction `Function` ou `Sub` contient un mot clé non valide.  
  
 **ID d’erreur :** BC30242  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez le mot clé non valide de l’instruction `Function` ou `Sub` .
