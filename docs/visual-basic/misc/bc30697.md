# <a name="membername1-cannot-override-membername2-because-they-differ-by-the-types-of-optional-parameters"></a>' \<membername1> 'ne peut pas se substituer \<membername2> à' ', car les types de paramètres facultatifs les différencient

Les membres surchargés doivent avoir des types de données différents pour un ou plusieurs paramètres obligatoires.  
  
 **ID d’erreur :** BC30697  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Définissez au moins un paramètre obligatoire en tant que type de données différent.
