# <a name="membername1-cannot-override-membername2-because-they-differ-by-the-types-of-optional-parameters"></a>«\<nom_membre1 >' ne peut pas remplacer '\<nom_membre2 >', car les types des paramètres optionnels les différencient
Les membres surchargés doivent avoir des types de données différents pour un ou plusieurs paramètres obligatoires.  
  
 **ID d’erreur :** BC30697  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Définissez au moins un paramètre obligatoire en tant que type de données différent.
