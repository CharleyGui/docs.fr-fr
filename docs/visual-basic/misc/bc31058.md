# <a name="conflicts-with-name1-which-is-implicitly-created-for-name2-in-type-name3"></a>Conflits avec' \<name1> ', qui est créé implicitement pour' \<name2> 'dans \<type> ' \<name3> '

Le nom d’un membre de type est en conflit avec un autre membre créé implicitement.  
  
 **ID d’erreur :** BC31058  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Renommez le membre déclaré explicitement pour supprimer le conflit de noms.
