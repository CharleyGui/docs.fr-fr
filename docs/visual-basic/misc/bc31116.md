# <a name="removehandler-declaration-must-end-with-a-matching-end-removehandler"></a>La déclaration 'RemoveHandler' doit se terminer par un 'End RemoveHandler' correspondant
Une déclaration `RemoveHandler` doit se terminer par une instruction `End RemoveHandler` .  
  
 **ID d’erreur :** BC31116  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez que la déclaration `RemoveHandler` se termine par une instruction `End RemoveHandler` .
