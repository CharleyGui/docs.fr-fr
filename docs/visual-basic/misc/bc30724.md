# <a name="error-statements-are-not-valid-in-the-immediate-window"></a>Les instructions 'Error' ne sont pas valides dans la fenêtre Exécution
Les instructions`Error` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30724  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez les instructions `Error` du code de débogage.
