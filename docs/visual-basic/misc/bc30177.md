# <a name="only-one-of-notoverridable-mustoverride-or-overridable-can-be-specified"></a>Vous pouvez spécifier uniquement une instance de type 'NotOverridable', 'MustOverride' ou 'Overridable'.

Un conflit de spécifications de substitution est présent dans une déclaration element. Les mots clés de substitution s’excluent mutuellement.  
  
 **ID d’erreur :** BC30177  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Choisissez un spécificateur de substitution, puis supprimez les autres mots clés de la déclaration.
