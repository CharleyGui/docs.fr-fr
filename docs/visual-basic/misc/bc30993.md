# <a name="property-propertyname-cannot-be-initialized-in-an-object-initializer-expression-because-all-accessible-overloads-require-arguments"></a><span data-ttu-id="83c0a-102">La propriété '\<propertyname>' ne peut pas être initialisée dans une expression d’initialiseur d’objet, car toutes les surcharges accessibles nécessitent des arguments</span><span class="sxs-lookup"><span data-stu-id="83c0a-102">Property '\<propertyname>' cannot be initialized in an object initializer expression because all accessible overloads require arguments</span></span>

<span data-ttu-id="83c0a-103">Les membres initialisés dans une liste d’initialiseurs d’objets doivent être des champs ou des propriétés.</span><span class="sxs-lookup"><span data-stu-id="83c0a-103">The members initialized in an object initializer list must be either fields or properties.</span></span> <span data-ttu-id="83c0a-104">De plus, les propriétés dans une liste d’initialiseurs ne peuvent pas avoir de paramètres.</span><span class="sxs-lookup"><span data-stu-id="83c0a-104">Additionally, properties in an initializer list cannot have parameters.</span></span> <span data-ttu-id="83c0a-105">La propriété à l’origine de cette erreur est surchargée, et chacune de ses versions requiert des arguments.</span><span class="sxs-lookup"><span data-stu-id="83c0a-105">The property causing this error is overloaded, and each of its versions requires arguments.</span></span> <span data-ttu-id="83c0a-106">Ainsi, la propriété ne peut pas être initialisée dans une liste d’initialiseurs d’objets.</span><span class="sxs-lookup"><span data-stu-id="83c0a-106">Therefore, the property cannot be initialized in an object initializer list.</span></span>  
  
 <span data-ttu-id="83c0a-107">**ID d’erreur :** BC30993</span><span class="sxs-lookup"><span data-stu-id="83c0a-107">**Error ID:** BC30993</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="83c0a-108">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="83c0a-108">To correct this error</span></span>  
  
- <span data-ttu-id="83c0a-109">Supprimez la propriété qui requiert des arguments de la liste d’initialiseurs.</span><span class="sxs-lookup"><span data-stu-id="83c0a-109">Remove the property that requires arguments from the initializer list.</span></span>  
  
## <a name="example"></a><span data-ttu-id="83c0a-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="83c0a-110">Example</span></span>  

 <span data-ttu-id="83c0a-111">La classe suivante contient trois définitions de propriétés : une pour `TotalItems` et deux pour `Item`, qui est surchargée.</span><span class="sxs-lookup"><span data-stu-id="83c0a-111">The following class contains three property definitions: one for `TotalItems` and two for `Item`, which is overloaded.</span></span>  
  
```vb  
Class CollectionOfItems  
    Property TotalItems() As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
    Property Item(ByVal Key As String) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
    Property Item(ByVal Index As Integer) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="83c0a-112">La propriété `TotalItems` ne requiert aucun argument et peut être initialisée dans une liste d’initialiseurs d’objets, comme indiqué dans la déclaration suivante.</span><span class="sxs-lookup"><span data-stu-id="83c0a-112">The `TotalItems` property requires no arguments and can be initialized in an object initialization list, as shown in the following declaration.</span></span>  
  
```vb  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
```  
  
 <span data-ttu-id="83c0a-113">La propriété `Item` est surchargée et chaque surcharge requiert un argument.</span><span class="sxs-lookup"><span data-stu-id="83c0a-113">The `Item` property is overloaded, and each overload requires an argument.</span></span> <span data-ttu-id="83c0a-114">Ainsi, `Item` ne peut pas apparaître dans une liste d’initialiseurs d’objets.</span><span class="sxs-lookup"><span data-stu-id="83c0a-114">Therefore, `Item` cannot appear in an object initializer list.</span></span>  
  
```vb  
' The following declaration is not valid.  
' Dim coinCollection As New CollectionOfItems With { .TotalItems = 0, _  
'    .Item = aCoinObject }  
```  
  
 <span data-ttu-id="83c0a-115">Pour éviter cette erreur, initialisez la propriété `Item` à l’extérieur de l’initialiseur d’objets.</span><span class="sxs-lookup"><span data-stu-id="83c0a-115">To avoid this error, initialize the `Item` property outside the object initializer.</span></span>  
  
```vb  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
coinCollection.Item(1) = aCoinObject  
```
