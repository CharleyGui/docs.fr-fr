# <a name="namespace-or-type-name-has-already-been-imported"></a>L’espace de noms ou le type '\<name>' a déjà été importé
L’utilisateur a tenté d’importer plusieurs fois un même espace de noms.  
  
 **ID d’erreur :** BC31051  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez les instructions redondantes `Imports` .
