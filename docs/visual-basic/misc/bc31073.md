# <a name="name1-conflicts-with-a-type-by-the-same-name-defined-in-name2"></a>' \<name1> 'est en conflit avec un portant \<type> le même nom défini dans' \<name2> '

Un nom qui est en conflit avec un type du même nom a été rencontré.  
  
 **ID d’erreur :** BC31073  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Déclarez un nom unique.  
  
2. Utilisez des crochets autour du nom du membre en conflit.
