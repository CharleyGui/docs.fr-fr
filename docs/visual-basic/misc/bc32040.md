# <a name="the-keyword-keyword-is-used-to-overload-inherited-members-do-not-use-the-keyword-keyword-when-overloading-sub-new"></a>Le mot\<clé' > 'est utilisé pour surcharger les membres hérités; n'\<utilisez pas le mot clé’mot clé > 'lors de la surcharge de’Sub New'
Un constructeur est déclaré avec le mot clé `Overloads` .  
  
 Visual Basic ne prend pas en charge l’héritage ni la surcharge des constructeurs.  
  
 **ID d’erreur:** BC32040  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez le mot clé `Overloads` de toutes les déclarations de constructeur.
