<span data-ttu-id="8b371-103">Soit un composant ne trouve pas de répertoire nommé TEMP, soit le lecteur ou la partition contenant le répertoire TEMP ne dispose pas de suffisamment d’espace pour enregistrer les informations.</span><span class="sxs-lookup"><span data-stu-id="8b371-103">Either a component cannot find a directory named TEMP, or the drive or partition containing the TEMP directory lacks sufficient space to save information.</span></span>
