<span data-ttu-id="be037-103">Une valeur de laquelle il n’est pas possible de déduire un nom de propriété a été incluse dans la liste d’initialiseurs d’une déclaration de type anonyme, alors qu’elle n’a pas été assignée à un nom de propriété.</span><span class="sxs-lookup"><span data-stu-id="be037-103">A value from which a property name cannot be inferred has been included in the initializer list of an anonymous type declaration without being assigned to a property name.</span></span>
