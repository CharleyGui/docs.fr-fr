# <a name="unable-to-sign-assembly-error-message"></a>Impossible de signer l’assembly : \<error message>

Le compilateur Visual Basic appelle Assembly Linker (Al.exe, également appelé ALink) pour générer un assembly avec un manifeste. L’éditeur de liens a signalé une erreur de signature de l’assembly après sa création. Cela peut être dû à une clé non valide.  
  
 **ID d’erreur :** BC30146  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Examinez le message d’erreur cité et consultez la rubrique  [Al.exe (Assembly Linker)](../../framework/tools/al-exe-assembly-linker.md) pour obtenir plus d’explications et de conseils.  
  
2. Si l'erreur persiste, rassemblez des informations sur ses circonstances et avertissez les services de support technique Microsoft.
