# <a name="xml-comment-type-parameter-must-have-a-name-attribute"></a>Le paramètre de type de commentaire XML doit avoir un attribut ’name’
Le paramètre de type de commentaire XML doit avoir un attribut 'name'. Le commentaire XML est ignoré.  
  
 La balise de commentaire XML exige un attribut `name` qui est manquant ou incomplet.  
  
 **ID d’erreur :** BC42318  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ajoutez l’attribut `name` obligatoire à la balise ou complétez-le.
