# <a name="typename1-cannot-be-converted-to-typename2"></a>«\<nom_type1 >' ne peut pas être convertie en '\<nom_type2 > »
Une conversion de type a été tentée entre deux types incompatibles.  
  
 **ID d’erreur :** BC30741  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez les types qui font l’objet d’un cast.
