# <a name="local-variable-cannot-have-the-same-name-as-the-function-containing-it"></a>Une variable locale ne peut pas porter le même nom que la fonction qui la contient
Une instruction `Dim` spécifie une variable avec le même nom que la procédure `Function` qui la contient.  
  
 **ID d’erreur :** BC30290  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez la déclaration de la variable ou modifiez le nom de la variable.
