# <a name="end-statements-are-not-valid-in-the-immediate-window"></a>Les instructions 'End' ne sont pas valides dans la fenêtre Exécution
Les instructions `Stop` et `End` interrompent l’exécution et ne sont pas autorisées dans un contexte de débogage.  
  
 **ID d’erreur :** BC30123  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- N’émettez pas d’instruction `End` ou `Stop` dans la fenêtre **Exécution** .
