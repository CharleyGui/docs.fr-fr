# <a name="operator-operator-is-not-defined-for-types-type1-and-type2"></a>L’opérateur 'opérateur' n’est pas défini pour les types '&lt;type1&gt;' et '&lt;type2&gt;'

L’opérateur 'opérateur' n’est pas défini pour les types '<type1>' et '<type2>'. Vous pouvez utiliser la propriété’value’pour récupérer la valeur de chaîne du premier élément de' \<parentElement> '.  
  
 Vous avez utilisé un littéral XML dans un fichier binaire ou une comparaison `Boolean` .  
  
 **ID d’erreur :** BC31195  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Utilisez la propriété `Value` du littéral XML pour le fichier binaire ou la comparaison `Boolean` .
