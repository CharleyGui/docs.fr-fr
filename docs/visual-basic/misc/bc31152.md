# <a name="xml-namespace-prefix-prefix-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>Le préfixe de l’espace de noms XML 'prefix' est réservé pour une utilisation par XML et l’URI de l’espace de noms ne peut pas être changé
Un préfixe d’espace de noms XML réservé, tel que `xmlns`, a été utilisé dans un littéral XML.  
  
 **ID d’erreur :** BC31152  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Remplacez le préfixe de l’espace de noms XML par une valeur qui n’est pas réservée.
