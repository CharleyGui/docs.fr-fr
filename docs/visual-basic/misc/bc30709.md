# <a name="variable-declaration-statements-are-not-valid-in-the-immediate-window"></a>Les instructions de déclaration de variable ne sont pas valides dans la fenêtre Exécution
Les déclarations de variable ne peuvent pas être utilisées dans la fenêtre **Exécution** .  
  
 **ID d’erreur :** BC30709  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ne déclarez pas de variables dans la fenêtre **Exécution** .
