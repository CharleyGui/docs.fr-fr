# <a name="end-enum-must-be-preceded-by-a-matching-enum"></a>'End Enum' doit être précédé d’un 'Enum' correspondant
Une instruction `End Enum` se produit sans instruction `Enum` correspondante. `End Enum` doit être précédée d’une instruction `Enum` correspondante.  
  
 **ID d’erreur :** BC30184  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Vérifiez l’orthographe ou la validité d’une instruction `Enum` précédente.  
  
2. Vérifiez que les membres `Enum` sont correctement mis en forme.
