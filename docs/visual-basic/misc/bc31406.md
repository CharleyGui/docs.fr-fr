# <a name="methodname-cannot-override-a-method-that-has-been-shadowed"></a>' \<methodname> 'ne peut pas substituer une méthode qui a été masquée

Une méthode a essayé de substituer une méthode qui masque un autre membre.  
  
 **ID d’erreur :** BC31406  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Utilisez le modificateur `Shadows` pour substituer les membres masqués.
