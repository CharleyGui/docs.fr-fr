# <a name="source-file-filename-specified-multiple-times"></a>Le fichier source \<filename> a été spécifié plusieurs fois

Un fichier a été spécifié plusieurs fois.  
  
 **ID d’erreur :** BC2002  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez les spécificateurs de fichiers redondants.
