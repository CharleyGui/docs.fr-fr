# <a name="method1-cannot-override-method2-because-they-differ-by-their-return-types"></a>' \<method1> 'ne peut pas se substituer \<method2> à' ', car ils diffèrent par leurs types de retour

Vous avez tenté de substituer une méthode par une autre dont le type de retour est différent. Un type peut se substituer à une méthode substituable héritée en déclarant celle-ci avec les mêmes nom et signature, et en marquant la déclaration avec le modificateur `Overrides` . La signature des deux méthodes doit correspondre.  
  
 **ID d’erreur :** BC30437  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez les types de retour des deux méthodes et modifiez-les si nécessaire pour qu’elles correspondent.
