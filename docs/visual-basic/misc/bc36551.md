## <a name="example"></a><span data-ttu-id="1f64a-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="1f64a-108">Example</span></span>

    <Extension()> _  
    Module StringExtensions  
        <Extension()> _

        End Sub  
    End Module
