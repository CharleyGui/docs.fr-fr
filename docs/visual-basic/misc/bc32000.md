# <a name="local-variable-variablename-cannot-be-referred-to-before-it-is-declared"></a>La variable locale '\<variablename>' ne peut pas être référencée avant d’être déclarée

Une variable à l’intérieur d’une procédure est utilisée avant sa déclaration.  
  
 **ID d’erreur :** BC32000  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Déplacez la déclaration de sorte qu’elle précède la première référence à la variable.
