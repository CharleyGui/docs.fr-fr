# <a name="only-one-of-public-private-protected-friend-or-protected-friend-can-be-specified"></a>Vous pouvez spécifier uniquement une instance de type 'Public', 'Private', 'Protected', 'Friend' ou 'Protected Friend'
Des conflits d’accessibilité se produisent dans une déclaration d’élément. Une seule accessibilité peut être spécifiée.  
  
 **ID d’erreur :** BC30176  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Choisissez une accessibilité et supprimez les autres mots clés de la déclaration.
