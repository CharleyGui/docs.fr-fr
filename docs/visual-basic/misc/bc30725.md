# <a name="throw-statements-are-not-valid-in-the-immediate-window"></a>Les instructions 'Throw' ne sont pas valides dans la fenêtre Exécution

Les instructions`Throw` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30725  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez les instructions `Throw` du code de débogage.
