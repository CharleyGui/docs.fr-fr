# <a name="catch-statements-are-not-valid-in-the-immediate-window"></a>Les instructions 'Catch' ne sont pas valides dans la fenêtre Exécution.

Les instructions`Catch` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30715  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez les instructions `Catch` du code de débogage.
