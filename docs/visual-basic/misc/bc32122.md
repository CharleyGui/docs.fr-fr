# <a name="cannot-inherit-interface-interfacename1-because-the-interface-interfacename2-from-which-it-inherits-could-be-identical-to-interface-interfacename3-from-which-the-interface-interfacename4-inherits-for-some-type-arguments"></a><span data-ttu-id="ef87b-102">Impossible d’hériter de l’interface '\<interfacename1>', car l’interface '\<interfacename2>' dont elle hérite peut être identique à l’interface '\<interfacename3>' dont l’interface '\<interfacename4>' hérite pour certains arguments de type</span><span class="sxs-lookup"><span data-stu-id="ef87b-102">Cannot inherit interface '\<interfacename1>' because the interface '\<interfacename2>' from which it inherits could be identical to interface '\<interfacename3>' from which the interface '\<interfacename4>' inherits for some type arguments</span></span>

<span data-ttu-id="ef87b-103">Une interface générique hérite d’au moins deux interfaces génériques, et deux des héritages peuvent entrer en conflit pour certaines valeurs d’arguments de type.</span><span class="sxs-lookup"><span data-stu-id="ef87b-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>

 <span data-ttu-id="ef87b-104">Les instructions suivantes peuvent générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="ef87b-104">The following statements can generate this error.</span></span>

```vb
Public Interface interfaceA(Of u)
End Interface
Public Interface interfaceX(Of v)
    Inherits interfaceA(Of v)
End Interface
Public Interface interfaceY(Of w)
    Inherits interfaceA(Of w)
End Interface
Public Interface derivedInterface(Of t1, t2)
    Inherits interfaceX(Of t1), interfaceY(Of t2)
End Interface
```

<span data-ttu-id="ef87b-105">Si `derivedInterface` est construit ou implémenté en fournissant le même type à `t1` et à `t2`, il doit hériter de deux versions de `interfaceA` avec des arguments de type identiques.</span><span class="sxs-lookup"><span data-stu-id="ef87b-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="ef87b-106">Cela crée une ambiguïté sur la version à laquelle accéder.</span><span class="sxs-lookup"><span data-stu-id="ef87b-106">Doing so would produce an ambiguity about which version to access.</span></span>

<span data-ttu-id="ef87b-107">**ID d’erreur :** BC32122</span><span class="sxs-lookup"><span data-stu-id="ef87b-107">**Error ID:** BC32122</span></span>

- <span data-ttu-id="ef87b-109">Modifiez l’un des arguments de type fournis à l’interface dérivée pour éviter tout conflit.</span><span class="sxs-lookup"><span data-stu-id="ef87b-109">Change one of the type arguments supplied to the derived interface so that there is no conflict.</span></span>
