# <a name="type-argument-inference-failed-for-type-parameter-typeparametername-of-genericproceduresignature"></a><span data-ttu-id="eb4c8-102">Échec de l’inférence d’argument de type pour le paramètre de type' \<typeparametername > 'de' \<genericproceduresignature > '</span><span class="sxs-lookup"><span data-stu-id="eb4c8-102">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'</span></span>

<span data-ttu-id="eb4c8-103">**ID d’erreur :** BC32051</span><span class="sxs-lookup"><span data-stu-id="eb4c8-103">**Error ID:** BC32051</span></span>

<span data-ttu-id="eb4c8-104">Échec de l’inférence d’argument de type pour le paramètre de type' \<typeparametername > 'de' \<genericproceduresignature > '.</span><span class="sxs-lookup"><span data-stu-id="eb4c8-104">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'.</span></span> <span data-ttu-id="eb4c8-105">Impossible de déduire l’argument de type à partir de l’argument passé au paramètre' \<parametername > '.</span><span class="sxs-lookup"><span data-stu-id="eb4c8-105">Type argument could not be inferred from the argument passed to parameter '\<parametername>'.</span></span>

 <span data-ttu-id="eb4c8-106">Une procédure générique a été appelée sans que des arguments de type aient été fournis et le compilateur ne peut pas déduire le type à passer à l’un des paramètres.</span><span class="sxs-lookup"><span data-stu-id="eb4c8-106">A generic procedure is called without supplying any type arguments, and the compiler cannot infer the type to pass to one of the parameters.</span></span>

 <span data-ttu-id="eb4c8-107">Normalement, quand vous appelez une procédure générique, vous fournissez un argument de type pour chaque paramètre de type que la procédure générique définit.</span><span class="sxs-lookup"><span data-stu-id="eb4c8-107">Normally, when you call a generic procedure, you supply a type argument for each type parameter that the generic procedure defines.</span></span> <span data-ttu-id="eb4c8-108">Si vous ne fournissez pas d’arguments de type, le compilateur tente de déduire les types à passer aux paramètres de type.</span><span class="sxs-lookup"><span data-stu-id="eb4c8-108">If you do not supply any type arguments, then the compiler attempts to infer the types to be passed to the type parameters.</span></span> <span data-ttu-id="eb4c8-109">Si le contexte de l’appel fournit des informations de type de données conflictuelles pour un paramètre de type, l’inférence de type échoue.</span><span class="sxs-lookup"><span data-stu-id="eb4c8-109">If the context of the call provides conflicting data type information for a type parameter, then the type inference fails.</span></span>

 <span data-ttu-id="eb4c8-110">Le code suivant peut générer cette erreur :</span><span class="sxs-lookup"><span data-stu-id="eb4c8-110">The following code can generate this error:</span></span>

```vb
Public Sub DoSomething(Of T)(arg1 As T(), arg2 As T)
End Sub

Public Sub Main()
    DoSomething(6, 42)
End Sub
```  
  
 <span data-ttu-id="eb4c8-111">Dans l’exemple précédent, le compilateur déduit le type `Integer` pour `t` en se basant sur la valeur 42 passée à `arg2`.</span><span class="sxs-lookup"><span data-stu-id="eb4c8-111">In the preceding example, the compiler infers type `Integer` for `t` based on the value of 42 passed to `arg2`.</span></span> <span data-ttu-id="eb4c8-112">Toutefois, cette inférence nécessite que `arg1` soit du type `Integer()`, autrement dit, un tableau `Integer`, mais la valeur 6 passée à `arg1` ne correspond pas à ce type.</span><span class="sxs-lookup"><span data-stu-id="eb4c8-112">However, that inference would require `arg1` to be of type `Integer()`, that is, an array of `Integer`, and the value 6 passed to `arg1` does not match that type.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="eb4c8-113">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="eb4c8-113">To correct this error</span></span>

- <span data-ttu-id="eb4c8-114">Fournissez des arguments de type à la procédure générique afin que le compilateur n’ait pas besoin de les déduire.</span><span class="sxs-lookup"><span data-stu-id="eb4c8-114">Supply type arguments to the generic procedure, so that the compiler does not have to infer them.</span></span>

- <span data-ttu-id="eb4c8-115">Fournissez des arguments normaux avec des types qui correspondent à ceux des arguments de type.</span><span class="sxs-lookup"><span data-stu-id="eb4c8-115">Supply normal arguments with types that match those of the type arguments.</span></span>
