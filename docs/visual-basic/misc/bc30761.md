# <a name="project-projectname-cannot-reference-project-projectname-because-projectname-directly-or-indirectly-references-projectname"></a>Projet '\<nom_projet >' ne peut pas référencer de projet '\<nom_projet >', car '\<nom_projet >' référence directement ou indirectement '\<nom_projet >'
Un projet inclut une référence à un deuxième projet. Le deuxième projet inclut, à son tour, une référence au projet dans lequel il est référencé.  
  
 **ID d’erreur :** BC30761  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez l’une des références.
