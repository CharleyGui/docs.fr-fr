# <a name="project-projectname-cannot-reference-project-projectname-because-projectname-directly-or-indirectly-references-projectname"></a>Le projet' \<projectname> 'ne peut pas faire référence au projet' \<projectname> ', car' \<projectname> 'référence directement ou indirectement' \<projectname> '

Un projet inclut une référence à un deuxième projet. Le deuxième projet inclut, à son tour, une référence au projet dans lequel il est référencé.  
  
 **ID d’erreur :** BC30761  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez l’une des références.
