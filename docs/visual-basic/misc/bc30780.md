# <a name="continue-statements-are-not-valid-in-the-immediate-window"></a>Les instructions 'Continue' ne sont pas valides dans la fenêtre Exécution
Instructions 'Continue' peuvent uniquement être utilisées dans le code source.  
  
 **ID d’erreur :** BC30780  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez l’instruction 'Continue' le code de débogage.
