# <a name="continue-statements-are-not-valid-in-the-immediate-window"></a>Les instructions 'Continue' ne sont pas valides dans la fenêtre Exécution
Les instructions’continue’ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30780  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez l’instruction’continue’du code de débogage.
