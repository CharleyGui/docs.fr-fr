# <a name="with-contexts-and-statements-are-not-valid-in-debug-windows"></a>Les contextes et instructions 'With' ne sont pas valides dans les fenêtres de débogage
Les instructions`With` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30726  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez les instructions `With` du code de débogage.
