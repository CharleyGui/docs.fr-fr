# <a name="required-attribute-version-missing-from-xml-declaration"></a>Attribut 'version' requis absent de la déclaration XML
L’attribut `version` d’un littéral de document XML est manquant.  
  
 **ID d’erreur :** BC31153  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ajoutez l’attribut `version` à la déclaration de document XML, comme dans l’exemple suivant : `<?xml version="1.0"?>`
