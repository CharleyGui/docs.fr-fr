# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-to-region-regionname"></a><span data-ttu-id="5bd4b-102">' \<derivedtypename> 'ne peut pas hériter de \<type> ' \<constructedbasetypename> ', car il étend l’accès du type' \<internaltypename> 'à \<region> ' \<regionname> '</span><span class="sxs-lookup"><span data-stu-id="5bd4b-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' to \<region> '\<regionname>'</span></span>
<span data-ttu-id="5bd4b-103">Une classe ou une interface dérivée essaie d’étendre le niveau d’accès d’un type interne en l’utilisant comme argument de type dans une classe ou une interface de base.</span><span class="sxs-lookup"><span data-stu-id="5bd4b-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="5bd4b-104">Le code suivant peut générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="5bd4b-104">The following code can generate this error.</span></span>  
  
```vb  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="5bd4b-105">Le code à l’extérieur de `containingClass` n’est pas autorisé à accéder à `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="5bd4b-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="5bd4b-106">Toutefois, `derivedClass` est accessible à partir de n’importe quel code dans le même assembly.</span><span class="sxs-lookup"><span data-stu-id="5bd4b-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="5bd4b-107">Par conséquent, `derivedClass` ne peut pas hériter de `baseClass` s’il utilise `internalStructure` comme argument de type, car cela peut exposer `internalStructure` dans l’ensemble de la région de code de définition.</span><span class="sxs-lookup"><span data-stu-id="5bd4b-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="5bd4b-108">**ID d’erreur :** BC30921</span><span class="sxs-lookup"><span data-stu-id="5bd4b-108">**Error ID:** BC30921</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5bd4b-109">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="5bd4b-109">To correct this error</span></span>  
  
- <span data-ttu-id="5bd4b-110">Modifiez les niveaux d’accès des classes ou des interfaces afin que le type dérivé n’étende pas le niveau d’accès du type interne.</span><span class="sxs-lookup"><span data-stu-id="5bd4b-110">Adjust the access levels of the classes or interfaces so that the derived type does not expand the access level of the internal type.</span></span>  
  
     <span data-ttu-id="5bd4b-111">-ou-</span><span class="sxs-lookup"><span data-stu-id="5bd4b-111">-or-</span></span>  
  
- <span data-ttu-id="5bd4b-112">Si vous ne pouvez pas modifier les niveaux d’accès, n’utilisez pas le type interne comme argument de type lors de la construction de la classe ou de l’interface de base.</span><span class="sxs-lookup"><span data-stu-id="5bd4b-112">If you cannot adjust the access levels, then do not use the internal type as a type argument when constructing the base class or interface.</span></span>
