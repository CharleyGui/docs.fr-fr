# <a name="this-inheritance-causes-circular-dependencies-between-type1-typename1-and-its-nested-type2-typename2"></a><span data-ttu-id="50f8b-102">Cet héritage provoque des dépendances circulaires entre \<type1> ' \<typename1> 'et son \<type2> ' \<typename2> 'imbriqué</span><span class="sxs-lookup"><span data-stu-id="50f8b-102">This inheritance causes circular dependencies between \<type1> '\<typename1>' and its nested \<type2> '\<typename2>'</span></span>

<span data-ttu-id="50f8b-103">Une structure d’héritage entraîne une dépendance circulaire entre les classes imbriquées, autrement dit, deux classes qui héritent l’une de l’autre.</span><span class="sxs-lookup"><span data-stu-id="50f8b-103">An inheritance structure results in circular dependency among nested classes, that is, two classes inheriting from each other.</span></span>  
  
 <span data-ttu-id="50f8b-104">Le code suivant peut générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="50f8b-104">The following code can generate this error message.</span></span>  
  
```vb  
Public Class c1  
    Inherits c3.c4  
    Public Class c2  
    End Class  
End Class  
Public Class c3  
    Inherits c1.c2  
    Public Class c4  
    End Class  
End Class  
```  
  
 <span data-ttu-id="50f8b-105">Dans le code précédent, la classe `c1` hérite de la classe `c4`, mais `c4` est imbriqué dans `c3`, qui hérite de `c2`qui est imbriqué dans `c1`.</span><span class="sxs-lookup"><span data-stu-id="50f8b-105">In the preceding code, class `c1` inherits from class `c4`, but `c4` is nested inside `c3`, which inherits from `c2`, nested inside `c1`.</span></span>  
  
 <span data-ttu-id="50f8b-106">**ID d’erreur :** BC30907</span><span class="sxs-lookup"><span data-stu-id="50f8b-106">**Error ID:** BC30907</span></span>
