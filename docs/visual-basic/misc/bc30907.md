# <a name="this-inheritance-causes-circular-dependencies-between-type1-typename1-and-its-nested-type2-typename2"></a>Cet héritage provoque des dépendances circulaires entre \<type1 > ' \<typename1 > 'et son @no__t >-2type2 @no__t' >-3typename2 '
Une structure d’héritage entraîne une dépendance circulaire entre les classes imbriquées, autrement dit, deux classes qui héritent l’une de l’autre.  
  
 Le code suivant peut générer cette erreur.  
  
```vb  
Public Class c1  
    Inherits c3.c4  
    Public Class c2  
    End Class  
End Class  
Public Class c3  
    Inherits c1.c2  
    Public Class c4  
    End Class  
End Class  
```  
  
 Dans le code précédent, la classe `c1` hérite de la classe `c4`, mais `c4` est imbriqué dans `c3`, qui hérite de `c2`qui est imbriqué dans `c1`.  
  
 **ID d’erreur :** BC30907  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez les dépendances circulaires de la structure d’héritage.
