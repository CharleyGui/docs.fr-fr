# <a name="expected-closing--for-xml-processor-instruction"></a>' ? > 'fermant attendu pour une instruction de processeur XML
Un littéral d’instruction de traitement XML ne possède pas les caractères `?>` de fermeture requis.  
  
 **ID d’erreur :** BC31160  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ajoutez les caractères `?>` de fermeture requis à la fin du littéral d’instruction de traitement XML.
