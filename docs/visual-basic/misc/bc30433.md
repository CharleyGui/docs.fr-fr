# <a name="methods-in-a-module-cannot-be-declared-specifier"></a>Les méthodes d’un module ne peuvent pas être déclarées' \<specifier> '

Vous avez utilisé un spécificateur non valide pour un méthode comprise dans une instruction `Module` . Les modules ne peuvent jamais être instanciés, ne prennent pas en charge l’héritage et ne peuvent pas implémenter d’interfaces.  
  
 **ID d’erreur :** BC30433  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez le spécificateur.
