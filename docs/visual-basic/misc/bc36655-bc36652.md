<span data-ttu-id="af674-103">Impossible de déduire le ou les types de données du ou des paramètres de type\<dans la méthode d’extension'\<MethodName > 'définie dans’TypeName > 'à partir de ces arguments, car plusieurs types sont possibles.</span><span class="sxs-lookup"><span data-stu-id="af674-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in '\<typename>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="af674-104">La spécification explicite du ou des types de données peut permettre de corriger cette erreur.</span><span class="sxs-lookup"><span data-stu-id="af674-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="af674-105">L’utilisateur a tenté d’utiliser l’inférence de type pour déterminer le ou les types du ou des paramètres de type dans un appel à une méthode d’extension générique.</span><span class="sxs-lookup"><span data-stu-id="af674-105">An attempt has been made to use type inference to determine the type (or types) of the type parameter (or parameters) in a call to a generic extension method.</span></span> <span data-ttu-id="af674-106">Le compilateur détecte plusieurs types de données possibles pour un ou plusieurs des paramètres de type, et il signale cette erreur.</span><span class="sxs-lookup"><span data-stu-id="af674-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>

> [!NOTE]
> <span data-ttu-id="af674-107">Quand la spécification des arguments n’est pas une option (par exemple, pour les opérateurs de requête dans les expressions de requête), le message d’erreur apparaît sans la deuxième phrase.</span><span class="sxs-lookup"><span data-stu-id="af674-107">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

```vb

Imports System.Runtime.CompilerServices
Module Module1
    Sub Main()

        Dim caller As New Class1
        '' Not valid.
        'caller.targetExtension(1, "2")

    End Sub

    <Extension()> _
    Sub targetExtension(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)

    Class Class1
    End Class
```
