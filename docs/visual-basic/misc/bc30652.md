# <a name="reference-required-to-assembly-assemblyname-containing-the-type-membername"></a>Une référence à l’assembly' \<assemblyname> 'contenant le type' 'est requise \<membername>

Une référence à l’assembly' \<assemblyname> 'contenant le type' 'est requise \<membername> . Ajoutez-en une à votre projet.  
  
 Le projet contient un type qui dépend d’un autre type d’un assembly qui n’est pas inclus dans les références de projet.  
  
 **ID d’erreur :** BC30652  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ajoutez une référence de projet pour les types dépendants contenus dans le projet.
