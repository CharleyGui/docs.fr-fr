# <a name="too-many-dll-application-clients"></a>Trop de clients pour cette DLL

La bibliothèque de liens dynamiques (DLL) de Visual Basic peut uniquement prendre en charge l’accès par un nombre limité d’applications hôtes. Votre application et d’autres applications Visual Basic hôtes (dont certaines sont accessibles par votre application) essaient d’accéder au Visual Basic DLL en même temps.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Réduisez le nombre d’applications ouvertes accédant à Visual Basic.
