# <a name="specifier-is-not-valid-on-a-structure-declaration"></a>' \<specifier> 'n’est pas valide dans une déclaration de structure
Vous avez utilisé un spécificateur, comme `Dim` ou `Override`, non valide dans une déclaration `Structure`  
  
 **ID d’erreur :** BC30395  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez le spécificateur.
