# <a name="type-of-argument-argumentname-is-typename-which-is-not-numeric"></a>Le type de l’argument '\<argumentname>' est '\<typename>', qui n’est pas numérique
Un argument non numérique apparaît dans une situation dans laquelle un argument numérique est attendu.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Assurez-vous que l’argument est numérique.
