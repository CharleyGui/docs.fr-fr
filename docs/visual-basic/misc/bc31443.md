<span data-ttu-id="9f380-103">Dans une méthode partielle qui inclut un ou plusieurs paramètres de type, les noms des paramètres de type doivent être identiques dans la déclaration et dans l’implémentation de la méthode.</span><span class="sxs-lookup"><span data-stu-id="9f380-103">In a partial method that includes one or more type parameters, the names of the type parameters must be the same in the declaration of the method and in the implementation of the method.</span></span>

```vb

```

```vb  
'Private Sub OnNameChanged(Of N)()

'End Sub  
```
