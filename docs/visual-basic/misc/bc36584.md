# <a name="parameter-parametername-in-extension-method-methodname-defined-in-typename-already-has-a-matching-argument"></a>Le paramètre '\<parametername>' de la méthode d’extension '\<methodname>' définie dans '\<typename>' a déjà un argument correspondant

Un argument est passé par nom plusieurs fois dans le même appel de procédure.  
  
 **ID d’erreur :** BC36584  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez si l’un des noms d’argument est mal orthographié.  
  
- Supprimez l’argument dupliqué de la liste d’arguments.
