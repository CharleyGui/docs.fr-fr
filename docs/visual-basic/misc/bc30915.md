# <a name="name-cannot-expose-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-outside-the-project-through-specifier-type"></a>' \<name> 'ne peut pas exposer le type délégué sous-jacent' \<delegatetype> 'de l’événement qu’il implémente en dehors du projet via \<specifier> ' \<type> '

Le code expose un type privé en dehors de la classe publique.  
  
 **ID d’erreur :** BC30915  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Déclarez le type `Public`.  
  
     - ou -  
  
- Utilisez un autre type.
