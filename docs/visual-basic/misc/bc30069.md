# <a name="for-loop-control-variable-variablename-already-in-use-by-an-enclosing-for-loop"></a>La variable de contrôle de boucle '\<variablename>' déjà utilisée par une boucle For englobante

Les boucles `For` imbriquées doivent toutes utiliser des variables de contrôle uniques.  
  
 **ID d’erreur :** BC30069  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Modifiez l’un des noms de variables de contrôle.
