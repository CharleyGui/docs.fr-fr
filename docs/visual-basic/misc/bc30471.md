# <a name="expression-is-not-an-array-or-a-method-and-cannot-have-an-argument-list"></a>L’expression n’est pas un tableau ni une méthode et elle ne peut pas avoir de liste d’arguments

Vous avez fourni une liste d’arguments pour une expression qui n’est pas un tableau ni une méthode. Seuls les tableaux ou les méthodes peuvent avoir des listes d’arguments.  
  
 **ID d’erreur :** BC30471  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez la liste d’arguments.
