# <a name="expression-cannot-be-evaluated-at-this-time"></a>Impossible d’évaluer l’expression pour l’instant

L’expression spécifiée n’a pas pu être évaluée dans le contexte actuel.  
  
 **ID d’erreur :** BC30736  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Modifiez la syntaxe de l’expression.
