# <a name="members-in-a-structure-cannot-be-declared-specifier"></a>Les membres d’une structure ne peuvent pas être déclarés' \<specifier> '

Vous avez utilisé un spécificateur qui n’est pas valide pour un membre compris dans une instruction `Structure` .  
  
 **ID d’erreur :** BC30435  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez le spécificateur.
