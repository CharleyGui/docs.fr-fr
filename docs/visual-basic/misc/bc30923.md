# <a name="name-cannot-expose-type-type-used-in-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-to-specifier1-type1-through-specifier2-type2"></a>' \<name> 'ne peut pas exposer \<type> le type' 'utilisé dans le type délégué sous-jacent' \<delegatetype> 'de l’événement qu’il implémente à' \<specifier1> \<type1> 'via \<specifier2> ' \<type2> '

Le code expose un type privé en dehors de la classe publique.  
  
 **ID d’erreur :** BC30923  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Déclarez le type `Public` ou utilisez un autre type.
