# <a name="cannot-calculate-number-of-periods-using-the-arguments-provided"></a>Impossible de calculer le nombre de périodes à l’aide des arguments fournis
Un appel à la `NPer` fonction ne contient-elle pas tous les arguments requis.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez que les valeurs `Rate`, `Prnt` et `PV` sont incluses dans l’appel de fonction.
