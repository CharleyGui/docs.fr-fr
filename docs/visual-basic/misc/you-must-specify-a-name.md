# <a name="you-must-specify-a-name"></a>Vous devez spécifier un nom
Lorsque vous effectuez des opérations de fichier, comme renommer un fichier, vous devez spécifier le nom.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Spécifiez le nom.
