# <a name="type1-typename-and-type2-typename-declared-in-name-conflict-in-namespace-namespacename"></a>\<type1> ' \<typename> 'et \<type2> ' \<typename> ', déclarés dans' \<name> ', sont en conflit dans \<namespace> ' \<namespacename> '

Deux éléments déclarés, tels qu’une classe et un module, ont le même nom dans un même espace de noms.  
  
 **ID d’erreur :** BC30175  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Déplacez la déclaration de l’un des éléments vers un autre espace de noms.  
  
- Modifiez le nom de l’un des éléments.
