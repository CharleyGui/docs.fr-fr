# <a name="reference-required-to-module-modulename-containing-the-implemented-interface-interfacename"></a>Une référence au module' \<modulename> 'contenant l’interface implémentée' 'est requise \<interfacename>

Une référence au module' \<modulename> 'contenant l’interface implémentée' 'est requise \<interfacename> . Ajoutez-en une à votre projet.  
  
 L’interface est définie dans un module qui n’est pas directement référencé dans votre projet. Le compilateur Visual Basic requiert une référence pour éviter toute ambiguïté au cas où l’interface serait définie dans plusieurs modules.  
  
 **ID d’erreur :** BC30010  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ajoutez le nom du module non référencé dans vos références de projet.
