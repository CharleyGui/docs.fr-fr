# <a name="xml-comment-parameter-parameter-does-not-match-a-parameter-on-the-corresponding-keyword-statement"></a>Le paramètre de commentaire XML '\<paramètre >' ne correspond pas à un paramètre sur le correspondant \<mot clé > instruction
Le paramètre de commentaire XML \<paramètre > ne correspond pas à un paramètre sur le correspondant \<mot clé > instruction. Le commentaire XML est ignoré.  
  
 Une balise de commentaire XML a été spécifiée pour certains (mais pas l’ensemble) des paramètres d’une méthode.  
  
 **ID d’erreur :** BC42307  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Fournissez la balise de commentaire XML pour tous les paramètres de la méthode.
