# <a name="all-projects-in-a-visual-basic-solution-must-target-the-same-platform-but-the-project-you-are-trying-to-add-targets-a-platform-other-than-the-one-specified-by-your-solution"></a>Tous les projets d’une solution Visual Basic doivent cibler la même plateforme, mais le projet que vous essayez d’ajouter cible une plateforme différente de celle spécifiée par votre solution

Tous les projets d’une solution Visual Basic doivent cibler la même plateforme, mais le projet que vous ajoutez cible une plateforme différente de celle spécifiée par votre solution.  
  
 **ID d’erreur :** BC30765  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ajoutez uniquement des projets qui ciblent la même plateforme spécifiée par votre solution.
