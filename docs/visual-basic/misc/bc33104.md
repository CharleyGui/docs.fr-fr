# <a name="if-operator-requires-either-two-or-three-operands"></a>L’opérateur 'If' exige deux ou trois opérandes
L’opérateur `If` accepte uniquement deux ou trois arguments.  
  
 **ID d’erreur :** BC33104  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Examinez l’expression `If` pour voir si le nombre d’opérateurs peut être réduit ou étendu à deux ou trois arguments.  
  
- Songez à utiliser une autre construction conditionnelle, comme une instruction `If...Then...Else` ou une instruction `If...Then...Else` imbriquée pour organiser les arguments.
