# <a name="error-setting-assembly-manifest-option-error-message"></a>Erreur lors de la définition de l’option du manifeste de l’assembly : \<error message>

Le compilateur Visual Basic appelle Assembly Linker (Al.exe, également appelé ALink) pour générer un assembly avec un manifeste. L’éditeur de liens a signalé une erreur lors de la recherche d’un fichier de clé ou d’un nom de clé avec lequel signer l’assembly.  
  
 **ID d’erreur :** BC30139  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Examinez le message d’erreur entre guillemets et consultez les erreurs listées sur [Al.exe (Assembly Linker)](../../framework/tools/al-exe-assembly-linker.md) pour obtenir une explication et des conseils supplémentaires.  
  
2. Si l'erreur persiste, rassemblez des informations sur ses circonstances et avertissez les services de support technique Microsoft.
