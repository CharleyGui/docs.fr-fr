# <a name="typename-must-be-declared-mustinherit-because-it-contains-methods-declared-mustoverride"></a>'\<nom_type >' doit être déclarée 'MustInherit', car il contient des méthodes déclarées 'MustOverride'
Les types qui contiennent des membres `MustOverride` doivent être déclarés `MustInherit`.  
  
 **ID d’erreur :** BC31411  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ajoutez le modificateur `MustInherit` au type.
