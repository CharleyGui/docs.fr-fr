# <a name="attribute-value-is-not-valid-expecting-value"></a>Valeur d’attribut non valide ; 'value' attendue
Une valeur non valide pour un attribut dans un littéral de document XML a été spécifiée. Les valeurs d’attributs valides pour un littéral de document XML sont déterminées dans la spécification XML 1.0.  
  
 **ID d’erreur :** BC31181  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Mettez à jour la valeur d’attribut vers une valeur valide, comme indiqué dans la spécification XML 1.0.
