# <a name="side-effects-not-valid-during-expression-evaluation-in-this-context"></a>Effets secondaires non valides pendant l’évaluation de l’expression dans ce contexte
Les instructions qui modifient l’état d’un objet ne sont pas autorisées dans ce contexte.  
  
 **ID d’erreur :** BC30700  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Modifiez votre code afin que les instructions soient évaluées sans modifier l’état d’autres éléments.
