# <a name="end-removehandler-must-be-preceded-by-a-matching-removehandler-declaration"></a>'End RemoveHandler' doit être précédé d’une instruction 'RemoveHandler' correspondante

Une instruction `End RemoveHandler` s’est produite sans instruction `RemoveHandler` correspondante. `End RemoveHandler` doit être précédée d’une instruction `RemoveHandler` correspondante.  
  
 **ID d’erreur :** BC31125  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez que l’instruction `RemoveHandler` précédente est valide et orthographiée correctement.
