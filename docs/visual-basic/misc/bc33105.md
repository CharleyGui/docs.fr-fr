# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="efbf2-102">Les opérandes 'If' ne peuvent pas être des arguments nommés</span><span class="sxs-lookup"><span data-stu-id="efbf2-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="efbf2-103">L’utilisation d’arguments nommés dans les opérandes de l’opérateur `If` n’est pas valide.</span><span class="sxs-lookup"><span data-stu-id="efbf2-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="efbf2-104">L’exemple suivant génère cette erreur :</span><span class="sxs-lookup"><span data-stu-id="efbf2-104">The following example causes this error:</span></span>  
  
```vb

Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="efbf2-105">Il diffère de la fonction `IIf` qui autorise des arguments nommés, comme indiqué dans le code suivant :</span><span class="sxs-lookup"><span data-stu-id="efbf2-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```vb

```  
  
 <span data-ttu-id="efbf2-106">**ID d’erreur :** BC33105</span><span class="sxs-lookup"><span data-stu-id="efbf2-106">**Error ID:** BC33105</span></span>
