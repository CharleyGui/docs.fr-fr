# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="06e7e-102">Les opérandes 'If' ne peuvent pas être des arguments nommés</span><span class="sxs-lookup"><span data-stu-id="06e7e-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="06e7e-103">L’utilisation d’arguments nommés dans les opérandes de l’opérateur `If` n’est pas valide.</span><span class="sxs-lookup"><span data-stu-id="06e7e-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="06e7e-104">L’exemple suivant génère cette erreur :</span><span class="sxs-lookup"><span data-stu-id="06e7e-104">The following example causes this error:</span></span>

```

```
