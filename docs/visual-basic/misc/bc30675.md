# <a name="synclock-statement-must-end-with-a-matching-end-synclock"></a>L’instruction 'SyncLock' doit se terminer par un 'End SyncLock' correspondant

`SyncLock` commencent par le mot clé `SyncLock` et se terminent par la construction `End SyncLock` .  
  
 **ID d’erreur :** BC30675  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez que le bloc `SyncLock` se termine par une construction `End SyncLock` .
