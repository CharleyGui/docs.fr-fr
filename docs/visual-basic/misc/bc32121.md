# <a name="cannot-inherit-interface-interfacename1-because-the-interface-interfacename2-from-which-it-inherits-could-be-identical-to-interface-interfacename3-for-some-type-arguments"></a><span data-ttu-id="41ce9-102">Impossible d’hériter de l’interface '\<interfacename1>', car l’interface '\<interfacename2>' dont elle hérite peut être identique à l’interface '\<interfacename3>' pour certains arguments de type</span><span class="sxs-lookup"><span data-stu-id="41ce9-102">Cannot inherit interface '\<interfacename1>' because the interface '\<interfacename2>' from which it inherits could be identical to interface '\<interfacename3>' for some type arguments</span></span>
<span data-ttu-id="41ce9-103">Une interface générique hérite d’au moins deux interfaces génériques, et deux des héritages peuvent entrer en conflit pour certaines valeurs d’arguments de type.</span><span class="sxs-lookup"><span data-stu-id="41ce9-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="41ce9-104">Les instructions suivantes peuvent générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="41ce9-104">The following statements can generate this error.</span></span>  
  
```vb  
Public Interface interfaceA(Of u)  
    Inherits interfaceX(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="41ce9-105">Si `derivedInterface` est construit ou implémenté en fournissant le même type à `t1` et à `t2`, il doit hériter de deux versions de `interfaceX` avec des arguments de type identiques.</span><span class="sxs-lookup"><span data-stu-id="41ce9-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceX` with identical type arguments.</span></span> <span data-ttu-id="41ce9-106">Cela crée une ambiguïté sur la version à laquelle accéder.</span><span class="sxs-lookup"><span data-stu-id="41ce9-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="41ce9-107">**ID d’erreur :** BC32121</span><span class="sxs-lookup"><span data-stu-id="41ce9-107">**Error ID:** BC32121</span></span>

- <span data-ttu-id="41ce9-109">Modifiez l’un des arguments de type fournis à l’interface dérivée pour éviter tout conflit.</span><span class="sxs-lookup"><span data-stu-id="41ce9-109">Change one of the type arguments supplied to the derived interface so that there is no conflict.</span></span>  
  
     <span data-ttu-id="41ce9-110">-ou-</span><span class="sxs-lookup"><span data-stu-id="41ce9-110">-or-</span></span>
