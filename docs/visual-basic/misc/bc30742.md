# <a name="value-valuename1-cannot-be-converted-to-valuename2"></a>La valeur' \<valuename1> 'ne peut pas être convertie en' \<valuename2> '

Une conversion de type a été tentée entre deux types incompatibles.  
  
 **ID d’erreur :** BC30742  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez les types qui font l’objet d’un cast.
