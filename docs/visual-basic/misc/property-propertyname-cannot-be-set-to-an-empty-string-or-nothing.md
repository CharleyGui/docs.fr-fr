# <a name="property-propertyname-cannot-be-set-to-an-empty-string-or-nothing"></a>La propriété \<propertyname> ne peut pas être définie en tant que chaîne vide ou Nothing

La propriété exige une valeur autre qu’une chaîne vide ("") ou `Nothing`. Ce peut être le résultat d’un calcul incorrect de la valeur.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Fournissez une valeur valide.
