# <a name="type-typename-is-imported-from-different-versions-of-assembly-assemblyname"></a>Le type' \<typename> 'est importé à partir de différentes versions de l’assembly' \<assemblyname> '

Le type' \<typename> 'est importé à partir de différentes versions de l’assembly' \<assemblyname> '. Un même projet ne peut pas utiliser plusieurs versions du même assembly.  
  
 Un type a été importé à partir de plusieurs versions du même assembly, créant ainsi une ambiguïté.  
  
 **ID d’erreur :** BC31525  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez les références aux différentes versions du même assembly.
