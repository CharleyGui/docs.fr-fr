# <a name="type-typename-is-imported-from-different-versions-of-assembly-assemblyname"></a>Type '\<nom_type >' est importé à partir de différentes versions d’assembly '\<nom_assembly >'
Type '\<nom_type >' est importé à partir de différentes versions d’assembly '\<nom_assembly >'. Un même projet ne peut pas utiliser plusieurs versions du même assembly.  
  
 Un type a été importé à partir de plusieurs versions du même assembly, créant ainsi une ambiguïté.  
  
 **ID d’erreur :** BC31525  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez les références aux différentes versions du même assembly.
