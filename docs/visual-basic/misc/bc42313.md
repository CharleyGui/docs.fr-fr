# <a name="xml-comment-tag-returns-is-not-permitted-on-a-writeonly-property"></a>La balise de commentaire XML 'returns' n'est pas autorisée pour une propriété 'WriteOnly'
La balise de commentaire XML 'returns' n'est pas autorisée pour une propriété 'WriteOnly'. Le commentaire XML est ignoré.  
  
 Un commentaire XML pour une déclaration de propriété en écriture seule ne peut pas contenir un \<retourne > balise.  
  
 **ID d’erreur :** BC42313  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimer la non prise en charge \<retourne > balise.
