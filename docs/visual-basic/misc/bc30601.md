# <a name="--expected"></a>'-' attendu
La syntaxe de cette ligne est incorrecte.  
  
 **ID d’erreur :** BC30601  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez la syntaxe du code qui a déclenché cette erreur.
