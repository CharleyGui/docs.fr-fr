# <a name="rsetstmt-is-not-declared"></a>' \<rsetstmt> 'n’est pas déclaré

'\<rsetstmt>' n’est pas déclaré. Les instructions RSet ne sont plus prises en charge ; utilisez Microsoft.VisualBasic.RSet à la place.  
  
 Plusieurs fonctions qui étaient intrinsèques à Visual Basic dans les versions précédentes ont été déplacées vers l' <xref:Microsoft.VisualBasic?displayProperty=nameWithType> espace de noms. Leurs fonctionnalités sont ainsi plus largement accessibles à tous les langages de programmation.  
  
 **ID d’erreur :** BC30821  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Utilisez plutôt la fonction `RSet` dans l’espace de noms `Microsoft.VisualBasic` .
