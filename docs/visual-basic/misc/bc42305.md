# <a name="xml-comment-tag-tag-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>La balise de commentaire XML '\<tag>' apparaît plusieurs fois avec des attributs identiques dans le même bloc de commentaire XML

La balise de commentaire XML \<tag> apparaît plusieurs fois avec des attributs identiques dans le même bloc de commentaires XML. Le commentaire XML sera ignoré.  
  
 Le commentaire XML contient plusieurs balises avec des attributs identiques.  
  
 **ID d’erreur :** BC42305  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez la balise en double.
