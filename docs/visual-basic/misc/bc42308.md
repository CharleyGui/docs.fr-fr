# <a name="xml-comment-parameter-must-have-a-name-attribute"></a>Le paramètre de commentaire XML doit avoir un attribut 'name'

Le paramètre de commentaire XML doit avoir un attribut 'name'. Le commentaire XML sera ignoré.  
  
 Il manque un attribut `name` obligatoire dans la balise XML.  
  
 **ID d’erreur :** BC42308  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Fournissez l’attribut `name` nécessaire pour la balise.
