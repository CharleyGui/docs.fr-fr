# <a name="methods-in-a-module-cannot-implement-interface-members"></a>Les méthodes d’un module ne peuvent pas implémenter des membres d’interface
Les membres d’interface doivent être implémentés par des classes ou des structures.  
  
 **ID d’erreur :** BC31083  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Implémentez les membres d’interface à partir de classes ou de structures.
