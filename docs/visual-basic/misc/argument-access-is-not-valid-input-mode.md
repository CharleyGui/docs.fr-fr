# <a name="argument-access-is-not-valid-valid-values-for-input-mode-are-openaccessread-and-openaccessdefault"></a>L’argument 'Access' n’est pas valide. Les valeurs valides pour le mode Input sont 'OpenAccess.Read' et 'OpenAccess.Default'.
Une valeur d’accès n’est pas valide en mode `Input` .  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Remplacez la valeur par `OpenAccess.Read` ou `OpenAccess.Default`.
