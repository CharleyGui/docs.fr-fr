# <a name="project-level-conditional-compilation-constant-is-not-valid-error"></a>Constante de compilation conditionnelle au niveau du projet non valide : \<error>

Une option de compilation conditionnelle non valide est spécifiée pour le projet.  
  
 **ID d’erreur :** BC31031  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Vérifiez les constantes de compilation conditionnelle utilisées avec ce projet.
