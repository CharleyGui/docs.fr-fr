# <a name="class-classname-cannot-be-indexed-because-it-has-no-default-property"></a>La classe\<nom_classe >' ne peut pas être indexée, car elle n’a aucune propriété par défaut
Vous n’avez pas fourni de propriété par défaut à une classe.  
  
 **ID d’erreur :** BC30367  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Spécifiez une propriété en tant que propriété par défaut. Pour cela, commencez sa déclaration par le mot clé `Default` .
