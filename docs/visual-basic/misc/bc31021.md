# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-error"></a>Le compilateur Visual Basic n’est pas en mesure de récupérer à partir de l’erreur suivante : \<error>

Le compilateur Visual Basic n’est pas en mesure de récupérer à partir de l’erreur suivante : <`error`>.  
  
 Certaines fonctionnalités de l’IDE ont été désactivées.  
  
 Une erreur interne s’est produite dans le compilateur de Visual Basic.  
  
 **ID d’erreur :** BC31021  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Enregistrez votre travail et redémarrez Visual Studio.  
  
2. Si l’erreur se reproduit, réinstallez Visual Studio.  
  
3. Si l'erreur persiste après la réinstallation, avertissez les services de support technique Microsoft.
