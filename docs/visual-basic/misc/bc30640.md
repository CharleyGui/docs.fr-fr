# <a name="option-explicit-can-be-followed-only-by-on-or-off"></a>'Option Explicit' ne peut être suivi que par 'On' ou 'Off'

Les instructions`Option Explicit` doivent spécifier `On` ou `Off`. Toutes les autres valeurs ne sont pas valides.  
  
 **ID d’erreur :** BC30640  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Spécifiez `Option Explicit On` ou `Option Explicit Off`.
