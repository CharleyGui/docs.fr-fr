# <a name="unable-to-include-xml-fragment-elementpath-of-file-commentfile"></a>Impossible d’inclure le fragment XML' \<elementpath> 'du fichier' \<commentfile> '
Une balise `<include>` au niveau d’un commentaire XML contient une expression XPath pour un élément qui n’existe pas, ou le fichier contient du code XML incorrect.  
  
 **ID d’erreur :** BC42320  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Vérifiez que le code XML contenu dans le fichier est valide.  
  
2. Vérifiez que le fragment XML existe bien dans le fichier.
