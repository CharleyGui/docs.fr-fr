# <a name="variable-declaration-without-an-as-clause-type-of-object-assumed"></a>Déclaration de variable sans clause 'As' ; type Object pris par défaut

Une déclaration de propriété ne spécifie pas de clause `As` .  
  
 Une clause `As` identifie un type de données à associer à un élément de programmation. Dans une [instruction Dim](../language-reference/statements/dim-statement.md), elle spécifie le type de données de la ou des variables. Si vous n’incluez pas de clause `As` dans l’instruction `Dim` , le type de données par défaut de la variable est `Object`.  
  
 Par défaut, ce message est un avertissement. Pour plus d’informations sur le masquage des avertissements ou leur traitement en tant qu’erreurs, consultez [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID d’erreur :** BC42020  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Incluez une clause `As` dans l’instruction `Dim` pour spécifier le type de données de la variable.
