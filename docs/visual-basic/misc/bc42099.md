# <a name="unused-local-constant-constant"></a>Constante locale inutilisée : ' \<constant> '
Vous avez déclaré une constante locale qui n’est utilisée nulle part dans votre code.  
  
 **ID d’erreur :** BC42099  
  
### <a name="to-remove-this-warning"></a>Pour supprimer cet avertissement  
  
1. Cette erreur est un avertissement. Vous pouvez l’ignorer. Pour supprimer l’avertissement, supprimez la déclaration de constante inutilisée de votre code.
