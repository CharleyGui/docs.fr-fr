# <a name="specifier1-and-specifier2-cannot-be-combined"></a>' \<specifier1> 'et' \<specifier2> 'ne peuvent pas être combinés
Deux spécificateurs incompatibles, tels que `Shadows` et `Overrides` , sont utilisés ensemble.  
  
 **ID d’erreur :** BC31408  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez l’un des deux spécificateurs.
