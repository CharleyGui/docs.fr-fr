# <a name="xml-literal-cannot-appear-here-unless-it-is-enclosed-in-parentheses"></a><span data-ttu-id="6f96a-102">Un littéral XML ne peut pas s’afficher ici à moins qu’il ne soit mis entre parenthèses</span><span class="sxs-lookup"><span data-stu-id="6f96a-102">XML literal cannot appear here unless it is enclosed in parentheses</span></span>

<span data-ttu-id="6f96a-103">Une déclaration de littéral XML est utilisée dans une expression à un emplacement ambigu pour le compilateur Visual Basic.</span><span class="sxs-lookup"><span data-stu-id="6f96a-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="6f96a-104">Autrement dit, le compilateur Visual Basic ne peut pas déterminer si le caractère « inférieur à » (<) est destiné à un opérateur de comparaison ou au début d’un littéral XML.</span><span class="sxs-lookup"><span data-stu-id="6f96a-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="6f96a-105">Le code suivant est fourni à titre d'exemple :</span><span class="sxs-lookup"><span data-stu-id="6f96a-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="6f96a-106">**ID d’erreur :** BC31198</span><span class="sxs-lookup"><span data-stu-id="6f96a-106">**Error ID:** BC31198</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="6f96a-107">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="6f96a-107">To correct this error</span></span>  
  
- <span data-ttu-id="6f96a-108">Placez la déclaration de littéral XML entre parenthèses, comme illustré dans l’exemple suivant :</span><span class="sxs-lookup"><span data-stu-id="6f96a-108">Enclose the XML literal declaration in parentheses, as shown in the following example:</span></span>  
  
```vb  
Dim queryResult = From element In elements _  
                  Where (<sample> Value</sample>) = "Value" _  
                  Select element  
```  
  
- <span data-ttu-id="6f96a-109">Modifiez votre code pour référencer un littéral XML existant, comme illustré dans l’exemple suivant :</span><span class="sxs-lookup"><span data-stu-id="6f96a-109">Modify your code to refer to an existing XML literal, as shown in the following example:</span></span>  
  
```vb  
Dim queryResult = From element In elements _  
                  Where e.<sample>.Value = "Value" _  
                  Select element  
```
