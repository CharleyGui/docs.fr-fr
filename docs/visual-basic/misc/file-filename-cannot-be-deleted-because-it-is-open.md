# <a name="file-filename-cannot-be-deleted-because-it-is-open"></a>Fichier '\<nom_fichier >' ne peut pas être supprimé car il est ouvert
L’utilisateur a tenté de supprimer un fichier ouvert.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Fermez le fichier avant de le supprimer.
