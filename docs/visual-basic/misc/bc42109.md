# <a name="variable-variablename-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>La variable '\<variablename>' est utilisée avant qu’une valeur lui soit affectée (Erreur Visual Basic)
Une variable est utilisée avant qu’une valeur ne lui ait été assignée, ce qui provoque une <xref:System.NullReferenceException>.  
  
 **ID d’erreur :** BC42109  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez qu’une valeur a été attribuée à la variable.
