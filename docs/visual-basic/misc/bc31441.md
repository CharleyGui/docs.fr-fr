# <a name="method-methodname1-must-be-declared-private-in-order-to-implement-partial-method-methodname2"></a>Méthode\<'methodname1>' doit être déclarée "privée" afin\<de mettre en œuvre la méthode partielle 'methodname2>'
L’implémentation d’une méthode partielle doit être déclarée `Private`. Par exemple, le code suivant génère cette erreur.  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub valueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' Implementation of the partial method, with Private missing,
    ' causes this error.
    'Sub valueChanged()  
    '    MsgBox(Value was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 **ID d’erreur :** BC31441  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Utilisez le modificateur d’accès `Private` dans l’implémentation de la méthode partielle, comme indiqué dans l’exemple suivant.  
  
```vb  
Private Sub valueChanged()  
    MsgBox("Value was changed to " & Me.Quantity)  
End Sub  
```
