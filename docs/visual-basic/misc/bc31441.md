<span data-ttu-id="ead18-103">L’implémentation d’une méthode partielle doit être déclarée `Private`.</span><span class="sxs-lookup"><span data-stu-id="ead18-103">The implementation of a partial method must be declared `Private`.</span></span> <span data-ttu-id="ead18-104">Par exemple, le code suivant génère cette erreur.</span><span class="sxs-lookup"><span data-stu-id="ead18-104">For example, the following code causes this error.</span></span>
