# <a name="keyword-is-not-valid-as-an-identifier"></a>Mot clé non valide en tant qu’identificateur

Un mot clé apparaît là où un nom d’élément est exigé. Le contexte nécessite un élément comme une variable ou une procédure.  
  
 **ID d’erreur :** BC30183  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Remplacez le mot clé par un nom d’élément valide.
