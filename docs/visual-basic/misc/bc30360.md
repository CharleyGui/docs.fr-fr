# <a name="property-propertyname-must-be-declared-default-because-it-overrides-a-default-property-on-the-base-name"></a>La propriété' \<propertyname> 'doit être déclarée’default', car elle se substitue à une propriété par défaut sur le' 'de base \<name>

Une instruction `Property` dans votre code remplace une propriété `Default` sur la classe, la structure ou l’interface de base. Elle doit donc être déclarée `Default`.  
  
 **ID d’erreur :** BC30360  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Déclarez la propriété `Default`.
