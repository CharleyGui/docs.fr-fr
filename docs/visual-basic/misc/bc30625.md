# <a name="module-statement-must-end-with-a-matching-end-module"></a>L’instruction 'Module' doit se terminer par un 'End Module' correspondant
Les blocs de module doivent se terminer par une instruction `End Module` .  
  
 **ID d’erreur :** BC30625  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ajoutez une instruction `End Module` à la fin du bloc `Module` .
