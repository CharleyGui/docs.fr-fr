# <a name="argument-not-specified-for-parameter-parametername"></a>Argument non spécifié pour le paramètre '\<nom_paramètre >'
Un argument pour un paramètre obligatoire n’a pas été fourni.  
  
 **ID d’erreur :** BC36569  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Fournissez un argument pour le paramètre spécifié.
