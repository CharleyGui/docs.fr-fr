# <a name="structures-cannot-have-inherits-statements"></a>Les structures ne peuvent pas utiliser d’instructions 'Inherits'
Contrairement aux classes, les structures ne prennent pas en charge l’héritage.  
  
 **ID d’erreur :** BC30628  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez l’instruction `Inherits` de la structure.  
  
- Modifiez la conception de votre application pour qu’elle utilise plutôt des classes.
