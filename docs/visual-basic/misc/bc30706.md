# <a name="expression-or-statement-is-not-valid-in-debug-windows"></a>L’expression ou l’instruction n’est pas valide dans les fenêtres de débogage
Vous ne pouvez pas utiliser d’instructions`Try...Catch...Finally` dans le contexte de débogage.  
  
 **ID d’erreur :** BC30706  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Supprimez les instructions `Try...Catch...Finally` du code de débogage.
