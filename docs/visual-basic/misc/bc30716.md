# <a name="finally-statements-are-not-valid-in-the-immediate-window"></a>Les instructions 'Finally' ne sont pas valides dans la fenêtre Exécution.

Les instructions`Finally` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30716  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez les instructions `Finally` du code de débogage.
