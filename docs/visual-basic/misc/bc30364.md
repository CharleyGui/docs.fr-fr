# <a name="sub-new-cannot-be-declared-modifier"></a>'Sub New' ne peut pas être déclaré '\<modificateur >'
Vous avez tenté de déclarer un `Sub New` avec un modificateur, tel que `Overrides`, qui n’est pas valide.  
  
 **ID d’erreur :** BC30364  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez le modificateur.
