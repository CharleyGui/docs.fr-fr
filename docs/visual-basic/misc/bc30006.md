# <a name="reference-required-to-module-modulename-containing-the-definition-for-event-eventname"></a>Une référence au module' \<modulename> 'contenant la définition de l’événement' 'est requise \<eventname>

Une référence au module' <`modulename`> 'contenant la définition de l’événement' <`eventname`> 'est requise. Ajoutez-en une à votre projet.  
  
 L’événement est défini dans un module qui n’est pas directement référencé dans votre projet. Le compilateur Visual Basic requiert une référence pour éviter toute ambiguïté au cas où l’événement serait défini dans plusieurs modules.  
  
 **ID d’erreur :** BC30006  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ajoutez le nom du module non référencé dans vos références de projet.
