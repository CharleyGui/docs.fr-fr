# <a name="mybase-cannot-be-used-with-method-methodname-because-it-is-declared-mustoverride"></a>'MyBase' ne peut pas être utilisé avec la méthode '\<methodname>', car il est déclaré 'MustOverride'
Vous avez essayé d’utiliser `MyBase` avec une méthode qui a été déclarée `MustOverride`.  
  
 **ID d’erreur :** BC30399  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez la déclaration `MustOverride` .
