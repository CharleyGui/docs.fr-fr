# <a name="method-methodname-is-already-declared-in-interface-interfacename"></a>La méthode' \<methodname> 'est déjà déclarée dans l’interface' \<interfacename> '

Vous avez déclaré une méthode d’interface une seconde fois, provoquant sa duplication.  
  
 **ID d’erreur :** BC30377  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez la déclaration dupliquée.
