# <a name="property-accessors-cannot-be-declared-accessmodifier-in-a-notoverridable-property"></a><span data-ttu-id="7db8b-102">Les accesseurs de propriété ne peuvent pas être déclarés '\<accessmodifier>' dans une propriété 'NotOverridable'.</span><span class="sxs-lookup"><span data-stu-id="7db8b-102">Property accessors cannot be declared '\<accessmodifier>' in a 'NotOverridable' property</span></span>
<span data-ttu-id="7db8b-103">Une [instruction d’extraction](../language-reference/statements/get-statement.md) ou une [instruction Set](../language-reference/statements/set-statement.md) dans une `NotOverridable` propriété comprend le `Private` mot clé.</span><span class="sxs-lookup"><span data-stu-id="7db8b-103">A [Get Statement](../language-reference/statements/get-statement.md) or [Set Statement](../language-reference/statements/set-statement.md) in a `NotOverridable` property includes the `Private` keyword.</span></span>  
  
 <span data-ttu-id="7db8b-104">La ligne de raisonnement ci-dessous explique la raison pour laquelle `NotOverridable` et `Private` ne peuvent pas être associés dans une [Property Statement](../language-reference/statements/property-statement.md):</span><span class="sxs-lookup"><span data-stu-id="7db8b-104">The following line of reasoning explains why `NotOverridable` and `Private` cannot be combined in a [Property Statement](../language-reference/statements/property-statement.md):</span></span>  
  
1. <span data-ttu-id="7db8b-105">Une propriété ou une procédure qui ne substitue pas de propriété ou de procédure de classe de base a le paramètre par défaut [NotOverridable](../language-reference/modifiers/notoverridable.md).</span><span class="sxs-lookup"><span data-stu-id="7db8b-105">A property or procedure that does not override a base class property or procedure has a default setting of [NotOverridable](../language-reference/modifiers/notoverridable.md).</span></span>  
  
2. <span data-ttu-id="7db8b-106">Toutefois, une propriété ou une procédure dans une classe dérivée qui substitue une propriété ou une procédure de classe de base a le paramètre par défaut [Overridable](../language-reference/modifiers/overridable.md).</span><span class="sxs-lookup"><span data-stu-id="7db8b-106">However, a property or procedure in a derived class that overrides a base class property or procedure has a default setting of [Overridable](../language-reference/modifiers/overridable.md).</span></span> <span data-ttu-id="7db8b-107">Pour mettre fin à la hiérarchie de substitution, vous pouvez la déclarer `NotOverridable`.</span><span class="sxs-lookup"><span data-stu-id="7db8b-107">To terminate the hierarchy of overriding, you can declare it `NotOverridable`.</span></span> <span data-ttu-id="7db8b-108">Ce contexte est le seul dans lequel vous pouvez utiliser `NotOverridable`.</span><span class="sxs-lookup"><span data-stu-id="7db8b-108">This is the only context in which you can use `NotOverridable`.</span></span> <span data-ttu-id="7db8b-109">En d’autres termes, vous pouvez utiliser `NotOverridable` uniquement en association avec [Overrides](../language-reference/modifiers/overrides.md).</span><span class="sxs-lookup"><span data-stu-id="7db8b-109">That is, you can use `NotOverridable` only in combination with [Overrides](../language-reference/modifiers/overrides.md).</span></span>  
  
3. <span data-ttu-id="7db8b-110">Si une propriété de classe de base ou une procédure est déclarée [Private](../language-reference/modifiers/private.md), une classe dérivée ne peut pas substituer cette propriété ou procédure car elle ne peut pas y accéder.</span><span class="sxs-lookup"><span data-stu-id="7db8b-110">If a base class property or procedure is declared [Private](../language-reference/modifiers/private.md), a derived class cannot override that property or procedure because it cannot access it.</span></span> <span data-ttu-id="7db8b-111">Pour cette raison, vous ne pouvez pas utiliser `Private` associé avec `Overridable`.</span><span class="sxs-lookup"><span data-stu-id="7db8b-111">Because of this, you cannot use `Private` in combination with `Overridable`.</span></span>  
  
4. <span data-ttu-id="7db8b-112">Pour substituer une propriété ou une procédure, la propriété ou la procédure de substitution doit avoir la même signature, mais également le même niveau d’accès.</span><span class="sxs-lookup"><span data-stu-id="7db8b-112">To override a property or procedure, the overriding property or procedure must have not only the identical signature but also the same access level.</span></span> <span data-ttu-id="7db8b-113">Cela signifie qu’une propriété ou une procédure de substitution ne peut pas spécifier `Private`, car une propriété ou une procédure qui peut être substituée ne peut pas spécifier `Private`.</span><span class="sxs-lookup"><span data-stu-id="7db8b-113">This means that an overriding property or procedure cannot specify `Private`, because an overridable property or procedure cannot specify `Private`.</span></span>  
  
5. <span data-ttu-id="7db8b-114">Étant donné que vous pouvez spécifier `NotOverridable` uniquement sur une propriété ou une procédure de substitution, vous ne pouvez pas l’associer à `Private`.</span><span class="sxs-lookup"><span data-stu-id="7db8b-114">Because you can specify `NotOverridable` only on an overriding property or procedure, you cannot combine it with `Private`.</span></span>  
  
 <span data-ttu-id="7db8b-115">En suivant le même raisonnement, les procédures de propriété (`Get` et `Set`) d’une propriété de substitution ne peuvent pas avoir la valeur `Private`.</span><span class="sxs-lookup"><span data-stu-id="7db8b-115">By the same reasoning, the individual property procedures (`Get` and `Set`) of an overriding property cannot be `Private`.</span></span>
