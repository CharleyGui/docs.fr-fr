# <a name="set-method-cannot-have-more-than-one-parameter"></a>La méthode 'Set' ne peut pas comporter plusieurs paramètres
Plusieurs paramètres sont définis pour des procédures `Set` de propriété.  
  
 **ID d’erreur :** BC31063  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez les paramètres en trop des procédures `Set` de propriété.
