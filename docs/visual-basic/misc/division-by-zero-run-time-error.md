# <a name="division-by-zero-visual-basic-run-time-error"></a>Division par zéro (erreur d’exécution Visual Basic)

Une expression utilisée en tant que diviseur possède une valeur égale à zéro.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Vérifiez l’orthographe des variables de l’expression. Un nom de variable mal orthographié peut créer implicitement une variable numérique dont la valeur est zéro.  
  
2. Vérifiez les opérations précédentes sur les variables de l’expression, surtout celles passées dans la procédure en tant qu’arguments à partir d’autres procédures.
