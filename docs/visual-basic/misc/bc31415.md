# <a name="implementsclause-cannot-implement-typename-because-typename-is-a-reserved-name"></a>'\<clause_implements >' ne peut pas implémenter '\<nom_type >', car '\<typename >' est un nom réservé
Une instruction `Implements` a spécifié un nom réservé.  
  
 **ID d’erreur :** BC31415  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Renommez le membre d’interface qui n’a pas pu être implémenté.
