# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="74602-102">Le paramètre 'ByRef' 'parametername\<>' ne peut pas être utilisé dans une expression lambda</span><span class="sxs-lookup"><span data-stu-id="74602-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>
<span data-ttu-id="74602-103">Une expression lambda déclarée dans un élément `Sub` ou une fonction ne peut pas utiliser les paramètres `ByRef` de cet élément `Sub` ou de cette fonction.</span><span class="sxs-lookup"><span data-stu-id="74602-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="74602-104">Par exemple, le code suivant provoquera cette erreur, car le paramètre `ByRef``n` est utilisé dans l’expression lambda.</span><span class="sxs-lookup"><span data-stu-id="74602-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```vb  
'' Not valid.
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="74602-105">**ID d’erreur :** BC36639</span><span class="sxs-lookup"><span data-stu-id="74602-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="74602-106">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="74602-106">To correct this error</span></span>

    ```vb  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
