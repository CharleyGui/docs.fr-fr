# <a name="assembly-assemblyname-cannot-be-created-because-its-path-is-longer-than-259-characters"></a>Impossible de créer l’assembly '\<assemblyname>', car la longueur de son chemin dépasse 259 caractères

Le chemin de cet assembly dépasse la longueur maximale de 259 caractères.  
  
 **ID d’erreur :** BC31518  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Créez un chemin plus court en utilisant des noms moins longs ou en sélectionnant une hiérarchie de fichiers moins profonde.
