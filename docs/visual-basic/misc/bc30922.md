# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-outside-the-assembly"></a><span data-ttu-id="be5b6-102">' \<derivedtypename > 'ne peut pas hériter de \<Type > ' \<constructedbasetypename > ', car il étend l’accès du type' \<internaltypename > 'en dehors de l’assembly</span><span class="sxs-lookup"><span data-stu-id="be5b6-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' outside the assembly</span></span>
<span data-ttu-id="be5b6-103">Une classe ou une interface dérivée tente d’étendre le niveau d’accès d’un type restreint en l’utilisant comme argument de type d’une classe de base ou d’une interface.</span><span class="sxs-lookup"><span data-stu-id="be5b6-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="be5b6-104">Le code suivant peut générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="be5b6-104">The following code can generate this error.</span></span>  
  
```vb  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="be5b6-105">Le code situé en dehors de l’assembly n’est pas autorisé à accéder à `restrictedStructure`.</span><span class="sxs-lookup"><span data-stu-id="be5b6-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="be5b6-106">Cependant, `derivedClass` est accessible à n’importe quel code qui peut le référencer.</span><span class="sxs-lookup"><span data-stu-id="be5b6-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="be5b6-107">`derivedClass` ne peut donc pas hériter de `baseClass` s’il utilise `restrictedStructure` comme argument de type, car cela peut exposer `restrictedStructure` au code dans n’importe quel assembly.</span><span class="sxs-lookup"><span data-stu-id="be5b6-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="be5b6-108">**ID d’erreur :** BC30922</span><span class="sxs-lookup"><span data-stu-id="be5b6-108">**Error ID:** BC30922</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="be5b6-109">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="be5b6-109">To correct this error</span></span>  
  
- <span data-ttu-id="be5b6-110">Ajustez les niveaux d’accès des classes ou des interfaces pour que le type dérivé ne développe pas le niveau d’accès du type restreint.</span><span class="sxs-lookup"><span data-stu-id="be5b6-110">Adjust the access levels of the classes or interfaces so the derived type does not expand the access level of the restricted type.</span></span>  
  
     <span data-ttu-id="be5b6-111">\- ou -</span><span class="sxs-lookup"><span data-stu-id="be5b6-111">-or-</span></span>  
  
- <span data-ttu-id="be5b6-112">Si vous ne pouvez pas ajuster les niveaux d’accès, n’utilisez pas le type restreint comme argument de type lors de la construction d’une classe de base ou d’une interface.</span><span class="sxs-lookup"><span data-stu-id="be5b6-112">If you cannot adjust the access levels, do not use the restricted type as a type argument when constructing the base class or interface.</span></span>
