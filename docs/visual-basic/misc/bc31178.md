# <a name="expected-closing--for-xml-entity"></a>';' fermant attendu pour une entité XML
Il manque un point-virgule (;) fermant dans une référence d’entité XML (par exemple, `>`).  
  
 **ID d’erreur :** BC31178  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Incluez le point-virgule à la fin de la référence d’entité XML.
