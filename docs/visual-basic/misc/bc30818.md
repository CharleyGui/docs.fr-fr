# <a name="function-is-not-declared"></a>' \<function> 'n’est pas déclaré

'\<function>' n’est pas déclaré. La fonction a été déplacée dans l’espace de noms Microsoft.VisualBasic.  
  
 **ID d’erreur :** BC30818  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Utilisez la fonction appropriée de l’espace de noms `Microsoft.VisualBasic` .
