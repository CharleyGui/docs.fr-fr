# <a name="default-can-be-applied-to-only-one-property-name-in-a-type"></a>'Default' peut être appliqué au nom qu’une seule propriété dans un \<type >
Vous avez tenté d’appliquer `Default` à plusieurs noms de propriétés dans une classe, interface ou structure.  
  
 **ID d’erreur :** BC30359  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez toutes les déclarations `Default` sauf une.
