# <a name="xml-comment-tag-include-must-have-a-attribute-attribute"></a>La balise de commentaire XML 'include' doit avoir un '\<attribut >' attribut
La balise de commentaire XML 'include' doit avoir un \<attribut > attribut. Le commentaire XML est ignoré.  
  
 Un attribut `file` ou `path` , passé à la balise `<`include`>` , est manquant ou incomplet.  
  
 **ID d’erreur :** BC42310  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ajoutez ou complétez l’attribut requis pour la \<inclure > balise.
