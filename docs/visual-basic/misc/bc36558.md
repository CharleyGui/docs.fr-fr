## <a name="example"></a><span data-ttu-id="54b52-111">Exemple</span><span class="sxs-lookup"><span data-stu-id="54b52-111">Example</span></span>

<span data-ttu-id="54b52-112">L’exemple suivant utilise l’attribut `AttributeUsage` pour spécifier les éléments de programme auxquels la nouvelle version de `ExtensionAttribute` peut s’appliquer.</span><span class="sxs-lookup"><span data-stu-id="54b52-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="54b52-113">L’exemple spécifie trois membres de l’énumération `AttributeTargets` : `Assembly`, `Class`et `Method`.</span><span class="sxs-lookup"><span data-stu-id="54b52-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="54b52-114">L’omission de l’un de ces éléments provoque cette erreur.</span><span class="sxs-lookup"><span data-stu-id="54b52-114">The omission of any one of these elements will cause this error.</span></span>

```vb
Namespace System.Runtime.CompilerServices
    <AttributeUsage(AttributeTargets.Assembly Or _
        AttributeTargets.Class Or AttributeTargets.Method)>
    Class ExtensionAttribute
        Inherits System.Attribute
        ' Definitions of methods, fields, and properties.
    End Class
End Namespace
```

<span data-ttu-id="54b52-115">Vous pouvez également autoriser l’application de `ExtensionAttribute` à tous les éléments de programme à l’aide du membre `All` de `AttributeTargets`.</span><span class="sxs-lookup"><span data-stu-id="54b52-115">Alternatively, you could allow `ExtensionAttribute` to apply to all program elements by using the `All` member of `AttributeTargets`.</span></span>

```xml

```

<span data-ttu-id="54b52-116">La suppression de la ligne `AttributeUsage` , comme indiqué dans le code suivant, produit le même résultat.</span><span class="sxs-lookup"><span data-stu-id="54b52-116">Deleting the `AttributeUsage` line, as shown in the following code, produces the same result.</span></span>

```vb
    End Class
End Namespace
```
