# <a name="friend-access-was-denied-to-assembly-if-you-want-the-compiler-to-consider-friend-types-in-assembly-add-the-public-key-information-in-the-project-settings-or-through-a-command-line-switch-to-the-compiler-for-the-project-project"></a>L’accès Friend a été refusé à \<assembly >. Si vous souhaitez que le compilateur considère les types Friend dans \<assembly >, ajoutez les informations de clé publique dans les paramètres du projet ou via un commutateur de ligne de commande au compilateur pour le projet \<project >.

**ID d’erreur :** BC31536
