# <a name="xml-elements-cannot-be-selected-from-type-type"></a><span data-ttu-id="1ac15-102">Les éléments XML ne peuvent pas être sélectionnés à partir du type 'type'</span><span class="sxs-lookup"><span data-stu-id="1ac15-102">XML elements cannot be selected from type 'type'</span></span>

<span data-ttu-id="1ac15-103">Un élément enfant XML a été référencé pour un objet qui n’est pas du type <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>ou `IEnumerable(Of XElement)`.</span><span class="sxs-lookup"><span data-stu-id="1ac15-103">An XML child element has been referenced for an object that is not of type <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, or `IEnumerable(Of XElement)`.</span></span> <span data-ttu-id="1ac15-104">Pour plus d'informations, consultez [XML Child Axis Property](../language-reference/xml-axis/xml-child-axis-property.md).</span><span class="sxs-lookup"><span data-stu-id="1ac15-104">For more information, see [XML Child Axis Property](../language-reference/xml-axis/xml-child-axis-property.md).</span></span>  
  
```vb  
' Generates an error.  
Dim var = "sample text".<child>  
```  
  
 <span data-ttu-id="1ac15-105">**ID d’erreur :** BC36807</span><span class="sxs-lookup"><span data-stu-id="1ac15-105">**Error ID:** BC36807</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="1ac15-106">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="1ac15-106">To correct this error</span></span>  
  
- <span data-ttu-id="1ac15-107">Assurez-vous que l’objet dont vous référencez un attribut est fortement typé en tant que <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>ou `IEnumerable(Of XElement)`.</span><span class="sxs-lookup"><span data-stu-id="1ac15-107">Ensure that the object of which you are referencing an attribute is strongly typed as <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, or `IEnumerable(Of XElement)`.</span></span> <span data-ttu-id="1ac15-108">Vous trouverez ci-dessous un exemple :</span><span class="sxs-lookup"><span data-stu-id="1ac15-108">Following is an example:</span></span>
