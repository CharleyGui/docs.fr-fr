# <a name="membername-cannot-be-declared-shadows-outside-of-a-class-structure-or-interface"></a>'\<nom_membre >' ne peut pas être déclaré 'Shadows' en dehors d’une classe, une structure ou une interface
Le mot clé `Shadows` apparaît dans une déclaration au niveau de l’espace de noms, du module ou du fichier. L’occultation est significative uniquement dans un élément de programmation qui peut hériter d’un élément de base.  
  
 **ID d’erreur :** BC32200  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez le mot clé `Shadows` ou déplacez la déclaration à l’intérieur d’une classe, d’une structure ou d’une interface.
