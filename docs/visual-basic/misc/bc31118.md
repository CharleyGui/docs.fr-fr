# <a name="end-addhandler-must-be-the-first-statement-on-a-line"></a>'End AddHandler' doit être la première instruction sur une ligne
Une instruction `End AddHandler` suit un séparateur d’instruction deux-points (:). `End AddHandler` doit être la seule instruction sur sa ligne source.  
  
 **ID d’erreur :** BC31118  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Séparez les instructions en les plaçant sur des lignes différentes.
