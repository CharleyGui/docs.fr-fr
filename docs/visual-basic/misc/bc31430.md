# <a name="expression-of-type-typename1-can-never-be-of-type-typename2"></a><span data-ttu-id="8c6aa-102">L’expression de type' \<typename1> 'ne peut jamais être de type' \<typename2> '</span><span class="sxs-lookup"><span data-stu-id="8c6aa-102">Expression of type '\<typename1>' can never be of type '\<typename2>'</span></span>
<span data-ttu-id="8c6aa-103">Une expression `TypeOf`...`Is` teste une variable de référence d’objet sur un type de données qu’elle ne peut pas contenir.</span><span class="sxs-lookup"><span data-stu-id="8c6aa-103">A `TypeOf`...`Is` expression tests an object reference variable to a data type that it cannot hold.</span></span>  
  
 <span data-ttu-id="8c6aa-104">Dans certains cas, le compilateur peut déterminer qu’un test `TypeOf`...`Is` ne peut qu’échouer, par exemple s’il n’y a aucune relation d’héritage entre deux classes.</span><span class="sxs-lookup"><span data-stu-id="8c6aa-104">In some cases the compiler can determine that a `TypeOf`...`Is` test can only fail, for example if there is no inheritance relationship between two classes.</span></span>  
  
 <span data-ttu-id="8c6aa-105">Le code suivant peut générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="8c6aa-105">The following code can generate this error.</span></span>  
  
 `Dim refVar as System.Windows.Forms.Form`  
  
 `If TypeOf refVar Is System.Array`  
  
 `End If`  
  
 <span data-ttu-id="8c6aa-106"><xref:System.Windows.Forms.Form> et <xref:System.Array> étant des types sans aucune relation entre eux, le compilateur peut déterminer que l’expression `TypeOf`...`Is` retourne `False` pour toutes les valeurs de `refVar`.</span><span class="sxs-lookup"><span data-stu-id="8c6aa-106">Because <xref:System.Windows.Forms.Form> and <xref:System.Array> are totally unrelated types, the compiler can determine that the `TypeOf`...`Is` expression returns `False` for any value of `refVar`.</span></span>  
  
 <span data-ttu-id="8c6aa-107">**ID d’erreur :** BC31430</span><span class="sxs-lookup"><span data-stu-id="8c6aa-107">**Error ID:** BC31430</span></span>
