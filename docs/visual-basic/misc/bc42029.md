# <a name="catch-block-never-reached-because-name1-inherits-from-name2"></a>Le bloc’Catch’n’a jamais été atteint, car' \<name1> 'hérite de' \<name2> '

Un bloc `Catch` du code n’est jamais atteint.  
  
 Par défaut, ce message est un avertissement. Pour plus d’informations sur le masquage des avertissements ou le traitement des avertissements en tant qu’erreurs, consultez [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID d’erreur :** BC42029  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez le bloc `Catch` redondant.
