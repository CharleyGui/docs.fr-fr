# <a name="type-statements-are-no-longer-supported-use-structure-statements-instead"></a>Les instructions 'Type' ne sont plus prises en charge ; utilisez les instructions 'Structure' à la place
Vous ne pouvez plus utiliser le mot clé `Type` pour définir des types définis par l’utilisateur.  
  
 **ID d’erreur :** BC30802  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Utilisez l’instruction `Structure` pour définir un nouveau type.
