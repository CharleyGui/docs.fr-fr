# <a name="definition-of-method-method-is-not-accessible-in-this-context"></a>Définition de méthode \<méthode > n’est pas accessible dans ce contexte
Une fonction d’agrégation qui n’existe pas a été incluse avec l’opérateur `Into` .  
  
 **ID d’erreur :** BC36594  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez l’orthographe du nom de la fonction d’agrégation et apportez les corrections nécessaires.  
  
- Ajoutez la méthode comme fonction d’agrégation personnalisée (méthode d’extension). Pour plus d’informations sur la création d’une fonction d’agrégation personnalisée, consultez [Aggregate, Clause](../../visual-basic/language-reference/queries/aggregate-clause.md).
