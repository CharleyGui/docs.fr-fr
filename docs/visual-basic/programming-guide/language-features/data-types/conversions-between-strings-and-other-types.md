 <span data-ttu-id="18866-112">Vous pouvez utiliser la fonction `Format` pour convertir un nombre en une chaîne mise en forme, ce qui peut inclure non seulement les chiffres appropriés, mais également la mise en forme des symboles tels qu’un signe monétaire (comme `$`), des séparateurs de milliers ou des *symboles de groupement de chiffres* (tels que `,`) et un séparateur décimal (par exemple, `.`).</span><span class="sxs-lookup"><span data-stu-id="18866-112">You can use the `Format` function to convert a number to a formatted string, which can include not only the appropriate digits but also formatting symbols such as a currency sign (such as `$`), thousands separators or *digit grouping symbols* (such as `,`), and a decimal separator (such as `.`).</span></span> <span data-ttu-id="18866-113">`Format` utilise automatiquement les symboles appropriés en fonction des paramètres **régionaux** spécifiés dans le **panneau**de configuration Windows.</span><span class="sxs-lookup"><span data-stu-id="18866-113">`Format` automatically uses the appropriate symbols according to the **Regional Options** settings specified in the Windows **Control Panel**.</span></span>
