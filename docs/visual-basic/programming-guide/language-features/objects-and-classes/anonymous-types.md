## <a name="an-anonymous-type-definition"></a><span data-ttu-id="4f38d-180">Définition de type anonyme</span><span class="sxs-lookup"><span data-stu-id="4f38d-180">An Anonymous Type Definition</span></span>

 <span data-ttu-id="4f38d-182">Si le type anonyme contient au moins une propriété de clé, la définition remplace trois membres hérités de <xref:System.Object>: <xref:System.Object.Equals%2A>, <xref:System.Object.GetHashCode%2A>et <xref:System.Object.ToString%2A>.</span><span class="sxs-lookup"><span data-stu-id="4f38d-182">If the anonymous type contains at least one key property, the definition overrides three members inherited from <xref:System.Object>: <xref:System.Object.Equals%2A>, <xref:System.Object.GetHashCode%2A>, and <xref:System.Object.ToString%2A>.</span></span> <span data-ttu-id="4f38d-183">Le code produit pour tester l’égalité et déterminer la valeur de code de hachage ne prend en compte que les propriétés de clé.</span><span class="sxs-lookup"><span data-stu-id="4f38d-183">The code produced for testing equality and determining the hash code value considers only the key properties.</span></span> <span data-ttu-id="4f38d-184">Si le type anonyme ne contient pas de propriétés de clé, seul <xref:System.Object.ToString%2A> est substitué.</span><span class="sxs-lookup"><span data-stu-id="4f38d-184">If the anonymous type contains no key properties, only <xref:System.Object.ToString%2A> is overridden.</span></span> <span data-ttu-id="4f38d-185">Les propriétés explicitement nommées d’un type anonyme ne peuvent pas entrer en conflit avec ces méthodes générées.</span><span class="sxs-lookup"><span data-stu-id="4f38d-185">Explicitly named properties of an anonymous type cannot conflict with these generated methods.</span></span> <span data-ttu-id="4f38d-186">Autrement dit, vous ne pouvez pas utiliser `.Equals`, `.GetHashCode`ou `.ToString` pour nommer une propriété.</span><span class="sxs-lookup"><span data-stu-id="4f38d-186">That is, you cannot use `.Equals`, `.GetHashCode`, or `.ToString` to name a property.</span></span>
