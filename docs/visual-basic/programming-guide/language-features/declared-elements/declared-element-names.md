## <a name="rules"></a><span data-ttu-id="a422f-104">Règles</span><span class="sxs-lookup"><span data-stu-id="a422f-104">Rules</span></span>  
 <span data-ttu-id="a422f-105">Un nom d’élément dans Visual Basic doit respecter les règles suivantes :</span><span class="sxs-lookup"><span data-stu-id="a422f-105">An element name in Visual Basic must observe the following rules:</span></span>

 `xyz$wv`

## <a name="case-sensitivity-in-names"></a><span data-ttu-id="a422f-133">Respecte la casse dans les noms</span><span class="sxs-lookup"><span data-stu-id="a422f-133">Case Sensitivity in Names</span></span>
