# <a name="overload-resolution-visual-basic"></a><span data-ttu-id="9cf20-102">Résolution de surcharge (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="9cf20-102">Overload Resolution (Visual Basic)</span></span>

1. <span data-ttu-id="9cf20-105">**Accessibilité.**</span><span class="sxs-lookup"><span data-stu-id="9cf20-105">**Accessibility.**</span></span> <span data-ttu-id="9cf20-106">Il élimine toute surcharge avec un niveau d’accès qui empêche le code d’appel de l’appeler.</span><span class="sxs-lookup"><span data-stu-id="9cf20-106">It eliminates any overload with an access level that prevents the calling code from calling it.</span></span>  
  
2. <span data-ttu-id="9cf20-107">**Nombre de paramètres.**</span><span class="sxs-lookup"><span data-stu-id="9cf20-107">**Number of Parameters.**</span></span> <span data-ttu-id="9cf20-108">Il élimine toute surcharge qui définit un nombre différent de paramètres que ceux fournis dans l’appel.</span><span class="sxs-lookup"><span data-stu-id="9cf20-108">It eliminates any overload that defines a different number of parameters than are supplied in the call.</span></span>

     <span data-ttu-id="9cf20-113">Dans cette étape, il élimine toute surcharge pour laquelle les types de données des arguments d’appel ne peuvent pas être convertis en types de paramètres définis dans la surcharge.</span><span class="sxs-lookup"><span data-stu-id="9cf20-113">In this step, it eliminates any overload for which the data types of the calling arguments cannot be converted to the parameter types defined in the overload.</span></span>

5. <span data-ttu-id="9cf20-117">**Moins d’élargissement.**</span><span class="sxs-lookup"><span data-stu-id="9cf20-117">**Least Widening.**</span></span> <span data-ttu-id="9cf20-118">Le compilateur considère les surcharges restantes par paires.</span><span class="sxs-lookup"><span data-stu-id="9cf20-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="9cf20-119">Pour chaque paire, il compare les types de données des paramètres définis.</span><span class="sxs-lookup"><span data-stu-id="9cf20-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="9cf20-120">Si les types dans l’une des surcharges s’élargissent tous aux types correspondants dans l’autre, le compilateur élimine ce dernier.</span><span class="sxs-lookup"><span data-stu-id="9cf20-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="9cf20-121">Autrement dit, il conserve la surcharge qui nécessite le moins d’élargissement.</span><span class="sxs-lookup"><span data-stu-id="9cf20-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6. <span data-ttu-id="9cf20-122">**Candidat unique.**</span><span class="sxs-lookup"><span data-stu-id="9cf20-122">**Single Candidate.**</span></span> <span data-ttu-id="9cf20-123">Il continue d’envisager des surcharges par paires jusqu’à ce qu’il ne reste qu’une surcharge, et il résout l’appel à cette surcharge.</span><span class="sxs-lookup"><span data-stu-id="9cf20-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="9cf20-124">Si le compilateur ne peut pas réduire les surcharges à un seul candidat, il génère une erreur.</span><span class="sxs-lookup"><span data-stu-id="9cf20-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>
