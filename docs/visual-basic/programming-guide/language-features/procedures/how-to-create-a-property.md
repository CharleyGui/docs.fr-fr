### <a name="to-create-a-get-procedure-that-retrieves-a-property-value"></a><span data-ttu-id="719a9-116">Pour créer une procédure d’extraction qui récupère une valeur de propriété</span><span class="sxs-lookup"><span data-stu-id="719a9-116">To create a Get procedure that retrieves a property value</span></span>

### <a name="to-create-a-set-procedure-that-writes-a-propertys-value"></a><span data-ttu-id="719a9-124">Pour créer une procédure Set qui écrit la valeur d’une propriété</span><span class="sxs-lookup"><span data-stu-id="719a9-124">To create a Set procedure that writes a property's value</span></span>

3. <span data-ttu-id="719a9-130">Placez les instructions de code pour stocker une valeur dans la propriété entre `Set` les `End Set` instructions et.</span><span class="sxs-lookup"><span data-stu-id="719a9-130">Place the code statements to store a value in the property between the `Set` and `End Set` statements.</span></span> <span data-ttu-id="719a9-131">Ce code peut inclure d’autres calculs et manipulations de données, en plus de la validation et du stockage de la valeur de la propriété.</span><span class="sxs-lookup"><span data-stu-id="719a9-131">This code can include other calculations and data manipulations in addition to validating and storing the property's value.</span></span>  
  
4. <span data-ttu-id="719a9-132">Utilisez le paramètre value pour accepter la valeur fournie par le code appelant.</span><span class="sxs-lookup"><span data-stu-id="719a9-132">Use the value parameter to accept the value supplied by the calling code.</span></span> <span data-ttu-id="719a9-133">Vous pouvez soit stocker cette valeur directement dans une instruction d’assignation, soit l’utiliser dans une expression pour calculer la valeur interne à stocker.</span><span class="sxs-lookup"><span data-stu-id="719a9-133">You can either store this value directly in an assignment statement, or use it in an expression to calculate the internal value to be stored.</span></span>  
  
 <span data-ttu-id="719a9-134">Vous devez écrire une `Set` procédure pour une propriété en lecture-écriture et pour une propriété en écriture seule.</span><span class="sxs-lookup"><span data-stu-id="719a9-134">You must write a `Set` procedure for a read-write property and for a write-only property.</span></span> <span data-ttu-id="719a9-135">Vous ne devez pas définir une `Set` procédure pour une propriété en lecture seule.</span><span class="sxs-lookup"><span data-stu-id="719a9-135">You must not define a `Set` procedure for a read-only property.</span></span>
