## <a name="perform-a-left-outer-join-by-using-the-group-join-clause"></a><span data-ttu-id="f7763-124">Effectuer une jointure externe gauche à l’aide de la clause Group Join</span><span class="sxs-lookup"><span data-stu-id="f7763-124">Perform a Left Outer Join by Using the Group Join Clause</span></span>
