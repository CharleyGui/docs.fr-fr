# <a name="tutorial-manipulating-content-in-a-wordprocessingml-document-visual-basic"></a>Didacticiel : manipulation de contenu dans un document WordprocessingML (Visual Basic)
Ce didacticiel montre comment appliquer l'approche transformationnelle fonctionnelle et LINQ to XML pour manipuler des documents XML. Les Visual Basic exemples interrogent et manipulent des informations dans des documents WordprocessingML Office Open XML enregistrés par Microsoft Word.  
  
 Pour plus d’informations, consultez le [blog d’Eric White](http://www.ericwhite.com).
