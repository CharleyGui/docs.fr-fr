# <a name="programming-guide-linq-to-xml-visual-basic"></a>Programming Guide (LINQ to XML) (Visual Basic)
Cette section fournit des informations conceptuelles et de procédure sur la programmation avec [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)].  
  
## <a name="who-should-read-this-documentation"></a>À qui cette documentation est-elle destinée ?  
 This documentation targets developers who already understand Visual Basic and some basic aspects of the .NET Framework.  
  
 L’objectif de cette documentation est de simplifier l’utilisation de [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] pour tous les développeurs. [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] facilite la programmation XML. Son utilisation n'est pas restreinte aux développeurs expérimentés.  
  
 [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] s’appuie essentiellement sur les classes génériques. Therefore, is very important that you understand the use of generic types and classes. For more information, see [Generic Types in Visual Basic](../../../../visual-basic/programming-guide/language-features/data-types/generic-types.md).
