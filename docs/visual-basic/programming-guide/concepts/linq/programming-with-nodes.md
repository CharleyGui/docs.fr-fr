 <span data-ttu-id="d1e3b-120">Cela est illustré par l'exemple suivant :</span><span class="sxs-lookup"><span data-stu-id="d1e3b-120">The following example demonstrates this:</span></span>

### <a name="empty-text-nodes-are-possible"></a><span data-ttu-id="d1e3b-122">Des nœuds de texte vides sont possibles</span><span class="sxs-lookup"><span data-stu-id="d1e3b-122">Empty Text Nodes are Possible</span></span>

### <a name="xpath-axis-methods-do-not-return-child-white-space-of-xdocument"></a><span data-ttu-id="d1e3b-137">Les méthodes d'axe XPath ne retournent pas les espaces blancs enfants de XDocument</span><span class="sxs-lookup"><span data-stu-id="d1e3b-137">XPath Axis Methods Do Not Return Child White Space of XDocument</span></span>
