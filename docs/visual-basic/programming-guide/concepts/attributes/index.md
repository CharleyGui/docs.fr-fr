## <a name="using-attributes"></a><span data-ttu-id="dd57c-116">Utilisation d'attributs</span><span class="sxs-lookup"><span data-stu-id="dd57c-116">Using Attributes</span></span>

 <span data-ttu-id="dd57c-121">Une méthode avec l’attribut <xref:System.Runtime.InteropServices.DllImportAttribute> est déclarée comme suit :</span><span class="sxs-lookup"><span data-stu-id="dd57c-121">A method with the attribute <xref:System.Runtime.InteropServices.DllImportAttribute> is declared like this:</span></span>
