# <a name="mycomputerclock-object"></a>My.Computer.Clock, objet

Fournit des propriétés permettant d’accéder à l’heure locale actuelle et à l’heure UTC (Universal Coordinated Time), équivalent à l’heure GMT (Greenwich Mean Time), à partir de l’horloge système.  
  
## <a name="remarks"></a>Notes  

 Pour plus d’informations sur les méthodes et les propriétés de l’objet `My.Computer.Clock` , consultez <xref:Microsoft.VisualBasic.Devices.Clock>.  
  
## <a name="requirements"></a>Spécifications  

 **Espace de noms :** <xref:Microsoft.VisualBasic.Devices>  
  
 **Classe :**<xref:Microsoft.VisualBasic.Devices.Clock>  
  
 **Assembly :** Visual Basic la bibliothèque Runtime (dans Microsoft.VisualBasic.dll)
