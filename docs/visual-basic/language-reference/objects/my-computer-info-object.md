# <a name="mycomputerinfo-object"></a>My.Computer.Info, objet

Fournit des propriétés permettant d’obtenir des informations sur la mémoire, les assemblys chargés, le nom et le système d’exploitation de l’ordinateur.  
  
## <a name="remarks"></a>Notes  

 Pour plus d’informations sur les propriétés de l' `My.Computer.Info` objet, consultez <xref:Microsoft.VisualBasic.Devices.ComputerInfo> .  
  
## <a name="requirements"></a>Spécifications  

 **Espace de noms :** <xref:Microsoft.VisualBasic.Devices>  
  
 **Classe :**<xref:Microsoft.VisualBasic.Devices.ComputerInfo>  
  
 **Assembly :** Visual Basic la bibliothèque Runtime (dans Microsoft.VisualBasic.dll)
