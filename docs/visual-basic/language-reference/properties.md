# <a name="properties-visual-basic"></a>Propriétés (Visual Basic)

Cette page répertorie les propriétés qui sont membres de Visual Basic modules. Les autres propriétés qui sont membres d’objets Visual Basic spécifiques sont répertoriées dans [objets](objects/index.md).  
  
## <a name="visual-basic-properties"></a>Propriétés de la Visual Basic  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.DateAndTime.DateString%2A>|Retourne ou définit une `String` valeur représentant la date actuelle en fonction de votre système.|  
|<xref:Microsoft.VisualBasic.DateAndTime.Now%2A>|Retourne une `Date` valeur contenant la date et l’heure actuelles en fonction de votre système.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngine%2A>|Retourne une `String` représentant le runtime en cours d'utilisation.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngineBuildVersion%2A>|Retourne un `Integer` contenant le numéro de version de build du runtime en cours d’utilisation.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngineMajorVersion%2A>|Retourne un `Integer` contenant le numéro de version principale du runtime en cours d’utilisation.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngineMinorVersion%2A>|Retourne un `Integer` contenant le numéro de version mineure du runtime en cours d’utilisation.|  
|<xref:Microsoft.VisualBasic.DateAndTime.TimeOfDay%2A>|Retourne ou définit une valeur `Date` contenant l'heure en cours indiquée par votre système.|  
|<xref:Microsoft.VisualBasic.DateAndTime.Timer%2A>|Retourne une valeur de type `Double` représentant le nombre de secondes écoulées depuis minuit.|  
|<xref:Microsoft.VisualBasic.DateAndTime.TimeString%2A>|Retourne ou définit une `String` valeur représentant l’heure actuelle de la journée en fonction de votre système.|  
|<xref:Microsoft.VisualBasic.DateAndTime.Today%2A>|Retourne ou définit une valeur `Date` représentant la date actuelle indiquée par votre système.|
