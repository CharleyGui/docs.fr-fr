# <a name="unable-to-link-to-resource-file-filename-error-message"></a>Impossible de créer un lien vers le fichier de ressources «\<filename > » : \<message d’erreur >
Le compilateur Visual Basic appelle Assembly Linker (al. exe, également appelé ALink) pour générer un assembly avec un manifeste. L'éditeur de liens a signalé une erreur de liaison à un fichier de ressources COM+ natif à partir de l'assembly.  
  
 **ID d’erreur :** BC30144  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Examinez le message d’erreur cité et consultez la rubrique [al. exe](../../../framework/tools/al-exe-assembly-linker.md). pour obtenir plus d’explications et de conseils.  
  
2. Si l'erreur persiste, rassemblez des informations sur ses circonstances et avertissez les services de support technique Microsoft.
