# <a name="device-io-error"></a>Erreur d'E/S de périphérique
Une erreur d’entrée ou de sortie s’est produite pendant que votre programme utilisait un appareil tel qu’une imprimante ou un lecteur de disque.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez que le périphérique fonctionne correctement, puis recommencez l’opération.
