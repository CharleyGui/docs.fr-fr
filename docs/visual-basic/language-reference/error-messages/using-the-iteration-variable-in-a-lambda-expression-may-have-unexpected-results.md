 <span data-ttu-id="2ddc5-105">Cet avertissement s’affiche lorsque vous utilisez une variable d’itération de boucle dans une expression lambda qui est déclarée à l’intérieur de la boucle.</span><span class="sxs-lookup"><span data-stu-id="2ddc5-105">This warning appears when you use a loop iteration variable in a lambda expression that is declared inside the loop.</span></span> <span data-ttu-id="2ddc5-106">Par exemple, l’exemple suivant provoque l’avertissement apparaît.</span><span class="sxs-lookup"><span data-stu-id="2ddc5-106">For example, the following example causes the warning to appear.</span></span>  
  
```vb

    ' The warning is given for the use of i.  
    Dim exampleFunc As Func(Of Integer) = Function() i  
Next  
```  
  
 <span data-ttu-id="2ddc5-107">L’exemple suivant montre les résultats inattendus peuvent se produire.</span><span class="sxs-lookup"><span data-stu-id="2ddc5-107">The following example shows the unexpected results that might occur.</span></span>  
  
```vb  
Module Module1  
    Sub Main()  
        Dim array1 As Func(Of Integer)() = New Func(Of Integer)(4) {}  
  
        For i As Integer = 0 To 4  
            array1(i) = Function() i  
        Next  
  
        For Each funcElement In array1  
            System.Console.WriteLine(funcElement())  
        Next  
  
    End Sub  
End Module  
```

 `5`  
  
 `5`  
  
 `5`  
  
 `5`  
  
 `5`  
  
 <span data-ttu-id="2ddc5-111">Par défaut, ce message est un avertissement.</span><span class="sxs-lookup"><span data-stu-id="2ddc5-111">By default, this message is a warning.</span></span> <span data-ttu-id="2ddc5-112">Pour plus d’informations sur le masquage des avertissements ou le traitement des avertissements en tant qu’erreurs, consultez [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).</span><span class="sxs-lookup"><span data-stu-id="2ddc5-112">For more information about hiding warnings or treating warnings as errors, see [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).</span></span>  
  
 <span data-ttu-id="2ddc5-113">**ID d’erreur :** BC42324</span><span class="sxs-lookup"><span data-stu-id="2ddc5-113">**Error ID:** BC42324</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="2ddc5-114">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="2ddc5-114">To correct this error</span></span>  
  
- <span data-ttu-id="2ddc5-115">Affectez la valeur de la variable d’itération à une variable locale et utilisez la variable locale dans l’expression lambda.</span><span class="sxs-lookup"><span data-stu-id="2ddc5-115">Assign the value of the iteration variable to a local variable, and use the local variable in the lambda expression.</span></span>  
  
```vb  
Module Module1  
    Sub Main()  
        Dim array1 As Func(Of Integer)() = New Func(Of Integer)(4) {}  
  
        For i As Integer = 0 To 4  
            Dim j = i  
            array1(i) = Function() j  
        Next  
  
        For Each funcElement In array1  
            System.Console.WriteLine(funcElement())  
        Next  
  
    End Sub  
End Module  
```
