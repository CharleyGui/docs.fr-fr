# <a name="labels-that-are-numbers-must-be-followed-by-colons"></a>Les étiquettes qui sont des nombres doivent être suivies du signe deux-points

Les numéros de ligne suivent les mêmes règles que les autres types d’étiquettes et doivent contenir un signe deux-points.  
  
 **ID d’erreur :** BC30801  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Placez le nombre suivi d’un signe deux-points au début d’une ligne de code. par exemple :  
  
    ```vb  
    400:    X += 1  
    ```
