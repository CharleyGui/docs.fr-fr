# <a name="constant-expression-not-representable-in-type-typename"></a>L'expression constante n'est pas représentable dans le type '\<typename>'

Vous essayez d’évaluer une constante qui ne tient pas dans le type cible, généralement parce qu’elle déborde de la plage.  
  
 **ID d’erreur :** BC30439  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Remplacez le type cible par un type qui peut gérer la constante.
