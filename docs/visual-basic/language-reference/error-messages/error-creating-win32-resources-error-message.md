# <a name="error-creating-win32-resources-error-message"></a>Erreur lors de la création de ressources Win32 : \<error message>

Le compilateur Visual Basic appelle Assembly Linker (Al.exe, également appelé ALink) pour générer un assembly avec un manifeste. L'éditeur de liens a signalé une erreur pendant la création d'une ressource en mémoire. L'environnement peut poser problème ou votre ordinateur peut manquer de mémoire.  
  
 **ID d’erreur :** BC30136  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Examinez le message d’erreur cité et consultez la rubrique [Al.exe](../../../framework/tools/al-exe-assembly-linker.md). pour obtenir plus d’explications et de conseils.  
  
2. Si l'erreur persiste, rassemblez des informations sur ses circonstances et avertissez les services de support technique Microsoft.
