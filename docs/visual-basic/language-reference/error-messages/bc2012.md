# <a name="cant-open-filename-for-writing"></a>Impossible d’ouvrir le fichier'\<> 'en écriture
Le fichier spécifié ne peut pas être ouvert pour l’écriture, peut-être parce qu’il a déjà été ouvert.  
  
 **ID d’erreur :** BC2012  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Fermez le fichier et rouvrez-le.  
  
2. Vérifiez les autorisations du fichier.
