# <a name="bc30722-evaluation-of-expression-or-statement-timed-out"></a>BC30722 : l’évaluation de l’expression ou de l’instruction a expiré

L’évaluation d’une expression ne s’est pas terminée en temps opportun.

 **ID d’erreur :** BC30722

## <a name="to-correct-this-error"></a>Pour corriger cette erreur

1. Vérifiez que le code entré est correct.

2. Simplifiez votre expression pour qu’elle prenne moins de temps à s’exécuter.
