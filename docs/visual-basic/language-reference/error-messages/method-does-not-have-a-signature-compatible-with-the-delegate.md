# <a name="method-does-not-have-a-signature-compatible-with-the-delegate"></a>La méthode n'a pas de signature compatible avec le délégué
Il existe une incompatibilité entre les signatures de la méthode et le délégué que vous essayez d’utiliser. L’instruction `Delegate` définit les types de paramètre et les types de retour d’une classe déléguée. Toute procédure ayant des paramètres correspondants de types et de retours compatibles peut être utilisée pour créer une instance de ce type délégué.  
  
 **ID d’erreur**: BC36563
