# <a name="bc30698-unable-to-write-temporary-file-because-temporary-path-is-not-available"></a>BC30698 : impossible d’écrire le fichier temporaire, car le chemin d’accès temporaire n’est pas disponible

Visual Basic n’a pas pu déterminer le chemin d’accès de stockage des fichiers temporaires.

 **ID d’erreur :** BC30698

## <a name="to-correct-this-error"></a>Pour corriger cette erreur

1. Démarrez Visual Studio.

2. Si le problème persiste, réinstallez Visual Studio.
