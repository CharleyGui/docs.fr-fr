# <a name="too-many-files"></a>Trop de fichiers
Soit des fichiers supplémentaires ont été créés dans le répertoire racine que le système d’exploitation le permet, soit des fichiers supplémentaires ont été ouverts que le nombre spécifié dans le paramètre **Files =** de votre configuration. Fichier SYS.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Si votre programme ouvre, ferme ou enregistre des fichiers dans le répertoire racine, modifiez votre programme afin qu’il utilise un sous-répertoire.  
  
2. Augmentez le nombre de fichiers spécifiés dans votre configuration de **fichiers =** . SYS, puis redémarrez votre ordinateur.
