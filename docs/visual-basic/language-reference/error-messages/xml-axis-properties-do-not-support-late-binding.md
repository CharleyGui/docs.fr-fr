# <a name="bc31168-xml-axis-properties-do-not-support-late-binding"></a>BC31168 : les propriétés d’axe XML ne prennent pas en charge la liaison tardive

Une propriété d’axe XML a été référencée pour un objet non typé.

 **ID d’erreur :** BC31168

## <a name="to-correct-this-error"></a>Pour corriger cette erreur

- Assurez-vous que l’objet est un objet de type fort <xref:System.Xml.Linq.XElement> avant de référencer la propriété d’axe XML.
