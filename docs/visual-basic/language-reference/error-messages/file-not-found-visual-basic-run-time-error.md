# <a name="file-not-found-visual-basic-run-time-error"></a>Fichier introuvable (erreur d'exécution Visual Basic)

Le fichier est introuvable à l’emplacement spécifié. L’erreur a les causes possibles suivantes :

- Une instruction fait référence à un fichier qui n’existe pas.

- Une tentative d’appel d’une procédure dans une bibliothèque de liens dynamiques (DLL) a été effectuée, mais la bibliothèque spécifiée dans la `Lib` clause de l' `Declare` instruction est introuvable.

- Vous avez tenté d’ouvrir un projet ou de charger un fichier texte qui n’existe pas.

## <a name="to-correct-this-error"></a>Pour corriger cette erreur

- Vérifiez l’orthographe du nom du fichier et de la spécification du chemin d’accès.
