# <a name="property-or-method-not-found"></a>Propriété ou méthode introuvable
La méthode de l’objet référencé ou de la propriété de l’objet n’est pas définie.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vous avez peut-être mal orthographié le nom de l’objet. Pour afficher les propriétés et méthodes sont définies pour un objet, afficher l’Explorateur d’objets. Sélectionnez la bibliothèque d’objets appropriée pour afficher la liste des propriétés et méthodes disponibles.
