# <a name="property-or-method-not-found"></a>Propriété ou méthode introuvable
La méthode d’objet ou la propriété d’objet référencée n’est pas définie.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vous avez peut-être mal orthographié le nom de l’objet. Pour connaître les propriétés et les méthodes définies pour un objet, affichez l’Explorateur d’objets. Sélectionnez la bibliothèque d’objets appropriée pour afficher la liste des propriétés et méthodes disponibles.
