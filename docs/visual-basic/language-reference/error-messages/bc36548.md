# <a name="bc36548-cannot-convert-anonymous-type-to-an-expression-tree-because-a-property-of-the-type-is-used-to-initialize-another-property"></a><span data-ttu-id="d73f5-102">BC36548 : impossible de convertir le type anonyme en arborescence de l’expression, car une propriété du type est utilisée pour initialiser une autre propriété</span><span class="sxs-lookup"><span data-stu-id="d73f5-102">BC36548: Cannot convert anonymous type to an expression tree because a property of the type is used to initialize another property</span></span>

<span data-ttu-id="d73f5-103">Le compilateur n’accepte pas la conversion d’un type anonyme en arborescence de l’expression lorsqu’une propriété du type anonyme est utilisée pour initialiser une autre propriété du type anonyme.</span><span class="sxs-lookup"><span data-stu-id="d73f5-103">The compiler does not accept conversion of an anonymous type to an expression tree when one property of the anonymous type is used to initialize another property of the anonymous type.</span></span> <span data-ttu-id="d73f5-104">Par exemple, dans le code suivant, `Prop1` est déclaré dans la liste d’initialisation, puis utilisé comme valeur initiale pour `Prop2` .</span><span class="sxs-lookup"><span data-stu-id="d73f5-104">For example, in the following code, `Prop1` is declared in the initialization list and then used as the initial value for `Prop2`.</span></span>

<span data-ttu-id="d73f5-105">**ID d’erreur :** BC36548</span><span class="sxs-lookup"><span data-stu-id="d73f5-105">**Error ID:** BC36548</span></span>

## <a name="example"></a><span data-ttu-id="d73f5-106"> Exemple</span><span class="sxs-lookup"><span data-stu-id="d73f5-106">Example</span></span>

```vb
Module M2

    Sub ExpressionExample(Of T)(x As Expressions.Expression(Of Func(Of T)))
    End Sub

    Sub Main()
        ' The following line causes the error.
        ExpressionExample(Function() New With {.Prop1 = 2, .Prop2 = .Prop1})
    End Sub
End Module
```

## <a name="to-correct-this-error"></a><span data-ttu-id="d73f5-107">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="d73f5-107">To correct this error</span></span>

<span data-ttu-id="d73f5-108">Assignez la valeur initiale de `Prop1` à une variable locale.</span><span class="sxs-lookup"><span data-stu-id="d73f5-108">Assign the initial value for `Prop1` to a local variable.</span></span> <span data-ttu-id="d73f5-109">Affectez cette variable à `Prop1` et à `Prop2` , comme illustré dans le code suivant :</span><span class="sxs-lookup"><span data-stu-id="d73f5-109">Assign that variable to both `Prop1` and `Prop2`, as shown in the following code:</span></span>

```vb
Sub Main()
    Dim temp = 2
    ExpressionExample(Function() New With {.Prop1 = temp, .Prop2 = temp})
End Sub
```
