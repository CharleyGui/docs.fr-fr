# <a name="bc31084-events-cannot-be-declared-with-a-delegate-type-that-has-a-return-type"></a>BC31084 : les événements ne peuvent pas être déclarés avec un type délégué ayant un type de retour

Un délégué a été spécifié pour une procédure de fonction.

 **ID d’erreur :** BC31084

## <a name="to-correct-this-error"></a>Pour corriger cette erreur

- Spécifiez un délégué pour une `Sub` procédure.
