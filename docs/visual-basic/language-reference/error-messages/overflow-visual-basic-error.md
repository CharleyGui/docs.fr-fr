# <a name="bc30036-overflow-visual-basic-error"></a>BC30036 : dépassement de capacité (erreur Visual Basic)

Un littéral représente une valeur en dehors des limites du type de données auquel il est assigné.

 **ID d’erreur :** BC30036

## <a name="to-correct-this-error"></a>Pour corriger cette erreur

- Consultez la plage de valeurs pour le type de données cible et réécrivez le littéral pour qu’il soit conforme à cette plage.
