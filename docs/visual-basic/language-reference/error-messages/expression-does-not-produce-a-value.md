# <a name="bc30491-expression-does-not-produce-a-value"></a>BC30491 : l’expression ne produit pas de valeur

Vous avez tenté d’utiliser une expression qui ne produit pas de valeur dans un contexte produisant des valeurs, comme appeler un `Sub` dans un contexte où un `Function` est attendu.

 **ID d’erreur :** BC30491

## <a name="to-correct-this-error"></a>Pour corriger cette erreur

- Remplacez l’expression par une expression qui produit une valeur.
