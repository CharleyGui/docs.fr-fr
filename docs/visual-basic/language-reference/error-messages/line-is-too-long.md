# <a name="line-is-too-long"></a>Ligne trop longue
Les lignes de texte source ne peut pas dépasser 65 535 caractères.  
  
 **ID d’erreur :** BC30494  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Raccourcissez la longueur de la ligne à 65 535 caractères ou moins.
