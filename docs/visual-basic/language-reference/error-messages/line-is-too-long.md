# <a name="line-is-too-long"></a>Ligne trop longue

Les lignes de texte source ne peuvent pas dépasser 65535 caractères.  
  
 **ID d’erreur :** BC30494  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Réduisez la longueur de la ligne à 65535 caractères au maximum.
