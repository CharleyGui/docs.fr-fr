# <a name="bc30766-functionname-is-not-declared-smart-devicevisual-basic-compiler-error"></a>BC30766 : ' \<functionname> 'n’est pas déclaré (erreur du compilateur Smart Device/Visual Basic)

<`functionname`> n’est pas déclaré. La fonctionnalité d’E/S de fichier est normalement disponible dans l’espace de noms `Microsoft.VisualBasic` , mais la version ciblée de .NET Compact Framework ne la prend pas en charge.

 **ID d’erreur :** BC30766

## <a name="to-correct-this-error"></a>Pour corriger cette erreur

- Effectuez des opérations sur les fichiers avec les fonctions définies dans l’espace de noms `System.IO` .
