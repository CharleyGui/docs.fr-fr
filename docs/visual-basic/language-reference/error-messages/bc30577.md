# <a name="bc30577-addressof-operand-must-be-the-name-of-a-method-without-parentheses"></a><span data-ttu-id="052c2-102">BC30577 : l’opérande’AddressOf’doit être le nom d’une méthode (sans parenthèses)</span><span class="sxs-lookup"><span data-stu-id="052c2-102">BC30577: 'AddressOf' operand must be the name of a method (without parentheses)</span></span>

<span data-ttu-id="052c2-103">L’opérateur `AddressOf` crée une instance de délégué de procédure qui fait référence à une procédure spécifique.</span><span class="sxs-lookup"><span data-stu-id="052c2-103">The `AddressOf` operator creates a procedure delegate instance that references a specific procedure.</span></span> <span data-ttu-id="052c2-104">La syntaxe est la suivante :</span><span class="sxs-lookup"><span data-stu-id="052c2-104">The syntax is as follows:</span></span>

```vb
AddressOf procedurename
```

<span data-ttu-id="052c2-105">Vous avez inséré des parenthèses autour de l’argument qui suit `AddressOf` , où aucune n’est nécessaire.</span><span class="sxs-lookup"><span data-stu-id="052c2-105">You inserted parentheses around the argument following `AddressOf`, where none are needed.</span></span>

<span data-ttu-id="052c2-106">**ID d’erreur :** BC30577</span><span class="sxs-lookup"><span data-stu-id="052c2-106">**Error ID:** BC30577</span></span>

## <a name="example"></a><span data-ttu-id="052c2-107"> Exemple</span><span class="sxs-lookup"><span data-stu-id="052c2-107">Example</span></span>

<span data-ttu-id="052c2-108">L’exemple suivant génère l’bc30577 :</span><span class="sxs-lookup"><span data-stu-id="052c2-108">The following example generates bc30577:</span></span>

```vb
Public Sub CountZeroToTen()
    For i = 0 To 10
        Console.WriteLine($"Counted: {i}")
        Threading.Thread.Sleep(500)
    Next
End Sub

Sub Main()
    ' Any of the following two lines generates bc30577.
    'Dim t As New Threading.Thread(AddressOf(CountZeroToTen))
    'Dim t As New Threading.Thread(AddressOf CountZeroToTen())
    t.Start()
End Sub
```

## <a name="to-correct-this-error"></a><span data-ttu-id="052c2-109">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="052c2-109">To correct this error</span></span>

1. <span data-ttu-id="052c2-110">Supprimez les parenthèses autour de l’argument `AddressOf` comme indiqué dans l’exemple suivant :</span><span class="sxs-lookup"><span data-stu-id="052c2-110">Remove the parentheses around the argument following `AddressOf` as shown in the following example:</span></span>

    ```vb
    Public Sub CountZeroToTen()
        For i = 0 To 10
            Console.WriteLine($"Counted: {i}")
            Threading.Thread.Sleep(500)
        Next
    End Sub

    Sub Main()
        Dim t As New Threading.Thread(AddressOf CountZeroToTen)
        t.Start()
    End Sub
    ```

2. <span data-ttu-id="052c2-111">Assurez-vous que l’argument est un nom de méthode.</span><span class="sxs-lookup"><span data-stu-id="052c2-111">Make sure the argument is a method name.</span></span>
