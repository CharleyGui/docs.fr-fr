<span data-ttu-id="26c33-103">Un tableau dans une structure est déclaré avec une taille initiale.</span><span class="sxs-lookup"><span data-stu-id="26c33-103">An array in a structure is declared with an initial size.</span></span> <span data-ttu-id="26c33-104">Vous ne pouvez pas initialiser un élément de structure, et la déclaration d’une taille de tableau est une forme d’initialisation.</span><span class="sxs-lookup"><span data-stu-id="26c33-104">You cannot initialize any structure element, and declaring an array size is one form of initialization.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="26c33-108">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="26c33-108">To correct this error</span></span>
