# <a name="type-of-variablename-cannot-be-inferred-because-the-loop-bounds-and-the-step-variable-do-not-widen-to-the-same-type"></a><span data-ttu-id="c5f4d-102">Type de '\<nom_variable >' ne peut pas être déduit, car les limites de la boucle et la variable étape s’étend pas au même type</span><span class="sxs-lookup"><span data-stu-id="c5f4d-102">Type of '\<variablename>' cannot be inferred because the loop bounds and the step variable do not widen to the same type</span></span>
<span data-ttu-id="c5f4d-103">Vous avez écrit un `For...Next` boucle dans laquelle le compilateur ne peut pas déduire un type de données pour la variable de contrôle de boucle, car les conditions suivantes sont remplies :</span><span class="sxs-lookup"><span data-stu-id="c5f4d-103">You have written a `For...Next` loop in which the compiler cannot infer a data type for the loop control variable because the following conditions are true:</span></span>  
  
- <span data-ttu-id="c5f4d-104">Le type de données de la variable de contrôle de boucle n’est pas spécifié avec une clause `As` .</span><span class="sxs-lookup"><span data-stu-id="c5f4d-104">The data type of the loop control variable is not specified with an `As` clause.</span></span>  
  
- <span data-ttu-id="c5f4d-105">Les limites de la boucle et l’incrémentation contiennent au moins deux types de données.</span><span class="sxs-lookup"><span data-stu-id="c5f4d-105">The loop bounds and step variable contain at least two data types.</span></span>  
  
- <span data-ttu-id="c5f4d-106">Il n’existe aucune conversion standard entre les types de données.</span><span class="sxs-lookup"><span data-stu-id="c5f4d-106">No standard conversions exist between the data types.</span></span>  
  
 <span data-ttu-id="c5f4d-107">Par conséquent, le compilateur ne peut pas déduire le type de données de variable de contrôle d’une boucle.</span><span class="sxs-lookup"><span data-stu-id="c5f4d-107">Therefore, the compiler cannot infer the data type of a loop's control variable.</span></span>  
  
 <span data-ttu-id="c5f4d-108">Dans l’exemple suivant, la variable de l’étape est un caractère et les limites de la boucle sont les deux entiers.</span><span class="sxs-lookup"><span data-stu-id="c5f4d-108">In the following example, the step variable is a character and the loop bounds are both integers.</span></span> <span data-ttu-id="c5f4d-109">Comme il n’existe aucune conversion standard entre les caractères et les entiers, cette erreur est signalée.</span><span class="sxs-lookup"><span data-stu-id="c5f4d-109">Because there is no standard conversion between characters and integers, this error is reported.</span></span>  
  
```vb  
Dim stepVar = "1"c  
Dim m = 0  
Dim n = 20  
  
' Not valid.  
' For i = 1 To 10 Step stepVar  
    ' Loop processing  
' Next  
```  
  
 <span data-ttu-id="c5f4d-110">**ID d’erreur :** BC30982</span><span class="sxs-lookup"><span data-stu-id="c5f4d-110">**Error ID:** BC30982</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c5f4d-111">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="c5f4d-111">To correct this error</span></span>  
  
- <span data-ttu-id="c5f4d-112">Modifier les types des limites de la boucle et selon vos besoins afin qu’au moins un d'entre eux est un type de la clause d’incrémentation.</span><span class="sxs-lookup"><span data-stu-id="c5f4d-112">Change the types of the loop bounds and step variable as necessary so that at least one of them is a type that the others widen to.</span></span> <span data-ttu-id="c5f4d-113">Dans l’exemple précédent, remplacez le type de `stepVar` à `Integer`.</span><span class="sxs-lookup"><span data-stu-id="c5f4d-113">In the preceding example, change the type of `stepVar` to `Integer`.</span></span>  
  
    ```  
    Dim stepVar = 1  
    ```  
  
     <span data-ttu-id="c5f4d-114">- ou -</span><span class="sxs-lookup"><span data-stu-id="c5f4d-114">—or—</span></span>  
  
    ```  
    Dim stepVar As Integer = 1  
    ```  
  
- <span data-ttu-id="c5f4d-115">Utiliser les fonctions de conversion explicite pour convertir les limites de la boucle et la variable de l’étape pour les types appropriés.</span><span class="sxs-lookup"><span data-stu-id="c5f4d-115">Use explicit conversion functions to convert the loop bounds and step variable to the appropriate types.</span></span> <span data-ttu-id="c5f4d-116">Dans l’exemple précédent, vous devez appliquer le `Val` à fonction `stepVar`.</span><span class="sxs-lookup"><span data-stu-id="c5f4d-116">In the preceding example, apply the `Val` function to `stepVar`.</span></span>  
  
    ```  
    For i = 1 To 10 Step Val(stepVar)  
        ' Loop processing  
    Next  
    ```
