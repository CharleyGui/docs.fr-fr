# <a name="error-saving-temporary-win32-resource-file-filename-error-message"></a>Erreur lors de l’enregistrement du fichier de ressources Win32 temporaire'\<filename > ' : \<message d’erreur >
Le compilateur Visual Basic appelle Assembly Linker (al. exe, également appelé ALink) pour générer un assembly avec un manifeste. L'éditeur de liens a signalé une erreur d'obtention d'un nom de fichier à utiliser dans l'écriture d'une ressource en mémoire.  
  
 **ID d’erreur :** BC30137  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Examinez le message d’erreur cité et consultez la rubrique [al. exe](../../../framework/tools/al-exe-assembly-linker.md). pour obtenir plus d’explications et de conseils.  
  
2. Si l'erreur persiste, rassemblez des informations sur ses circonstances et avertissez les services de support technique Microsoft.
