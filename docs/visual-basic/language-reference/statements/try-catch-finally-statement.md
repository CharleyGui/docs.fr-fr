## <a name="syntax"></a><span data-ttu-id="26437-105">Syntaxe</span><span class="sxs-lookup"><span data-stu-id="26437-105">Syntax</span></span>

```vb
Try
    [ tryStatements ]
    [ Exit Try ]
[ Catch [ exception [ As type ] ] [ When expression ]
    [ catchStatements ]
    [ Exit Try ] ]
[ Catch ... ]
[ Finally
    [ finallyStatements ] ]
End Try
```
