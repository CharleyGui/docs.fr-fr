# <a name="in-clause-visual-basic"></a>In, clause (Visual Basic)

Spécifie le groupe que la variable de boucle doit parcourir dans une `For Each` boucle, ou spécifie la collection à interroger dans une `From` `Join` clause, ou `Group Join` .  
  
## <a name="remarks"></a>Notes  

 Le `In` mot clé peut être utilisé dans les contextes suivants :  
  
 [For Each...Next (instruction)](for-each-next-statement.md)  
  
 [From, clause](../queries/from-clause.md)  
  
 [Join (clause)](../queries/join-clause.md)  
  
 [Group Join (clause)](../queries/group-join-clause.md)
