# <a name="scaling-cloud-native-applications"></a>Mise à l’échelle des applications Cloud natives

[!INCLUDE [book-preview](../../../includes/book-preview.md)]

L’évolutivité est l’un des avantages les plus fréquents du passage à un environnement d’hébergement cloud. L’évolutivité, ou la possibilité pour une application d’accepter une charge utilisateur supplémentaire sans compromettre les performances de chaque utilisateur. La plupart du temps, il est possible de fractionner une application en petites parties, chacune d’entre elles pouvant être affectée à toutes les ressources dont elles ont besoin. Les fournisseurs de Cloud permettent une évolutivité massive à tout moment et partout dans le monde.

 Dans ce chapitre, nous discutons des technologies qui permettent aux applications Cloud natives de se mettre à l’échelle pour répondre à la demande des utilisateurs. Ces technologies sont les suivantes :

- Containers
- Orchestrators
- Informatique Serverless
