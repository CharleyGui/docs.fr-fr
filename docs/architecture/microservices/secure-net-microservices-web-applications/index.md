### <a name="authenticate-with-an-openid-connect-or-oauth-20-identity-provider"></a><span data-ttu-id="12f7c-176">S’authentifier avec un fournisseur d’identité OpenID Connect ou OAuth 2.0</span><span class="sxs-lookup"><span data-stu-id="12f7c-176">Authenticate with an OpenID Connect or OAuth 2.0 Identity provider</span></span>

### <a name="consume-security-tokens"></a><span data-ttu-id="12f7c-200">Consommer des jetons de sécurité</span><span class="sxs-lookup"><span data-stu-id="12f7c-200">Consume security tokens</span></span>
