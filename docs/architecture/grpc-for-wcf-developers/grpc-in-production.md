# <a name="grpc-in-production"></a>gRPC en production

Vous pouvez exécuter des applications ASP.NET Core 3,0, y compris gRPC services, sur Windows, sur Linux et dans des conteneurs à l’aide de plateformes modernes telles que Dockr et Kubernetes. Ce chapitre explore les différentes options d’exécution de vos services gRPC en production et examine les options de surveillance et de journalisation pour garantir un fonctionnement optimal des systèmes.
