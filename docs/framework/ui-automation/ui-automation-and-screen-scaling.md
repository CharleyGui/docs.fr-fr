> [!NOTE]
> <span data-ttu-id="2a326-123">Par défaut, le DWM n’effectue pas de mise à l’échelle pour les applications qui ne prennent pas en charge DPI lorsque l’utilisateur définit la valeur PPP sur 120, mais l’exécute lorsque la valeur PPP est définie sur une valeur personnalisée 144 ou supérieure.</span><span class="sxs-lookup"><span data-stu-id="2a326-123">By default, the DWM does not perform scaling for non-dpi-aware applications when the user sets the dpi to 120, but does perform it when the dpi is set to a custom value of 144 or higher.</span></span> <span data-ttu-id="2a326-124">Toutefois, l’utilisateur peut remplacer le comportement par défaut.</span><span class="sxs-lookup"><span data-stu-id="2a326-124">However, the user can override the default behavior.</span></span>  
  
 <span data-ttu-id="2a326-125">La mise à l’échelle de l’écran pose de nouveaux défis pour les applications qui sont concernées d’une manière ou d’une autre par les coordonnées d’écran.</span><span class="sxs-lookup"><span data-stu-id="2a326-125">Screen scaling creates new challenges for applications that are concerned in any way with screen coordinates.</span></span> <span data-ttu-id="2a326-126">L’écran contient maintenant deux systèmes de coordonnées : l’un physique et l’autre logique.</span><span class="sxs-lookup"><span data-stu-id="2a326-126">The screen now contains two coordinate systems: physical and logical.</span></span> <span data-ttu-id="2a326-127">Les coordonnées physiques d’un point correspondent au décalage réel en pixels par rapport au point d’origine en haut à gauche.</span><span class="sxs-lookup"><span data-stu-id="2a326-127">The physical coordinates of a point are the actual offset in pixels from the top left of the origin.</span></span> <span data-ttu-id="2a326-128">Les coordonnées logiques correspondent aux décalages tels qu’ils seraient si les pixels eux-mêmes étaient mis à l’échelle.</span><span class="sxs-lookup"><span data-stu-id="2a326-128">The logical coordinates are the offsets as they would be if the pixels themselves were scaled.</span></span>
