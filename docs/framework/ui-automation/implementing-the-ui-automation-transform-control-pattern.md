## <a name="implementation-guidelines-and-conventions"></a><span data-ttu-id="04338-109">Conventions et recommandations en matière d'implémentation</span><span class="sxs-lookup"><span data-stu-id="04338-109">Implementation Guidelines and Conventions</span></span>
