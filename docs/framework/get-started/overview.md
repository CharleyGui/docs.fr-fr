## <a name="features-of-the-common-language-runtime"></a><span data-ttu-id="73f72-129">Fonctionnalités du Common Language Runtime</span><span class="sxs-lookup"><span data-stu-id="73f72-129">Features of the common language runtime</span></span>

<span data-ttu-id="73f72-132">Concernant la sécurité, les composants managés se voient attribuer divers niveaux de confiance en fonction d’un nombre de facteurs qui comprennent leur origine (comme Internet, un réseau d’entreprise ou un ordinateur local).</span><span class="sxs-lookup"><span data-stu-id="73f72-132">Regarding security, managed components are awarded varying degrees of trust, depending on a number of factors that include their origin (such as the Internet, enterprise network, or local computer).</span></span> <span data-ttu-id="73f72-133">Cela signifie qu’un composant managé peut ou ne peut pas effectuer des opérations d’accès au fichier, des opérations d’accès au Registre ou d’autres fonctions délicates, même si ce composant est utilisé dans la même application active.</span><span class="sxs-lookup"><span data-stu-id="73f72-133">This means that a managed component might or might not be able to perform file-access operations, registry-access operations, or other sensitive functions, even if it's used in the same active app.</span></span>
