# <a name="icorruntimehostmapfile-method"></a>ICorRuntimeHost::MapFile, méthode
Mappe le fichier spécifié en mémoire. Cette méthode est obsolète.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT MapFile(  
    [in]  HANDLE    hFile,  
    [out] HMODULE*  hMapAddress  
);  
```  
  
## <a name="parameters"></a>Paramètres  
 `hFile`  
 dans Handle du fichier à mapper.  
  
 `hMapAddress`  
 à Adresse de départ de la mémoire à laquelle commencer le mappage du fichier.
