# <a name="hosting-structures"></a>Structures d'hébergement
Cette section décrit les structures non managées utilisées par l’API d’hébergement.  
  
## <a name="in-this-section"></a>Dans cette section  
 [AssemblyBindInfo, structure](../../../../docs/framework/unmanaged-api/hosting/assemblybindinfo-structure.md)  
 Fournit des informations détaillées sur l’assembly référencé.  
  
 [BucketParameters, structure](../../../../docs/framework/unmanaged-api/hosting/bucketparameters-structure.md)  
 Stocke le nom de type d’un événement et les paramètres de l’exception actuelle qui est associée à l’événement.  
  
 [COR_GC_STATS, structure](../../../../docs/framework/unmanaged-api/hosting/cor-gc-stats-structure.md)  
 Fournit des statistiques sur le mécanisme de garbage collection du common language runtime (CLR).  
  
 [COR_GC_THREAD_STATS, structure](../../../../docs/framework/unmanaged-api/hosting/cor-gc-thread-stats-structure.md)  
 Contient des statistiques par thread relatives à garbage collection.  
  
 [CustomDumpItem, structure](../../../../docs/framework/unmanaged-api/hosting/customdumpitem-structure.md)  
 Décrit un élément à ajouter à un dump personnalisé dans le rapport d’erreurs.  
  
 [MDAInfo, structure](../../../../docs/framework/unmanaged-api/hosting/mdainfo-structure.md)  
 Fournit des détails sur l’événement `Event_MDAFired`, qui déclenche la création d’un Assistant Débogage managé (MDA).  
  
 [ModuleBindInfo, structure](../../../../docs/framework/unmanaged-api/hosting/modulebindinfo-structure.md)  
 Fournit des informations détaillées sur le module référencé et l’assembly qui le contient.  
  
 [StackOverflowInfo, structure](../../../../docs/framework/unmanaged-api/hosting/stackoverflowinfo-structure.md)  
 Stocke le type de dépassement qui s’est produit et les informations sur l’exception levée en raison du dépassement de capacité.
