# <a name="_corimageunloading-function"></a>_CorImageUnloading, fonction
Informe la chargeuse lorsque les images du module géré sont déchargées.  
  
 Cette fonction n’est pas implémentée. S’il est appelé, il revient E_NOTIMPL.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
STDAPI (VOID) _CorImageUnloading(
   [in] PVOID* ImageBase  
);  
```  
  
## <a name="parameters"></a>Paramètres  
 `ImageBase`  
 [dans] Un pointeur à l’emplacement de départ de l’image à décharger.
