# <a name="iapartmentcallback-interface"></a>IApartmentCallback, interface
Fournit des méthodes pour effectuer des rappels dans un cloisonnement. Un *cloisonnement* est un conteneur logique au sein d’un processus pour les objets qui partagent les mêmes exigences d’accès aux threads.  
  
## <a name="methods"></a>Méthodes  
  
|Méthode|Description|  
|------------|-----------------|  
|[DoCallback, méthode](../../../../docs/framework/unmanaged-api/hosting/iapartmentcallback-docallback-method.md)|Exécute la fonction spécifiée dans un cloisonnement.|
