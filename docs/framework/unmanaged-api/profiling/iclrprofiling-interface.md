# <a name="iclrprofiling-interface"></a>ICLRProfiling, interface
Fournit la méthode [ICLRProfiling :: AttachProfiler](iclrprofiling-attachprofiler-method.md) , qui permet à un profileur de s’attacher à un processus en cours d’exécution.  
  
## <a name="methods"></a>Méthodes  
  
|Méthode|Description|  
|------------|-----------------|  
|[AttachProfiler, méthode](iclrprofiling-attachprofiler-method.md)|Attache le profileur spécifié au processus spécifié.|
