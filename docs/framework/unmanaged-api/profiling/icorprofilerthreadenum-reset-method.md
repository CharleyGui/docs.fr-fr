# <a name="icorprofilerthreadenumreset-method"></a>ICorProfilerThreadEnum::Reset, méthode
Déplace le curseur de l'énumérateur à la position de départ de la séquence.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT Reset();  
```
