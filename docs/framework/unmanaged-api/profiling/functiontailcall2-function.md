## <a name="remarks"></a><span data-ttu-id="b73d3-110">Notes</span><span class="sxs-lookup"><span data-stu-id="b73d3-110">Remarks</span></span>

 <span data-ttu-id="b73d3-113">La valeur de la `func` paramètre n’est pas valide après la `FunctionTailcall2` fonction retourne, car la valeur peut changer ou être détruite.</span><span class="sxs-lookup"><span data-stu-id="b73d3-113">The value of the `func` parameter is not valid after the `FunctionTailcall2` function returns because the value may change or be destroyed.</span></span>  
  
 <span data-ttu-id="b73d3-114">Le `FunctionTailcall2` fonction est un rappel ; vous devez l’implémenter.</span><span class="sxs-lookup"><span data-stu-id="b73d3-114">The `FunctionTailcall2` function is a callback; you must implement it.</span></span> <span data-ttu-id="b73d3-115">L’implémentation doit utiliser le `__declspec`(`naked`) attribut de classe de stockage.</span><span class="sxs-lookup"><span data-stu-id="b73d3-115">The implementation must use the `__declspec`(`naked`) storage-class attribute.</span></span>  
  
 <span data-ttu-id="b73d3-116">Le moteur d’exécution n’enregistre pas les registres avant d’appeler cette fonction.</span><span class="sxs-lookup"><span data-stu-id="b73d3-116">The execution engine does not save any registers before calling this function.</span></span>  
  
- <span data-ttu-id="b73d3-117">À l’entrée, vous devez enregistrer tous les registres que vous utilisez, y compris celles dans l’unité de virgule flottante (FPU).</span><span class="sxs-lookup"><span data-stu-id="b73d3-117">On entry, you must save all registers that you use, including those in the floating-point unit (FPU).</span></span>  
  
- <span data-ttu-id="b73d3-118">À la sortie, vous devez restaurer la pile en dépilant tous les paramètres qui ont été envoyés par son appelant.</span><span class="sxs-lookup"><span data-stu-id="b73d3-118">On exit, you must restore the stack by popping off all the parameters that were pushed by its caller.</span></span>  
  
 <span data-ttu-id="b73d3-119">L’implémentation de `FunctionTailcall2` ne doivent pas bloquer, car il sera retarder le garbage collection.</span><span class="sxs-lookup"><span data-stu-id="b73d3-119">The implementation of `FunctionTailcall2` should not block because it will delay garbage collection.</span></span> <span data-ttu-id="b73d3-120">L’implémentation ne doit pas tenter un garbage collection, car la pile est peut-être pas à l’état garbage collection convivial.</span><span class="sxs-lookup"><span data-stu-id="b73d3-120">The implementation should not attempt a garbage collection because the stack may not be in a garbage collection-friendly state.</span></span> <span data-ttu-id="b73d3-121">Si un garbage collection est tenté, le runtime bloque jusqu'à ce que `FunctionTailcall2` retourne.</span><span class="sxs-lookup"><span data-stu-id="b73d3-121">If a garbage collection is attempted, the runtime will block until `FunctionTailcall2` returns.</span></span>
