# <a name="iassemblycacheitemabortitem-method"></a>IAssemblyCacheItem::AbortItem, méthode
Permet à l’assembly dans le Global Assembly Cache d’effectuer des opérations de nettoyage avant qu’il ne soit libéré.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT AbortItem ();  
```
