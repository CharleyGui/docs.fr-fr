# <a name="iassemblyenumclone-method"></a>IAssemblyEnum::Clone, méthode
Crée une copie superficielle de cet objet [IAssemblyEnum](iassemblyenum-interface.md) .  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT Clone (  
    [out] IAssemblyEnum   **ppEnum  
);  
```  
  
## <a name="parameters"></a>Paramètres  
 `ppEnum`  
 à Pointeur vers la copie.
