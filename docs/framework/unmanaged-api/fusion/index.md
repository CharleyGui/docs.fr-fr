# <a name="fusion-unmanaged-api-reference"></a>Fusion (Informations de référence sur les API non managées)
L'API de fusion permet à un hôte de runtime d'accéder aux propriétés des ressources d'une application afin de pouvoir localiser les versions correctes de ces ressources pour l'application.
