# <a name="asm_display_flags-enumeration"></a>ASM_DISPLAY_FLAGS, énumération
Indique la version, la construction, la culture, la signature, et ainsi de suite, de l’assemblage dont le nom d’affichage sera récupéré par [l’IAssemblyName::GetDisplayName](iassemblyname-getdisplayname-method.md) méthode.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
typedef enum {  
  
    ASM_DISPLAYF_VERSION                 = 0x01,  
    ASM_DISPLAYF_CULTURE                 = 0x02,  
    ASM_DISPLAYF_PUBLIC_KEY_TOKEN        = 0x04,  
    ASM_DISPLAYF_PUBLIC_KEY              = 0x08,  
    ASM_DISPLAYF_CUSTOM                  = 0x10,  
    ASM_DISPLAYF_PROCESSORARCHITECTURE   = 0x20,  
    ASM_DISPLAYF_LANGUAGEID              = 0x40,  
    ASM_DISPLAYF_RETARGET                = 0x80,  
    ASM_DISPLAYF_CONFIG_MASK             = 0x100,  
    ASM_DISPLAYF_MVID                    = 0x200,  
    ASM_DISPLAYF_FULL                    =
                      ASM_DISPLAYF_VERSION           |
                      ASM_DISPLAYF_CULTURE           |
                      ASM_DISPLAYF_PUBLIC_KEY_TOKEN  |
                      ASM_DISPLAYF_RETARGET          |
                      ASM_DISPLAYF_PROCESSORARCHITECTURE  
  
} ASM_DISPLAY_FLAGS;  
```  
  
## <a name="remarks"></a>Notes   
 `ASM_DISPLAYF_FULL`reflète toute modification apportée à la version de l’objet [IAssemblyName.](iassemblyname-interface.md) Ne présumez pas que la valeur retournée est immuable.
