# <a name="icordebugmodulegetassembly-method"></a>ICorDebugModule::GetAssembly, méthode

Obtient l’assembly conteneur pour ce module.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetAssembly(  
    [out] ICorDebugAssembly **ppAssembly  
);  
```  
  
## <a name="parameters"></a>Paramètres  

 `ppAssembly`  
 à Pointeur vers un objet ICorDebugAssembly qui représente l’assembly contenant ce module.
