# <a name="icordebugprocess8-interface"></a>ICorDebugProcess8, interface
[Pris en charge dans le .NET Framework 4,6 et versions ultérieures]  
  
 Étend logiquement l’interface ICorDebugProcess pour activer ou désactiver certains types de rappels d’exception [ICorDebugManagedCallback2](icordebugmanagedcallback2-interface.md) .  
  
## <a name="methods"></a>Méthodes  
  
|Méthode|Description|  
|------------|-----------------|  
|[EnableExceptionCallbacksOutsideOfMyCode, méthode](icordebugprocess8-enableexceptioncallbacksoutsideofmycode-method.md)|Active ou désactive certains types de rappels d’exception [ICorDebugManagedCallback2](icordebugmanagedcallback2-interface.md) .|
