# <a name="icordebugterminate-method"></a>ICorDebug::Terminate, méthode
Met fin à `ICorDebug` l’objet.  
  
> [!NOTE]
> `Terminate`ne doit pas être appelé tant qu’un rappel [ICorDebugManagedCallback :: ExitProcess](icordebugmanagedcallback-exitprocess-method.md) n’a pas été reçu pour tous les processus en cours de débogage.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT Terminate ();  
```  
  
## <a name="remarks"></a>Notes   
 `Terminate`doit être appelé lorsque l' `ICorDebug` objet n’est plus nécessaire.
