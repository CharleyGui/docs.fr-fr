# <a name="icorpublishenumreset-method"></a>ICorPublishEnum::Reset, méthode
Place le curseur au début de l’énumération.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT Reset ();  
```  
  
## <a name="parameters"></a>Paramètres  
 `Reset` ne prend aucun paramètre.
