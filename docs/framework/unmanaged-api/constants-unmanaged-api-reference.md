# <a name="constants-unmanaged-api-reference"></a>Constantes (référence des API non managées)

Cette rubrique décrit le type de langage, le fournisseur de langage et les constantes de type de document qui sont définis dans CorSym. idl.  
  
## <a name="language-type-constants"></a>Constantes de type de langage  

 Le tableau suivant montre les constantes de type de langage, qui représentent des GUID qui identifient les langages de programmation.  
  
|Symbole|Description|  
|------------|-----------------|  
|CorSym_LanguageType_C|Indique le langage C.|  
|CorSym_LanguageType_CPlusPlus|Indique le langage C++.|  
|CorSym_LanguageType_CSharp|Indique le langage C#.|  
|CorSym_LanguageType_Basic|Indique la langue de base.|  
|CorSym_LanguageType_Java|Indique le langage Java.|  
|CorSym_LanguageType_Cobol|Indique le langage COBOL.|  
|CorSym_LanguageType_Pascal|Indique le langage Pascal.|  
|CorSym_LanguageType_ILAssembly|Indique le code assembleur MSIL (Microsoft Intermediate Language).|  
|CorSym_LanguageType_JScript|Indique le langage JScript.|  
|CorSym_LanguageType_SMC|Indique le langage SMC.|  
|CorSym_LanguageType_MCPlusPlus|Indique que le langage C++ est activé pour le .NET Framework.|  
  
## <a name="language-vendor-constants"></a>Constantes de fournisseur de langue  

 Le tableau suivant montre les constantes de fournisseur de langages, qui représentent des GUID qui identifient les fournisseurs de langage de programmation.  
  
|Symbole|Description|  
|------------|-----------------|  
|CorSym_LanguageVendor_Microsoft|Indique Microsoft.|  
  
## <a name="document-type-constants"></a>Constantes de type de document  

 Le tableau suivant présente les constantes de type de document, qui représentent des GUID qui identifient les types de documents.  
  
|Symbole|Description|  
|------------|-----------------|  
|CorSym_DocumentType_Text|Indique un document texte.|  
|CorSym_DocumentType_MC|Indique un document non-texte.|
