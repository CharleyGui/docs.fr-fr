# <a name="imaptoken-interface"></a>IMapToken, interface
Fournit des fonctionnalités de mappage entre les assemblys.  
  
> [!NOTE]
> L' `IMapToken` interface doit être implémentée par le client.  
  
## <a name="methods"></a>Méthodes  
  
|Méthode|Description|  
|------------|-----------------|  
|[Map, méthode](imaptoken-map-method.md)|Mappe une relation entre deux assemblys à l’aide de signatures de métadonnées.|
