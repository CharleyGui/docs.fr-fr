# <a name="imetadataimportresetenum-method"></a>IMetaDataImport::ResetEnum Method
Réinitialise l'énumérateur spécifié à la position spécifiée.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT ResetEnum (  
   [in] HCORENUM    hEnum,
   [in] ULONG       ulPos  
);  
```  
  
## <a name="parameters"></a>Paramètres  
 `hEnum`  
 [dans] L’enumérateur à réinitialiser.  
  
 `ulPos`  
 [dans] La nouvelle position à laquelle placer l’énumérateur.
