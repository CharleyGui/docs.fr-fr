# <a name="imetadataassemblyimportenummanifestresources-method"></a>IMetaDataAssemblyImport::EnumManifestResources, méthode
Obtient un pointeur vers un énumérateur pour les ressources référencées dans le manifeste de l’assembly actuel.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT EnumManifestResources (  
    [in, out] HCORENUM         *phEnum,   
    [out] mdManifestResource   rManifestResources[],   
    [in]  ULONG                cMax,   
    [out] ULONG                *pcTokens  
);   
```  
  
## <a name="parameters"></a>Paramètres  
 `phEnum`  
 [in, out] Pointeur vers l’énumérateur. Il doit s’agir d’une valeur NULL lorsque la méthode `EnumManifestResources` est appelée pour la première fois.  
  
 `rManifestResources`  
 à Tableau utilisé pour stocker les jetons de métadonnées de `mdManifestResource`.  
  
 `cMax`  
 dans Nombre maximal de jetons de `mdManifestResource` qui peuvent être placés dans `rManifestResources`.  
  
 `pcTokens`  
 à Nombre de jetons de `mdManifestResource` placés en `rManifestResources`.  
  
## <a name="return-value"></a>Valeur de retour  
  
|HRESULT|Description|  
|-------------|-----------------|  
|`S_OK`|`EnumManifestResources` retourné avec succès.|  
|`S_FALSE`|Il n’y a aucun jeton à énumérer. Dans ce cas, `pcTokens` a la valeur zéro.|
