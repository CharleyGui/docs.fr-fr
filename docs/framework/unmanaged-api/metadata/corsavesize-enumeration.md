# <a name="corsavesize-enumeration"></a>CorSaveSize, énumération
Contient des valeurs indiquant le niveau de précision requis lors de l'interrogation de la taille d'une opération d'enregistrement.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
typedef enum CorSaveSize {  
    cssAccurate                = 0x0000,   
    cssQuick                   = 0x0001,   
    cssDiscardTransientCAs     = 0x0002  
} CorSaveSize;  
```  
  
## <a name="members"></a>Membres  
  
|Membre|Description|  
|------------|-----------------|  
|`cssAccurate`|Specifies that the return value should be exact.|  
|`cssQuick`|Specifies that the return value should be estimated.|  
|`cssDiscardTransientCAs`|Specifies that discardable types should be removed.|
