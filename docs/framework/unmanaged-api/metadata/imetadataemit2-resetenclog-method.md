# <a name="imetadataemit2resetenclog-method"></a>IMetaDataEmit2::ResetENCLog, méthode
Réinitialise le journal Edit-and-continue et démarre une nouvelle session.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT ResetENCLog ();  
```  
  
## <a name="parameters"></a>Paramètres  
 `ResetENCLog` n’accepte aucun paramètre.
