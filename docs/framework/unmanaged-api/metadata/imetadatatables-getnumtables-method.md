# <a name="imetadatatablesgetnumtables-method"></a>IMetaDataTables::GetNumTables, méthode
Obtient le nombre de tables dans la portée de l’instance de `IMetaDataTables` actuelle.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetNumTables (  
    [out]  ULONG   *pcTables  
);  
```  
  
## <a name="parameters"></a>Paramètres  
 `pcTables`  
 à Pointeur vers le nombre de tables dans la portée de l’instance actuelle.
