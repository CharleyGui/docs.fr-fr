# <a name="imetadataimportcountenum-method"></a>IMetaDataImport::CountEnum, méthode
Obtient le nombre d’éléments dans le recensement qui a été récupéré par l’énumérateur spécifié.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT CountEnum (  
   [in]  HCORENUM    hEnum,
   [out] ULONG       *pulCount  
);  
```  
  
## <a name="parameters"></a>Paramètres  
 `hEnum`  
 [dans] La poignée de l’enumérateur.  
  
 `pulCount`  
 [out] Le nombre d’éléments énumérés.  
  
## <a name="remarks"></a>Notes   
 La poignée `hEnum` spécifiée par est `Enum`obtenue à partir d’un précédent appel *nom* (par exemple, [IMetaDataImport::EnumTypeDefs](../../../../docs/framework/unmanaged-api/metadata/imetadataimport-enumtypedefs-method.md)).
