# <a name="imetadataimportgetsigfromtoken-method"></a>IMetaDataImport::GetSigFromToken, méthode
Obtient la signature de métadonnées binaires associée au jeton spécifié.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetSigFromToken (   
   [in]   mdSignature        mdSig,   
   [out]  PCCOR_SIGNATURE    *ppvSig,   
   [out]  ULONG              *pcbSig   
);  
```  
  
## <a name="parameters"></a>Paramètres  
 `mdSig`  
 [in] The token to return the binary metadata signature for.  
  
 `ppvSig`  
 [out] A pointer to the returned metadata signature.  
  
 `pcbSig`  
 [out] The size in bytes of the binary metadata signature.
