# <a name="imetadataimportenumfields-method"></a>IMetaDataImport::EnumFields, méthode
Énumère les jetons FieldDef pour le type référencé par le jeton TypeDef spécifié.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT EnumFields (
   [in, out] HCORENUM    *phEnum,
   [in]      mdTypeDef   cl,
   [out]     mdFieldDef  rFields[],
   [in]      ULONG       cMax,
   [out]     ULONG       *pcTokens  
);  
```  
  
## <a name="parameters"></a>Paramètres  
 `phEnum`  
 [dans, dehors] Un pointeur à l’enumérateur.  
  
 `cl`  
 [dans] Le jeton TypeDef de la classe dont les champs doivent être énumérés.  
  
 `rFields`  
 [out] La liste des jetons FieldDef.  
  
 `cMax`  
 [in] Taille maximale du tableau `rFields`.  
  
 `pcTokens`  
 [out] Le nombre réel de jetons FieldDef retourné dans `rFields`.  
  
## <a name="return-value"></a>Valeur de retour  
  
|HRESULT|Description|  
|-------------|-----------------|  
|`S_OK`|`EnumFields`retourné avec succès.|  
|`S_FALSE`|Il n’y a pas de champs à énumérer. Dans ce `pcTokens` cas, c’est zéro.|
