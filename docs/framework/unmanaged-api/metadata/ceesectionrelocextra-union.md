# <a name="ceesectionrelocextra-union"></a>CeeSectionRelocExtra, union

Représente un décalage d’adresse utilisé par l’interface [ICeeGen](iceegen-interface.md) pour déplacer une section.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
typedef union  {  
    USHORT highAdj;  
} CeeSectionRelocExtra;  
```  
  
## <a name="members"></a>Membres  
  
|Membre|Description|  
|------------|-----------------|  
|`highAdj`|Ajustement d’adresse supérieure pour la section.|
