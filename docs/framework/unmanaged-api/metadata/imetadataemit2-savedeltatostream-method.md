# <a name="imetadataemit2savedeltatostream-method"></a>IMetaDataEmit2::SaveDeltaToStream, méthode
Enregistre les modifications de la session actuelle de modification et de poursuite au flux spécifié.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT SaveDeltaToStream (  
    [in] IStream     *pIStream,
    [in] DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>Paramètres  
 `pIStream`  
 [dans] Un pointeur d’interface pour le flux bref auquel enregistrer les modifications.  
  
 `dwSaveFlags`  
 [in] Réservée. Cette valeur doit être zéro.
