# <a name="imetadatatablesgettableindex-method"></a>IMetaDataTables::GetTableIndex, méthode
Gets the index for the table referenced by the specified token.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetTableIndex (  
    [in]  ULONG   token,  
    [out] ULONG   *pixTbl  
);  
```  
  
## <a name="parameters"></a>Paramètres  
 `token`  
 [in] The token that references the table.  
  
 `pixTbl`  
 [out] A pointer to the returned index for the referenced table.  
  
## <a name="remarks"></a>Notes  
 We do not recommend the use of this method, because it does not return consistent results. For information about the GUID table, see the Common Language Infrastructure (CLI) documentation, especially "Partition II: Metadata Definition and Semantics".
