# <a name="metadata-global-static-functions"></a>Fonctions statiques globales des métadonnées
Commençant par le .NET Framework 4, la documentation relative à la `_Cor*`, `CoInitialize*`, `CoUninitialize*`, et `CoEEShutDownCOM` fonctions relatives à l’hébergement, se trouve dans [déconseillée fonctions d’hébergement CLR](../../../../docs/framework/unmanaged-api/hosting/deprecated-clr-hosting-functions.md). Il n'existe aucune fonction statique globale relative aux métadonnées.
