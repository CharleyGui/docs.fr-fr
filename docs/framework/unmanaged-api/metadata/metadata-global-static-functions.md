# <a name="metadata-global-static-functions"></a>Fonctions statiques globales des métadonnées
À partir de la .NET Framework 4, la documentation des `_Cor*` fonctions,, `CoInitialize*` `CoUninitialize*` et `CoEEShutDownCOM` , qui se rapportent à l’hébergement, est située dans les [fonctions d’hébergement CLR dépréciées](../hosting/deprecated-clr-hosting-functions.md). Il n'existe aucune fonction statique globale relative aux métadonnées.
