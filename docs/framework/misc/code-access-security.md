 <span data-ttu-id="238ad-111">Le .NET Framework fournit un mécanisme de sécurité appelé sécurité d'accès du code, qui aide à protéger les systèmes informatiques contre du code mobile malveillant, ## <a name="walking-the-call-stack"></a><span data-ttu-id="238ad-140">Parcours de la pile des appels</span><span class="sxs-lookup"><span data-stu-id="238ad-140">Walking the Call Stack</span></span>  
 <span data-ttu-id="238ad-141">Pour déterminer si le code est autorisé à accéder à une ressource ou à exécuter une opération, le système de sécurité du runtime parcourt la pile des appels, en
