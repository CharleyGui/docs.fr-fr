# <a name="mitigate-new-behaviors-in-net-framework-46-and-later"></a>Atténuer les nouveaux comportements dans le cadre .NET 4.6 et plus tard

Cette section contient des articles qui décrivent les mesures d’atténuation des comportements qui ont été introduits dans le cadre .NET 4.6 et plus tard.

Envisagez d’atténuer un nouveau comportement s’il cause des problèmes pour votre application ou est autrement indésirable. Dans certains cas, vous pouvez également *activer* un nouveau comportement sur les applications qui ciblent une ancienne version de .NET Framework, mais sont en cours d’exécution sur .NET Framework 4.6 ou plus tard.
