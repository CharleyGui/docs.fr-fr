# <a name="feedback-and-community"></a>Commentaires et communauté
Nous apprécions vos commentaires et suggestions concernant la documentation de Windows Communication Foundation (WCF) ou de CardSpace. En bas de chaque page dans le Kit de développement logiciel figure la phrase « Envoyer des commentaires sur cette rubrique à Microsoft ». Cliquez sur le lien « Commentaires » pour envoyer votre opinion ou vos suggestions sur la rubrique spécifique.  
  
 Si vous recherchez plus d’informations, ou que vous souhaitez partager des techniques avec d’autres personnes intéressées dans WCF, essayez le [forum MSDN Windows Communication Foundation](https://go.microsoft.com/fwlink/?LinkId=186463).
