 <span data-ttu-id="61c3c-110">Cette fonctionnalité est disponible uniquement pour les services WCF hébergés sous IIS.</span><span class="sxs-lookup"><span data-stu-id="61c3c-110">This feature is only available to WCF services that are hosted under IIS.</span></span> <span data-ttu-id="61c3c-111">Cette fonctionnalité n’est pas activée par défaut.</span><span class="sxs-lookup"><span data-stu-id="61c3c-111">This feature is not enabled by default.</span></span> <span data-ttu-id="61c3c-112">Pour l’activer `multipleSiteBindingsEnabled` `true`, vous devez ajouter l’attribut à l'`serviceHostingEnvironment`élément < > dans votre fichier Web. config et lui affecter la valeur, comme illustré dans l’exemple suivant.</span><span class="sxs-lookup"><span data-stu-id="61c3c-112">To enable it you must add the `multipleSiteBindingsEnabled` attribute to the <`serviceHostingEnvironment`> element in your Web.config file and set it to `true`, as shown in the following example.</span></span>
