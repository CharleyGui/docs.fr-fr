## <a name="distributed-processing-and-compute-clusters"></a><span data-ttu-id="71567-112">Traitement distribué et clusters de calcul</span><span class="sxs-lookup"><span data-stu-id="71567-112">Distributed Processing and Compute Clusters</span></span>
