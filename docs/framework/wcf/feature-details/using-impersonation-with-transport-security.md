|<span data-ttu-id="7b2bb-113">None</span><span class="sxs-lookup"><span data-stu-id="7b2bb-113">None</span></span>|<span data-ttu-id="7b2bb-114">L'application serveur n'essaie pas d'emprunter l'identité du client.</span><span class="sxs-lookup"><span data-stu-id="7b2bb-114">The server application does not attempt to impersonate the client.</span></span>|
