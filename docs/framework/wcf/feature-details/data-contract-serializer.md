# <a name="data-contract-serializer"></a>Sérialiseur de contrat de données
L’infrastructure Windows Communication Foundation (WCF) utilise la <xref:System.Runtime.Serialization.DataContractSerializer> classe comme sérialiseur par défaut. Les rubriques de cette section fournissent davantage d'informations sur son utilisation en tant que sérialiseur autonome.  
  
## <a name="in-this-section"></a>Dans cette section  
 [Sérialisation et désérialisation](serialization-and-deserialization.md)  
 Explique comment utiliser la classe <xref:System.Runtime.Serialization.DataContractSerializer> pour sérialiser des graphiques d'objet et désérialiser des documents XML dans des objets.  
  
 [Importation et exportation de schémas](schema-import-and-export.md)  
 Explique comment générer un schéma en langage XSD (XML Schema Definition) à partir d’une classe .NET Framework, ou comment générer du code à partir d’un document de schéma XSD.
