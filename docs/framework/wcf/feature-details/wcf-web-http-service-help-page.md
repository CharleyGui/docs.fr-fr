## <a name="using-the-wcf-web-http-help-page"></a><span data-ttu-id="4870b-111">Utilisation de la page d'aide HTTP Web WCF</span><span class="sxs-lookup"><span data-stu-id="4870b-111">Using the WCF Web HTTP Help Page</span></span>  
 <span data-ttu-id="4870b-112">La page d'aide HTTP Web WCF affiche une brève description de chaque opération, à condition d'en spécifier une à l'aide de l'objet <xref:System.ComponentModel.DescriptionAttribute>.</span><span class="sxs-lookup"><span data-stu-id="4870b-112">The WCF WEB HTTP Help page displays a short description for each operation provided that you specify one using the <xref:System.ComponentModel.DescriptionAttribute>.</span></span> <span data-ttu-id="4870b-113">Cet attribut prend une chaîne contenant une courte description de l'opération à laquelle il est appliqué.</span><span class="sxs-lookup"><span data-stu-id="4870b-113">This attribute takes a string that contains a short description of the operation it is applied to.</span></span> <span data-ttu-id="4870b-114">Par exemple, le code suivant indique comment utiliser l'attribut <xref:System.ComponentModel.DescriptionAttribute> pour fournir une description courte.</span><span class="sxs-lookup"><span data-stu-id="4870b-114">For example, the following code shows how to use the <xref:System.ComponentModel.DescriptionAttribute> to provide a short description.</span></span>  
  
```csharp
[OperationContract]  
[WebGet(UriTemplate="/template1", BodyStyle = WebMessageBodyStyle.Bare)]  
[Description("Description for GET /template1")]  
SyndicationFeedFormatter GetTemplate1();  
```  
  
 <span data-ttu-id="4870b-115">Pour activer la page d'aide HTTP WEB WCF, vous devez ajouter un comportement de point de terminaison aux points de terminaison de votre service.</span><span class="sxs-lookup"><span data-stu-id="4870b-115">To turn on the WCF WEB HTTP Help page, you must add an endpoint behavior to your service's endpoints.</span></span> <span data-ttu-id="4870b-116">Cela peut s'effectuer dans la configuration ou dans le code.</span><span class="sxs-lookup"><span data-stu-id="4870b-116">This can be done in configuration or code.</span></span> <span data-ttu-id="4870b-117">Pour activer la page d'aide HTTP WEB WCF dans la configuration, ajoutez un comportement de point de terminaison avec un élément `<webHttp>`, définissez `enableHelp` à la valeur `true`, puis ajoutez un point de terminaison et configurez-le pour utiliser le comportement de point de terminaison.</span><span class="sxs-lookup"><span data-stu-id="4870b-117">To enable the WCF WEB HTTP Help age in configuration, add an endpoint behavior with a `<webHttp>` element, set `enableHelp` to `true`, and add an endpoint and configure it to use the endpoint behavior.</span></span> <span data-ttu-id="4870b-118">Le code de configuration suivant montre comment effectuer cela.</span><span class="sxs-lookup"><span data-stu-id="4870b-118">The following configuration code shows how to do this.</span></span>  
  
```xml  
<endpointBehaviors>  
   <behavior name="RESTEndpointBehavior">  
      <webHttp enableHelp="true"/>  
   </behavior>  
</endpointBehaviors>  
<!-- ... -->  
<services>  
   <service behaviorConfiguration="RESTWebServiceBehavior" name="RESTWebService">      <endpoint address="" kind="webHttpEndpoint" behaviorConfiguration="RESTEndpointBehavior" contract="IHello" />  
  
      <!-- ... -->  
   </service>  
</services>  
```  
  
 <span data-ttu-id="4870b-119">Pour activer la page d'aide HTTP WEB WCF dans le code, ajoutez un point de terminaison de service et ajoutez un objet <xref:System.ServiceModel.Description.WebHttpBehavior> au point de terminaison, en affectant à la propriété <xref:System.ServiceModel.Description.WebHttpBehavior.HelpEnabled%2A> la valeur `true`.</span><span class="sxs-lookup"><span data-stu-id="4870b-119">To enable the WCF Web HTTP Help page in code, add a service endpoint and add a <xref:System.ServiceModel.Description.WebHttpBehavior> to the endpoint setting <xref:System.ServiceModel.Description.WebHttpBehavior.HelpEnabled%2A> to `true`.</span></span> <span data-ttu-id="4870b-120">Le code suivant montre comment procéder.</span><span class="sxs-lookup"><span data-stu-id="4870b-120">The following code shows how to do this.</span></span>  
  
```csharp
using (WebServiceHost host = new WebServiceHost(typeof(Service), new Uri("http://localhost:8000/Customers")))  
{  
   host.AddServiceEndpoint(typeof(ICustomerCollection), new WebHttpBinding(), "");
   host.Description.Endpoints[0].Behaviors.Add(new WebHttpBehavior { EnableHelp = true });  
   // ...  
}  
```  
  
 <span data-ttu-id="4870b-121">La page d'aide est basée sur XHTML avec un balisage qui identifie les différentes parties de la page.</span><span class="sxs-lookup"><span data-stu-id="4870b-121">The help page is XHTML based with mark-up that identifies the different parts of the page.</span></span> <span data-ttu-id="4870b-122">Cela permet aux clients d'accéder à la page par programme, à l'aide de <xref:System.Xml.Linq.XElement> ou d'autres API XLinq.</span><span class="sxs-lookup"><span data-stu-id="4870b-122">This enables clients to programmatically access the page using <xref:System.Xml.Linq.XElement> or other XLinq APIs.</span></span>
