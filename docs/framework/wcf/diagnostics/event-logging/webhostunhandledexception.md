# <a name="webhostunhandledexception"></a>WebHostUnhandledException

ID : 109  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  

 Cet événement indique qu'une exception non prise en charge WebHost s'est produite. L'événement répertorie les informations sur l'émetteur, l'exception, le nom de processus et l'ID de processus.
