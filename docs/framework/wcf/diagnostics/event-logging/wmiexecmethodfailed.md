# <a name="wmiexecmethodfailed"></a>WmiExecMethodFailed

ID : 125  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  

 Cet événement indique que la demande ExecMethod WMI n'a pas été traitée. L'événement répertorie l'exception, le nom de processus et l'ID de processus.
