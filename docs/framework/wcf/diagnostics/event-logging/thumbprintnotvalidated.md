# <a name="thumbprintnotvalidated"></a>ThumbPrintNotValidated
Id : 153  
  
 Gravité : Error  
  
 Catégorie : TransactionBridge  
  
## <a name="description"></a>Description  
 Cet événement indique que le certificat d'identité avec l'empreinte numérique spécifiée n'a pas pu être validé. Cet événement répertorie le nom et l'ID de processus.
