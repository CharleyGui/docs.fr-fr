# <a name="transactionbridgerecoveryfailure"></a>TransactionBridgeRecoveryFailure

ID : 146  
  
 Gravité : Erreur  
  
 Catégorie : TransactionBridge  
  
## <a name="description"></a>Description  

 Cet événement indique que MSDTC TransactionBridge a échoué pendant la récupération. Cet incident constituant une condition irrécupérable, le service MSDTC a donc été arrêté. L'événement répertorie l'exception, le nom de processus et l'ID de processus. .
