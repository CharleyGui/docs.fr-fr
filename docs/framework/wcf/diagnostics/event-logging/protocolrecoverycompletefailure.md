# <a name="protocolrecoverycompletefailure"></a>ProtocolRecoveryCompleteFailure
Id : 145  
  
 Gravité : Error  
  
 Catégorie : TransactionBridge  
  
## <a name="description"></a>Description  
 Cet événement indique que le service de protocole WS-Atomic Transaction n’a pas pu démarrer et récupérer. Par conséquent, les fonctionnalités WS-Atomic Transaction sont désactivées. L'événement répertorie l'ID de protocole, le nom de protocole, l'exception, le nom de processus et l'ID de processus.
