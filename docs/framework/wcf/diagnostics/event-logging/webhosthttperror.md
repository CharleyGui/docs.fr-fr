# <a name="webhosthttperror"></a>WebHostHttpError

ID : 110  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  

 Cet événement indique qu'une exception WebHost HTTP s'est produite. L'événement répertorie les informations sur l'émetteur, HtmlErrorMessage, l'exception, le nom de processus et l'ID de processus.
