# <a name="protocolstopped"></a>ProtocolStopped

ID : 151  
  
 Gravité : Erreur  
  
 Catégorie : TransactionBridge  
  
## <a name="description"></a>Description  

 Cet événement indique que le service de protocole WS-Atomic Transaction a été stoppé. L'événement répertorie l'ID de protocole, le nom de protocole, le nom de processus et l'ID de processus. .
