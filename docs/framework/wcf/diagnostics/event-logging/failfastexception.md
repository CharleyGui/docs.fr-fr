# <a name="failfastexception"></a>FailFastException
Id : 103  
  
 Gravité : Error  
  
 Catégorie : Traçage  
  
## <a name="description"></a>Description  
 Une exception a été levée pendant FailFast. L'événement répertorie l'exception, le nom de processus et l'ID de processus.
