# <a name="webhostfailedtolisten"></a>WebHostFailedToListen
Id : 112  
  
 Gravité : Error  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  
 Cet événement indique qu'une erreur a eu lieu lors de la tentative d'écoute de l'URL spécifique. Ce processus de travail est terminé. L'événement répertorie l'exception, le nom de processus et l'ID de processus.
