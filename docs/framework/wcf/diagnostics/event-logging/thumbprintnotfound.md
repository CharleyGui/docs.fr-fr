# <a name="thumbprintnotfound"></a>ThumbPrintNotFound
Id : 152  
  
 Gravité : Error  
  
 Catégorie : TransactionBridge  
  
## <a name="description"></a>Description  
 Cet événement indique que le certificat d'identité contenant l'empreinte numérique spécifiée n'a pas pu être trouvé. Cet événement répertorie le nom et l'ID du processus.
