# <a name="fatalunexpectedstatemachineevent"></a>FatalUnexpectedStateMachineEvent
Id : 137  
  
 Gravité : Error  
  
 Catégorie : TransactionBridge  
  
## <a name="description"></a>Description  
 Cet événement indique qu'un ordinateur d'état a essayé de traiter un événement inattendu. L'événement a été considéré comme fatal. L'événement répertorie l'ID de transaction, l'ordinateur d'état, l'état actuel, l'historique, le nom de l'événement, le détails de l'événement, le nom de processus et l'ID de processus.
