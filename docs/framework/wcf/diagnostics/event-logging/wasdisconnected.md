# <a name="wasdisconnected"></a>WasDisconnected

ID : 162  
  
 Gravité : Erreur  
  
 Catégorie : SMSvcHost  
  
## <a name="description"></a>Description  

 Cet événement indique que le service WAS (Windows Process Activation) a été déconnecté. L'événement répertorie le HRESULT, le nom de processus et l'ID de processus.
