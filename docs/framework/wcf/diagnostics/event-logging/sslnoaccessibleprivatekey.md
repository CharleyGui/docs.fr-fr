# <a name="sslnoaccessibleprivatekey"></a>SslNoAccessiblePrivateKey

ID : 155  
  
 Gravité : Erreur  
  
 Catégorie : TransactionBridge  
  
## <a name="description"></a>Description  

 Cet événement indique qu'un certificat d'identité avec le nom de sujet et l'empreinte numérique spécifiques n'a pas de clé privée accessible. Cet événement répertorie le nom et l'ID de processus.
