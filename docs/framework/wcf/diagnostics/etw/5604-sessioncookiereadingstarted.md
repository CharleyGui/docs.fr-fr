## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|5604|  
|Mots clés|Sécurité|  
|Level|Commentaires|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque la lecture du jeton de session à partir du cookie de session a démarré.  
  
## <a name="message"></a>Message  

 Début de la lecture du jeton de session à partir du cookie de session.
