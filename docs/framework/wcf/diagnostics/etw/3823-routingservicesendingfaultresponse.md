# <a name="3823---routingservicesendingfaultresponse"></a>3823 - RoutingServiceSendingFaultResponse

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|3823|  
|Mots clés|RoutingServices|  
|Level|Avertissement|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque le service de routage renvoie un message de réponse d'erreur.  
  
## <a name="message"></a>Message  

 Le service de routage renvoie un message de réponse d'erreur avec l'action « %1 ».
