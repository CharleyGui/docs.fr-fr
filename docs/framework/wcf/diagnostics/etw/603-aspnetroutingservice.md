## <a name="description"></a>Description  

 Cet événement est émis lorsque la demande entrante correspond à un service WCF défini dans l’itinéraire ASP.NET avec l’adresse %1.  
  
## <a name="message"></a>Message  

 La demande entrante correspond à un service WCF défini dans ASP.NET route avec l’adresse %1.
