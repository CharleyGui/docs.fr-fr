# <a name="3305---clientbasecachedchannelfactorycount"></a>3305 - ClientBaseCachedChannelFactoryCount

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|3305|  
|Mots clés|ServiceModel|  
|Level|Informations|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis pour indiquer le nombre de fabriques de canaux mises en cache par base cliente.  
  
## <a name="message"></a>Message  

 Le nombre de fabriques de canaux mises en cache est de %1.  Seules «%2 » fabriques de canaux peuvent être mises en cache.
