# <a name="3815---routingserviceprocessingmessage"></a>3815 - RoutingServiceProcessingMessage

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|3815|  
|Mots clés|RoutingServices|  
|Level|Informations|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque le service de routage traite un message.  
  
## <a name="message"></a>Message  

 Le service de routage traite un message avec l’ID : « %1 », action : « %2 », URL entrante : « %3 », Reçu dans la transaction : %4.
