# <a name="3416---websocketconnectionrequestsendstop"></a>3416 - WebSocketConnectionRequestSendStop

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|3416|  
|Mots clés|HTTP|  
|Level|Commentaires|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque l'envoi d'une requête de connexion WebSocket s'est arrêté.  
  
## <a name="message"></a>Message  

 Demande de connexion à l'élément WebSocketId %1 envoyée.
