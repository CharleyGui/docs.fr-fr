# <a name="3321---connectionpoolmiss"></a>3321 - ConnectionPoolMiss

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|3321|  
|Mots clés|Channel|  
|Level|Commentaires|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque le pool n'a aucune connexion disponible.  
  
## <a name="message"></a>Message  

 Le pool de %1 ne dispose d'aucune connexion disponible et dispose de %2 connexions occupées.
