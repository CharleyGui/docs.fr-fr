|||  
|-|-|  
|id|3832|  
|Mots clés|RoutingServices|  
|Level|Informations|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque le service de routage a transmis le message avec un ID spécifié.  
  
## <a name="message"></a>Message  

 Le service de routage a correctement transmis le message avec l'ID « %1 » [opération %2] à « %3 ».
