# <a name="4010---pendingsessionqueuefull"></a>4010 - PendingSessionQueueFull

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|4010|  
|Mots clés|ActivationServices|  
|Level|Avertissement|  
|Channel|Microsoft-Windows-Application Server-Applications/Analyse|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque la distribution de session a échoué, car la file d'attente des sessions en attente est pleine.  
  
## <a name="message"></a>Message  

 La distribution de sessions pour « %1 » a échoué, car la file d'attente des sessions en attente contient « %2 » éléments en attente, et est donc pleine.
