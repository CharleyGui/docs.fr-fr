# <a name="4016---messagequeueduplicatedsocketerror"></a>4016 - MessageQueueDuplicatedSocketError

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|4016|  
|Mots clés|ActivationServices|  
|Level|Error|  
|Channel|Microsoft-Windows-Application Server-Applications/Analyse|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque la file des messages n'a pas pu dupliquer de socket.  
  
## <a name="message"></a>Message  

 Échec de la duplication de socket par la file d'attente des messages.
