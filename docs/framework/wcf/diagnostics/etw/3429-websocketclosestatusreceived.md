|||  
|-|-|

|Mots clés|HTTP|  
|Level|Commentaires|

## <a name="description"></a>Description  

 Cet événement est émis lorsque l'état de fermeture WebSocket est reçu.  
  
## <a name="message"></a>Message  

 Message de fermeture de la connexion à l'élément WebSocketId %1 reçu avec l'état « %2 ».
