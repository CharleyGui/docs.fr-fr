# <a name="4805---discoverymessagewithinvalidcontent"></a>4805 - DiscoveryMessageWithInvalidContent

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|4805|  
|Mots clés|Découverte|  
|Level|Avertissement|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque le message de découverte contient du contenu non valide.  
  
## <a name="message"></a>Message  

 Un message %1 avec messageId='%2' a été déposé, car son contenu n’était pas valide.
