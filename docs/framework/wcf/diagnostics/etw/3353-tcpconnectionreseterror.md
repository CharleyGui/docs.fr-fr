|||

|Mots clés|TCP|  
|Level|Avertissement|

## <a name="description"></a>Description  

 Cet événement est émis avec l'erreur de réinitialisation de connexion TCP.  
  
## <a name="message"></a>Message  

 Le SocketId %1 à l'adresse distante %2 a rencontré une erreur lors de la réinitialisation de la connexion.
