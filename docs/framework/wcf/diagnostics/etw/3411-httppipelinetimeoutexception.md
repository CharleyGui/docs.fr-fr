# <a name="3411---httppipelinetimeoutexception"></a>3411 - HttpPipelineTimeoutException

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|3411|  
|Mots clés|HTTP|  
|Level|Error|  
|Channel|Microsoft-Windows-Application Server-Applications/Analyse|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque la connexion WebSocket a dépassé le délai imparti.  
  
## <a name="message"></a>Message  

 La connexion WebSocket a expiré.
