# <a name="4807---discoverymessagewithinvalidreplyto"></a>4807 - DiscoveryMessageWithInvalidReplyTo

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|4807|  
|Mots clés|Découverte|  
|Level|Avertissement|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque le message de découverte a été supprimé, car son adresse de réponse ReplyTo n'était pas valide.  
  
## <a name="message"></a>Message  

 Un message de demande de découverte avec messageId='%1' a été supprimé, car son adresse de réponse ReplyTo n’était pas valide.
