# <a name="4813---duplicatediscoverymessage"></a>4813 - DuplicateDiscoveryMessage

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|4813|  
|Mots clés|Découverte|  
|Level|Avertissement|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque le message a été supprimé, car il s’agissait d’un doublon.  
  
## <a name="message"></a>Message  

 Un message %1 avec messageId='%2' a été supprimé, car il s'agissait d'un doublon.
