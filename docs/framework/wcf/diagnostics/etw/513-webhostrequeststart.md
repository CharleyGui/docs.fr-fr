## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|513|  
|Mots clés|WebHost|  
|Level|Informations|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque WebHostRequest est démarré.  
  
## <a name="message"></a>Message  

 Réception d'une requête avec le chemin d'accès virtuel « %2 » en provenance de l'AppDomain « %1 ».
