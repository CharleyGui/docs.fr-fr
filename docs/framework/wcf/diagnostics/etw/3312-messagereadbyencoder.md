## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|3312|  
|Mots clés|Channel|  
|Level|Informations|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque le message a été lu par l’encodeur.  
  
## <a name="message"></a>Message  

 Un message d'une taille de « %1 » octets a été lu par l'encodeur.
