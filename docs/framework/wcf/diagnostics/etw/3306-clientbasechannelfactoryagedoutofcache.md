## <a name="description"></a>Description  

 Cet événement est émis lorsqu'une fabrique de canaux a été échue du cache, car le cache a atteint sa limite.  
  
## <a name="message"></a>Message  

 Une fabrique de canaux a été échue du cache, car le cache a atteint sa limite de « %1 ».
