|||  
|-|-|

|Mots clés|Channel|  
|Level|Commentaires|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque le MtomMessageEncoder a commencé à décoder le message.  
  
## <a name="message"></a>Message  

 MtomMessageEncoder a démarré le décodage du message.
