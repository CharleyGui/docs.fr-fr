# <a name="3803---routingserviceprocessingfailure"></a>3803 - RoutingServiceProcessingFailure

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|3803|  
|Mots clés|RoutingServices|  
|Level|Error|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque le service de routage n'est pas parvenu à traiter un message sur le point de terminaison.  
  
## <a name="message"></a>Message  

 Échec du service de routage lors du traitement d'un message sur le point de terminaison avec l'adresse « %1 ».
