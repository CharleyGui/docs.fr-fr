# <a name="3819---routingservicemovedtobackup"></a>3819 - RoutingServiceMovedToBackup

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|3819|  
|Mots clés|RoutingServices|  
|Level|Informations|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque le service de routage a été déplacé vers les points de terminaison de sauvegarde.  
  
## <a name="message"></a>Message  

 Message du service de routage avec l'ID %1 [opération %2] déplacé vers le point de terminaison de sauvegarde « %3 ».
