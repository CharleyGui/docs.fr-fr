# <a name="62326---httphandlerpickedforurl"></a>62326 - HttpHandlerPickedForUrl

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|62326|  
|Mots clés|WebHost|  
|Level|Informations|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Cet événement est émis lorsque le gestionnaire HTTP est sélectionné pour traiter toutes les requêtes adressées à cette URL.  
  
## <a name="message"></a>Message  

 L'URL « %1 » héberge un document XAML avec le type d'élément racine « %2 ». Le type de gestionnaire HTTP « %3 » est sélectionné pour traiter toutes les requêtes adressées à cette URL.
