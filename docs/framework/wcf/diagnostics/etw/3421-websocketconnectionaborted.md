# <a name="3421---websocketconnectionaborted"></a>3421 - WebSocketConnectionAborted

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|3421|  
|Mots clés|HTTP|  
|Level|Error|  
|Channel|Microsoft-Windows-Application Server-Applications/Analyse|  
  
## <a name="description"></a>Description  

 Cet événement est émis lors de l'abandon de la connexion WebSocket.  
  
## <a name="message"></a>Message  

 La connexion à l'élément WebSocketId %1 est abandonnée.
