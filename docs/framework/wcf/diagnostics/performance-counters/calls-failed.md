# <a name="calls-failed"></a>Appels ayant échoué
Nom du compteur : Appels ayant échoué  
  
## <a name="description"></a>Description  
 Nombre d'appels avec exceptions non traitées dans cette opération. Ce compteur est incrémenté chaque fois qu’une exception non gérée dans cette opération.
