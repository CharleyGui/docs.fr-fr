# <a name="endpoint-calls-failed"></a>Point de terminaison : appels ayant échoué

Nom du compteur : appels ayant échoué.  
  
## <a name="description"></a>Description  

 Nombre d'appels à ce point de terminaison avec des exceptions non prises en charge. Ce compteur est incrémenté à chaque exception non traitée à ce point de terminaison.
