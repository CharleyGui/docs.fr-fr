# <a name="queue-dropped-messages"></a>Messages mis en file d’attente déposés

Nom du compteur : dépôt de messages en file d’attente.  
  
## <a name="description"></a>Description  

 Nombre de messages déposés par le transport de mise en file d’attente au niveau de ce service.  
  
 Pour plus d’informations sur le moment où les messages sont supprimés, consultez [gestion des messages incohérents](../../feature-details/poison-message-handling.md).
