# <a name="queued-poison-messages"></a>Messages incohérents mis en file d'attente
Nom du compteur : Messages incohérents mis en file d'attente.  
  
## <a name="description"></a>Description  
 Nombre de messages vers ce service marqués comme étant incohérents par le transport de mise en file d'attente.  
  
 Pour plus d’informations sur le moment où les messages sont marqués empoisonnés, voir [Poison-Message Handling](../../feature-details/poison-message-handling.md).
