# <a name="microsofttransactionstransactionbridgerecoveredparticipantinvalidmetadata"></a>Microsoft.Transactions.TransactionBridge.RecoveredParticipantInvalidMetadata
La référence du point de terminaison trouvée dans une entrée du journal de récupération de participant contenait des métadonnées non valides ou incompatibles.  
  
## <a name="description"></a>Description  
 Le système ne peut pas créer de proxy pour une inscription de participant récupéré.  
  
## <a name="troubleshooting"></a>Dépannage  
 Contactez Microsoft si le problème persiste.
