# <a name="systemservicemodelchannelssocketconnectioncreate"></a>System.ServiceModel.Channels.SocketConnectionCreate

System.ServiceModel.Channels.SocketConnectionCreate  
  
## <a name="description"></a>Description  

 Ce suivi est émis pour la première activité de processus effectuée par le client et pour l'activité correspondant à la réception d'octets du service. Il fournit les adresses IP locales et distantes. Il est émis au niveau Informations.
