# <a name="systemservicemodelportsharingreadnettcpconfig"></a>System.ServiceModel.PortSharing.ReadNetTcpConfig

System.ServiceModel.PortSharing.ReadNetTcpConfig  
  
## <a name="description"></a>Description  

 Les données de configuration pour NetTcpSection ont été lues avec les données spécifiées : listenBacklog, maxConnectionsPendingDispatch, maxPendingAccepts, receiveTimeout et teredoEnabled.
