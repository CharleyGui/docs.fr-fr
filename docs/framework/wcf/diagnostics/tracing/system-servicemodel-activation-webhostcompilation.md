# <a name="systemservicemodelactivationwebhostcompilation"></a>System.ServiceModel.Activation.WebHostCompilation

System.ServiceModel.Activation.WebHostCompilation  
  
## <a name="description"></a>Description  

 Se produit lorsqu'un service IIS (Internet Information Service) ou qu'un service d'activation de processus de Windows (WAS, Windows Process Activation Service) hébergé est compilé à partir du code incorporé présent dans le fichier .svc.
