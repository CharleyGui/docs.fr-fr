# <a name="microsofttransactionstransactionbridgeregistrationcoordinatorfaulted"></a>Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorFaulted
Le service du protocole WS-AT a reçu une faute de son coordinateur en réponse à un message Register.  
  
## <a name="description"></a>Description  
 Suivi si le TransactionManager local n’est pas en mesure de s’inscrire auprès de son TransactionManager supérieur car une erreur est retournée.  
  
## <a name="troubleshooting"></a>Dépannage  
 Inspectez le message de suivi pour connaître l'erreur retournée.
