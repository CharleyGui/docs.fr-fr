# <a name="systemservicemodelservicehosttimeoutonclose"></a>System.ServiceModel.ServiceHostTimeoutOnClose

System.ServiceModel.ServiceHostTimeoutOnClose  
  
## <a name="description"></a>Description  

 Le délai d’attente de l’opération de fermeture d’un ServiceHost est dépassé. Cela indique que certains canaux de session n’ont pas été correctement arrêtés par le client et ont peut-être été abandonnés. Modifiez la durée par défaut afin d'ajuster le délai d'attente de fermeture.
