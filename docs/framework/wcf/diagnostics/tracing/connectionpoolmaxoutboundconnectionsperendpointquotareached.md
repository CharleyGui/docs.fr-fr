System.ServiceModel.Channels.ConnectionPoolMaxOutboundConnectionsPerEndpointQuotaReached  
  
## <a name="description"></a>Description  
 Le quota MaxOutboundConnectionsPerEndpoint ayant été atteint, la connexion a été fermée et celle-ci n'a pas été stockée dans ce pool de connexions.
