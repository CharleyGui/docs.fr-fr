## <a name="propagating-the-activity-id-to-a-service"></a><span data-ttu-id="c7841-123">Propager l'ID d'activité à un service</span><span class="sxs-lookup"><span data-stu-id="c7841-123">Propagating the Activity ID to A Service</span></span>

> [!NOTE]
> <span data-ttu-id="c7841-127">Si l' `propagateActivity` attribut a la valeur `true` sur le client et le service, l’activité ambiante dans l’étendue de l’opération du service est définie par WCF.</span><span class="sxs-lookup"><span data-stu-id="c7841-127">If the `propagateActivity` attribute is set to `true` on both the client and service, the ambient activity in the operation scope of the service is set by WCF.</span></span>

```csharp
    Guid newGuid = Guid.NewGuid();
    Trace.CorrelationManager.ActivityId = newGuid;
}

// Emit the processing traces for that request.

                            + n1 + ", " + n2);

```
