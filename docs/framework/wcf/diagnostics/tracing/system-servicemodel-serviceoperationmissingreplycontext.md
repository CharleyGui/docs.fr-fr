# <a name="systemservicemodelserviceoperationmissingreplycontext"></a>System.ServiceModel.ServiceOperationMissingReplyContext

System.ServiceModel.ServiceOperationMissingReplyContext  
  
## <a name="description"></a>Description  

 ReplyContext est null pour une opération bidirectionnelle. Les données de suivi affichent le message suivant : « L'opération Reply n'a pas de IRequestContext à utiliser pour la réponse. »  
  
 Ce suivi est utile pour déboguer pendant le développement.
