# <a name="systemservicemodelactivationwebhostprotocolmisconfigured"></a>System.ServiceModel.Activation.WebHostProtocolMisconfigured
Le protocole est configuré incorrectement dans le service WAS (Windows Process Activation Service).  
  
## <a name="description"></a>Description  
 Cette erreur se produit lorsque le protocole de transport est configuré incorrectement.  
  
## <a name="troubleshooting"></a>Dépannage  
 Vérifiez et corrigez les paramètres de configuration de transport dans web.config.
