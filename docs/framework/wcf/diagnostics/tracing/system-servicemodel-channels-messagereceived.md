# <a name="systemservicemodelchannelsmessagereceived"></a>System.ServiceModel.Channels.MessageReceived

System.ServiceModel.Channels.MessageReceived  
  
## <a name="description"></a>Description  

 Réception d'un message sur un canal.  
  
 Cette trace est émise lors de la réception d’un message WCF si la `System.ServiceModel` source de suivi est activée au niveau des informations ou des commentaires. Ce suivi est nécessaire pour consulter la flèche de corrélation du message dans la vue du graphique d'activité.
