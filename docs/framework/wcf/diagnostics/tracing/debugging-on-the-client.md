# <a name="debugging-on-the-client"></a>Débogage sur le client

Pour permettre aux utilisateurs d’écrire plus facilement des applications clientes pour votre service WCF, vous pouvez ajouter le [\<serviceDebug>](../../../configure-apps/file-schema/wcf/servicedebug.md) comportement de service au fichier de configuration de votre service. Ce comportement permet de publier des pages d'aide et de retourner des informations d'exception gérées dans les détails des erreurs SOAP renvoyées au client.
