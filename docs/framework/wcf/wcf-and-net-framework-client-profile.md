# <a name="wcf-and-net-framework-client-profile"></a>WCF et .NET Framework Client Profile

.NET Framework profil client est une version allégée du .NET Framework complet conçu pour les clients qui n’ont pas besoin de la totalité de l’infrastructure. Windows Communication Foundation n'est pas intégralement pris en charge par l'infrastructure cliente.  
  
## <a name="wcf-features-supported-by-the-net-framework-client-profile"></a>Fonctionnalités WCF prises en charge par le profil client .NET Framework  

 Les fonctionnalités de Windows Communication Foundation suivantes sont prises en charge par .NET Framework profil client :  
  
- WCF est intégralement pris en charge, à l'exception de Cardspace et l'hébergement Web.  
  
- Les canaux TCP/IP de communication à distance sont pris en charge.  
  
- Les services Web Asmx ne sont pas pris en charge.
