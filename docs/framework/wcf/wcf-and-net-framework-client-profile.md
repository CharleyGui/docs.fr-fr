# <a name="wcf-and-net-framework-client-profile"></a>WCF et .NET Framework Client Profile
.NET framework Client Profile est une version légère du .NET Framework complet destinée aux clients qui n’avez pas besoin de l’infrastructure entière. Windows Communication Foundation n'est pas intégralement pris en charge par l'infrastructure cliente.  
  
## <a name="wcf-features-supported-by-the-net-framework-client-profile"></a>Fonctionnalités WCF prises en charge par le .NET Framework Client Profile  
 Les fonctionnalités de Windows Communication Foundation suivantes sont prises en charge par .NET Framework Client Profile :  
  
- WCF est intégralement pris en charge, à l'exception de Cardspace et l'hébergement Web.  
  
- Les canaux TCP/IP de communication à distance sont pris en charge.  
  
- Les services Web Asmx ne sont pas pris en charge.
