# <a name="wcf-system-requirements"></a>Configuration requise WCF

Windows Communication Foundation (WCF) est une infrastructure de communication utilisée pour créer des applications distribuées. La liste suivante répertorie les spécifications qui permettent à WCF de s’exécuter.

## <a name="system-requirements"></a>Configuration système requise

WCF est installé par défaut sur [!INCLUDE[wv](../../../includes/wv-md.md)].

WCF peut également être installé sur [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], Windows Server 2003 R2 ou [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].

> [!NOTE]
> La fonctionnalité de Message Queuing (MSMQ) de WCF est prise en charge uniquement sur [!INCLUDE[wv](../../../includes/wv-md.md)], Windows Server 2003 R2, [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)] et Windows XP professionnel.
