# <a name="security-extensibility"></a>Extensibilité de la sécurité
Cette section contient des exemples qui illustrent la sécurité personnalisée dans Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>Dans cette section  
 [Durable Issued Token Provider](durable-issued-token-provider.md)  
 Montre comment implémenter un fournisseur personnalisé de jetons émis pour le client.  
  
 [SAML Token Provider](saml-token-provider.md)  
 Montre comment implémenter un fournisseur de jetons SAML personnalisé par client.  
  
 [Supporting Tokens](supporting-tokens.md)  
 Montre comment ajouter des jetons supplémentaires à un message qui utilise WS-Security.  
  
 [Token Authenticator](token-authenticator.md)  
 Montre comment implémenter un authentificateur de jetons personnalisé.  
  
 [Token Provider](token-provider.md)  
 Montre comment implémenter un fournisseur de jetons personnalisé.  
  
 [Validateur de nom d'utilisateur et de mot de passe](user-name-password-validator.md)  
 Montre comment implémenter un validateur UserNamePassword personnalisé.  
  
 [X.509 Certificate Validator](x-509-certificate-validator.md)  
 Montre comment implémenter un validateur de certificat X.509 personnalisé.  
  
 [Authorization Policy](authorization-policy.md)  
 Montre comment implémenter une stratégie d'autorisation de revendication personnalisée et un gestionnaire d'autorisations de service personnalisé associé.  
  
 [Custom Token](custom-token.md)  
 Montre comment ajouter une implémentation de jeton personnalisée dans une application Windows Communication Foundation (WCF).  
  
 [Client Validation](client-validation.md)  
 Montre comment écrire un comportement de point de terminaison client qui valide le point de terminaison de service pour garantir que le point de terminaison de service est sûr à utiliser.
