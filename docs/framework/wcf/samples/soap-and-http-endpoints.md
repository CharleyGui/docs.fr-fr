## <a name="discussion"></a><span data-ttu-id="e23d9-108">Discussions</span><span class="sxs-lookup"><span data-stu-id="e23d9-108">Discussion</span></span>  
 <span data-ttu-id="e23d9-109">Cet exemple se compose de deux composants : un projet d’application Web (service) qui contient un service WCF et une application console (client) qui appelle des opérations de service à l’aide de liaisons SOAP et HTTP.</span><span class="sxs-lookup"><span data-stu-id="e23d9-109">This sample consists of two components: a Web Application project (Service) that contains a WCF service and a console application (Client) that invokes service operations using SOAP and HTTP bindings.</span></span>

 <span data-ttu-id="e23d9-115">Le fichier Web. config configure le service WCF avec 3 points de terminaison :</span><span class="sxs-lookup"><span data-stu-id="e23d9-115">The Web.config file configures the WCF service with 3 endpoints:</span></span>

- <span data-ttu-id="e23d9-118">le point de terminaison ~/service.svc/soap qui permet aux clients d’accéder au service en utilisant la liaison SOAP sur HTTP.</span><span class="sxs-lookup"><span data-stu-id="e23d9-118">The ~/service.svc/soap endpoint that allows the clients to access the service using the SOAP over HTTP binding.</span></span>

 <span data-ttu-id="e23d9-121">Le projet client illustre l’accès au service à l’aide d’un proxy SOAP (généré via **Ajouter une référence de service**) et l’accès au service à l’aide de <xref:System.Net.WebClient> .</span><span class="sxs-lookup"><span data-stu-id="e23d9-121">The client project demonstrates accessing the service using a SOAP proxy (generated through **Add Service Reference**) and accessing the service using <xref:System.Net.WebClient>.</span></span>  
  
 <span data-ttu-id="e23d9-122">L'exemple se compose d'un service hébergé sur le Web et d'une application console.</span><span class="sxs-lookup"><span data-stu-id="e23d9-122">The sample consists of a Web-hosted service and a console application.</span></span> <span data-ttu-id="e23d9-123">Lorsque l'application console s'exécute, le client adresse des requêtes au service et affiche les informations pertinentes des réponses dans la fenêtre de console.</span><span class="sxs-lookup"><span data-stu-id="e23d9-123">As the console application runs, the client makes requests to the service and writes the pertinent information from the responses to the console window.</span></span>
