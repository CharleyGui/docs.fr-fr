# <a name="develop-client-applications-with-net-framework"></a>Développer des applications clientes avec .NET Framework

Il existe plusieurs façons de développer des applications Windows avec .NET Framework. Vous pouvez utiliser l’un de ces outils et de ces frameworks :

- [Plateforme Windows universelle (UWP)](/windows/uwp/)
- [Windows Presentation Foundation (WPF)](./wpf/index.md)
- [Windows Forms](./winforms/index.md)

Cette section contient des articles qui décrivent comment créer des applications Windows à l’aide de Windows Presentation Foundation ou Windows Forms. Toutefois, vous pouvez également créer des applications Web à l’aide d' .NET Framework et d’applications clientes pour les ordinateurs ou les appareils que vous mettez à disposition via Microsoft Store (applications UWP).
