# <a name="develop-client-applications-with-net-framework"></a>Développer des applications client avec .NET Framework

Il existe plusieurs façons de développer des applications basées sur Windows avec .NET Framework. Vous pouvez utiliser l’un de ces outils et de ces frameworks :

- [Plateforme Windows universelle (UWP)](/windows/uwp/)
- [Windows Presentation Foundation (WPF)](./wpf/index.md)
- [Windows Forms](./winforms/index.md)

Cette section contient des articles qui décrivent comment créer des applications basées sur Windows en utilisant Windows Presentation Foundation ou Windows Forms. Cependant, vous pouvez également créer des applications Web à l’aide de .NET Framework et d’applications client pour ordinateurs ou appareils que vous rendez disponibles via Microsoft Store (applications UWP).
