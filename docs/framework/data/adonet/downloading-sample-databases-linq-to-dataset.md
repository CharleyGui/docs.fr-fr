# <a name="download-sample-databases-linq-to-dataset"></a>Télécharger des extraits de données (LINQ sur DataSet)

Les échantillons et les procédures pas à pas de la documentation LINQ à DataSet utilisent la base de données de l’échantillon AdventureWorks. Vous pouvez télécharger cette base de données d’échantillons gratuitement. Naviguez vers [les bases de données d’échantillons d’AdventureWorks](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks) pour télécharger la base de données. Ensuite, suivez les instructions sur cette page pour attacher ou restaurer une base de données.
  
## <a name="sql-server-express"></a>SQL Server Express

Les échantillons et les procédures pas à pas de la section LINQ à DataSet utilisent SQL Server 2005 comme magasin de données, mais vous pouvez les modifier pour utiliser [SQL Server Express](https://go.microsoft.com/fwlink/?linkid=866658) à la place. SQL Server Express est gratuit, et vous pouvez le redistribuer avec des applications.
