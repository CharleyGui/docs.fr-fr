# <a name="clr-triggers"></a>déclencheurs CLR

Un déclencheur est un type spécial de procédure stockée qui s'exécute automatiquement lorsqu'un événement de langage s'exécute. En raison de l'intégration de Microsoft SQL Server avec le Common Language Runtime (CLR) de .NET Framework, vous pouvez utiliser tout langage de .NET Framework pour créer des déclencheurs CLR.  
  
Pour plus d’informations, consultez [créer des déclencheurs CLR](/sql/relational-databases/triggers/create-clr-triggers).
