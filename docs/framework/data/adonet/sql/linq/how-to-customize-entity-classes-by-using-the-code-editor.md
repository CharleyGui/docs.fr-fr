# <a name="how-to-customize-entity-classes-by-using-the-code-editor"></a>Procédure : Personnaliser des classes d’entité à l’aide de l’éditeur de code
Les développeurs qui utilisent Visual Studio peuvent utiliser la Concepteur Objet Relationnel pour créer ou personnaliser leurs classes d’entité.  
  
 Vous pouvez également utiliser l’éditeur de code Visual Studio pour écrire votre propre code de mappage ou personnaliser du code qui a déjà été généré. Pour plus d’informations, consultez [mappage basé sur les attributs](attribute-based-mapping.md).  
  
 Les rubriques de cette section décrivent comment personnaliser votre modèle objet.  
  
 [Guide pratique : Spécifier les noms des bases de données](how-to-specify-database-names.md)  
 Explique comment utiliser <xref:System.Data.Linq.Mapping.DatabaseAttribute.Name%2A>.  
  
 [Guide pratique pour Représenter des tables en tant que classes](how-to-represent-tables-as-classes.md)  
 Explique comment utiliser <xref:System.Data.Linq.Mapping.TableAttribute>.  
  
 [Guide pratique pour Représenter des colonnes en tant que membres de classe](how-to-represent-columns-as-class-members.md)  
 Explique comment utiliser <xref:System.Data.Linq.Mapping.ColumnAttribute>.  
  
 [Guide pratique : Représenter les clés primaires](how-to-represent-primary-keys.md)  
 Explique comment utiliser <xref:System.Data.Linq.Mapping.ColumnAttribute.IsPrimaryKey%2A>.  
  
 [Guide pratique pour Mapper les relations de base de données](how-to-map-database-relationships.md)  
 Fournit des exemples d'utilisation de l'attribut <xref:System.Data.Linq.Mapping.AssociationAttribute>.  
  
 [Guide pratique : Représenter les colonnes comme générées par la base de données](how-to-represent-columns-as-database-generated.md)  
 Explique comment utiliser <xref:System.Data.Linq.Mapping.ColumnAttribute.IsDbGenerated%2A>.  
  
 [Guide pratique : Représenter des colonnes en tant que colonnes timestamp ou version](how-to-represent-columns-as-timestamp-or-version-columns.md)  
 Explique comment utiliser <xref:System.Data.Linq.Mapping.ColumnAttribute.IsVersion%2A>.  
  
 [Guide pratique pour Spécifier les types de données de base de données](how-to-specify-database-data-types.md)  
 Explique comment utiliser <xref:System.Data.Linq.Mapping.ColumnAttribute.DbType%2A>.  
  
 [Guide pratique : Représenter des colonnes calculées](how-to-represent-computed-columns.md)  
 Explique comment utiliser <xref:System.Data.Linq.Mapping.ColumnAttribute.Expression%2A>.  
  
 [Guide pratique pour Spécifier des champs de stockage privés](how-to-specify-private-storage-fields.md)  
 Explique comment utiliser <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A>.  
  
 [Guide pratique pour Représenter les colonnes comme autorisant les valeurs null](how-to-represent-columns-as-allowing-null-values.md)  
 Explique comment utiliser <xref:System.Data.Linq.Mapping.ColumnAttribute.CanBeNull%2A>.  
  
 [Guide pratique : Hiérarchies d’héritage de mappage](how-to-map-inheritance-hierarchies.md)  
 Décrit les mappages requis pour spécifier une hiérarchie d'héritage.  
  
 [Guide pratique pour Spécifier la vérification de conflit d’accès concurrentiel](how-to-specify-concurrency-conflict-checking.md)  
 Explique comment utiliser <xref:System.Data.Linq.Mapping.ColumnAttribute.UpdateCheck%2A>.
