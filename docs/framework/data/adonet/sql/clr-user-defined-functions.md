# <a name="clr-user-defined-functions"></a>Fonctions CLR définies par l'utilisateur

Les fonctions définies par l'utilisateur sont des routines qui acceptent des paramètres, effectuent des calculs ou d'autres actions et retournent un résultat. Vous pouvez écrire des fonctions définies par l'utilisateur dans tout langage de programmation Microsoft .NET Framework, tel que Microsoft Visual Basic .NET ou Microsoft Visual C#.  
  
 Pour plus d’informations, consultez [fonctions CLR définies par l’utilisateur](/sql/relational-databases/clr-integration-database-objects-user-defined-functions/clr-user-defined-functions).
