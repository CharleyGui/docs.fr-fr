# <a name="linq-to-dataset-examples"></a>Exemples de LINQ to DataSet
Cette section fournit des exemples de programmation LINQ to DataSet qui utilisent les opérateurs de requête standard. Le <xref:System.Data.DataSet> utilisé dans ces exemples est rempli à l’aide `FillDataSet` de la méthode, qui est spécifiée lors [du chargement des données dans un DataSet](loading-data-into-a-dataset.md). Pour plus d’informations, consultez [vue d’ensemble desC#opérateurs de requête standard ()](../../../csharp/programming-guide/concepts/linq/standard-query-operators-overview.md) ou [vue d’ensemble des opérateurs de requête standard (Visual Basic)](../../../visual-basic/programming-guide/concepts/linq/standard-query-operators-overview.md).
