- <span data-ttu-id="645c4-111">Clé d'entité de la terminaison principale.</span><span class="sxs-lookup"><span data-stu-id="645c4-111">The entity key of the principal end.</span></span>  
  
- <span data-ttu-id="645c4-112">Terminaison dépendante de la contrainte.</span><span class="sxs-lookup"><span data-stu-id="645c4-112">The dependent end of the constraint.</span></span> <span data-ttu-id="645c4-113">(Type d'entité dont une ou plusieurs propriétés référencent la clé d'entité de la terminaison principale.)</span><span class="sxs-lookup"><span data-stu-id="645c4-113">(An entity type that has a property or properties that reference the entity key of the principal end.)</span></span>
