# <a name="entity-framework-resources"></a>Ressources Entity Framework
Les ressources externes suivantes fournissent des informations et une aide pour créer des applications Entity Framework.  
  
 [Centre de développement des données](https://go.microsoft.com/fwlink/?LinkId=213876)  
 Emplacement central pour le développement de données avec les technologies Microsoft.  
  
 [Blog de l’équipe ADO.NET](https://go.microsoft.com/fwlink/?LinkId=91905)  
 Blog contenant des mises à jour et des discussions sur les caractéristiques et les fonctionnalités d'ADO.NET.  
  
 [Blog sur la conception de Entity Framework](https://go.microsoft.com/fwlink/?LinkId=186888)  
 Discussions relatives à la conception et aperçus des fonctionnalités des futures versions d’Entity Framework.  
  
 [Guide pratique de la plateforme de données Vidéos : Série Entity Framework](https://go.microsoft.com/fwlink/?LinkId=124600)  
 Contient un ensemble de présentations vidéo qui montrent comment générer des applications à l’aide de différents composants d’Entity Framework.  
  
 [FAQ Entity Framework](https://social.technet.microsoft.com/wiki/contents/articles/3737.entity-framework-faq.aspx)  
 Section wiki contenant les questions fréquemment posées sur le Entity Framework.
