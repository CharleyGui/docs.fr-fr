# <a name="working-with-objects"></a>Utilisation des objets

Le Entity Framework vous permet d’interroger, d’insérer, de mettre à jour et de supprimer des données, exprimées sous forme d’objets CLR (common language runtime typés) qui sont des instances de types d’entité. Les types d'entité représentent les entités définies dans le modèle conceptuel. Le Entity Framework mappe les entités et les relations définies dans un modèle conceptuel à une source de données. Le Entity Framework fournit des fonctionnalités permettant d’effectuer les opérations suivantes : matérialise les données retournées par la source de données sous forme d’objets ; suivre les modifications apportées aux objets ; gérer l’accès concurrentiel ; propager les modifications de l’objet vers la source de données ; et de lier des objets à des contrôles.  
  
 Pour plus d’informations sur l’utilisation des objets dans la dernière version du Entity Framework consultez [utilisation des objets](/previous-versions/gg696163(v=vs.103)).
