# <a name="getting-started-entity-framework"></a>Mise en route (Entity Framework)
ADO.NET Entity Framework prend en charge les applications et les services centrés sur les données, et fournit une plateforme pour la programmation sur des données qui élève le niveau d’abstraction du niveau relationnel logique au niveau conceptuel. En permettant aux développeurs d’utiliser des données à un niveau supérieur d’abstraction, le Entity Framework prend en charge le code qui est indépendant d’un moteur de stockage de données ou d’un schéma relationnel particulier. Pour plus d’informations, consultez [Entity Framework vue d’ensemble](overview.md).  
  
 Pour commencer rapidement à utiliser la dernière version du Entity Framework, consultez la page [prise en main](https://go.microsoft.com/fwlink/?LinkId=235280)
