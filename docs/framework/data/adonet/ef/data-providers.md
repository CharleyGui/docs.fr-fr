# <a name="entity-framework-data-providers"></a>Fournisseurs de données Entity Framework
Cette section fournit des informations sur les fournisseurs de données qui prennent en charge [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)].  
  
## <a name="in-this-section"></a>Dans cette section  
 [Fournisseur EntityClient pour Entity Framework](../../../../../docs/framework/data/adonet/ef/entityclient-provider-for-the-entity-framework.md)  
 Décrit le fournisseur de données EntityClient. Ce fournisseur transforme des requêtes sur un modèle de données en une arborescence de commandes canonique. L’arborescence de commandes peut ensuite être consommé par un fournisseur de données .NET Framework pour le [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)].  
  
 [SqlClient pour Entity Framework](../../../../../docs/framework/data/adonet/ef/sqlclient-for-the-entity-framework.md)  
 Décrit le fournisseur de données .NET Framework qui prend en charge le [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] pour une utilisation avec une base de données SQL Server.
