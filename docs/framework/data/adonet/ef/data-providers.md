# <a name="entity-framework-data-providers"></a>Fournisseurs de données Entity Framework
Cette section fournit des informations sur les fournisseurs de données qui prennent en charge le Entity Framework.  
  
## <a name="in-this-section"></a>Dans cette section  
 [Fournisseur EntityClient pour Entity Framework](entityclient-provider-for-the-entity-framework.md)  
 Décrit le fournisseur de données EntityClient. Ce fournisseur transforme des requêtes sur un modèle de données en une arborescence de commandes canonique. L'arborescence de commandes peut ensuite être utilisée par un fournisseur de données .NET Framework pour Entity Framework.  
  
 [SqlClient pour Entity Framework](sqlclient-for-the-entity-framework.md)  
 Décrit le fournisseur de données .NET Framework qui prend en charge la Entity Framework pour une utilisation avec une base de données SQL Server.
