# <a name="querying-a-conceptual-model"></a>Interrogation d'un modèle conceptuel
Le Entity Framework ADO.NET vous permet d’interroger un modèle conceptuel. Pour interroger le modèle conceptuel à l’aide de la version la plus récente du Entity Framework, consultez [interrogation des données](https://go.microsoft.com/fwlink/?LinkId=235282).
