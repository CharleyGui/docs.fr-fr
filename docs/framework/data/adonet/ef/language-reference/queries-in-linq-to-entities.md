# <a name="queries-in-linq-to-entities"></a><span data-ttu-id="14549-102">Requêtes dans LINQ to Entities</span><span class="sxs-lookup"><span data-stu-id="14549-102">Queries in LINQ to Entities</span></span>
<span data-ttu-id="14549-103">Une requête est une expression qui récupère des données d'une source de données.</span><span class="sxs-lookup"><span data-stu-id="14549-103">A query is an expression that retrieves data from a data source.</span></span> <span data-ttu-id="14549-104">En général, les requêtes sont exprimées dans un langage de requête spécialisé, tel que SQL pour les bases de données relationnelles et Xquery pour XML.</span><span class="sxs-lookup"><span data-stu-id="14549-104">Queries are usually expressed in a specialized query language, such as SQL for relational databases and XQuery for XML.</span></span> <span data-ttu-id="14549-105">Par conséquent, les développeurs ont dû apprendre un nouveau langage de requête pour chaque type de source de données ou format de données qu'ils interrogent.</span><span class="sxs-lookup"><span data-stu-id="14549-105">Therefore, developers have had to learn a new query language for each type of data source or data format that they query.</span></span> <span data-ttu-id="14549-106">LINQ (Language-Integrated Query) propose un modèle cohérent plus simple qui permet d'utiliser des données de types de sources et de formats divers.</span><span class="sxs-lookup"><span data-stu-id="14549-106">Language-Integrated Query (LINQ) offers a simpler, consistent model for working with data across various kinds of data sources and formats.</span></span> <span data-ttu-id="14549-107">Dans une requête LINQ, vous travaillez toujours avec des objets de programmation.</span><span class="sxs-lookup"><span data-stu-id="14549-107">In a LINQ query, you always work with programming objects.</span></span>  
  
 <span data-ttu-id="14549-108">Une opération de requête LINQ se compose de trois actions : obtenir la ou les sources de données, créer la requête, puis exécuter cette dernière.</span><span class="sxs-lookup"><span data-stu-id="14549-108">A LINQ query operation consists of three actions: obtain the data source or sources, create the query, and execute the query.</span></span>  
  
 <span data-ttu-id="14549-109">Les sources de données qui implémentent l'interface générique <xref:System.Collections.Generic.IEnumerable%601> ou l'interface générique <xref:System.Linq.IQueryable%601> peuvent être interrogées via LINQ.</span><span class="sxs-lookup"><span data-stu-id="14549-109">Data sources that implement the <xref:System.Collections.Generic.IEnumerable%601> generic interface or the <xref:System.Linq.IQueryable%601> generic interface can be queried through LINQ.</span></span> <span data-ttu-id="14549-110">Instances du paramètre générique <xref:System.Data.Objects.ObjectQuery%601> classe qui implémente le modèle générique <xref:System.Linq.IQueryable%601> l’interface, servir de la source de données pour LINQ aux requêtes d’entités.</span><span class="sxs-lookup"><span data-stu-id="14549-110">Instances of the generic <xref:System.Data.Objects.ObjectQuery%601> class, which implements the generic <xref:System.Linq.IQueryable%601> interface, serve as the data source for LINQ to Entities queries.</span></span> <span data-ttu-id="14549-111">La classe générique <xref:System.Data.Objects.ObjectQuery%601> représente une requête qui retourne une collection de zéro, un ou plusieurs objets typés.</span><span class="sxs-lookup"><span data-stu-id="14549-111">The <xref:System.Data.Objects.ObjectQuery%601> generic class represents a query that returns a collection of zero or more typed objects.</span></span> <span data-ttu-id="14549-112">Vous pouvez également laisser le compilateur déduire le type d’une entité en utilisant le mot clé c# `var` (Dim en Visual Basic).</span><span class="sxs-lookup"><span data-stu-id="14549-112">You can also let the compiler infer the type of an entity by using the C# keyword `var` (Dim in Visual Basic).</span></span>  
  
 <span data-ttu-id="14549-113">Dans la requête, vous spécifiez exactement les informations que vous voulez récupérer à partir de la source de données.</span><span class="sxs-lookup"><span data-stu-id="14549-113">In the query, you specify exactly the information that you want to retrieve from the data source.</span></span> <span data-ttu-id="14549-114">Une requête peut également spécifier la manière dont ces informations doivent être triées, regroupées et mises en forme avant d'être retournées.</span><span class="sxs-lookup"><span data-stu-id="14549-114">A query can also specify how that information should be sorted, grouped, and shaped before it is returned.</span></span> <span data-ttu-id="14549-115">Dans LINQ, une requête est stockée dans une variable.</span><span class="sxs-lookup"><span data-stu-id="14549-115">In LINQ, a query is stored in a variable.</span></span> <span data-ttu-id="14549-116">Si la requête retourne une séquence de valeurs, la variable de requête doit elle-même être de type requêtable.</span><span class="sxs-lookup"><span data-stu-id="14549-116">If the query returns a sequence of values, the query variable itself must be a queryable type.</span></span> <span data-ttu-id="14549-117">Cette variable de requête n'effectue aucune action et ne retourne aucune donnée ; elle stocke uniquement les informations de requête.</span><span class="sxs-lookup"><span data-stu-id="14549-117">This query variable takes no action and returns no data; it only stores the query information.</span></span> <span data-ttu-id="14549-118">Une fois que vous avez créé une requête, vous devez l'exécuter pour extraire des données.</span><span class="sxs-lookup"><span data-stu-id="14549-118">After you create a query you must execute that query to retrieve any data.</span></span>  
  
## <a name="query-syntax"></a><span data-ttu-id="14549-119">Syntaxe de requête</span><span class="sxs-lookup"><span data-stu-id="14549-119">Query Syntax</span></span>  
 <span data-ttu-id="14549-120">Les requêtes LINQ to Entities peuvent être composées dans deux syntaxes différentes : la syntaxe d'expression de requête et la syntaxe de requête fondée sur une requête.</span><span class="sxs-lookup"><span data-stu-id="14549-125">As a developer, you have the option of calling them directly by using method syntax, instead of using query syntax.</span></span> <span data-ttu-id="14549-126">Pour plus d’informations, consultez [Syntaxe de requête et syntaxe de méthode dans
