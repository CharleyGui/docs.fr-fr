# <a name="exists-entity-sql"></a>EXISTS (Entity SQL)
Détermine si une collection est vide.  
  
## <a name="syntax"></a>Syntaxe  
  
```sql  
[NOT] EXISTS ( expression )  
```  
  
## <a name="arguments"></a>Arguments  
 `expression`  
 Toute expression valide qui retourne une collection.  
  
 NOT  
 Indique que la valeur du résultat de l'opérateur EXISTS est inversée.  
  
## <a name="return-value"></a>Valeur de retour  
 `true` si la collection n’est pas vide ; Sinon, `false`.  
  
## <a name="remarks"></a>Remarques  
 EXISTS est l'un des opérateurs de jeu [!INCLUDE[esql](../../../../../../includes/esql-md.md)]. Tous les opérateurs de jeu [!INCLUDE[esql](../../../../../../includes/esql-md.md)] sont évalués de gauche à droite. Pour obtenir des informations de précédence pour les opérateurs de jeu de [!INCLUDE[esql](../../../../../../includes/esql-md.md)], consultez [except](except-entity-sql.md).
