Sinon, une exception est levée.</span><span class="sxs-lookup"><span data-stu-id="74e27-109">In such as case, a reference to one of the columns by name must match the case of the column name exactly; otherwise an exception is thrown.</span></span> <span data-ttu-id="74e27-110">Par exemple, si la table **MyTable** contient les colonnes **col1** et **col1**, vous devez référencer **col1** par nom comme **MyTable.
