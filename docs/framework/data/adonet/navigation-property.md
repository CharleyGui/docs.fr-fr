# <a name="navigation-property"></a><span data-ttu-id="c0326-102">Propriété de navigation</span><span class="sxs-lookup"><span data-stu-id="c0326-102">Navigation property</span></span>

<span data-ttu-id="c0326-103">Une *propriété de navigation* est une propriété facultative sur [un type d’entité](entity-type.md) qui permet de naviguer d’une [terminaison](association-end.md) d’une [Association](association-type.md) à l’autre terminaison.</span><span class="sxs-lookup"><span data-stu-id="c0326-103">A *navigation property* is an optional property on an [entity type](entity-type.md) that allows for navigation from one [end](association-end.md) of an [association](association-type.md) to the other end.</span></span> <span data-ttu-id="c0326-104">Contrairement aux autres [Propriétés](property.md), les propriétés de navigation ne transmettent pas de données.</span><span class="sxs-lookup"><span data-stu-id="c0326-104">Unlike other [properties](property.md), navigation properties do not carry data.</span></span>

<span data-ttu-id="c0326-105">Une définition de propriété de navigation comprend les éléments suivants :</span><span class="sxs-lookup"><span data-stu-id="c0326-105">A navigation property definition includes the following:</span></span>

- <span data-ttu-id="c0326-106">Nom.</span><span class="sxs-lookup"><span data-stu-id="c0326-106">A name.</span></span> <span data-ttu-id="c0326-107">(Requis)</span><span class="sxs-lookup"><span data-stu-id="c0326-107">(Required)</span></span>

- <span data-ttu-id="c0326-108">Association faisant l'objet de la navigation.</span><span class="sxs-lookup"><span data-stu-id="c0326-108">The association that it navigates.</span></span> <span data-ttu-id="c0326-109">(Requis)</span><span class="sxs-lookup"><span data-stu-id="c0326-109">(Required)</span></span>

- <span data-ttu-id="c0326-110">Terminaisons de l'association faisant l'objet de la navigation.</span><span class="sxs-lookup"><span data-stu-id="c0326-110">The ends of the association that it navigates.</span></span> <span data-ttu-id="c0326-111">(Requis)</span><span class="sxs-lookup"><span data-stu-id="c0326-111">(Required)</span></span>

<span data-ttu-id="c0326-112">Notez que les propriétés de navigation sont facultatives sur les deux types d'entités au niveau des terminaisons d'une association.</span><span class="sxs-lookup"><span data-stu-id="c0326-112">Note that navigation properties are optional on both entity types at the ends of an association.</span></span> <span data-ttu-id="c0326-113">Si vous définissez une propriété de navigation sur un type d'entité au niveau de la terminaison d'une association, il n'est pas nécessaire de définir une propriété de navigation sur le type d'entité à l'autre terminaison de l'association.</span><span class="sxs-lookup"><span data-stu-id="c0326-113">If you define a navigation property on one entity type at the end of an association, you do not have to define a navigation property on the entity type at the other end of the association.</span></span>

<span data-ttu-id="c0326-114">Le type de données d’une propriété de navigation est déterminé par la [multiplicité](association-end-multiplicity.md) de sa [terminaison d’association](association-end.md)distante.</span><span class="sxs-lookup"><span data-stu-id="c0326-114">The data type of a navigation property is determined by the [multiplicity](association-end-multiplicity.md) of its remote [association end](association-end.md).</span></span> <span data-ttu-id="c0326-115">Par exemple, considérez une propriété de navigation, `OrdersNavProp`, qui existe sur un type d'entité `Customer` et qui navigue dans une association un-à-plusieurs entre `Customer` et `Order`.</span><span class="sxs-lookup"><span data-stu-id="c0326-115">For example, suppose a navigation property, `OrdersNavProp`, exists on a `Customer` entity type and navigates a one-to-many association between `Customer` and `Order`.</span></span> <span data-ttu-id="c0326-116">Étant donné que la terminaison d’association distante pour la propriété de navigation a une multiplicité de plusieurs (\*), son type de données est une collection (de `Order`).</span><span class="sxs-lookup"><span data-stu-id="c0326-116">Because the remote association end for the navigation property has multiplicity of many (\*), its data type is a collection (of `Order`).</span></span> <span data-ttu-id="c0326-117">De même, si une propriété de navigation, `CustomerNavProp`, existe sur le type d'entité `Order`, son type de données est `Customer`, car la multiplicité de la terminaison distante est un (1).</span><span class="sxs-lookup"><span data-stu-id="c0326-117">Similarly, if a navigation property, `CustomerNavProp`, exists on the `Order` entity type, its data type would be `Customer`, because the multiplicity of the remote end is one (1).</span></span>
