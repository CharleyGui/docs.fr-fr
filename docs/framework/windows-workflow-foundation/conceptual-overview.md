# <a name="windows-workflow-conceptual-overview"></a>Vue d'ensemble conceptuelle de Windows Workflow

Cette section contient un ensemble de rubriques traitant des concepts plus importants sous-jacents à Windows Workflow Foundation (WF).  
  
## <a name="in-this-section"></a>Dans cette section  

 [Vue d'ensemble de Windows Workflow](overview.md)  
 Décrit la base qui permet aux utilisateurs de créer des flux de travail système ou humains dans leurs applications écrites pour les systèmes d’exploitation Windows Vista, Windows XP, Windows Server 2003 et Windows Server 2008.  
  
 [Concepts Windows Workflow fondamentaux](fundamental-concepts.md)  
 Décrit les concepts fondamentaux utilisés dans Windows Workflow Foundation développement qui peuvent être nouveaux pour certains développeurs.  
  
 [Architecture Windows Workflow](architecture.md)  
 Décrit les composants utilisés dans le développement de Windows Workflow Foundation.
