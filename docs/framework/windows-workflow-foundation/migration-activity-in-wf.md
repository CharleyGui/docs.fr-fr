# <a name="migration-activity-in-wf"></a>Activité de migration dans le WF

[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] fournit l' <xref:System.Activities.Statements.Interop> activité pour exécuter les activités qui dérivent d’une activité dans un workflow basé sur <xref:System.Activities.Activity> . Pour plus d’informations, consultez la section conseils sur la [migration](migration-guidance.md) .  
  
> [!NOTE]
> L' <xref:System.Activities.Statements.Interop> activité n’apparaît pas dans la boîte à outils du concepteur de flux de travail, sauf si le paramètre **Framework cible** du projet du flux de travail est défini sur **.NET Framework 4** ou version ultérieure.
