# <a name="guide-to-the-windows-workflow-documentation"></a>Guide de la documentation Windows Workflow
Cette rubrique contient des informations sur l'utilisation de la documentation de Windows Workflow Foundation. Les documents liés sont des points de départ recommandés regroupés en fonction d'intérêts et de niveaux d'expertise spécifiques.  
  
## <a name="new-to-windows-workflow-foundation-programming"></a>Nouveautés concernant la programmation Windows Workflow Foundation  
  
- Si vous débutez en programmation avec Windows Workflow Foundation et que vous souhaitez simplement voir des exemples d’applications qui fonctionnent, consultez les rubriques répertoriées sous [exemples de flux](./samples/index.md)de travail Windows.  
  
- Pour plus d’informations sur les différents modèles de contrôle de flux de Windows Workflow Foundation, consultez [conception de flux de travail](designing-workflows.md).  
  
- Pour obtenir un didacticiel qui vous guide dans les étapes de base de la création d’une application Windows Workflow Foundation, consultez [prise en main didacticiel](getting-started-tutorial.md).  
  
- Si vous êtes intéressé par les concepts qui sous-tendent Windows Workflow Foundation, consultez les rubriques de la section [vue d’ensemble conceptuelle de Windows Workflow](conceptual-overview.md) .  
  
- Pour obtenir la liste des termes utilisés dans la documentation Windows Workflow Foundation, consultez [Windows Workflow Foundation Glossaire pour .NET Framework 4,5](glossary.md).  
  
- Pour obtenir la liste des nouveaux concepts et fonctionnalités, consultez [Nouveautés de Windows Workflow Foundation](whats-new.md).
