# <a name="windows-workflow-wf-samples"></a>Exemples Windows Workflow (WF)

Vous pouvez [Télécharger des exemples Windows Workflow](https://www.microsoft.com/download/details.aspx?id=21459) qui fournissent des instructions sur les différents aspects de Windows Workflow Foundation (WF).

Les Articles de cette section décrivent certains des exemples du package de téléchargement. Pour obtenir un ensemble complet de documentation qui couvre tous les exemples, consultez la section [exemples de WF .NET Framework 4](/previous-versions/dotnet/netframework-4.0/dd483375(v=vs.100)) .

> [!NOTE]
> Les exemples téléchargeables ont été créés avec Visual Studio 2010 et .NET Framework 4, mais ils sont compatibles avec les versions ultérieures de Visual Studio et le .NET Framework. Vous trouverez des exemples supplémentaires pour Windows Workflow Foundation dans dans les [!INCLUDE[netfx_current_short](../../../../includes/netfx-current-short-md.md)] [exemples de code MSDN](/samples/browse/?redirectedfrom=MSDN-samples).
