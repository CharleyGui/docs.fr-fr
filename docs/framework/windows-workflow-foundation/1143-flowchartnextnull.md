# <a name="1143---flowchartnextnull"></a>1143 - FlowchartNextNull

## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|id|1143|  
|Mots clés|WFActivities|  
|Level|Informations|  
|Channel|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  

 Indique que le nœud suivant dans un Flowchart est Null. L'exécution de Flowchart va se terminer.  
  
## <a name="message"></a>Message  

 Flowchart « %1 »/FlowStep - Le nœud suivant est Null. L'exécution de Flowchart va se terminer.

|AppDomain|xs:string|Chaîne retournée par AppDomain.CurrentDomain.FriendlyName.|
