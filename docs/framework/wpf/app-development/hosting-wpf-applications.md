# <a name="hosting-wpf-applications"></a>Hébergement d'applications WPF
Les applications de navigateur XAML WPF (XBAP) sont des applications clientes riches qui peuvent être déployées sur un serveur Web et démarrées dans un navigateur. L’hôte WPF (PresentationHost. exe) est inscrit en tant qu’interpréteur de commandes et le gestionnaire MIME pour les fichiers XBAP et XAML. Par conséquent, Internet Explorer sait démarrer l’hôte WPF lorsqu’une application XBAP est lancée. Les utilisateurs de Firefox peuvent installer des modules complémentaires Firefox qui permettent à Firefox d’héberger également des applications XBAP. Une application XBAP peut être hébergée dans d’autres navigateurs ou applications autonomes à l’aide des API d’hébergement de navigateur natives fournies par WPF.
