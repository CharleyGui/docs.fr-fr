# <a name="styles-and-templates"></a>Styles et modèles
Les styles et les modèles de Windows Presentation Foundation (WPF) font référence à une suite de fonctionnalités (styles, modèles, déclencheurs et storyboards) qui permettent à une application, un document ou un concepteur d’interface utilisateur de créer des applications visuellement attrayantes et à Standardisez un aspect particulier du produit.
