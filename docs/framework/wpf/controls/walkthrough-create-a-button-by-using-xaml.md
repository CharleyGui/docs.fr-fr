5. <span data-ttu-id="122a0-209">**Ajouter une animation lorsque le bouton est cliqué :** L’étape finale consiste à ajouter un déclencheur pour lorsque l’utilisateur clique sur le bouton.</span><span class="sxs-lookup"><span data-stu-id="122a0-209">**Add an animation for when the button is clicked:** The final step is to add a trigger for when the user clicks the button.</span></span> <span data-ttu-id="122a0-210">Ajoutez le balisage suivant n’importe où à l’intérieur de la `ControlTemplate.Triggers` bloc :</span><span class="sxs-lookup"><span data-stu-id="122a0-210">Add the following markup anywhere inside of the `ControlTemplate.Triggers` block:</span></span>

    ```xaml
      </EventTrigger.Actions>

    ```
