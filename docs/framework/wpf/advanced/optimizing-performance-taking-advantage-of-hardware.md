### <a name="graphics-rendering-tiers"></a><span data-ttu-id="60882-118">Couches de rendu graphiques</span><span class="sxs-lookup"><span data-stu-id="60882-118">Graphics Rendering Tiers</span></span>

 <span data-ttu-id="60882-121">Pour ce faire, [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] fournit des fonctionnalités permettant de déterminer la capacité graphique d’un système au moment de l’exécution.</span><span class="sxs-lookup"><span data-stu-id="60882-121">To achieve this, [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] provides functionality to determine the graphics capability of a system at runtime.</span></span> <span data-ttu-id="60882-122">La fonctionnalité graphique est déterminée en classant la carte vidéo comme l’un des trois niveaux de capacité de rendu.</span><span class="sxs-lookup"><span data-stu-id="60882-122">Graphics capability is determined by categorizing the video card as one of three rendering capability tiers.</span></span> [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]<span data-ttu-id="60882-123">expose une API qui permet à une application d’interroger le niveau de fonctionnalité de rendu.</span><span class="sxs-lookup"><span data-stu-id="60882-123">exposes an API that allows an application to query the rendering capability tier.</span></span> <span data-ttu-id="60882-124">Votre application peut ensuite prendre des chemins de code différents au moment de l’exécution en fonction de la couche de rendu prise en charge par le matériel.</span><span class="sxs-lookup"><span data-stu-id="60882-124">Your application can then take different code paths at run time depending on the rendering tier supported by the hardware.</span></span>  
  
 <span data-ttu-id="60882-125">Les fonctionnalités du matériel graphique qui ont le plus d’impact sur les niveaux de la couche de rendu sont les suivantes :</span><span class="sxs-lookup"><span data-stu-id="60882-125">The features of the graphics hardware that most impact the rendering tier levels are:</span></span>  
  
- <span data-ttu-id="60882-126">**RAM vidéo** La quantité de mémoire vidéo sur le matériel graphique détermine la taille et le nombre de mémoires tampon qui peuvent être utilisées pour la composition de graphiques.</span><span class="sxs-lookup"><span data-stu-id="60882-126">**Video RAM** The amount of video memory on the graphics hardware determines the size and number of buffers that can be used for compositing graphics.</span></span>

- <span data-ttu-id="60882-135">**Couche de rendu 0** Aucune accélération matérielle graphique.</span><span class="sxs-lookup"><span data-stu-id="60882-135">**Rendering Tier 0** No graphics hardware acceleration.</span></span> <span data-ttu-id="60882-136">Le niveau de version de DirectX est inférieur à la version 7,0.</span><span class="sxs-lookup"><span data-stu-id="60882-136">The DirectX version level is less than version 7.0.</span></span>  
  
- <span data-ttu-id="60882-137">**Niveau de rendu 1** Accélération matérielle graphique partielle.</span><span class="sxs-lookup"><span data-stu-id="60882-137">**Rendering Tier 1** Partial graphics hardware acceleration.</span></span> <span data-ttu-id="60882-138">Le niveau de version de DirectX est supérieur ou égal à la version 7,0 **, et inférieur** à la version 9,0.</span><span class="sxs-lookup"><span data-stu-id="60882-138">The DirectX version level is greater than or equal to version 7.0, and **lesser** than version 9.0.</span></span>
