# <a name="printing-how-to-topics"></a>Rubriques "Comment" relatives à l'impression
Les rubriques de cette section montrent comment utiliser les fonctionnalités d’impression et de gestion du système d’impression incluses avec [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] ainsi que le nouveau chemin d’impression XPS (XML Paper Specification).  
  
## <a name="in-this-section"></a>Dans cette section  
 [Appeler une boîte de dialogue Imprimer](how-to-invoke-a-print-dialog.md)  
 Instructions pour le balisage XAML permettant de déclarer un objet de boîte de dialogue d’impression Microsoft Windows et d’utiliser du code pour appeler la boîte de dialogue à partir d’une application Windows Presentation Foundation (WPF).  
  
 [Cloner une imprimante](how-to-clone-a-printer.md)  
 Instructions relatives à l’installation d’une deuxième file d’attente à l’impression avec exactement les mêmes propriétés qu’une file d’attente à l’impression existante.  
  
 [Diagnostiquer un travail d'impression problématique](how-to-diagnose-problematic-print-job.md)  
 Instructions pour l’utilisation des propriétés des files d’attente à l’impression et des travaux d’impression pour diagnostiquer un travail d’impression qui n’est pas en cours d’impression.  
  
 [Déterminer si un travail d'impression peut être imprimé à cette heure de la journée](how-to-discover-whether-a-print-job-can-be-printed-at-this-time-of-day.md)  
 Instructions relatives à l’utilisation des propriétés des files d’attente à l’impression et des travaux d’impression pour déterminer par programme les heures de la journée auxquelles le travail peut être imprimé.  
  
 [Énumérer un sous-ensemble de files d'attente à l'impression](how-to-enumerate-a-subset-of-print-queues.md)  
 Instructions pour la génération d’une liste d’imprimantes présentant certaines caractéristiques.  
  
 [Obtenir les propriétés de l'objet de système d'impression sans réflexion](how-to-get-print-system-object-properties-without-reflection.md)  
 Instructions sur la façon de découvrir les propriétés et les types de l’objet du système d’impression du Runtime.  
  
 [Imprimer des fichiers XPS par programmation](how-to-programmatically-print-xps-files.md)  
 Instructions pour l’impression rapide de fichiers XPS (XML Paper Specification) sans nécessiter de [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)].  
  
 [Observer à distance l'état d'imprimantes](how-to-remotely-survey-the-status-of-printers.md)  
 Instructions relatives à la création d’un utilitaire qui enquêtera sur les imprimantes afin de découvrir les problèmes liés à un bourrage papier ou à un autre problème.  
  
 [Valider et fusionner des PrintTicket](how-to-validate-and-merge-printtickets.md)  
 Instructions pour vérifier qu’un ticket d’impression est valide et qu’il ne demande rien qui n’est pas pris en charge par l’imprimante.
