# <a name="input-wpf"></a>Entrée (WPF)
Windows Presentation Foundation (WPF) prend en charge plusieurs types d’entrée. Cette entrée comprend le texte, les fonctions tactiles, la souris, les commandes, le focus, le toucher, le glisser-déplacer et l’encre numérique. Cette section décrit les rubriques relatives à l’entrée dans WPF.
