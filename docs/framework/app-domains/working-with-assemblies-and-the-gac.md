# <a name="working-with-assemblies-and-the-global-assembly-cache"></a><span data-ttu-id="d3966-102">Utilisation d'assemblys et du Global Assembly Cache</span><span class="sxs-lookup"><span data-stu-id="d3966-102">Working with Assemblies and the Global Assembly Cache</span></span>
<span data-ttu-id="d3966-103">Si vous prévoyez de partager un assembly entre plusieurs applications, vous pouvez l’installer dans le Global Assembly Cache.</span><span class="sxs-lookup"><span data-stu-id="d3966-103">If you intend to share an assembly among several applications, you can install it into the global assembly cache.</span></span> <span data-ttu-id="d3966-104">Chaque ordinateur où le common language runtime est installé a ce cache de code à l’échelle de l’ordinateur.</span><span class="sxs-lookup"><span data-stu-id="d3966-104">Each computer where the common language runtime is installed has this machine-wide code cache.</span></span> <span data-ttu-id="d3966-105">Le Global Assembly Cache stocke des assemblys spécialement destinés à être partagés entre plusieurs applications sur l’ordinateur.</span><span class="sxs-lookup"><span data-stu-id="d3966-105">The global assembly cache stores assemblies specifically designated to be shared by several applications on the computer.</span></span> <span data-ttu-id="d3966-106">Un assembly doit avoir un nom fort pour être installé dans le Global Assembly Cache.</span><span class="sxs-lookup"><span data-stu-id="d3966-106">An assembly must have a strong name to be installed in the global assembly cache.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="d3966-107">Les assemblys placés dans le Global Assembly Cache doivent avoir le même nom d’assembly et le même nom de fichier (sans l’extension du nom de fichier).</span><span class="sxs-lookup"><span data-stu-id="d3966-107">Assemblies placed in the global assembly cache must have the same assembly name and file name (not including the file name extension).</span></span> <span data-ttu-id="d3966-108">Par exemple, un assembly avec le nom d’assembly monAssembly doit avoir le nom de fichier monAssembly.exe ou monAssembly.dll.</span><span class="sxs-lookup"><span data-stu-id="d3966-108">For example, an assembly with the assembly name of myAssembly must have a file name of either myAssembly.exe or myAssembly.dll.</span></span>  
  
 <span data-ttu-id="d3966-109">Il est recommandé de partager des assemblys en les installant dans le Global Assembly Cache seulement quand c’est nécessaire.</span><span class="sxs-lookup"><span data-stu-id="d3966-109">You should share assemblies by installing them into the global assembly cache only when necessary.</span></span> <span data-ttu-id="d3966-110">Une règle générale est de conserver les dépendances d’assembly privées et de placer les assemblys dans le répertoire de l’application, sauf si le partage d’un assembly est explicitement nécessaire.</span><span class="sxs-lookup"><span data-stu-id="d3966-110">As a general guideline, keep assembly dependencies private and locate assemblies in the application directory unless sharing an assembly is explicitly required.</span></span> <span data-ttu-id="d3966-111">En outre, il n’est pas nécessaire d’installer des assemblys dans le Global Assembly Cache pour les rendre accessibles à COM Interop ou à du code non managé.</span><span class="sxs-lookup"><span data-stu-id="d3966-111">In addition, you do not have to install assemblies into the global assembly cache to make them accessible to COM interop or unmanaged code.</span></span>  
  
 <span data-ttu-id="d3966-112">Vous pouvez souhaiter installer un assembly dans le Global Assembly Cache pour plusieurs raisons :</span><span class="sxs-lookup"><span data-stu-id="d3966-112">There are several reasons why you might want to install an assembly into the global assembly cache:</span></span>  
  
- <span data-ttu-id="d3966-113">Emplacement partagé.</span><span class="sxs-lookup"><span data-stu-id="d3966-113">Shared location.</span></span>  
  
     <span data-ttu-id="d3966-114">Les assemblys qui doivent être utilisés par des applications peuvent être placés dans le Global Assembly Cache.</span><span class="sxs-lookup"><span data-stu-id="d3966-114">Assemblies that should be used by applications can be put in the global assembly cache.</span></span> <span data-ttu-id="d3966-115">Par exemple, si toutes les applications doivent utiliser un assembly qui se trouve dans le Global Assembly Cache, vous pouvez ajouter une instruction de stratégie de version au fichier Machine.config pour rediriger les références à l’assembly.</span><span class="sxs-lookup"><span data-stu-id="d3966-115">For example, if all applications should use an assembly located in the global assembly cache, a version policy statement can be added to the Machine.config file that redirects references to the assembly.</span></span>  
  
- <span data-ttu-id="d3966-116">Sécurité des fichiers.</span><span class="sxs-lookup"><span data-stu-id="d3966-116">File security.</span></span>  
  
     <span data-ttu-id="d3966-117">Les administrateurs protègent souvent le répertoire systemroot en utilisant une liste de contrôle d’accès pour contrôler l’accès en écriture et en exécution.</span><span class="sxs-lookup"><span data-stu-id="d3966-117">Administrators often protect the systemroot directory using an Access Control List (ACL) to control write and execute access.</span></span> <span data-ttu-id="d3966-118">Le Global Assembly Cache étant installé dans le répertoire systemroot, il hérite de la liste de contrôle d’accès de ce répertoire.</span><span class="sxs-lookup"><span data-stu-id="d3966-118">Because the global assembly cache is installed in the systemroot directory, it inherits that directory's ACL.</span></span> <span data-ttu-id="d3966-119">Il est recommandé que seuls les utilisateurs disposant de privilèges d’administrateur soient autorisés à supprimer des fichiers du Global Assembly Cache.</span><span class="sxs-lookup"><span data-stu-id="d3966-119">It is recommended that only users with Administrator privileges be allowed to delete files from the global assembly cache.</span></span>  
  
- <span data-ttu-id="d3966-120">Contrôle de version côte à côte.</span><span class="sxs-lookup"><span data-stu-id="d3966-120">Side-by-side versioning.</span></span>  
  
     <span data-ttu-id="d3966-121">Plusieurs copies des assemblys avec le même nom mais avec des informations de version différentes peuvent être conservées dans le Global Assembly Cache.</span><span class="sxs-lookup"><span data-stu-id="d3966-121">Multiple copies of assemblies with the same name but different version information can be maintained in the global assembly cache.</span></span>  
  
- <span data-ttu-id="d3966-122">Emplacements de recherche supplémentaires.</span><span class="sxs-lookup"><span data-stu-id="d3966-122">Additional search location.</span></span>  
  
     <span data-ttu-id="d3966-123">Le common language runtime recherche dans le Global Assembly Cache un assembly qui correspond à l’assembly demandé avant de chercher ailleurs ou d’utiliser les informations du code base dans un fichier de configuration.</span><span class="sxs-lookup"><span data-stu-id="d3966-123">The common language runtime checks the global assembly cache for an assembly that matches the assembly request before probing or using the codebase information in a configuration file.</span></span>  
  
 <span data-ttu-id="d3966-124">Notez que dans certains scénarios, vous ne voulez explicitement pas installer un assembly dans le Global Assembly Cache.</span><span class="sxs-lookup"><span data-stu-id="d3966-124">Note that there are scenarios where you explicitly do not want to install an assembly into the global assembly cache.</span></span> <span data-ttu-id="d3966-125">Si vous placez un des assemblys composant une application dans le Global Assembly Cache, vous ne pouvez plus répliquer ni installer l’application en utilisant XCOPY pour copier le répertoire de l’application.</span><span class="sxs-lookup"><span data-stu-id="d3966-125">If you place one of the assemblies that make up an application into the global assembly cache, you can no longer replicate or install the application by using XCOPY to copy the application directory.</span></span> <span data-ttu-id="d3966-126">Dans ce cas, vous devez également déplacer l’assembly dans le Global Assembly Cache.</span><span class="sxs-lookup"><span data-stu-id="d3966-126">In this case, you must also move the assembly into the global assembly cache.</span></span>
