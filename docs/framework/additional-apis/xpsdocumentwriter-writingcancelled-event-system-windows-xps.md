# <a name="xpsdocumentwriter_writingcancelled-event"></a>XpsDocumentWriter. Événement \_WritingCancelled

Se produit lorsqu’une opération de <xref:System.Windows.Xps.XpsDocumentWriter.Write%2A> ou de <xref:System.Windows.Xps.XpsDocumentWriter.WriteAsync%2A> est annulée. 

## <a name="syntax"></a>Syntaxe

``` csharp
internal event WritingCancelledEventHandler _WritingCancelled
```
