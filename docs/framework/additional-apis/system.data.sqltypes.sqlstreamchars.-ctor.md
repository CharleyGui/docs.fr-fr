# <a name="sqlstreamchars-constructor"></a>SqlStreamChars, constructeur

Initialise une nouvelle instance de la classe `SqlStreamChars` . L’assembly qui contient ce constructeur a une relation Friend avec SQLAccess. dll. Elle est destinée à être utilisée par SQL Server. Pour les autres bases de données, utilisez le mécanisme d’hébergement fourni par cette base de données.

```csharp
protected SqlStreamChars ();
```

## <a name="remarks"></a>Notes

> [!WARNING]
> Le constructeur `SqlStreamChars` est protégé et n’est pas destiné à être utilisé directement dans votre code.
>
> Microsoft ne prend pas en charge l’utilisation de ce constructeur dans une application de production, quel que soit le cas.
