# <a name="tlsstreamm_worker-field"></a>Champ TlsStream. m_Worker

Représente l’état du flux SSL.

## <a name="syntax"></a>Syntaxe

```csharp
private SslState m_Worker;
```

## <a name="field-value"></a>Valeur de champ

`System.Net.Security.SslState`  
État du flux SSL.

## <a name="remarks"></a>Notes

> [!WARNING]
> Le champ `TlsStream.m_Worker` est privé et n’est pas destiné à être utilisé directement dans votre code.
>
> Microsoft ne prend pas en charge l’utilisation de ce champ dans une application de production en l’absence de toute circonstance.
