# <a name="exceptionhelper-class"></a>ExceptionHelper, classe

Fournit des exceptions avec des messages d’erreur standardisés. Cette classe ne peut pas être héritée.

```csharp
internal static class ExceptionHelper
```

> [!WARNING]
> Cette classe est interne et n’est pas destinée à être utilisée directement dans votre code.
>
> Microsoft ne prend pas en charge l’utilisation de cette classe dans une application de production en l’absence de toute circonstance.

## <a name="webpermissionunrestricted-field"></a>Champ WebPermissionUnrestricted

Spécifie que l’application peut se connecter aux ressources Internet.

```csharp
internal static readonly WebPermission WebPermissionUnrestricted
```
