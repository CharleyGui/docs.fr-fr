# <a name="makeparentitemscontrolgotfocus-method"></a>Méthode MakeParentItemsControlGotFocus

Donne le focus au contrôle parent de l’élément. Si le contrôle parent est un <xref:System.Windows.Controls.ListBox> , donne le focus à l’élément le plus récemment accédé dans le <xref:System.Windows.Controls.ListBox> .

```csharp
internal void MakeParentItemsControlGotFocus()
```

> [!WARNING]
> Cette méthode est interne et n’est pas destinée à être utilisée directement dans votre code.
>
> Microsoft ne prend pas en charge l’utilisation de cette méthode dans une application de production en l’absence de toute circonstance.
