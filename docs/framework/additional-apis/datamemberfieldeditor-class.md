# <a name="datamemberfieldeditor-class"></a>Classe DataMemberFieldEditor

Fournit l’interface utilisateur pour les propriétés d’édition des objets liés aux données en énumérant toutes les propriétés de l’objet `DataSource` pour sélectionner la source de valeur à partir.  
  
## <a name="syntax"></a>Syntaxe
  
```csharp
internal class DataMemberFieldEditor : UITypeEditor
```

> [!WARNING]
> La `DataMemberFieldEditor` classe est interne et n’est pas destinée à être utilisée directement dans votre code.
>
> Microsoft ne prend en charge l’utilisation de cette classe dans une application de production en aucune circonstance.
