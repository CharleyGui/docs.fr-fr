# <a name="datamemberfieldeditor-class"></a>DataMemberFieldEditor, classe

Fournit une interface utilisateur pour modifier les propriétés des objets liés aux données en répertoriant toutes les propriétés de l’objet `DataSource` pour sélectionner la source de la valeur à partir de.  
  
## <a name="syntax"></a>Syntaxe
  
```csharp
internal class DataMemberFieldEditor : UITypeEditor
```

> [!WARNING]
> La classe `DataMemberFieldEditor` est interne et n’est pas destinée à être utilisée directement dans votre code.
> 
> Microsoft ne prend pas en charge l’utilisation de cette classe dans une application de production en l’absence de toute circonstance.
