# <a name="textbox-control-windows-forms"></a>TextBox, contrôle (Windows Forms)
Les zones de texte Windows Forms sont utilisées pour obtenir l’entrée de l’utilisateur ou pour afficher du texte. Le contrôle `TextBox` est généralement utilisé pour le texte modifiable, bien qu’il puisse également être mis en lecture seule. Les zones de texte peuvent afficher plusieurs lignes, ajuster le texte à la taille du contrôle et ajouter une mise en forme de base. Le contrôle `TextBox` permet d’afficher ou d’entrer un seul format de texte dans le contrôle.
