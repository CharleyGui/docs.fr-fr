# <a name="panel-control-windows-forms"></a>Panel, contrôle (Windows Forms)
Les contrôles de `Panel` Windows Forms sont utilisés pour fournir un regroupement identifiable pour d’autres contrôles. En règle générale, vous utilisez des panneaux pour subdiviser un formulaire par fonction. Le contrôle `Panel` est semblable au contrôle <xref:System.Windows.Forms.GroupBox> ; Toutefois, seul le contrôle `Panel` peut avoir des barres de défilement, et seul le contrôle <xref:System.Windows.Forms.GroupBox> affiche une légende.
