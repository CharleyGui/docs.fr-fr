# <a name="panel-control-windows-forms"></a>Panel, contrôle (Windows Forms)
Les `Panel` contrôles Windows Forms sont utilisés pour fournir un regroupement identifiable pour d’autres contrôles. En règle générale, vous utilisez des panneaux pour subdiviser un formulaire par fonction. Le `Panel` contrôle est semblable au <xref:System.Windows.Forms.GroupBox> contrôle ; Toutefois, seul le `Panel` contrôle peut avoir des barres de défilement, et seul le <xref:System.Windows.Forms.GroupBox> contrôle affiche une légende.
