## <a name="to-bind-the-control-to-a-data-source"></a><span data-ttu-id="41d21-114">Pour lier le contrôle à une source de données</span><span class="sxs-lookup"><span data-stu-id="41d21-114">To bind the control to a data source</span></span>

1. <span data-ttu-id="41d21-115">Cliquez sur le glyphe de balise active (![glyphe de balise active](./media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph")) dans le <xref:System.Windows.Forms.DataGridView> coin supérieur droit du contrôle.</span><span class="sxs-lookup"><span data-stu-id="41d21-115">Click the smart tag glyph (![Smart Tag Glyph](./media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph")) on the upper-right corner of the <xref:System.Windows.Forms.DataGridView> control.</span></span>

2. <span data-ttu-id="41d21-116">Cliquez sur la flèche déroulante correspondant à l’option **Choisir la Source de données**.</span><span class="sxs-lookup"><span data-stu-id="41d21-116">Click the drop-down arrow for the **Choose Data Source** option.</span></span>

3. <span data-ttu-id="41d21-117">Si votre projet ne dispose pas déjà d’une source de données, cliquez **Ajouter la source de données projet** et suivez les étapes indiquées par l’Assistant.</span><span class="sxs-lookup"><span data-stu-id="41d21-117">If your project does not already have a data source, click **Add Project Data Source** and follow the steps indicated by the wizard.</span></span>

     <span data-ttu-id="41d21-118">Pour plus d’informations, consultez la page [Assistant Configuration de source de données](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-2013/w4dd7z6t(v=vs.120)).</span><span class="sxs-lookup"><span data-stu-id="41d21-118">For more information, see [Data Source Configuration Wizard](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-2013/w4dd7z6t(v=vs.120)).</span></span> <span data-ttu-id="41d21-119">Votre nouvelle source de données s’affiche dans la liste déroulante **Choisir la source de données**.</span><span class="sxs-lookup"><span data-stu-id="41d21-119">Your new data source will appear in the **Choose Data Source** drop-down window.</span></span> <span data-ttu-id="41d21-120">Si votre nouvelle source de données contient un seul membre, comme une table de base de données unique, le contrôle est automatiquement lié à ce membre.</span><span class="sxs-lookup"><span data-stu-id="41d21-120">If your new data source contains only one member, such as a single database table, the control will automatically bind to that member.</span></span> <span data-ttu-id="41d21-121">Sinon, passez à l'étape suivante.</span><span class="sxs-lookup"><span data-stu-id="41d21-121">Otherwise, continue to the next step.</span></span>

4. <span data-ttu-id="41d21-122">Développez les nœuds **Autres sources de données** et **Sources de données du projet** si cela n’est pas déjà fait, puis sélectionnez la source de données à laquelle lier le contrôle.</span><span class="sxs-lookup"><span data-stu-id="41d21-122">Expand the **Other Data Sources** and **Project Data Sources** nodes if they are not already expanded, and then select the data source to bind the control to.</span></span>

5. <span data-ttu-id="41d21-123">Si votre source de données contient plusieurs membres, par exemple si vous avez créé un <xref:System.Data.DataSet?displayProperty=nameWithType> qui contient plusieurs tables, développez la source de données, puis sélectionnez le membre spécifique à lier.</span><span class="sxs-lookup"><span data-stu-id="41d21-123">If your data source contains more than one member, such as if you have created a <xref:System.Data.DataSet?displayProperty=nameWithType> that contains multiple tables, expand the data source, and then select the specific member to bind to.</span></span>

6. <span data-ttu-id="41d21-124">Pour créer une relation maître/détail, dans la fenêtre déroulante **choisir une source de données** pour <xref:System.Windows.Forms.DataGridView> un deuxième contrôle, <xref:System.Windows.Forms.BindingSource> développez le créé pour la table parente, puis sélectionnez la table enfant associée dans la liste affichée.</span><span class="sxs-lookup"><span data-stu-id="41d21-124">To create a master/detail relationship, in the **Choose Data Source** drop-down window for a second <xref:System.Windows.Forms.DataGridView> control, expand the <xref:System.Windows.Forms.BindingSource> created for the parent table, and then select the related child table from the list shown.</span></span>

    > [!NOTE]
    > <span data-ttu-id="41d21-125">Si votre projet a déjà une source de données, vous pouvez également utiliser la fenêtre **Sources de données** pour créer un formulaire de données.</span><span class="sxs-lookup"><span data-stu-id="41d21-125">If your project already has a data source, you can also use the **Data Sources** window to create a data form.</span></span> <span data-ttu-id="41d21-126">Pour plus d’informations, consultez la page [Fenêtre Sources de données](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-2013/6ckyxa83(v=vs.120)).</span><span class="sxs-lookup"><span data-stu-id="41d21-126">For more information, see [Data Sources Window](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-2013/6ckyxa83(v=vs.120)).</span></span>
