# <a name="how-to-align-multiple-controls-on-windows-forms"></a>Procédure : aligner plusieurs contrôles dans des Windows Forms

Pour normaliser la disposition de l’interface utilisateur (IU) de votre application Windows, vous pouvez positionner des groupes de contrôles avec une seule commande.

## <a name="to-align-multiple-controls-on-a-form"></a>Pour aligner plusieurs contrôles sur un formulaire

1. Dans Visual Studio, ouvrez le formulaire qui contient les contrôles que vous souhaitez positionner dans le **Concepteur Windows Forms**.

2. Sélectionnez les contrôles que vous souhaitez aligner. Le premier contrôle que vous sélectionnez est le contrôle principal sur lequel les autres contrôles seront alignés.

3. Dans le menu **format** , sélectionnez **Aligner**, puis sélectionnez l’une des sept options disponibles.
