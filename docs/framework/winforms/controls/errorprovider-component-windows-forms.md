# <a name="errorprovider-component-windows-forms"></a>ErrorProvider, composant (Windows Forms)
Le composant `ErrorProvider` Windows Forms est utilisé pour montrer à l’utilisateur de manière non intrusive un problème. Elle est généralement utilisée conjointement avec la validation de l’entrée d’utilisateur sur un formulaire ou l’affichage d’erreurs dans un jeu de données.
