# <a name="treeview-control-windows-forms"></a>TreeView, contrôle (Windows Forms)
Le contrôle Windows Forms `TreeView` affiche une hiérarchie de nœuds, un peu comme les fichiers et dossiers sont affichés dans le volet gauche de la fonctionnalité Explorateur Windows du système d’exploitation Windows.
