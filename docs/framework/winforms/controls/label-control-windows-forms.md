# <a name="label-control-windows-forms"></a>Label, contrôle (Windows Forms)
> [!IMPORTANT]
> Le <xref:System.Windows.Forms.ToolStripLabel> contrôle remplace et ajoute des fonctionnalités <xref:System.Windows.Forms.Label> au contrôle. Vous pouvez utiliser <xref:System.Windows.Forms.ToolStripLabel> avec d’autres nouveaux contrôles tels <xref:System.Windows.Forms.ToolStripDropDown>que. Toutefois, le <xref:System.Windows.Forms.Label> contrôle est conservé pour la compatibilité descendante et l’utilisation ultérieure, si vous le souhaitez.  
  
 Les <xref:System.Windows.Forms.Label> contrôles Windows Forms sont utilisés pour afficher du texte ou des images qui ne peuvent pas être modifiés par l’utilisateur. Ils sont utilisés pour identifier des objets sur un formulaire — pour fournir une description de ce que fait un certain contrôle en cas de clic, par exemple, ou pour afficher des informations en réponse à un événement ou à un processus d’exécution dans votre application. Étant donné <xref:System.Windows.Forms.Label> que le contrôle ne peut pas recevoir le focus, il peut également être utilisé pour créer des clés d’accès pour d’autres contrôles.
