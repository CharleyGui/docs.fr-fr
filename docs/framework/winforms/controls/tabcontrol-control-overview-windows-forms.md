# <a name="tabcontrol-control-overview-windows-forms"></a>Vue d'ensemble du contrôle TabControl (Windows Forms)
Le <xref:System.Windows.Forms.TabControl> Windows Forms affiche plusieurs onglets comparables aux intercalaires d’un agenda ou aux étiquettes d’un ensemble de dossiers dans un classeur. Les onglets peuvent contenir des images et d'autres contrôles. Vous pouvez utiliser le contrôle onglet pour produire le type de boîte de dialogue à plusieurs pages qui apparaît à de nombreux emplacements dans le système d’exploitation Windows, tels que les propriétés d’affichage du panneau de configuration. En outre, le <xref:System.Windows.Forms.TabControl> peut être utilisé pour créer des pages de propriétés, qui sont utilisées pour définir un groupe de propriétés associées.  
  
## <a name="working-with-tabcontrol"></a>Utilisation de TabControl  
 La propriété la plus importante de l' <xref:System.Windows.Forms.TabControl> est <xref:System.Windows.Forms.TabControl.TabPages%2A>, qui contient les onglets individuels. Chaque onglet individuel est un objet <xref:System.Windows.Forms.TabPage>. Lorsqu’un utilisateur clique sur un onglet, il déclenche l’événement <xref:System.Windows.Forms.Control.Click> pour cet objet <xref:System.Windows.Forms.TabPage>.
