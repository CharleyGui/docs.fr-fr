# <a name="develop-a-composite-windows-forms-control"></a>Développer un contrôle de Windows Forms composite

Vous pouvez développer un contrôle Windows Forms composite en combinant d'autres contrôles Windows Forms. Les contrôles composites qui dérivent de <xref:System.Web.UI.UserControl> sont appelés contrôles utilisateur. La classe de base, <xref:System.Windows.Forms.UserControl>, fournit le routage clavier pour les contrôles enfants, ce qui permet de garantir que les contrôles enfants peuvent recevoir le focus. Pour obtenir un exemple de contrôle utilisateur, consultez l’exemple <xref:System.Windows.Forms.UserControl> dans [Comment : appliquer des attributs dans des contrôles Windows Forms](how-to-apply-attributes-in-windows-forms-controls.md).
