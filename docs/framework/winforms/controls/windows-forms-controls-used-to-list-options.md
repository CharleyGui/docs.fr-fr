# <a name="windows-forms-controls-used-to-list-options"></a>Contrôles Windows Forms utilisés pour l'affichage de listes d'options
Vous pouvez ajouter divers contrôles à un Windows Form si vous souhaitez fournir aux utilisateurs une liste d’options parmi lesquelles choisir. Selon la quantité de données que vous souhaitez limiter à l’entrée de vos utilisateurs, vous pouvez ajouter un contrôle de <xref:System.Windows.Forms.ListBox>, un contrôle de <xref:System.Windows.Forms.ComboBox> ou un contrôle de <xref:System.Windows.Forms.CheckedListBox>. Utilisez les liens suivants pour déterminer le contrôle qui répond le mieux à vos besoins.
