# <a name="linklabel-control-windows-forms"></a>LinkLabel, contrôle (Windows Forms)
Le contrôle `LinkLabel` Windows Forms vous permet d’ajouter des liens de style Web vers les applications Windows Forms. Vous pouvez `LinkLabel` utiliser le contrôle pour <xref:System.Windows.Forms.Label> tout ce que vous pouvez utiliser le contrôle pour; vous pouvez également définir une partie du texte en lien vers un objet ou une page Web.
