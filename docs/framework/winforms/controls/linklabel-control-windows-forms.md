# <a name="linklabel-control-windows-forms"></a>LinkLabel, contrôle (Windows Forms)
Le contrôle Windows Forms `LinkLabel` vous permet d’ajouter des liens de style Web aux applications Windows Forms. Vous pouvez utiliser le contrôle `LinkLabel` pour tout ce que vous pouvez utiliser pour le contrôle <xref:System.Windows.Forms.Label> ; vous pouvez également définir une partie du texte sous la forme d’un lien vers un objet ou une page Web.
