# <a name="how-to-set-grid-options-for-all-windows-forms"></a>Comment : définir des options de grille pour tous les Windows Forms

Lorsque vous êtes familiarisé avec l’environnement de développement Visual Studio, vous pouvez définir des préférences pour tous les formulaires et projets avec lesquels vous travaillez dans la Concepteur Windows Forms.

## <a name="set-global-windows-forms-options"></a>Définir les options de Windows Forms global

1. Dans Visual Studio, dans le menu **Outils** , sélectionnez **options**.

2. Dans le volet gauche de la boîte de dialogue **options** , cliquez sur **Concepteur Windows Forms**.

   Dans le volet droit, sous le titre **paramètres de disposition** , vous pouvez définir les paramètres de grille par défaut pour tous les nouveaux formulaires que vous créez. Ces paramètres incluent la taille de la grille, si les contrôles s’y alignent et si elle est activée par défaut. En outre, vous pouvez choisir entre les modes de disposition **SnapToGrid** et **SnapLines** . Pour plus d’informations sur les lignes d’alignement, consultez [procédure pas à pas : Organisation des contrôles sur Windows Forms à l’aide des lignes d’alignement](walkthrough-arranging-controls-on-windows-forms-using-snaplines.md).
