# <a name="mainmenu-component-windows-forms"></a>MainMenu, composant (Windows Forms)
> [!NOTE]
> Bien que `MenuStrip` et `ContextMenuStrip` remplacent et ajoutent des fonctionnalités aux contrôles `MainMenu` et <xref:System.Windows.Forms.ContextMenu> des versions précédentes, `MainMenu` et les <xref:System.Windows.Forms.ContextMenu> sont conservés pour la compatibilité descendante et l’utilisation future si vous le souhaitez.  
  
 Le composant Windows Forms `MainMenu` affiche un menu au moment de l’exécution.
