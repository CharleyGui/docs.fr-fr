# <a name="combobox-control-windows-forms"></a>ComboBox, contrôle (Windows Forms)
Le contrôle Windows Forms `ComboBox` permet d’afficher des données dans une zone de liste déroulante. Par défaut, le contrôle `ComboBox` apparaît en deux parties : la partie supérieure est une zone de texte qui permet à l’utilisateur de taper un élément de liste. La deuxième partie est une zone de liste qui affiche une liste d’éléments à partir desquels l’utilisateur peut en sélectionner une.
