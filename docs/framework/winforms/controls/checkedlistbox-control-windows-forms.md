# <a name="checkedlistbox-control-windows-forms"></a>CheckedListBox, contrôle (Windows Forms)
Le contrôle Windows Forms `CheckedListBox` affiche une liste d’éléments, comme le contrôle <xref:System.Windows.Forms.ListBox>, et peut également afficher une coche en regard des éléments de la liste.
