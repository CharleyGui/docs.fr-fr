### <a name="to-add-a-table-to-the-datagrid-control-in-the-designer"></a><span data-ttu-id="e8df4-113">Pour ajouter une table au contrôle DataGrid dans le concepteur</span><span class="sxs-lookup"><span data-stu-id="e8df4-113">To add a table to the DataGrid control in the designer</span></span>

2. <span data-ttu-id="e8df4-116">Sélectionnez la propriété <xref:System.Windows.Forms.DataGrid.TableStyles%2A> du contrôle <xref:System.Windows.Forms.DataGrid> dans le Fenêtre Propriétés, puis cliquez sur le bouton de sélection (![le bouton de sélection (...) dans la Fenêtre Propriétés de Visual Studio.](./media/visual-studio-ellipsis-button.png)) en regard de la propriété pour afficher l **'éditeur de collections DataGridTableStyle**.</span><span class="sxs-lookup"><span data-stu-id="e8df4-116">Select the <xref:System.Windows.Forms.DataGrid> control's <xref:System.Windows.Forms.DataGrid.TableStyles%2A> property in the Properties window, and then click the ellipsis button (![The Ellipsis button (...) in the Properties window of Visual Studio.](./media/visual-studio-ellipsis-button.png)) next to the property to display the **DataGridTableStyle Collection Editor**.</span></span>

3. <span data-ttu-id="e8df4-117">Dans l’éditeur de collections, cliquez sur **Ajouter** pour insérer un style de tableau.</span><span class="sxs-lookup"><span data-stu-id="e8df4-117">In the collection editor, click **Add** to insert a table style.</span></span>

4. <span data-ttu-id="e8df4-118">Cliquez sur **OK** pour fermer l’éditeur de collections, puis rouvrez-le en cliquant sur le bouton de sélection en regard de la propriété <xref:System.Windows.Forms.DataGrid.TableStyles%2A>.</span><span class="sxs-lookup"><span data-stu-id="e8df4-118">Click **OK** to close the collection editor, and then reopen it by clicking the ellipsis button next to the <xref:System.Windows.Forms.DataGrid.TableStyles%2A> property.</span></span>

     <span data-ttu-id="e8df4-119">Lorsque vous rouvrez l’éditeur de collections, toutes les tables de données liées au contrôle s’affichent dans la liste déroulante de la propriété <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> du style de table.</span><span class="sxs-lookup"><span data-stu-id="e8df4-119">When you reopen the collection editor, any data tables bound to the control will appear in the drop-down list for the <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> property of the table style.</span></span>

5. <span data-ttu-id="e8df4-120">Dans la zone **membres** de l’éditeur de collections, cliquez sur le style de table.</span><span class="sxs-lookup"><span data-stu-id="e8df4-120">In the **Members** box of the collection editor, click the table style.</span></span>

6. <span data-ttu-id="e8df4-121">Dans la zone **Propriétés** de l’éditeur de collections, sélectionnez la valeur <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> de la table que vous souhaitez afficher.</span><span class="sxs-lookup"><span data-stu-id="e8df4-121">In the **Properties** box of the collection editor, select the <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> value for the table you want to display.</span></span>

5. <span data-ttu-id="e8df4-130">Dans la zone **Propriétés** de l’éditeur de collections, sélectionnez la valeur <xref:System.Windows.Forms.DataGridColumnStyle.MappingName%2A> pour la colonne que vous souhaitez afficher.</span><span class="sxs-lookup"><span data-stu-id="e8df4-130">In the **Properties** box of the collection editor, select the <xref:System.Windows.Forms.DataGridColumnStyle.MappingName%2A> value for the column you want to display.</span></span>
