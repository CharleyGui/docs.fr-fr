# <a name="contextmenustrip-control-overview"></a>Vue d'ensemble du contrôle ContextMenuStrip
> [!NOTE]
> Le <xref:System.Windows.Forms.ContextMenuStrip> contrôle remplace et ajoute des fonctionnalités <xref:System.Windows.Forms.ContextMenu> au contrôle; Toutefois, le <xref:System.Windows.Forms.ContextMenu> contrôle est conservé pour la compatibilité descendante et l’utilisation future si tel est votre choix.  
  
 Les menus contextuels, également appelés menus contextuels, apparaissent à la position de la souris lorsque l’utilisateur clique avec le bouton droit de la souris. Les *menus* contextuels fournissent des options pour la zone cliente ou le contrôle à l’emplacement du pointeur de la souris.  
  
 Le <xref:System.Windows.Forms.ContextMenuStrip> contrôle est conçu pour fonctionner de manière transparente avec les <xref:System.Windows.Forms.ToolStrip> nouveaux contrôles et associés, mais vous pouvez aussi <xref:System.Windows.Forms.ContextMenuStrip> facilement associer un à d’autres contrôles.  
  
 Le tableau suivant présente les classes <xref:System.Windows.Forms.ContextMenuStrip> auxiliaires importantes.  
  
|Classe|Description|  
|-----------|-----------------|  
|<xref:System.Windows.Forms.ToolStripMenuItem>|Représente une option pouvant être sélectionnée, qui est affichée sur un <xref:System.Windows.Forms.MenuStrip> ou un <xref:System.Windows.Forms.ContextMenuStrip>.|  
|<xref:System.Windows.Forms.ToolStripDropDown>|Représente un contrôle qui permet à l’utilisateur de sélectionner un élément unique dans une liste qui s’affiche lorsque l’utilisateur clique sur un <xref:System.Windows.Forms.ToolStripDropDownButton> ou un élément de menu de niveau supérieur.|  
|<xref:System.Windows.Forms.ToolStripDropDownItem>|Fournit des fonctionnalités de base pour les <xref:System.Windows.Forms.ToolStripItem> contrôles dérivés de qui affichent des éléments déroulants lorsque l’utilisateur clique dessus.|
