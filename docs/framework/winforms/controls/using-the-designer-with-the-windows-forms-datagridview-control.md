# <a name="using-the-designer-with-the-windows-forms-datagridview-control"></a>Utilisation du concepteur avec le contrôle DataGridView Windows Forms
Visual Studio fournit la prise en charge du concepteur pour le contrôle `DataGridView` qui vous permet d’effectuer de nombreuses tâches d’installation sans écrire de code. Ces tâches incluent la liaison du contrôle à une source de données, la modification des colonnes utilisées pour afficher des données et l’ajustement de l’apparence et du comportement de base du contrôle.
