# <a name="radiobutton-control-windows-forms"></a>RadioButton, contrôle (Windows Forms)
Windows Forms contrôles de `RadioButton` présentent un ensemble de deux ou de plusieurs choix mutuellement exclusifs à l’utilisateur. Alors que les cases d’option et les cases à cocher peuvent sembler fonctionner de la même façon, il existe une différence importante : lorsqu’un utilisateur sélectionne une case d’option, les autres cases d’option du même groupe ne peuvent pas être sélectionnées.
