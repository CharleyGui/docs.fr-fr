# <a name="layout-in-windows-forms-controls"></a>Disposition dans les contrôles Windows Forms

Le positionnement précis des contrôles sur votre formulaire constitue une haute priorité pour de nombreuses applications. L’espace de noms <xref:System.Windows.Forms?displayProperty=nameWithType> vous offre de nombreux outils de disposition pour y parvenir.

## <a name="in-this-section"></a>Dans cette section

[Vue d’ensemble de la propriété AutoSize](autosize-property-overview.md)\
Décrit la propriété <xref:System.Windows.Forms.Control.AutoSize%2A> et son rôle dans la disposition.

[Marge et marge intérieure dans les contrôles de Windows Forms](margin-and-padding-in-windows-forms-controls.md)\
Décrit les propriétés <xref:System.Windows.Forms.Control.Margin%2A> et <xref:System.Windows.Forms.Control.Padding%2A> et leurs rôles dans la disposition.

[Comment : aligner un contrôle sur les bords des formulaires](how-to-align-a-control-to-the-edges-of-forms.md)\
Montre comment utiliser la propriété <xref:System.Windows.Forms.Control.Dock%2A> pour aligner votre contrôle sur le bord du formulaire qu’il occupe.

[Comment : créer une bordure autour d’un contrôle Windows Forms à l’aide de la marge intérieure](how-to-create-a-border-around-a-windows-forms-control-using-padding.md)\
Montre comment utiliser la propriété <xref:System.Windows.Forms.Control.Padding%2A> pour encadrer un contrôle.

[Comment : implémenter un moteur de disposition personnalisé](how-to-implement-a-custom-layout-engine.md)\
Montre comment implémenter une <xref:System.Windows.Forms.Layout.LayoutEngine> pour réorganiser les contrôles Windows Forms.
