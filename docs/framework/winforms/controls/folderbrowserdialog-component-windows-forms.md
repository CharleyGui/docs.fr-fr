# <a name="folderbrowserdialog-component-windows-forms"></a>FolderBrowserDialog, composant (Windows Forms)
Le composant Windows Forms `FolderBrowserDialog` affiche une interface avec laquelle les utilisateurs peuvent parcourir et sélectionner un dossier ou en créer un nouveau. Il s’agit d’un complément du composant [OpenFileDialog](openfiledialog-component-windows-forms.md) qui est utilisé pour parcourir et sélectionner des fichiers.
