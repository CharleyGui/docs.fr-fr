> [!NOTE]
>  <span data-ttu-id="e8a44-105">Les boîtes de dialogue et les commandes de menu qui s'affichent peuvent être différentes de celles qui sont décrites dans l'aide, en fonction de vos paramètres actifs ou de l'édition utilisée.</span><span class="sxs-lookup"><span data-stu-id="e8a44-105">The dialog boxes and menu commands you see might differ from those described in Help depending on your active settings or edition.</span></span> <span data-ttu-id="e8a44-106">Pour modifier vos paramètres, choisissez **Importation et exportation de paramètres** dans le menu **Outils** .</span><span class="sxs-lookup"><span data-stu-id="e8a44-106">To change your settings, choose **Import and Export Settings** on the **Tools** menu.</span></span> <span data-ttu-id="e8a44-107">Pour plus d’informations, consultez [Personnaliser l’IDE Visual Studio](/visualstudio/ide/personalizing-the-visual-studio-ide).</span><span class="sxs-lookup"><span data-stu-id="e8a44-107">For more information, see [Personalize the Visual Studio IDE](/visualstudio/ide/personalizing-the-visual-studio-ide).</span></span>  
  
### <a name="to-simple-bind-a-control"></a><span data-ttu-id="e8a44-108">Pour un contrôle à liaison simple</span><span class="sxs-lookup"><span data-stu-id="e8a44-108">To simple-bind a control</span></span>  
  
1. <span data-ttu-id="e8a44-109">Connectez-vous à une source de données.</span><span class="sxs-lookup"><span data-stu-id="e8a44-109">Connect to a data source.</span></span> <span data-ttu-id="e8a44-110">Pour plus d’informations, consultez [connexion à une Source de données](../data/adonet/connecting-to-a-data-source.md).</span><span class="sxs-lookup"><span data-stu-id="e8a44-110">For more information, see [Connecting to a Data Source](../data/adonet/connecting-to-a-data-source.md).</span></span>  
  
2. <span data-ttu-id="e8a44-111">Dans le formulaire, sélectionnez le contrôle et affichez la **propriétés** fenêtre.</span><span class="sxs-lookup"><span data-stu-id="e8a44-111">In the form, select the control and display the **Properties** window.</span></span>  
  
3. <span data-ttu-id="e8a44-112">Développez le **(DataBindings)** propriété.</span><span class="sxs-lookup"><span data-stu-id="e8a44-112">Expand the **(DataBindings)** property.</span></span>  
  
     <span data-ttu-id="e8a44-113">Les propriétés les plus souvent liées sont affichées sous la **(DataBindings)** propriété.</span><span class="sxs-lookup"><span data-stu-id="e8a44-113">The properties most often bound are displayed underneath the **(DataBindings)** property.</span></span> <span data-ttu-id="e8a44-114">Par exemple, dans la plupart des contrôles, le **texte** propriété est liée plus fréquemment.</span><span class="sxs-lookup"><span data-stu-id="e8a44-114">For example, in most controls, the **Text** property is most frequently bound.</span></span>  
  
4.  <span data-ttu-id="e8a44-115">Si la propriété dont vous souhaitez lier ne fait pas partie des propriétés couramment liées, cliquez sur le **points de suspension** bouton (![bouton les points de suspension (...) dans la fenêtre Propriétés de Visual Studio.](./media/how-to-create-a-simple-bound-control-on-a-windows-form/visual-studio-ellipsis-button.png)) dans le **() Avancé)** à cocher pour afficher le **mise en forme et liaison avancée** boîte de dialogue avec une liste complète des propriétés pour ce contrôle.</span><span class="sxs-lookup"><span data-stu-id="e8a44-115">If the property you want to bind is not one of the commonly bound properties, click the **Ellipsis** button (![The Ellipsis button (...) in the Properties window of Visual Studio.](./media/how-to-create-a-simple-bound-control-on-a-windows-form/visual-studio-ellipsis-button.png)) in the **(Advanced)** box to display the **Formatting and Advanced Binding** dialog box with a complete list of properties for that control.</span></span>  
  
5. <span data-ttu-id="e8a44-116">Sélectionnez la propriété que vous souhaitez lier, puis cliquez sur la flèche déroulante sous **liaison**.</span><span class="sxs-lookup"><span data-stu-id="e8a44-116">Select the property you want to bind and click the drop-down arrow under **Binding**.</span></span>  
  
     <span data-ttu-id="e8a44-117">Une liste des sources de données disponibles s'affiche.</span><span class="sxs-lookup"><span data-stu-id="e8a44-117">A list of available data sources is displayed.</span></span>  
  
6. <span data-ttu-id="e8a44-118">Développez la source de données avec laquelle vous voulez établir une liaison jusqu’à trouver l’élément de données souhaité.</span><span class="sxs-lookup"><span data-stu-id="e8a44-118">Expand the data source you want to bind to until you find the single data element you want.</span></span> <span data-ttu-id="e8a44-119">Par exemple, si vous établissez une liaison à une valeur de colonne dans une table de dataset, développez le nom du dataset, puis développez le nom de la table pour afficher les noms des colonnes.</span><span class="sxs-lookup"><span data-stu-id="e8a44-119">For example, if you are binding to a column value in a dataset's table, expand the name of the dataset, and then expand the table name to display column names.</span></span>  
  
7. <span data-ttu-id="e8a44-120">Cliquez sur le nom d'un élément avec lequel établir une liaison.</span><span class="sxs-lookup"><span data-stu-id="e8a44-120">Click the name of an element to bind to.</span></span>  
  
8. <span data-ttu-id="e8a44-121">Si vous travailliez le **mise en forme et liaison avancée** boîte de dialogue, cliquez sur **OK** pour revenir à la **propriétés** fenêtre.</span><span class="sxs-lookup"><span data-stu-id="e8a44-121">If you were working in the **Formatting and Advanced Binding** dialog box, click **OK** to return to the **Properties** window.</span></span>  
  
9. <span data-ttu-id="e8a44-122">Si vous souhaitez lier d’autres propriétés du contrôle, répétez les étapes 3 à 7.</span><span class="sxs-lookup"><span data-stu-id="e8a44-122">If you want to bind additional properties of the control, repeat steps 3 through 7.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="e8a44-123">Étant donné que les contrôles de liaison simple affichent uniquement un seul élément de données, il est très courant d’inclure une logique de navigation dans un formulaire Windows avec des contrôles de liaison simple.</span><span class="sxs-lookup"><span data-stu-id="e8a44-123">Because simple-bound controls show only a single data element, it is very typical to include navigation logic in a Windows Form with simple-bound controls.</span></span>
