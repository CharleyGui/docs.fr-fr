# <a name="graphics-overview-windows-forms"></a>Vue d'ensemble des graphismes (Windows Forms)
GDI + est une Interface graphique qui permet aux programmeurs d’écrire des applications indépendantes du périphérique. Les services de GDI + sont exposés via un ensemble de classes managées.  
  
## <a name="in-this-section"></a>Dans cette section  
 [Vue d'ensemble des graphismes](overview-of-graphics.md)  
 Fournit une introduction générale à GDI +.  
  
 [Trois catégories de services graphiques](three-categories-of-graphics-services.md)  
 Décrit les trois catégories qui constituent la programmation avec GDI +.  
  
 [Structure de l'interface graphique](structure-of-the-graphics-interface.md)  
 Décrit l’interface de classe managée de GDI +.
