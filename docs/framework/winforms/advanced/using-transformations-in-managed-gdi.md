# <a name="using-transformations-in-managed-gdi"></a>Utilisation des transformations dans GDI+ managé
Transformations affines incluent la rotation, la mise à l’échelle, la réflexion, l’inclinaison et la translation. Dans GDI +, le <xref:System.Drawing.Drawing2D.Matrix> classe fournit la base pour effectuer des transformations affines sur des dessins de vecteurs, des images et texte.
