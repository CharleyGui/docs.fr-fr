# <a name="about-gdi-managed-code"></a>À propos du code managé GDI+

GDI+ est la partie du système d’exploitation Windows qui fournit les graphiques vectoriels à deux dimensions, la création d’images et la typographie. GDI+ améliore le GDI (le Graphics Device Interface inclus avec les versions antérieures de Windows) en ajoutant de nouvelles fonctionnalités et en optimisant les fonctionnalités existantes.

L’interface de classe managée GDI+ (un ensemble de wrappers) fait partie de la .NET Framework, un environnement pour la génération, le déploiement et l’exécution de services Web XML et d’autres applications.

Cette section fournit des informations sur l’API GDI+ pour les programmeurs qui utilisent du code managé.
