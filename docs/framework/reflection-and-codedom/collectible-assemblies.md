# <a name="collectible-assemblies-for-dynamic-type-generation"></a><span data-ttu-id="bcf39-102">Assemblys pouvant être collectés pour la génération de type dynamique</span><span class="sxs-lookup"><span data-stu-id="bcf39-102">Collectible assemblies for dynamic type generation</span></span>

## <a name="lifetime-of-collectible-assemblies"></a><span data-ttu-id="bcf39-110">Durée de vie des assemblys pouvant être collectés</span><span class="sxs-lookup"><span data-stu-id="bcf39-110">Lifetime of collectible assemblies</span></span>

<span data-ttu-id="bcf39-111">La durée de vie d’un assembly pouvant être collecté est contrôlée par l’existence de références aux types qu’il contient et aux objets qui sont créés à partir de ces types.</span><span class="sxs-lookup"><span data-stu-id="bcf39-111">The lifetime of a collectible assembly is controlled by the existence of references to the types it contains and the objects that are created from those types.</span></span> <span data-ttu-id="bcf39-112">Le common language runtime ne décharge pas d’assembly tant qu’un ou que plusieurs des éléments suivants existent (`T` correspond à tout type qui est défini dans l’assembly) :</span><span class="sxs-lookup"><span data-stu-id="bcf39-112">The common language runtime does not unload an assembly as long as one or more of the following exist (`T` is any type that is defined in the assembly):</span></span>

- <span data-ttu-id="bcf39-113">Instance de `T`.</span><span class="sxs-lookup"><span data-stu-id="bcf39-113">An instance of `T`.</span></span>

- <span data-ttu-id="bcf39-114">Instance d’un tableau de `T`.</span><span class="sxs-lookup"><span data-stu-id="bcf39-114">An instance of an array of `T`.</span></span>

- <span data-ttu-id="bcf39-115">Instance d’un type générique qui a `T` comme l’un de ses arguments de type.</span><span class="sxs-lookup"><span data-stu-id="bcf39-115">An instance of a generic type that has `T` as one of its type arguments.</span></span> <span data-ttu-id="bcf39-116">Cela inclut les collections génériques de `T`, même si cette collection est vide.</span><span class="sxs-lookup"><span data-stu-id="bcf39-116">This includes generic collections of `T`, even if that collection is empty.</span></span>

- <span data-ttu-id="bcf39-117">Instance de <xref:System.Type> ou <xref:System.Reflection.Emit.TypeBuilder> qui représente `T`.</span><span class="sxs-lookup"><span data-stu-id="bcf39-117">An instance of <xref:System.Type> or <xref:System.Reflection.Emit.TypeBuilder> that represents `T`.</span></span>

   > [!IMPORTANT]
   > <span data-ttu-id="bcf39-118">Vous devez libérer tous les objets qui représentent des parties de l’assembly.</span><span class="sxs-lookup"><span data-stu-id="bcf39-118">You must release all objects that represent parts of the assembly.</span></span> <span data-ttu-id="bcf39-119"><xref:System.Reflection.Emit.ModuleBuilder> qui définit `T` conserve une référence à <xref:System.Reflection.Emit.TypeBuilder>, et l’objet <xref:System.Reflection.Emit.AssemblyBuilder> conserve une référence à <xref:System.Reflection.Emit.ModuleBuilder>, de sorte que les références à ces objets doivent être libérées.</span><span class="sxs-lookup"><span data-stu-id="bcf39-119">The <xref:System.Reflection.Emit.ModuleBuilder> that defines `T` keeps a reference to the <xref:System.Reflection.Emit.TypeBuilder>, and the <xref:System.Reflection.Emit.AssemblyBuilder> object keeps a reference to the <xref:System.Reflection.Emit.ModuleBuilder>, so references to these objects must be released.</span></span> <span data-ttu-id="bcf39-120">Même l’existence d’un <xref:System.Reflection.Emit.LocalBuilder> ou <xref:System.Reflection.Emit.ILGenerator> utilisé dans la construction de `T` empêche le déchargement.</span><span class="sxs-lookup"><span data-stu-id="bcf39-120">Even the existence of a <xref:System.Reflection.Emit.LocalBuilder> or an <xref:System.Reflection.Emit.ILGenerator> used in the construction of `T` prevents unloading.</span></span>

- <span data-ttu-id="bcf39-121">Référence statique à `T` par un autre type défini dynamiquement `T1` qui est encore accessible par l’exécution de code.</span><span class="sxs-lookup"><span data-stu-id="bcf39-121">A static reference to `T` by another dynamically defined type `T1` that is still reachable by executing code.</span></span> <span data-ttu-id="bcf39-122">Par exemple, `T1` peut dériver de `T`, ou `T` peut être le type d’un paramètre dans une méthode de `T1`.</span><span class="sxs-lookup"><span data-stu-id="bcf39-122">For example, `T1` might derive from `T`, or `T` might be the type of a parameter in a method of `T1`.</span></span>

- <span data-ttu-id="bcf39-123">**ByRef** pour un champ statique qui appartient à `T`.</span><span class="sxs-lookup"><span data-stu-id="bcf39-123">A **ByRef** to a static field that belongs to `T`.</span></span>

- <span data-ttu-id="bcf39-124"><xref:System.RuntimeTypeHandle>, <xref:System.RuntimeFieldHandle> ou <xref:System.RuntimeMethodHandle> qui fait référence à `T` ou à un composant de `T`.</span><span class="sxs-lookup"><span data-stu-id="bcf39-124">A <xref:System.RuntimeTypeHandle>, <xref:System.RuntimeFieldHandle>, or <xref:System.RuntimeMethodHandle> that refers to `T` or to a component of `T`.</span></span>

- <span data-ttu-id="bcf39-125">Instance de tout objet reflection qui peut être utilisée indirectement ou directement pour accéder à l’objet <xref:System.Type> qui représente `T`.</span><span class="sxs-lookup"><span data-stu-id="bcf39-125">An instance of any reflection object that could be used indirectly or directly to access the <xref:System.Type> object that represents `T`.</span></span> <span data-ttu-id="bcf39-126">Par exemple, l’objet <xref:System.Type> pour `T` peut être obtenu à partir d’un type tableau dont le type d’élément est `T`, ou à partir d’un type générique qui a `T` comme argument de type.</span><span class="sxs-lookup"><span data-stu-id="bcf39-126">For example, the <xref:System.Type> object for `T` can be obtained from an array type whose element type is `T`, or from a generic type that has `T` as a type argument.</span></span>

- <span data-ttu-id="bcf39-127">Méthode `M` sur la pile des appels de tout thread, où `M` est une méthode de `T` ou une méthode de niveau module qui est définie dans l’assembly.</span><span class="sxs-lookup"><span data-stu-id="bcf39-127">A method `M` on the call stack of any thread, where `M` is a method of `T` or a module-level method that is defined in the assembly.</span></span>

- <span data-ttu-id="bcf39-128">Délégué d’une méthode statique qui est définie dans un module de l’assembly.</span><span class="sxs-lookup"><span data-stu-id="bcf39-128">A delegate to a static method that is defined in a module of the assembly.</span></span>

<span data-ttu-id="bcf39-129">Si un seul élément de cette liste existe pour un seul type ou une seule méthode dans l’assembly, le runtime ne peut pas décharger l’assembly.</span><span class="sxs-lookup"><span data-stu-id="bcf39-129">If only one item from this list exists for only one type or one method in the assembly, the runtime cannot unload the assembly.</span></span>

> [!NOTE]
> <span data-ttu-id="bcf39-130">Le runtime ne décharge pas réellement l’assembly tant que les finaliseurs n’ont pas été exécutés pour tous les éléments de la liste.</span><span class="sxs-lookup"><span data-stu-id="bcf39-130">The runtime does not actually unload the assembly until finalizers have run for all items in the list.</span></span>

<span data-ttu-id="bcf39-131">À des fins de suivi de durée de vie, un type générique construit comme `List<int>` (en C#) ou `List(Of Integer)` (en Visual Basic), qui est créé et utilisé dans la génération d’un assembly pouvant être collecté est considéré comme ayant été défini dans l’assembly qui contient la définition de type générique ou dans un assembly qui contient la définition de l’un de ses arguments de type.</span><span class="sxs-lookup"><span data-stu-id="bcf39-131">For purposes of tracking lifetime, a constructed generic type such as `List<int>` (in C#) or `List(Of Integer)` (in Visual Basic) that is created and used in the generation of a collectible assembly is considered to have been defined either in the assembly that contains the generic type definition or in an assembly that contains the definition of one of its type arguments.</span></span> <span data-ttu-id="bcf39-132">L’assembly exact utilisé est un détail d’implémentation et est susceptible de changer.</span><span class="sxs-lookup"><span data-stu-id="bcf39-132">The exact assembly that is used is an implementation detail and subject to change.</span></span>

## <a name="restrictions-on-collectible-assemblies"></a><span data-ttu-id="bcf39-133">Restrictions sur les assemblys pouvant être collectés</span><span class="sxs-lookup"><span data-stu-id="bcf39-133">Restrictions on collectible assemblies</span></span>

<span data-ttu-id="bcf39-134">Les restrictions suivantes s’appliquent aux assemblys pouvant être collectés :</span><span class="sxs-lookup"><span data-stu-id="bcf39-134">The following restrictions apply to collectible assemblies:</span></span>

- <span data-ttu-id="bcf39-135">**Références statiques** Les types d’un assembly dynamique ordinaire ne peuvent pas avoir de références statiques à des types définis dans un assembly pouvant être collecté.</span><span class="sxs-lookup"><span data-stu-id="bcf39-135">**Static references** Types in an ordinary dynamic assembly cannot have static references to types that are defined in a collectible assembly.</span></span> <span data-ttu-id="bcf39-136">Par exemple, si vous définissez un type simple qui hérite d’un type dans un assembly pouvant être collecté, une exception <xref:System.NotSupportedException> est levée.</span><span class="sxs-lookup"><span data-stu-id="bcf39-136">For example, if you define an ordinary type that inherits from a type in a collectible assembly, a <xref:System.NotSupportedException> exception is thrown.</span></span> <span data-ttu-id="bcf39-137">Un type dans un assembly pouvant être collecté peut avoir des références statiques à un type dans un autre assembly pouvant être collecté, mais cela permet d’étendre la durée de vie de l’assembly référencé à la durée de vie de l’assembly de référence.</span><span class="sxs-lookup"><span data-stu-id="bcf39-137">A type in a collectible assembly can have static references to a type in another collectible assembly, but this extends the lifetime of the referenced assembly to the lifetime of the referencing assembly.</span></span>

- <span data-ttu-id="bcf39-138">**COM Interop** Aucune interface COM ne peut être définie dans un assembly pouvant être collecté, et aucune instance de type dans un assembly pouvant être collecté ne peut être convertie en objets COM.</span><span class="sxs-lookup"><span data-stu-id="bcf39-138">**COM interop** No COM interfaces can be defined within a collectible assembly, and no instances of types within a collectible assembly can be converted into COM objects.</span></span> <span data-ttu-id="bcf39-139">Un type dans un assembly pouvant être collecté ne peut pas servir de wrapper CCW (COM Callable Wrapper) ni de wrapper RCW (Runtime Callable Wrapper).</span><span class="sxs-lookup"><span data-stu-id="bcf39-139">A type in a collectible assembly cannot serve as a COM callable wrapper (CCW) or runtime callable wrapper (RCW).</span></span> <span data-ttu-id="bcf39-140">Toutefois, les types dans les assemblys pouvant être collectés peuvent utiliser des objets qui implémentent les interfaces COM.</span><span class="sxs-lookup"><span data-stu-id="bcf39-140">However, types in collectible assemblies can use objects that implement COM interfaces.</span></span>

- <span data-ttu-id="bcf39-141">**Appel** de code non managé Les méthodes qui ont <xref:System.Runtime.InteropServices.DllImportAttribute> l’attribut ne sont pas compilées quand elles sont déclarées dans un assembly pouvant être collecté.</span><span class="sxs-lookup"><span data-stu-id="bcf39-141">**Platform invoke** Methods that have the <xref:System.Runtime.InteropServices.DllImportAttribute> attribute will not compile when they are declared in a collectible assembly.</span></span> <span data-ttu-id="bcf39-142">L’instruction <xref:System.Reflection.Emit.OpCodes.Calli?displayProperty=nameWithType> ne peut pas être utilisée dans l’implémentation d’un type dans un assembly pouvant être collecté, et ces types ne peuvent pas être marshalés en code non managé.</span><span class="sxs-lookup"><span data-stu-id="bcf39-142">The <xref:System.Reflection.Emit.OpCodes.Calli?displayProperty=nameWithType> instruction cannot be used in the implementation of a type in a collectible assembly, and such types cannot be marshaled to unmanaged code.</span></span> <span data-ttu-id="bcf39-143">Toutefois, vous pouvez effectuer un appel en code natif à l’aide d’un point d’entrée qui est déclaré dans un assembly ne pouvant pas être collecté.</span><span class="sxs-lookup"><span data-stu-id="bcf39-143">However, you can call into native code by using an entry point that is declared in a non-collectible assembly.</span></span>

- <span data-ttu-id="bcf39-144">**Marshaling** Les objets (en particulier, les délégués) définis dans les assemblys pouvant être collectés ne peuvent pas être marshalés.</span><span class="sxs-lookup"><span data-stu-id="bcf39-144">**Marshaling** Objects (in particular, delegates) that are defined in collectible assemblies cannot be marshaled.</span></span> <span data-ttu-id="bcf39-145">Il s’agit d’une restriction sur tous les types émis transitoires.</span><span class="sxs-lookup"><span data-stu-id="bcf39-145">This is a restriction on all transient emitted types.</span></span>

- <span data-ttu-id="bcf39-146">**Chargement d’assembly** L’émission de réflexion est le seul mécanisme pris en charge pour le chargement des assemblys pouvant être collectés.</span><span class="sxs-lookup"><span data-stu-id="bcf39-146">**Assembly loading** Reflection emit is the only mechanism that is supported for loading collectible assemblies.</span></span> <span data-ttu-id="bcf39-147">Les assemblys chargés à l’aide de toute autre forme de chargement d’assembly ne peuvent pas être déchargés.</span><span class="sxs-lookup"><span data-stu-id="bcf39-147">Assemblies that are loaded by using any other form of assembly loading cannot be unloaded.</span></span>

- <span data-ttu-id="bcf39-148">**Objets liés au contexte** Les variables statiques de contexte ne sont pas prises en charge.</span><span class="sxs-lookup"><span data-stu-id="bcf39-148">**Context-bound objects** Context-static variables are not supported.</span></span> <span data-ttu-id="bcf39-149">Les types dans un assembly pouvant être collecté ne peuvent pas étendre <xref:System.ContextBoundObject>.</span><span class="sxs-lookup"><span data-stu-id="bcf39-149">Types in a collectible assembly cannot extend <xref:System.ContextBoundObject>.</span></span> <span data-ttu-id="bcf39-150">Toutefois, le code dans les assemblys pouvant être collectés peut utiliser des objets liés au contexte définis ailleurs.</span><span class="sxs-lookup"><span data-stu-id="bcf39-150">However, code in collectible assemblies can use context-bound objects that are defined elsewhere.</span></span>

- <span data-ttu-id="bcf39-151">**Données statiques de thread** Les variables statiques de thread ne sont pas prises en charge.</span><span class="sxs-lookup"><span data-stu-id="bcf39-151">**Thread-static data** Thread-static variables are not supported.</span></span>
