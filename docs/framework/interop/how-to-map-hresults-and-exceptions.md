### <a name="to-create-a-new-exception-class-and-map-it-to-an-hresult"></a><span data-ttu-id="6efb0-114">Pour créer une classe d’exception et la mapper à un HRESULT</span><span class="sxs-lookup"><span data-stu-id="6efb0-114">To create a new exception class and map it to an HRESULT</span></span>

    ```cpp  
    }  
    ```
