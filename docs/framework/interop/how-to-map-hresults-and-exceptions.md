### <a name="to-create-a-new-exception-class-and-map-it-to-an-hresult"></a><span data-ttu-id="7603a-116">Pour créer une classe d’exception et la mapper à un HRESULT</span><span class="sxs-lookup"><span data-stu-id="7603a-116">To create a new exception class and map it to an HRESULT</span></span>  
  
1. <span data-ttu-id="7603a-117">Utilisez le code suivant pour créer une classe d’exception appelée `NoAccessException` et mappez-la à la valeur HRESULT `E_ACCESSDENIED`.</span><span class="sxs-lookup"><span data-stu-id="7603a-117">Use the following code to create a new exception class called `NoAccessException` and map it to the HRESULT `E_ACCESSDENIED`.</span></span>

    ```cpp  
    {  
        NoAccessException () {  
        HResult = E_ACCESSDENIED;

    {  
    throw new NoAccessException();  
    }  
    ```
