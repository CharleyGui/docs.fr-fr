# <a name="wcf-directive-syntax"></a>Syntaxe de directive WCF
Spécifie les paramètres utilisés par les directives Windows Communication Foundation (WCF) dans les fichiers. svc pour diriger les compilateurs. Chaque directive peut contenir un ou plusieurs attributs (associés à des valeurs) qui sont spécifiques à cette directive. WCF n’a que [ \@](servicehost.md) la directive ServiceHost.  
  
## <a name="in-this-section"></a>Dans cette section  
 [@ServiceHost](servicehost.md)  
 Définit des attributs spécifiques à une page qui sont utilisés par le compilateur .svc. Ne peut être inclus que dans des fichiers .svc.
