 <span data-ttu-id="ec473-109">L’assembly partagé doit être détectable par le Runtime.</span><span class="sxs-lookup"><span data-stu-id="ec473-109">The shared assembly must be discoverable by the runtime.</span></span>  <span data-ttu-id="ec473-110">Pour spécifier un répertoire privé pour la résolution des références d’assembly, utilisez l' [ \<élément CodeBase >](./file-schema/runtime/codebase-element.md) ou [ \<le > de sonde](./file-schema/runtime/probing-element.md) dans un fichier de configuration, comme décrit dans spécification de l' [emplacement d’un assembly](specify-assembly-location.md).</span><span class="sxs-lookup"><span data-stu-id="ec473-110">To specify a private directory for resolving assembly references use the [\<codeBase> Element](./file-schema/runtime/codebase-element.md) or [\<probing> Element](./file-schema/runtime/probing-element.md) in a configuration file, as described in [Specifying an Assembly's Location](specify-assembly-location.md).</span></span>  <span data-ttu-id="ec473-111">Vous pouvez également placer l’assembly dans un sous-répertoire du répertoire de l’application.</span><span class="sxs-lookup"><span data-stu-id="ec473-111">You can also put the assembly in a subdirectory of the application directory.</span></span> <span data-ttu-id="ec473-112">Pour plus d’informations, consultez [Méthode de localisation des assemblys par le runtime](../deployment/how-the-runtime-locates-assemblies.md).</span><span class="sxs-lookup"><span data-stu-id="ec473-112">For more information, see [How the Runtime Locates Assemblies](../deployment/how-the-runtime-locates-assemblies.md).</span></span>
