## <a name="timespan-values"></a><span data-ttu-id="7dd41-108">Valeurs Timespan</span><span class="sxs-lookup"><span data-stu-id="7dd41-108">Timespan Values</span></span>  
 <span data-ttu-id="7dd41-109">Si <xref:System.TimeSpan> est utilisé comme type d’attribut, consultez la méthode <xref:System.TimeSpan.Parse%28System.String%29> pour connaître le format autorisé.</span><span class="sxs-lookup"><span data-stu-id="7dd41-109">Where <xref:System.TimeSpan> is used as the type of an attribute, see the <xref:System.TimeSpan.Parse%28System.String%29> method to see the allowed format.</span></span> <span data-ttu-id="7dd41-110">Ce format respecte la spécification suivante.</span><span class="sxs-lookup"><span data-stu-id="7dd41-110">This format conforms to the following specification.</span></span>
