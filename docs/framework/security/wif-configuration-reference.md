# <a name="wif-configuration-reference"></a>Référence de configuration de WIF
Vous pouvez configurer WIF (Windows Identity Foundation) dans vos applications en ajoutant les éléments de configuration nécessaires dans un fichier de configuration. Cette rubrique contient des liens vers des rubriques de référence relatives aux éléments de configuration de WIF.  
  
 [Schéma de configuration de WIF](../configure-apps/file-schema/windows-identity-foundation/index.md)  
 Contient des informations de référence sur les éléments de configuration de WIF.  
  
 [Conventions de schéma de configuration de WIF](wif-configuration-schema-conventions.md)  
 Contient des informations sur les attributs et formats généralement utilisés par les éléments de configuration de WIF.
