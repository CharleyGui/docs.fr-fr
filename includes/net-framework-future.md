> [!NOTE]
> .NET Framework 4,8 est la dernière version de .NET Framework et aucune autre version n’est publiée. Toutefois, .NET Framework [continueront d’être](https://devblogs.microsoft.com/dotnet/?s=%22.net+framework%22) résolus par des correctifs de bogues de sécurité et de fiabilité mensuels. En outre, il continuera à être inclus avec Windows, sans prévoir de le supprimer. Vous n’avez pas besoin de migrer vos applications .NET Framework, mais pour un nouveau développement, utilisez [.net 5,0 ou une version ultérieure](~/docs/core/introduction.md).
