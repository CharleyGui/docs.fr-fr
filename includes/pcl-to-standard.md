> [!IMPORTANT]
> Nous déconseillons vivement d’utiliser les bibliothèques de classes portables dans le nouveau développement d’applications, car elles ne ciblent qu’un sous-ensemble spécifique d’implémentations .NET. Si vous créez une bibliothèque réutilisable pour partager du code entre .NET Framework et d’autres implémentations .NET, le remplacement recommandé est une bibliothèque .NET Standard 2,0. Dans le cas contraire, utilisez .NET 5 ou une version ultérieure. Pour plus d'informations, consultez [.NET Standard](~/docs/standard/net-standard.md).
