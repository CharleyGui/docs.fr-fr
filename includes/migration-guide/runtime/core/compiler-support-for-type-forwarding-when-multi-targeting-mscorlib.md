### <a name="compiler-support-for-type-forwarding-when-multi-targeting-mscorlib"></a>Prise en charge par le compilateur du transfert de type lors du multiciblage de mscorlib

#### <a name="details"></a>Détails

Une nouvelle fonctionnalité CodeDOM permet à un compilateur d’effectuer une compilation sur la version ciblée de mscorlib.dll au lieu de la version .NET Framework 4.5 de mscorlib.dll.

| Name    | Valeur       |
|:--------|:------------|
| Étendue   |Edge|
|Version|4,5|
|Type|Runtime|

#### <a name="affected-apis"></a>API affectées

Non détectable via l’analyse des API.
