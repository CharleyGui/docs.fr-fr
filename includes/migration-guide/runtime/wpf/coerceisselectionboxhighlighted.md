### <a name="coerceisselectionboxhighlighted"></a>CoerceIsSelectionBoxHighlighted

#### <a name="details"></a>Détails

Certaines séquences d’actions impliquant un <xref:System.Windows.Controls.ComboBox?displayProperty=fullName> et sa source de données peut entraîner un <xref:System.NullReferenceException?displayProperty=fullName>.

#### <a name="suggestion"></a>Suggestion

Si possible, mettre à niveau vers .NET Framework 4.6.2.

| Name    | Valeur       |
|:--------|:------------|
| Étendue   |Secondaire|
|Version|4,6|
|Type|Runtime|
